## 2. 共鳴箱の動作原理

### 2.1 1段変換

```
機械励振 -> 共鳴板振動 -> 空気放射
```

電気を経由しない。電源・配線・故障の電気的経路がない。

### 2.2 励振源の種類

| 楽器 | 励振源 | 励振の性質 |
|------|------|------|
| ギター | 弦の撥弦 | 過渡的 (打撃的) |
| バイオリン | 弦の弓擦り | 連続的 (有限時間) |
| ピアノ | 弦の打撃 | 過渡的 |
| ピアニカv3 (設計) | フリーリードの空気流振動 | 連続的 (足踏み持続中) |

ピアニカv3 が連続励振であることは設計上の特徴である。サスティン制約が
ないため、エネルギー結合 T を高く設計しても弦楽器のような短いサスティン
にならない。

### 2.3 駆動点と放射点

共鳴箱では **駆動点と放射点が分離** している。
- 駆動点: 駒、リード基部など (励振が入る点)
- 放射点: 共鳴板全体 (空気を押す面)

スピーカでは駆動点 (ボイスコイル) と放射点 (コーン全体) は同軸であるが、
共鳴箱では駆動点が板の端部にある。これは「板の固有モードを励振する」
という発想そのものである。

## 3. 共鳴板の物理

### 3.1 板方程式 (Kirchhoff-Love)

薄板の支配方程式:

```
D * ∇⁴w + rho_mat * h * ∂²w/∂t² = f(x, y, t)

D = E*h³ / (12*(1-nu²))
```

ここで w(x,y,t) は板の変位、D は曲げ剛性、rho_mat は板密度、h は板厚、
E はヤング率、nu はポアソン比。

### 3.2 単純支持矩形板の固有周波数

```
f_mn = (pi/2) * sqrt(D/(rho_mat*h)) * ((m/a)² + (n/b)²)
```

a, b は板の寸法、m, n はモード番号。

### 3.3 ピアニカv3 (設計値) での f_11 計算

300×300 mm, h=2 mm, 桐 (rho=280 kg/m³, nu=0.3) で、E に幅をもたせて計算:

| E [GPa] | f_11 [Hz] |
|------|------|
| 5 | 89.3 |
| 6 | 97.8 |
| 7 | 105.6 |

原典0430f の f_11=96 Hz はこの範囲に入る。妥当な設計値。
ただし境界条件は実際には単純支持ではないため、実機での実測値は前後する。

### 3.4 桐 (Paulownia tomentosa) の物性 (文献値)

- 密度: 約 280 kg/m³ (スプルース約 400 kg/m³ より低い)
- ヤング率: 5-7 GPa (繊維方向、文献により幅あり)
- ポアソン比: 約 0.3

スプルースより低密度。同じ厚さなら単位面積質量 (rho*h) が小さく、
共鳴板として高い放射効率が期待される方向の物性である。

## 4. 放射効率 eta_rad の取り扱い

### 4.1 原典0430f の主張

原典は次の式を提示している:

```
Gamma = eta * rho_mat * h / (rho_air * c)   [単位 s]
eta_rad = 1 / (1 + Gamma)                     [無次元]
```

| 楽器 | 原典Gamma | 原典eta_rad |
|------|------|------|
| Guitar | 3.0 | 25% |
| Piano | 1.5 | 40% |
| Violin | 2.0 | 33% |
| Pianica v3 | 1.8 | 36% |

### 4.2 検算で発覚した問題

**Gamma の式の単位は [s] であり無次元ではない**。
ピアニカv3 のパラメータ (eta=0.05, rho_mat=280, h=0.002) を式に代入すると
6.8×10⁻⁵ s となる。原典の Gamma=1.8 とは 4-5桁の差がある。

つまり原典の Gamma 値は **物理式から導出された値ではなく、目標の eta_rad
(36%) から `Gamma = 1/eta_rad - 1` で逆算した無次元数** である。
原典0430f の式自体に誤りが存在する。

### 4.3 本書での取り扱い

**eta_rad=36% は設計目標値である**。
- 物理式からの導出ではない
- 実測値ではない (未製作)
- 「桐の物性なら高い放射効率が期待できる」という定性的な期待値

実機検証で測定する予定の値であり、未確定である。
本書ではこれを「設計目標」と明記し、「物理式から導出された値」と扱わない。

### 4.4 Gamma 概念そのものの位置付け

Gamma という量を導入すること自体は、放射効率を整理する上で有用である。
ただし「式の形と次元と値が一致した状態」で提示すべきであり、原典0430f の
式は次元が合わない。式の正しい形を導出する作業は本書の範囲を超える
(楽器音響学の専門文献を別途参照する必要がある)。

## 5. インピーダンス整合 (T_coupling)

### 5.1 標準式

伝送線路のインピーダンス整合:

```
T = 4 * Z1 * Z2 / (Z1 + Z2)²
```

Z1 = Z2 で T=1 (完全整合)。

### 5.2 原典0430f の値

| 楽器 | T (原典) |
|------|------|
| Guitar | 2.3% |
| Piano | 15% (平均) |
| Violin | 2% |
| Pianica v3 | 97% |

### 5.3 検算で確認できなかったこと

**ピアニカv3 の T=97% の根拠**:
- 原典には ABS-桐界面の Z_source と Z_board の値が記載されていない
- そのため T=97% を逆算的に検証することができない

ギターのみ Z1=0.3, Z2=50 Ns/m と原典に記載があり、これで計算すると
T=2.37% となり原典の 2.3% と一致する。式自体は正しい。

ピアニカv3 については **実測待ち**。

### 5.4 連続励振との関係

弦楽器は T を低く設計する (サスティン延長のため、エネルギーを少しずつ
放出する)。ピアニカv3 は連続励振 (足踏みが続く間リードが鳴り続ける) なので
サスティン延長の必要がなく、T を高く設計できる。これは設計上の論理として
正しい。ただし「実際に T=97% が得られるか」は別問題で、実測待ちである。

## 6. 共鳴板の支持構造

### 6.1 バイオリンの構造要素

- 表板 (スプルース、放射の主体)
- 裏板 (メイプル、剛性)
- 駒 (励振の入力点)
- 魂柱 (sound post、表板と裏板を結ぶ柱)
- 力木 (bass bar、表板の補強)

### 6.2 魂柱の機能

魂柱は表板と裏板の間に立つ細い円柱である。機能としては:

- 駒からの振動を裏板に伝達 (構造的経路)
- 表板と裏板の振動モードを結合
- 駒の片側を回転中心化することで、表板の運動を非対称化

### 6.3 「魂柱=スピーカ」アナロジーについて

旧版 Appendix E は「魂柱とスピーカは数学的同型 (isomorphic)」と記述したが、
これは誤りである:

- 魂柱: 1段の機械的伝達 (機械→機械)
- スピーカ: 2段の変換 (電気→機械→音響)
- 両者は伝達関数を持つ点で類似するが、これは「あらゆる線形系が伝達関数を持つ」
  程度の話で、「同型」と呼べる対応関係ではない

「機能的アナロジー」は成立する (両者とも振動を別の場所に伝える)。しかし
「数学的同型」と書くのは学術用語の濫用である。本書ではこの記述を撤回し、
「機能的アナロジー」と表現する。

### 6.4 ピアニカv3 における魂柱

原典0430f には「バイオリンの sound post 構造のピアニカへの応用可能性」が
**今後の課題** として記載されている。原典では未解決の課題であり、
旧版 Appendix D が「魂柱なし設計を決定」と書いたのは原典の主張を歪めている。

本書では「魂柱の応用は未検討」とのみ記載する。

## 7. ピアニカv3 の楽器分類

### 7.1 原典0430f の主張

原典は次の核心的観察を述べている:

> 「ピアニカv3 の物理パラメータはバイオリンに最も近い」

理由:
- 板面積が小さい (バイオリン 0.035 m² と同オーダーの 0.09 m²)
- 共鳴板単一 (両側に板があるピアノとは異なる)
- 連続励振 (バイオリンの弓と同じく持続的)

### 7.2 旧版の誤り

旧版 Appendix D は「ピアニカv3 はピアノ響板系」と書いた。これは原典の主張
「バイオリンに最も近い」を勝手に再分類した誤りである。

### 7.3 本書の立場

ピアニカv3 はバイオリン系に最も近い。ただし次の違いがある:
- 弦ではなくフリーリード励振 (連続性の点では弓に近い)
- 板面積はバイオリンより大きい (0.09 m² vs 0.035 m²)
- 板の境界条件は弦楽器と異なる (固定方法が異なる)

完全に一致する楽器は存在しない。「最も近い」は厳密な分類ではなく、
パラメータ空間内での距離の話である。

## 8. スピーカとの対比 (鏡像)

| 観点 | 共鳴箱 (アコースティック楽器) | スピーカ |
|------|------|------|
| エネルギー段数 | 1段 (機械→音響) | 3段 (電気→機械→音響) |
| 駆動方式 | 端部駆動 (駒・リード基部) | 中心軸駆動 (ボイスコイル) |
| 放射体 | 共鳴板全体 (受動) | コーン (能動) |
| 周波数応答 | 共鳴ピーク多数 (個性) | 設計目標としてフラット |
| 効率 (文献値) | アコースティック楽器 1-6%、ピアニカv3 設計値 35% | 0.4-5% |
| 電源 | 不要 | 必要 |
| 故障モード | 構造劣化 (長時間スケール) | 電気故障 (短時間スケール) |

ここで「ピアニカv3 設計値 35%」は実測ではない (§4.3 参照)。

「90倍効率差」のような誇張は撤回する。アコースティック楽器とスピーカは
**別の物理問題を解く装置** であり、効率比較は条件次第で逆転する。

旧版 Appendix C/E は主聖堂を「矩形 25×18×10 m, 4,500 m³」と仮定し、

しかし Web で確認できる事実は次の通り:

**確認できた事実**:
- 阪神淡路大震災から10年後の2004年完成 (旧版は2005年と誤記)
- 村上晶子アトリエ＆パートナーズ設計
- 被災した3教会 (灘、中山手、下山手) の統合再建
- 外観: 白煉瓦、緑青銅板葺きの大屋根
- 内部: 白煉瓦の壁、16枚のステンドグラス

**確認できなかったこと**:

### 9.2 本書での扱い

形状仮定が誤りなので無効である。

本書では主聖堂の音響モード計算は行わない。実測なしには論じられない。
ピアニカv3 の設置先が主聖堂であるという話は **設計設備配置の話** であって、

## 10. ピアニカv3 設計判断 (本書から導けるもの)

### 10.1 確実に言えること

- 共鳴箱パラダイムは電気を経由しない (1段変換)
- 桐の密度 (280 kg/m³) は共鳴板として有利な方向
- 連続励振 (フリーリード+足踏み) は弦楽器のサスティン制約から自由
- 板方程式から f_11 は 89-106 Hz の範囲が見込まれる (E=5-7 GPa, 単純支持仮定)

### 10.2 設計値として採用するもの

- 板寸法: 300×300 mm (面積 0.09 m²)
- 板厚: 2 mm
- 板材: 桐
- リード励振: 37音、A4=440Hz、平均律
- 駆動: リード基部から板への伝達

### 10.3 撤回するもの

- 「Gamma=1.8 を物理式から導出」(実は逆算値、§4.2)
- 「90倍効率差」(比較基準ずれ、§8)
- 「魂柱=スピーカは数学的同型」(機能的アナロジーに変更、§6.3)
- 「魂柱なし設計を決定」(原典では未検討、§6.4)
- 「ピアニカv3 はピアノ系」(原典ではバイオリン系、§7.2)
- 主聖堂を矩形と仮定した固有モード計算 (§9.2)
- リード→板 1-2 mW、足踏み 80 mW (出典不明、撤回)

### 10.4 実機後に確定する値

- f_11 の実値 (Spectroid FFT)
- Q 値 (リング法または半値幅法)
- E の実値 (打撃応答)
- T_coupling の実値 (励振点と板上の応答)
- SPL @1m (騒音計または校正済みマイク)
- 倍音構成 (Spectroid FFT)

## 11. 結論

共鳴箱は機械的励振を空気放射に直結する装置である。スピーカは電気を経由する。
両者は異なる物理問題を解く装置であり、優劣ではなく使い分けの対象である。

ピアニカv3 の採用根拠は次の事実で十分である:

1. 電気を経由しない (停電時運用、長寿命、電源設備不要)
2. 桐の物性が共鳴板として有利な方向にある
3. 連続励振という励振様式とフリーリードの整合
4. アコースティック楽器の系譜内に位置付けられる

これらは設計判断の出発点であり、定量的な数値は実機実測で確定する。
本書は **設計プロセス論の整理** であって、定量的な性能保証ではない。
