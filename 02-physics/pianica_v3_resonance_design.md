## 2. ピアニカv3 共鳴板の物理パラメータ

### 2.1 板材料: 桐 [A/B]

| パラメータ | 値 | 信頼度 | 備考 |
|-----------|-----|--------|------|
| 材料 | 桐 (Paulownia) | [A] | 設計選択 |
| 密度 rho | 260-300 kg/m^3 (代表値 280) | [A] | 木材物性表 |
| ヤング率 E | 3.5-7.0 GPa | [A] | 試料・繊維方向で大幅に変動 |
| 損失係数 eta | 0.03-0.06 (代表値 0.05) | [A] | 文献値、スプルースより高い |
| ポアソン比 nu | 0.3 (推定) | [B] | 木材の一般的値 |

桐はスプルース (rho=430, E=10-12 GPa) より軽量低弾性。rho*h が小さいため、同じ板厚で放射に有利な軽い振動質量を持つ。ただし損失係数が高い（スプルースの2-5倍）。

### 2.2 板の設計寸法 [C]

| パラメータ | 設計値 | 信頼度 | 備考 |
|-----------|--------|--------|------|
| 寸法 | 300 x 300 mm | [C] | 実機なし |
| 板厚 h | 1.5 - 2 mm | [C] | 実機なし |
| 面積 S | 0.09 m^2 | [C] | 上記から計算 |
| 面密度 rho*h | 0.42-0.60 kg/m^2 | [B] | rho と h の積 |

### 2.3 f_11 (基本振動数) [B]

Python検算結果（単純支持矩形板、300x300mm）:

```
f_mn = (pi/2) * sqrt(D / (rho*h)) * ((m/a)^2 + (n/b)^2)
D = E * h^3 / (12 * (1 - nu^2))

h = 2 mm, rho = 280 kg/m^3, nu = 0.3:

  E = 3.5 GPa -> f_11 =  74.7 Hz
  E = 4.5 GPa -> f_11 =  84.7 Hz
  E = 5.0 GPa -> f_11 =  89.3 Hz
  E = 5.8 GPa -> f_11 =  96.2 Hz  <-- 0430fの96Hzに一致
  E = 7.0 GPa -> f_11 = 105.6 Hz
```

**結論**: f_11 = 96 Hz は E ≈ 5.8 GPa の桐で実現可能。桐のE範囲 (3.5-7.0 GPa) 内であり、**設計目標値としては妥当**。ただし「実測値」ではない。実機製作時に Spectroid FFT で実測すべき。

**補足**: 実際の板は「単純支持矩形」ではなく、支持条件（固定端、自由端、混合）によりf_11は変動する。上記は理想化された計算であり、実機との差は 10-30% ありうる。

### 2.4 Gamma (放射減衰パラメータ) [D: 要再検討]

0430fでは以下の式が使用された:

```
Gamma = eta * rho_mat * h / (rho_air * c)
eta_rad = 1 / (1 + Gamma)
```

**問題: 次元不整合**

```
Gamma = [1] * [kg/m^3] * [m] / ([kg/m^3] * [m/s])
      = [kg/m^2] / [kg/(m^2*s)]
      = [s]
```

物理式を代入すると Gamma = 6.8e-5 [s]。一方、eta_rad = 1/(1+Gamma) は Gamma が無次元であることを要求する。

**0430fのGamma=1.8の正体**:

eta_rad = 36% を逆算すると Gamma = (1/0.36) - 1 = 1.78 ≈ 1.8。つまり Gamma=1.8 は**目標効率36%からの逆算値**であり、物理式からの導出ではない。

**是正方針**:

本パラメータは根本的な次元解析の問題を含む。0430fの比較表における他の楽器のGamma値（ギター3.0、ピアノ1.5、バイオリン2.0）も同じ問題を持つ。放射効率の正確な計算には、モード別の放射インピーダンス計算が必要であり、単一パラメータでの近似には限界がある。

**現状の位置付け**: Gamma=1.8 および eta_rad=36% は**設計目標値**として扱う。物理的根拠は不十分であり、実機製作後の実測で検証すべき。

### 2.5 T_coupling (駒結合伝達率) [C]

0430fの主張: T = 97%

```
T = 4 * Z_source * Z_board / (Z_source + Z_board)^2

T = 0.97 が成立する条件:
  Z1/Z2 比 = 0.70 - 1.42 (インピーダンス比が ~1.4以内)
```

ピアニカv3の構造（ABS壁から桐板への直接結合）では、弦楽器のような細い駒を介さず、広い面積で接触する。この構造がT=97%を物理的に正当化する論拠だが、**実測値ではなく推定値**。

弦楽器でTが低い (2-30%) のは意図的設計（サスティン確保）だが、連続励振のピアニカではこの制約がないため、T を最大化する設計が合理的である。

### 2.6 臨界周波数 f_c [B]

```
h = 2mm, E = 5.0 GPa -> f_c = 7,321 Hz
h = 2mm, E = 5.8 GPa -> f_c = 6,798 Hz
```

f_c >> 4,000 Hz なので、音楽的に重要な帯域 (100-4,000 Hz) 全体が f < f_c（放射効率が周波数依存で変動する領域）にある。

### 3.1 原典の核心主張: 「バイオリンに最も近い」

0430fは以下の根拠で「バイオリンに最も近い」と結論した:

1. **連続励振**: バイオリン（ボウイング）とピアニカ（フリーリード）はいずれも連続的エネルギー供給。ギター/ピアノは衝撃励振
2. **小型共鳴箱**: バイオリン (V ≈ 2L) とピアニカv3 の設計容積は同じオーダー
3. **空洞結合**: 両者とも内部空気のHelmholtz共鳴を利用する可能性
4. **T制約なし**: 連続励振のため、サスティン確保のためにTを下げる必要がない

**構造的差異**: バイオリンは双面構造（表板+裏板+魂柱）、ピアニカv3は単面構造（桐表板のみ）。この点では**ピアノ響板に構造的に近い**。

v1文書のAppendix Dはこの構造的差異を以て「ピアノ系に再分類」としたが、これは原典の「数値的特性でバイオリンに最も近い」を否定するものではない。**数値的特性（励振方式、サイズ）ではバイオリンに近く、構造（単面）ではピアノに近い**。両者は異なる観察軸であり、矛盾しない。

3. **実機検証**: CAHAYA改造 + Spectroid FFT測定 -> 未着手

### 5.1 魂柱応用の議論 [D: 未解決]

v1 Appendix Dでは連続励振との整合性を根拠に「魂柱なし」を最適解としたが、以下の理由で判断を保留する:

- 原典0430fが「次の課題」として残した問題を、十分な検証なく「解決済み」とした
- 連続励振で魂柱が不利な点（裏板へのエネルギー流出が不要）は物理的に正当だが、魂柱の他の機能（表板のモード制御、剛性補強）は検討されていない
- バイオリンの魂柱は表板と裏板を機械的に結合する剛体棒であり、スピーカ（電気-機械-音響の3段変換器）とは変換段数が異なる。「機能的アナロジー」（分離した振動体間の励振点という共通点）としては成立するが、「数学的同型 (isomorphism)」と呼ぶのは学術用語の濫用

### 5.2 効率比較の是正 [D: 基準未統一]

v1では「共鳴箱はスピーカの90倍効率的」と主張したが:

- **スピーカの効率**: 電気入力 -> 音響出力 = 0.5-2% (Hi-Fi)。分母は電気入力パワー
- **共鳴箱の「放射効率」**: 板の機械振動 -> 音響出力 ≈ 25-40%。分母は板の機械入力パワー

分母が異なる物理量であるため、直接比較は**誤導的**。共鳴箱には「電気->機械」段がなく、その段のロスを含まないため、高い数値が出る。

公平な比較をするなら:
- スピーカ: 電気->音響 = 0.5-2%
- 共鳴箱 (弦楽器全体): 弦の運動エネルギー -> 音響 = T * eta_rad ≈ 0.5-12%

弦楽器全体の効率はスピーカと同オーダーであり、「90倍」は成立しない。

## 7. 検算スクリプト

本文書の計算に使用したPythonスクリプト:

```python
import math

# === Kiri plate parameters ===
rho = 280       # kg/m^3
h = 0.002       # m
a, b = 0.30, 0.30  # m
nu = 0.3
rho_air, c = 1.2, 343  # kg/m^3, m/s

# f_11 vs E
for E in [3.5e9, 4.5e9, 5.0e9, 5.8e9, 7.0e9]:
    D = E * h**3 / (12 * (1 - nu**2))
    f_11 = (math.pi/2) * math.sqrt(D/(rho*h)) * ((1/a)**2 + (1/b)**2)
    f_c = c**2 / (2*math.pi) * math.sqrt(rho*h/D)
    print(f"E={E/1e9:.1f}GPa -> f_11={f_11:.1f}Hz, f_c={f_c:.0f}Hz")

# Gamma dimensional analysis
eta = 0.05
Gamma_dim = eta * rho * h / (rho_air * c)
print(f"Gamma (formula) = {Gamma_dim:.6e} [s] -- NOT dimensionless")

# Gamma from eta_rad target
eta_rad_target = 0.36
Gamma_reverse = (1/eta_rad_target) - 1
print(f"Gamma (from eta_rad=36%) = {Gamma_reverse:.2f} -- dimensionless, matches 0430f")

# T_coupling check
T = 0.97
b_coeff = 2 - 4/T
disc = b_coeff**2 - 4
r = (-b_coeff + math.sqrt(disc)) / 2
print(f"T=0.97 requires Z1/Z2 = {r:.3f} or {1/r:.3f}")
```
