## 1. 本文書の目的と限界

### 1.1 目的

ピアニカv3 (足踏みリードオルガン) の設計上、「共鳴箱」と「スピーカ」を
混同しないことが重要である。両者は音を出す装置という点で似ているが、
エネルギーの流れ方と物理モデルが異なる。本書はこの違いを最小限の事実だけで
整理する。

### 1.2 限界

本書は次の制約下で書かれている:

- ピアニカv3 は未製作であり、本書中の数値はすべて設計値・推定値である
- スピーカ側の数値は文献値を採用するが、本プロジェクトではスピーカを実装しない
- 比較は「概念の整理」であり、定量的な優劣判定ではない

---

## 2. エネルギー変換の段数

### 2.1 スピーカ (3段変換)

```
電気 [W] -> ボイスコイル磁力 [N] -> コーン振動 [m/s] -> 空気放射 [Pa]
```

各段で損失が生じる。

### 2.2 共鳴箱 (1段変換)

```
機械励振 [N] -> 共鳴板振動 [m/s] -> 空気放射 [Pa]
```

電気を経由しない。

### 2.3 段数の違いから言えること

段数が少ないほうが効率は高くなる傾向がある。ただし「段数が少ない=必ず効率が高い」
ではない。各段の整合 (インピーダンスマッチング) が支配的である。

---

## 3. 効率の比較

### 3.1 文献値 (信頼度: 文献に依拠、本プロジェクトでは未検証)

| 装置 | 全体効率 (電気/機械入力 -> 音響出力) | 出典 |
|------|--------------------|------|
| 一般的なダイナミックスピーカ | 0.4 - 5% | 標準的なオーディオ工学文献 |
| アコースティック楽器 (ギター・バイオリン) | 0.6 - 0.7% | 原典0430f の合成計算 |
| アコースティック楽器 (ピアノ) | 約 6% | 原典0430f の合成計算 |
| ピアニカv3 (設計値) | 約 35% | 原典0430f の合成計算、未製作のため未実測 |

### 3.2 ピアニカv3 の 35% について

原典0430f は ピアニカv3 の P_rad/P_input を `T_coupling × eta_radiation` で計算し、
T=97%, eta_rad=36% から 35% を得ている。両入力とも実測ではないため、35% も
実測ではない。詳細は §5 を参照。

### 3.3 旧版が主張した「90倍効率差」について

旧版 speaker_vs_resonance_box.md は「ピアニカv3 36% vs スピーカ 0.4% で
90倍効率差」と書いた。これは比較基準が異なる:

- 「ピアニカv3 36%」は **板から空気への局所的な放射効率** (eta_rad)
- 「スピーカ 0.4%」は **電気から空気への全体効率**

これらを直接比較すると 90倍だが、ピアニカv3 の **全体効率** で比較すると
原典0430f の合成計算で約 35%、スピーカの 0.4-5% と比べて同オーダーから1桁差。
「90倍」は誇張であった。本書では誇張を撤回する。

## 4. 共鳴板/コーンの構造

### 4.1 スピーカコーン

| 項目 | 一般的なダイナミックスピーカ |
|------|------|
| 材質 | 紙、ポリプロピレン、アルミ等 |
| 厚さ | 0.3 - 1 mm |
| 形状 | 円錐形 (能動駆動部) |
| 駆動 | ボイスコイル (中心軸方向の力) |
| 周波数応答 | 設計目標としてフラット |

### 4.2 楽器の共鳴板

| 項目 | バイオリン表板 | ピアノ響板 | ピアニカv3 (設計) |
|------|------|------|------|
| 材質 | スプルース | スプルース | 桐 |
| 厚さ | 約 2.5 mm | 5-8 mm (中央厚め) | 2 mm (設計値) |
| 面積 | 約 350 cm² | 約 1.5-2 m² | 約 900 cm² (設計値) |
| 駆動 | 駒経由 (端部から) | 駒経由 (端部から) | リード経由 (設計、未製作) |
| 周波数応答 | 共鳴ピーク多数 | 共鳴ピーク多数 | 未製作 |

### 4.3 桐の物性 (文献値)

- 密度: 約 280 kg/m³
- ヤング率 E: 5 - 7 GPa (繊維方向、文献により幅あり)
- ポアソン比 nu: 約 0.3

スプルースと比べると密度が低く、同じ厚さなら単位面積質量が小さい。これは
共鳴板として高い放射効率を期待できる方向の物性である。ただし「期待できる」
であって実測ではない。

## 5. 原典0430f の数値の検算結果

ここが旧版の最大の問題箇所である。本書ではすべて Python で検算した結果を記す。

### 5.1 Gamma の式 (誤り)

原典0430f には次の式が記載されている:

```
Gamma = eta * rho_mat * h / (rho_air * c)
eta_radiation = 1 / (1 + Gamma)
```

ここで eta は損失係数、rho_mat は板の密度、h は板厚、rho_air は空気密度、
c は音速。

**検算結果**: この式の単位は [s] (時間) であり、無次元ではない。値を代入すると
ピアニカv3 で 6.8×10⁻⁵ s となる。原典が記載する Gamma=1.8 とは4-5桁の差がある。

**結論**: 原典の式は次元が合わない。Gamma=1.8 は物理式から導出された値ではなく、
目標の eta_rad=36% から `Gamma = 1/eta_rad - 1` で逆算した無次元数である。
原典0430f 自体に誤りが存在する。

### 5.2 eta_rad = 1/(1+Gamma) (数学的整合)

| 楽器 | 原典Gamma | 1/(1+Gamma) | 原典eta_rad | 整合 |
|------|------|------|------|------|
| Guitar | 3.0 | 0.250 | 0.25 | 一致 |
| Piano | 1.5 | 0.400 | 0.40 | 一致 |
| Violin | 2.0 | 0.333 | 0.33 | 一致 |
| Pianica v3 | 1.8 | 0.357 | 0.36 | 一致 |

数学的には整合している。ただし Gamma 自体が逆算値なので、
これは「逆算で得たΓを再び逆変換した」結果に過ぎない。

### 5.3 f_11 = 96 Hz (設計値、妥当範囲)

単純支持矩形板の共振周波数:

```
f_mn = (pi/2) * sqrt(D/(rho_mat*h)) * ((m/a)^2 + (n/b)^2)
D = E*h^3 / (12*(1-nu^2))
```

ピアニカv3 想定 (300×300 mm, h=2 mm, 桐, 単純支持仮定) で計算:

| E [GPa] | f_11 [Hz] |
|------|------|
| 5 | 89.3 |
| 6 | 97.8 |
| 7 | 105.6 |

原典の 96 Hz は E≒6 GPa の範囲に入る。妥当な設計値。
ただし実際の境界条件は単純支持ではないため、実機での実測値は前後する。
未製作のため確定できない。

### 5.4 T_coupling = 97% (検算不可)

T_coupling = 4·Z1·Z2 / (Z1+Z2)² は標準的なインピーダンス整合の式である。

ピアニカv3 の T=97% はリード基部の ABS と桐共鳴板の界面での値とされるが、
**原典0430f には Z_source と Z_board の値が記載されていない**。
そのため本書では検算できない。実測待ち。

### 5.5 P_rad/P_input = 35% (合成値、実測ではない)

T × eta_rad = 0.97 × 0.36 = 0.349 ≒ 35%

数学的には正しい。ただし T も eta_rad も実測値ではない。

### 5.6 リード→板伝達 1-2 mW、足踏み 80 mW (出典不明の推定値)

旧版 Appendix B で使われたこれらの数値は **原典0430f に記載がない**。
旧版執筆時点での推定値であり、根拠となる物理式や測定データは旧版にも示されていない。
本書では採用しない。

## 6. 主聖堂 (カトリック神戸中央教会) について

### 6.1 旧版の問題点

旧版 Appendix C は主聖堂を「矩形 25×18×10 m, 4,500 m³」と仮定し、
矩形固有モードの計算を行った。しかし Web で確認できる事実は次の通り:

- 阪神淡路大震災から10年後の2004年完成、村上晶子アトリエ＆パートナーズ設計、被災した3教会 (灘、中山手、下山手) の統合再建
- 外観: 白煉瓦、緑青銅板葺きの大屋根
- 内部: 白煉瓦の壁、16枚のステンドグラス
- 形状: 「聖堂は楕円形でほとんど窓がなく」(出典: いこーよ)、「箱舟をイメージ」(出典: フォートラベル、トリップアドバイザー)
- 「3枚の壁で囲まれた聖堂」(出典: トリップアドバイザー)

つまり実態は **楕円形/箱舟形** であり、矩形ではない。

### 6.2 本書での扱い

主聖堂の体積、残響時間、座席数、固有モードなどの音響データは公開情報で
確認できなかった。本書ではこれらを推定・計算しない。

主聖堂を音響的に論じるには実測が必要である。本書では「楕円形/箱舟形の建築物」
という事実のみ記載し、共鳴箱パラダイムとの関連付けは行わない。

## 7. ピアニカv3 設計への含意

### 7.1 確実に言えること

- 共鳴箱パラダイムは電気を経由しない (1段変換)
- 桐は密度が低い (280 kg/m³) ため、放射効率の物理的有利性は期待できる方向
- スピーカと共鳴箱は異なる物理問題を解いている

### 7.2 設計値として採用するもの

- 板寸法 300×300 mm, h=2 mm 桐 (設計値、検証は実機後)
- f_11 推定 89-106 Hz の範囲 (E=5-7 GPa の範囲、単純支持仮定)
- リード励振による駆動 (37音、A4=440Hz、平均律)

### 7.3 採用しないもの (撤回)

- Gamma=1.8 を「物理式から導出」とする記述 (実は逆算値)
- 「90倍効率差」(比較基準のずれ、撤回)
- リード→板 1-2 mW、足踏み 80 mW (出典不明、撤回)
- 主聖堂を矩形と仮定した固有モード計算 (実態は楕円形/箱舟形、撤回)

### 7.4 実機後に確定すべき値

- 板の実際の f_11 (Spectroid FFT で測定)
- 効果的な Q 値 (リング法または半値幅法)
- 板の境界条件と E の実値 (打撃応答測定)
- T_coupling の実態 (励振点と板上の応答比較)
- 1m地点の SPL (騒音計または校正済みマイク)

## 8. 結論

スピーカと共鳴箱は異なる物理問題を解く装置であり、優劣ではなく使い分けの対象
である。ピアニカv3 は共鳴箱パラダイムを採用するが、その採用根拠を
「定量的な90倍効率差」のような誇張で正当化する必要はない。

採用根拠は次の事実で十分である:

1. 電気を経由しない (停電時運用、長寿命、電源設備不要)
2. 桐の物性が共鳴板として有利な方向にある (密度低)
3. 楽器音響学の体系内で位置付けられる (アコースティック楽器の系譜)

これらは実機検証を経て確認される予定の項目を含む。本書は **設計判断の出発点**
であり、最終的な数値は実機実測で確定する。
