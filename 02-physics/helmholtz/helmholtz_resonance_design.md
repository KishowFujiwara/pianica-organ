# ヘルムホルツ共鳴をピアニカの味方にする

## 問題: 低音域で板が鳴らない

桐薄板箱の放射面は上面+下面で1800cm^2。
しかし低音域では波長に対して板が小さすぎ、放射効率が落ちる。

```
| 音名 | 周波数 | 波長 | 板寸法/波長 | 板の放射効率sigma |
|------|--------|------|------------|-----------------|
| F3   | 175 Hz | 1.96m | 0.20 | 29% -- 弱い |
| G3   | 196 Hz | 1.75m | 0.23 | 37% -- 弱い |
| A3   | 220 Hz | 1.56m | 0.26 | 47% -- まだ弱い |
| B3   | 247 Hz | 1.39m | 0.29 | 59% |
| C4   | 262 Hz | 1.31m | 0.31 | 66% |
| E4   | 330 Hz | 1.04m | 0.38 | 100% |
| A4   | 440 Hz | 0.78m | 0.51 | 100% |
```

F3-A3(175-220Hz)が問題帯域。板だけでは音量の30-47%しか
出ない。これが「低音が薄い」原因になる。

**ギターが同じ問題をどう解決しているか:**
ギターの表板面積は800cm^2、最低音E2=82Hz(波長4.2m)。
板寸法/波長=0.07。板だけでは低音がほぼ出ない。
サウンドホール+内部空洞のヘルムホルツ共鳴(f_H~100Hz)が
低音域を担当する。ギターの低音はほぼ全てヘルムホルツが鳴らしている。

## ヘルムホルツ共鳴の物理

```
f_H = (c / 2*pi) * sqrt(S_port / (l_eff * V))

S_port: ポート面積 [m^2]
l_eff:  ポートの実効長 = 壁厚 + 0.85 * d (端部補正)
V:      箱の内部容積 [m^3]
c:      音速 343 m/s
```

桐箱の内部容積:

```
箱寸法: 400 x 225 x 50 mm
V = 4.5 L = 0.0045 m^3
```


### ポート設計

f_H を F3(175Hz)に合わせる:

```
2 x d=30mm 円形ポート
S_port = 2 * pi * (0.015)^2 = 14.1 cm^2
l_eff = 5mm(壁厚) + 0.85 * 30mm = 30.5mm

f_H = 343/(2*pi) * sqrt(0.00141 / (0.0305 * 0.0045))
    = 54.6 * sqrt(3.81)
    = 54.6 * 1.95
    = 175 Hz   [PASS: F3に一致]
```

## ヘルムホルツの2つの効果

### 効果1: ポートからの直接放射

ポートは小さな開口だが、ヘルムホルツ共鳴でQ倍に増幅された
空気の流れが噴出する。ポートはモノポール(点音源)として
放射し、前後の打ち消しがない。

f_H での板 vs ポートの放射パワー比較:

```
板(上面+下面, sigma=0.29):  P_plate = 0.053
ポート(2x30mm, Q=8):       P_port  = 0.091
合計:                       P_total = 0.144

ポートは板の1.7倍のパワーを放射する。
```

**F3ではポートが主たる放射体になる。**

ブースト: +4.3 dB at F3


### 効果2: 全壁面のモノポール駆動

ヘルムホルツの真の力はポート放射だけではない。

f_H において、箱内部の空気全体が同位相で圧力変動する。
この変動圧力が箱の全壁面を内側から同時に押す。
全壁面が同位相で「膨らむ-縮む」を繰り返す。

```
通常の板振動:
  各パネルが独立に振動
  パネル内でもモード形状により位相が異なる
  部分的に打ち消し合う

ヘルムホルツ駆動:
  内部圧力が全壁面を同位相で駆動
  全パネルが同時に外に膨らみ、同時に内に縮む
  打ち消しなし -> 完全なモノポール放射
```

これはスピーカーのバスレフ筐体と同じ原理。
しかしピアニカの桐箱では壁面全体が薄い振動板なので、
スピーカー筐体(振動しない剛体)とは異なり、
壁面自体が積極的に放射に参加する。

f_H での壁面ごとの放射寄与:

```
| 壁面 | 面積[cm^2] | sigma | 放射寄与 |
|------|-----------|-------|---------|
| 上面 | 900 | 0.29 | 49% |
| 下面 | 900 | 0.29 | 49% |
| 側面(長) x2 | 200 x2 | 0.07 | 1% |
| 側面(短) x2 | 113 x2 | 0.04 | 0.1% |
| ポート 2x30mm | 14 | 1.00 | (別計算) |
```

壁面駆動とポート放射を合わせると:

```
ポート寄与: 71%
壁面寄与:   29%
```

ポートが主役だが、壁面も無視できない。
壁面のGammaが小さいほど壁面寄与が増える。
桐箱(Gamma=0.66)は壁面寄与がABS箱(Gamma=93)の
数十倍大きい。

**だから箱の材料が低音域で重要になる。**

## ポートの位置と形状

### 円形ポート vs スロットポート

ギターは円形サウンドホール(直径約86mm)を1つ。
バイオリンはf字孔(スロット形状)を2つ。

```
| 形状 | メリット | デメリット |
|------|---------|----------|
| 円形 | 計算が容易、製作容易 | 面積を稼ぐと大穴になる |
| スロット | 面積を分散配置可能 | 端部補正が複雑 |
```

スロットポートの候補:

```
| 構成 | S_total | f_H | Boost@F3 |
|------|---------|-----|----------|
| 2 slots 10x40mm | 8.0 cm^2 | 198 Hz | +1.0 dB |
| 2 slots 5x60mm | 6.0 cm^2 | 207 Hz | +0.5 dB |
| 2 slots 8x50mm | 8.0 cm^2 | 212 Hz | +0.6 dB |
```

スロットの端部補正 l_eff はスロット幅 w で決まるため、
幅が狭いと l_eff が小さくなり、f_H が上がりすぎる。
F3(175Hz)にチューニングするには十分な幅が必要。

推奨: **円形ポート 2x30mm** がシンプルかつ最適。


### ポートの配置

ポートの位置は音の指向性と見た目に影響する。

```
案A: 上面(リードプレート側) -- ギター型
  + 音が奏者に向かう
  - リードプレートの配置と干渉する可能性

案B: 側面(短辺) -- バスレフスピーカー型
  + リードプレートと干渉しない
  + 低音は無指向性なので位置はあまり問題にならない

案C: 底面 -- 下向き放射
  + テーブル面で反射して増幅
  - テーブルがないと弱い
```

低音の波長(F3で1.96m)は箱のどの寸法より遥かに大きいため、
ポートの位置は放射パワーにほとんど影響しない。
実装上の都合で決めてよい。

推奨: **側面(短辺)に左右対称に1つずつ** が最も干渉しにくい。

## ヘルムホルツが効く帯域と効かない帯域

```
f < 0.7*f_H (< 120Hz): 帯域外、効果なし
f = f_H (175Hz):         最大ブースト +4.3dB
f = 1.1*f_H (193Hz):     ブースト +1.0dB
f = 1.3*f_H (228Hz):     ブースト +0.2dB
f > 1.5*f_H (> 260Hz):   効果ほぼゼロ、板だけで十分
```

ヘルムホルツは**F3の一点集中砲火**。
f_Hから半音~全音上がると効果が急減する。

しかしこれで十分。なぜなら:
- F3-G3 が最も板放射が弱い帯域(sigma 29-37%)
- A3以上は板だけでsigma 47%以上あり、実用上問題ない
- ヘルムホルツが F3-G3 を持ち上げれば、音量バランスが揃う

## ギターとの対比

```
| パラメータ | ギター | ピアニカ桐箱 |
|-----------|--------|------------|
| 内部容積 | 13 L | 4.5 L |
| ポート | 86mm円形 x1 | 30mm円形 x2 |
| f_H | ~100 Hz | 175 Hz |
| 最低音 | E2 = 82 Hz | F3 = 175 Hz |
| f_H/f_low | 1.2 | 1.0 |
| 板面積 | 800 cm^2 | 1800 cm^2 |
| 低音域の板sigma | <5% | 29% |
| ヘルムホルツ依存度 | 極めて高い | 中程度 |
```

ギターはヘルムホルツなしでは低音が全く出ない(sigma<5%)。
ピアニカはヘルムホルツなしでもsigma29%で「弱いが出る」。
ヘルムホルツは「ないと困る」のではなく「あると嬉しい」。

しかし+4.3dBは知覚上明確な差であり、
「低音が薄い」と「低音が豊か」の分岐点になり得る。

## 容積と f_H の関係

箱の高さ(=内部容積)を変えると f_H が変わる。

```
同じポート(2x30mm)で:

| 箱高 | V [L] | f_H [Hz] | 最低音との関係 |
|------|-------|----------|--------------|
| 30mm | 2.7 | 226 Hz | A3 付近 |
| 40mm | 3.6 | 196 Hz | G3 付近 |
| 50mm | 4.5 | 175 Hz | F3 [PASS] |
| 60mm | 5.4 | 160 Hz | E3 |
| 70mm | 6.3 | 148 Hz | D3 |
```

箱高50mmで f_H が F3(175Hz)にちょうど合う。
箱を浅くすると f_H が上がり、低音域からずれる。
箱を深くすると f_H は下がるが、箱が大きくなりすぎる。

箱高50mm = 最低音F3にチューニングされた最適サイズ。

## 全体設計へのフィードバック

桐薄板箱(0.7mm, リブ60mm間隔)にヘルムホルツポートを追加:

```
構造: 桐箱 400x225x50mm
壁厚: 0.7mm (リブ補強, 60mm間隔)
ポート: 側面短辺に 30mm円形 x2
f_H:   175 Hz (F3 にチューニング)
```

帯域別の放射メカニズム:

```
F3-G3 (175-196Hz): ヘルムホルツ + 板 (ポート主導, +4dB)
A3-C4 (220-262Hz): 板放射 (sigma 47-66%, 十分)
E4以上 (330Hz+):   板放射 (sigma 100%, 問題なし)
```

全体効率:

```
E4以上: BN1(18%) * BN2(68%) * BN3(60%) = 7.3%
F3:     BN1(18%) * BN2(68%) * BN3(60%) * 1.7(Helmholtz) = ~12%
```

F3でのヘルムホルツ効果は約1.7倍(+4.3dB)の増幅。
板のsigma=29%がヘルムホルツのモノポール放射で補完され、
実効的にsigma~50%相当になる。

ヘルムホルツは低音域の「最後の1ピース」。
これで全帯域にわたって均一な音量バランスが達成される。

## ピアニカの3つの放射メカニズム統合

```
1. 板モード放射 (BN3)
   担当帯域: 全域、特に中高音
   機構: 壁面の曲げ振動 -> 空気放射
   制御パラメータ: Gamma (材料, 板厚, スパン)

2. ヘルムホルツ共鳴 (ポート)
   担当帯域: f_H 付近の低音域
   機構: 箱全体の空気が共鳴 -> ポート+壁面のモノポール放射
   制御パラメータ: V(容積), S_port(ポート面積), l_eff(ポート長)

3. 固体伝導 (BN2)
   担当帯域: 全域
   機構: リード -> プレート -> 壁面への振動伝達
   制御パラメータ: Z整合, 接触面積, 経路数
```

3つ全てが桐薄板箱で改善される:
- 板モード: Gamma=0.66 (ABS比 1/140)
- ヘルムホルツ: 桐壁が柔らかいため壁面寄与が大きい
- 固体伝導: 桐の Z_point がプレートに近く BN2=68%

P-32Dでは3つ全てが悪かった:
- 板モード: Gamma=93
- ヘルムホルツ: ABS壁が硬くて重く、壁面寄与ゼロ
- 固体伝導: ABS->空気の界面でBN3=1%

**桐薄板箱 + ヘルムホルツポートは、3メカニズムを
同時に最適化する設計。** これが「ヘルムホルツを味方にする」
ことの物理的意味。
