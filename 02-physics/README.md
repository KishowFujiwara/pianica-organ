# 02-physics: 原理・物理的検証

## 章の目的

本章は、本プロジェクトで展開した**音響物理の検証作業**を集約する。
ピアニカP-32Dを起点に、22の振動伝達経路を同定し、フリーリードの発音原理、共鳴構造、3楽器(ギター・ピアノ・バイオリン)との比較まで、設計v3の物理的根拠を構成する。

## 主要発見の要約

### L1: 22経路の同定 (`22-paths/`)

ピアニカP-32Dの「リード振動 → 外部音」の経路を網羅的に列挙し、22経路を同定:
- A系 (空気経由): 鍵盤穴・ホース・空気室から外部へ
- B系 (固体伝導 → 壁面放射): リードプレート → ABS壁 → 外部
- C系 (筐体振動): ABS底面・側面の曲げモード

**核心**: 外部放射の95.2%はC系(壁面放射)。A1(鍵盤穴)はわずか0.8%。

### L2: パワーフロー定量化 (`powerflow/`)

22経路を5軸(構造インピーダンス・放射効率・周波数依存・位相・結合度)で評価。
v2.0で**構造パワー/放射パワーの混合**を正しく扱う修正。

### L3: 因果機構 (`causal-mechanism/`)

v1/v2/v3の放射効率を3段ボトルネック(BN1/BN2/BN3)で記述。
v3の35% (P-32D比98倍) を達成する因果連鎖を整理。

### L4: 材料・構造物理 (`materials/`, `helmholtz/`)

- 空気室材料: スパンが支配パラメータ。桐0.7mm + リブ60mmで効率7.3%
- 魂柱: マスローディング問題。全面接着Gamma=40 vs 魂柱Gamma=1.8
- ヘルムホルツ: F3に+4.3dB、ポート2×30mm、壁面モノポール駆動

### L5: 3楽器比較 (`three-instruments/`)

ギター・グランドピアノ・バイオリンの共鳴構造を統一フレームワーク (励振 → 結合 → 共鳴板 → 共鳴空洞 → 放射) で比較。
**結論**: ピアニカが学ぶべきはギター製作 (フラットトップ3DOF結合系)。バイオリンではない。
理由は「励振の連続性」ではなく「エネルギースケール」(本プロジェクトのクラスI微弱励振分類)。
