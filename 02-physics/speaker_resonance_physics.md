## 1. 序論 -- 本文書の目的

スピーカと共鳴箱（楽器の響板+空洞）は、いずれも「振動を空気中の音に変換する装置」である。しかし、エネルギー変換の段数、効率の定義、設計自由度において根本的に異なる。

本文書は両者の物理的差異を標準的な音響工学の枠組みで整理する。

**本文書に含まないもの**:
- ピアニカv3固有の数値（別文書に分離）
- 神戸中央教会の音響設計（別文書に分離）
- アナロジーや比喩による解釈

## 2. スピーカの原理 -- 3段エネルギー変換

### 2.1 全体構造

スピーカ（動電型）は以下の3段変換で電気信号を音波に変える:

```
電気 -> 機械 -> 音響
(アンプ)  (コーン)  (空気)
```

各段は異なる物理法則に支配される。

### 2.2 電気-機械変換 (Lorentz力)

永久磁石の磁束密度 B [T] の磁気ギャップ中に長さ l [m] のボイスコイルを配置し、電流 i [A] を流す:

```
F = B * l * i   [N]
```

この力がコーン（振動板）を駆動する。典型的なパラメータ:

- B = 1.0-1.5 T
- l = 5-15 m（コイル総長）
- B*l = 5-20 N/A（力係数, Bl product）

### 2.3 機械系の運動方程式

コーン+ボイスコイルの機械系は質量-ばね-ダンパ系でモデル化される:

```
M * x'' + R * x' + (1/C) * x = F(t)
```

- M: 振動質量（コーン + 空気負荷）[kg]
- R: 機械抵抗 [N*s/m]
- C: サスペンションのコンプライアンス [m/N]
- 共振周波数: f_s = 1 / (2*pi*sqrt(M*C))

Thiele-Small パラメータ (f_s, Q_ts, V_as) がこの機械系を完全に記述する。

### 2.4 機械-音響変換

コーンが面積 S_d [m^2] で前後にピストン運動すると、体積速度 U = S_d * v [m^3/s] を生成し、放射インピーダンス Z_rad を通じて音波を放射する:

```
P_rad = Re(Z_rad) * |U|^2
Z_rad ≈ rho * c * (k*a)^2 / 2 + j * (8/3*pi) * rho * c * k*a   (ka << 1)
```

- k = 2*pi*f / c (波数)
- a: コーン等価半径

低周波（ka << 1）では放射抵抗が (ka)^2 に比例するため、効率が極めて低い。

### 2.5 エンクロージャの役割

裸のスピーカユニットはコーン前後が音響的に短絡（双極子放射、前後の位相差でキャンセル）するため、エンクロージャ（箱）が不可欠:

- **密閉型**: 後面放射を完全に封じる。箱内空気がばねとして作用し f_s を上昇させる
- **バスレフ型**: ポート（ダクト）でHelmholtz共鳴を利用し低域を補完。ポート共振周波数 f_b 付近でコーン振幅が最小化
- **バックロードホーン**: 後面放射をホーンで導き低域効率を向上

### 2.6 スピーカの効率

典型的な効率（電気入力 -> 音響出力）:

```
eta = P_acoustic / P_electric ≈ 0.1% - 5%
```

大半のエネルギーはボイスコイルの抵抗熱として散逸する。Hi-Fiスピーカの典型値は 0.5-2%、PA/SRスピーカは 2-5%。

Hofmannの鉄則: 低域限界周波数・効率・箱容積の3変数にはトレードオフがあり、いずれか2つを固定すると残り1つが決まる。

## 3. 共鳴箱の原理 -- 1段エネルギー変換

### 3.1 全体構造

楽器の共鳴箱は以下の1段変換で機械振動を音波に変える:

```
機械 -> 音響
(弦/リード)  (板振動 -> 空気)
```

電気段がない。励振源（弦、リード等）の機械振動が駒（bridge）を介して板に伝わり、板全面が空気を駆動する。

### 3.2 駒の役割 -- インピーダンス変換

弦と板のインピーダンス比は典型的に 1:100 - 1:200。駒はこの不整合を緩和する:

```
T_coupling = 4 * Z_source * Z_board / (Z_source + Z_board)^2
```

| 楽器 | Z_source [Ns/m] | Z_board [Ns/m] | T |
|------|-----------------|----------------|---|
| ギター | 0.3 | 50 | 2.3% |
| ピアノ | 1-20 | 200 | 2-30% |
| バイオリン | 0.15 | 30 | 2% |

弦楽器でTが低い（2-30%）のは意図的設計である。Tが高すぎると弦エネルギーが即座に板へ流出し、サスティン（減衰時間）が短くなる:

```
tau_decay = 2 * m_string / (T * Z_bridge)
```

連続励振楽器（バイオリンのボウイング、フリーリード等）ではこの制約がないため、T を高くできる。

### 3.3 板振動のモード理論

共鳴板（矩形、単純支持の理想化）の固有振動数:

```
f_mn = (pi/2) * sqrt(D / (rho * h)) * ((m/a)^2 + (n/b)^2)

D = E * h^3 / (12 * (1 - nu^2))   [曲げ剛性]
```

- E: ヤング率 [Pa]
- h: 板厚 [m]
- rho: 材料密度 [kg/m^3]
- nu: ポアソン比
- a, b: 板の寸法 [m]
- m, n: モード次数 (1, 2, 3, ...)

最低モード (m=n=1) が基本振動で、板全面が同位相で振動する。高次モードでは節線が現れ、板が領域ごとに逆位相で振動する。

### 3.4 放射効率

板が空気中に放射する効率は、振動モードの空間パターンと音波の波長の関係で決まる。

**臨界周波数 f_c**: 板の屈曲波速度が空気中の音速と一致する周波数:

```
f_c = c^2 / (2*pi) * sqrt(rho * h / D)
```

- f < f_c: 屈曲波速度 < 音速。放射効率が低い（モードが空気波長より細かい場合キャンセル）
- f > f_c: 屈曲波速度 > 音速。放射効率が高い（coincidence効果）
- f ≈ f_c: 最大放射（板表面が音波面と整合）

楽器の共鳴板は f_c が高い（桐板2mm厚: f_c ≈ 7,300 Hz、スプルース2.5mm: f_c ≈ 5,000 Hz）ため、音楽的に重要な周波数帯域（100-4,000 Hz）では f < f_c であり、放射はモードごとの体積速度に依存する。

### 3.5 Helmholtz共鳴 -- 内部空洞の役割

共鳴箱の内部空気は、サウンドホール（f字孔等）と結合してHelmholtz共振器を形成する:

```
f_H = (c / (2*pi)) * sqrt(S_hole / (l_eff * V))
```

- S_hole: 開口面積 [m^2]
- l_eff: 開口の実効長さ [m]
- V: 内部容積 [m^3]

これにより板振動と空洞共鳴の二重の放射機構が実現する。ギターではf_H ≈ 100 Hz、バイオリンではf_H ≈ 290 Hz（A0モード）。

### 3.6 内部摩擦と材料特性

共鳴板材料には低い内部摩擦（損失係数 eta）が要求される:

| 材料 | 密度 [kg/m^3] | E [GPa] | eta | 用途 |
|------|--------------|---------|-----|------|
| スプルース | 400-430 | 10-12 | 0.01-0.02 | ギター、ピアノ、バイオリン |
| 桐 | 260-300 | 3.5-7.0 | 0.03-0.06 | 箏、琵琶 |
| メープル | 650 | 10-14 | 0.02-0.03 | バイオリン裏板 |

桐はスプルースより密度が低く（軽い）が、損失係数がやや高い。E/rho（比弾性率）が設計上の重要パラメータであり、高いほど同じ板厚で高い共振周波数を実現できる。

## 4. 両者の本質的違い

### 4.1 エネルギー変換構造の対比

```
スピーカ:   電気 -> [ボイスコイル] -> 機械 -> [コーン] -> 音響
            (アンプ出力)              (振動)          (放射)

共鳴箱:     機械 -> [駒] -> 板振動 -> 音響
            (弦/リード)     (モード重畳)  (面放射)
```

| 項目 | スピーカ | 共鳴箱 |
|------|---------|--------|
| エネルギー変換段数 | 3段（電気-機械-音響） | 1段（機械-音響） |
| 励振方式 | 電磁力（ピストン運動） | 機械的結合（板モード励振） |
| 放射面 | コーン（1点制御） | 板全面（多モード重畳） |
| 帯域制御 | 電気的（クロスオーバー、EQ） | 構造的（板厚、材料、形状） |
| 効率 | 0.1-5%（電気-音響） | 定義困難（機械入力の定義による） |
| 再現性 | 高い（工業製品） | 低い（材料個体差） |
| 設計変数 | Thiele-Small (f_s, Q_ts, V_as) | 板材料、形状、駒位置 |
| 熱損失 | 大きい（コイル抵抗熱） | 小さい（内部摩擦のみ） |

### 4.2 効率の定義の非対称性

**スピーカの効率は明確に定義できる**: eta = P_acoustic / P_electric。電気入力は電力計で測定でき、音響出力は無響室で測定できる。

**共鳴箱の効率は定義が曖昧**: 「機械入力」が一意に定義できない。弦のエネルギーは弦の運動エネルギーとポテンシャルエネルギーの和だが、駒を通じて板に伝わる瞬時パワーは周波数・振幅・位相に依存する。

したがって、スピーカと共鳴箱の効率を直接比較する場合は**比較基準を明示する必要がある**。スピーカの「電気-音響効率 0.5%」と共鳴箱の「放射効率」は異なる物理量であり、単純に比較すると誤導的な結論を生む。

### 4.3 熱力学的差異

**スピーカ**: ボイスコイルの電気抵抗 Re にジュール熱 I^2*Re が常に発生する。定格入力100Wのスピーカで95-99Wが熱になる。パワーコンプレッション（温度上昇によるRe増加 -> 効率低下）が実用上の問題。

**共鳴箱**: 機械エネルギーが板の内部摩擦で散逸する量は微小（eta = 0.01-0.06）。板の温度上昇は無視できる。エネルギーの大半は音響放射か構造的伝搬で消費される。

### 4.4 流体力学的差異

**スピーカ**: コーンがピストン運動で空気を直接圧縮する。低周波ではka << 1の非効率な放射。バスレフポートでは高SPL時に乱流が発生し歪みの原因となる。

**共鳴箱**: 板の屈曲振動が表面に沿った圧力変動を生成する。多数のモードが重畳し、各モードが固有の放射パターンを持つ。ピストン的な「一様圧縮」ではなく「面状の複雑なパターン放射」。

## 5. 設計哲学の違い

### 5.1 スピーカ設計

スピーカ設計は**電気工学の延長**:

- Thiele-Small モデルにより箱の寸法・ポート寸法を解析的に決定できる
- 周波数特性、歪率、指向性がスペックシートで定量化される
- 同一設計の再現性が高い（工業製品）
- クロスオーバーネットワーク、DSP等で後段補正が可能

設計自由度: 高い。電気的制御で応答を広範に調整できる。

### 5.2 共鳴箱設計

共鳴箱設計は**材料科学と経験的知見の融合**:

- 板厚、木目方向、含水率が特性を大きく左右する
- 個体差が大きく、解析的予測と実物の差が大きい
- 「タップトーン」（板を叩いた音）による職人的調整が不可欠
- 設計後の電気的補正ができない（パッシブ系）

設計自由度: 低い（物理法則と材料特性に強く拘束される）。ただし、この拘束が「楽器の個性」を生む。

### 5.3 空間規模による設計言語の分類

音響設計は空間規模により使用する概念体系（設計言語）が異なる:

| 空間規模 | 代表例 | 主たる物理 | 設計言語 |
|---------|--------|-----------|---------|
| ~0.01 m^3 | 楽器共鳴箱 | 板モード、Helmholtz共鳴 | 楽器音響学 |
| ~10-100 m^3 | リスニングルーム | 室内音響、残響 | Hi-Fi/スタジオ音響 |
| ~1,000+ m^3 | 大ホール、教会 | 統計音響、STI | SR/PA設計 |

これらは規模による分類であり、「階層」（上位が下位を包含する）ではない。

## 6. 主要数式一覧

### スピーカ系

```
Lorentz力:        F = B * l * i
機械系:           M*x'' + R*x' + (1/C)*x = F(t)
共振周波数:        f_s = 1 / (2*pi*sqrt(M*C))
放射インピーダンス: Z_rad = rho*c*(ka)^2/2 + j*(8/(3*pi))*rho*c*k*a  (ka<<1)
効率:             eta = P_acoustic / P_electric
Hofmann鉄則:       低域限界 * 効率 * 箱容積 = 一定（トレードオフ）
```

### 共鳴箱系

```
板固有振動数:      f_mn = (pi/2)*sqrt(D/(rho*h)) * ((m/a)^2 + (n/b)^2)
曲げ剛性:         D = E*h^3 / (12*(1-nu^2))
臨界周波数:        f_c = c^2/(2*pi) * sqrt(rho*h/D)
Helmholtz共鳴:     f_H = (c/(2*pi)) * sqrt(S_hole/(l_eff*V))
結合伝達率:        T = 4*Z1*Z2 / (Z1+Z2)^2
```

### 共通（音響・流体）

```
音速:             c = 343 m/s (20 deg C)
空気密度:          rho_air = 1.2 kg/m^3
空気特性インピーダンス: rho*c = 412 kg/(m^2*s)
波数:             k = 2*pi*f / c
Sabine残響時間:    RT60 = 0.161 * V / A
Schroeder周波数:   f_S = 2000 * sqrt(RT60 / V)
```
