# ピアニカP-32D 全振動伝達経路の解析（22経路）

## 0. 正しい構造

```
外部（聴取側）
 ↑  ↑  ↑（穴から音が出る）
[鍵盤バルブ] 鍵を押すと穴が開く
+--□--□--□--□--□--□--□--□--+
|  ABS上面（穴あり、32個の開口）  |
|                                |
|  +--□--□--□--□--□--□--□-+  |
|  | リードプレート(金属、穴+リード)|  |  <- 箱の「中」にある
|  +----------------------------+  |
|                                |
|    ABS箱 内部空間（加圧）       |
|    呼気 -> ここに入る            |
|                                |
|    <- パイプ差し込み口           |
+--------------------------------+
  ABS底面・側面（穴なし、密閉壁）
```

空気の流れ:
**パイプ -> ABS箱内（加圧 1-2 kPa） -> リードプレートの穴を下から通過 -> リードが振動（発音） -> ABS上面の穴 -> 外部**

鍵盤が押されると穴が開き、加圧された空気が出ていく。その流れでリードが振動し音が発生する。

---

## 1. リードからリードプレートへの駆動（2経路）

リードはリードプレートを **2系統で同時に駆動** している。

### B1a: 固体伝導（リベット経由）

リード根元はリベットでリードプレートにクランプされている。リードの振動による**反力がクランプ点を通じてリードプレートに直接伝わる**。金属同士の接触で内部損失は極めて小さい（$\eta < 0.001$）。

クランプ点反力（A4 = 440 Hz、先端振幅 0.5 mm の場合）:

$$F_\text{clamp} = M_\text{eff} \cdot \omega^2 \cdot x_\text{tip} \approx 0.2\ \text{mN}$$

### B1b: スロット隙間空気経由

リードがスロットを出入りするとき、隙間（0.05-0.10 mm）の空気が激しく変動する。リードの振幅が 0.5 mm でスロット隙間が 0.1 mm なら、リードがスロットを通過する瞬間に隙間がほぼ閉じ、局所音圧が急激に上昇する。この音圧がリードプレートの**スロット周囲の面を直接叩く**。

### B1a と B1b の位相差

固体伝導（B1a）は即座に伝わるが、空気経由（B1b）はわずかに遅れる。この位相差がリードプレートの振動パターンを複雑にし、特定の周波数で強め合ったり打ち消し合ったりする。

---

## 2. フリーリードの指向性

フリーリードの放射パターンは**等方的ではなく、指向性を持つ**。

リードがスロットの上に出る半周期で上面に正圧、下面に負圧。戻る半周期で逆。これは音響学の**ダイポール（双極子）音源**に相当し、放射パターンは $\cos\theta$（$\theta$ はリード振動方向からの角度）。

ただし整流作用（一方向にしか空気が有意に通過しない圧力式）により、**モノポール成分も重畳**する。実際のフリーリードは**モノポール + ダイポールの混合音源**。

箱内での壁面駆動強度:

| 壁面 | 駆動強度 | 理由 |
|------|---------|------|
| ABS上面 | **最大** | ダイポール最大方向 + 最短距離。ただし穴だらけ |
| ABS底面 | **大（逆位相）** | ダイポール最大方向。密閉壁で最大面積 |
| ABS側面 | 小 | ダイポールのヌル方向 |

**底面がテープ処置の最重要ターゲット**: ダイポールの強駆動方向にあり、穴がなく密閉で最大面積（192 cm²）。

## 3. 全22経路一覧

### A系列: 空気伝搬経路（5経路）

#### A1: リード -> 箱内空気 -> 開いた穴 -> 外部

**楽音の主経路。** リードの振動で変調された空気が、鍵盤バルブで開放された穴を通じて外部に放射される。ピアニカの音のほぼすべてがこの経路を通る。

箱全体がヘルムホルツ共鳴器として機能する:

$$f_H = \frac{c}{2\pi}\sqrt{\frac{S_\text{hole}}{V \cdot L_\text{eff}}}$$

1鍵開放時 $f_H \approx 332$ Hz。複数鍵開放で $f_H$ は上昇する。

#### A2: 箱内空気 -> ABS壁面を振動させる

箱内の音圧変動がABS壁面を**外側から叩く**（音圧駆動）。ダイポール指向性により底面が最も強く駆動される。壁面のモード周波数で共振が生じ、不快な着色音の原因となる。

#### A3: 箱内空気 -> 閉じているバルブを振動させる

押していない鍵のバルブも箱内音圧で揺れる。バルブが微振動すると、閉じた穴から微小な音漏れが発生する。パッキンの劣化で顕著になる。

#### A4: 箱内空気 -> パイプ -> マウスピース -> 奏者の口

箱内の音圧変動がパイプを逆流して奏者の口に戻る。奏者は口腔で振動を感じる。パイプの管内共鳴と結合する可能性がある。

#### A5: 箱内定在波 -> 隣のリードとの空気的連成

箱内の定在波が、押していないリードも微弱に振動させる。定在波の腹に位置するリードが影響を受けやすい。

箱内定在波:

$$f_{n_x n_y n_z} = \frac{c}{2}\sqrt{\left(\frac{n_x}{L_x}\right)^2 + \left(\frac{n_y}{L_y}\right)^2 + \left(\frac{n_z}{L_z}\right)^2}$$

主要モード: (1,0,0) = 536 Hz, (2,0,0) = 1072 Hz, (3,0,0) = 1608 Hz

---

### B系列: 固体伝搬経路（9経路、うち2経路はユーザー指摘による追加）

#### B1a: リード -> リベット -> リードプレート [NEW]

固体伝導。リード根元のクランプ点からリードプレート金属に直接伝わる。金属同士なのでほぼ無損失。ユーザー指摘により追加。

#### B1b: リード -> スロット隙間空気 -> リードプレート表面 [NEW]

空気経由の力学的駆動。スロット隙間（0.05-0.10 mm）の空気の急激な圧力変動がリードプレート面を叩く。B1aとは位相が異なる。ユーザー指摘により追加。

#### B2: リードプレート -> ネジ -> ABS上面

リードプレートを固定するネジが振動を直接ABS上面に伝える。剛結合。

#### B3: リードプレート -> プレートパッキン -> ABS上面

リードプレートとABS上面の間のパッキン（紙製）を通じた振動伝導。4本ネジで圧縮されているので伝導率は高い。パッキンが湿っているとやや減衰する。

#### B4: ABS上面 -> ABS側面 -> ABS底面

ABS箱は射出成型の一体品なので、上面・側面・底面がコーナーですべて連結している。上面に伝わった振動がそのまま底面まで到達する。

#### B5: ABS箱 -> フレームパッキン(ゴム) -> フレーム -> 鍵盤・バネ

箱の振動がフレームに伝わり、鍵盤機構を振動させる。ゴムパッキン（$\eta \approx 0.3$）がある程度減衰するが、圧縮されているので完全には遮断しない。

#### B6: ABS箱 -> 4本のネジ -> 外装カバー

箱をケースに固定するネジが振動を外装プラスチックカバーに伝える。

#### B7: 外装カバー -> テーブル面 / 奏者の手

外装の振動が接触面を通じて外部構造に伝わる。テーブルに置いて演奏する場合、テーブルがサウンドボードとして機能する可能性がある。

---

### C系列: 再放射経路（5経路）

固体伝搬で振動を受けた部品が、空気に音として再放射する経路。

#### C1: ABS底面・側面の振動 -> 外部空気へ直接放射

壁面が振動板として外部に音を放射する。底面が最大面積（192 cm²）で、かつダイポール指向性の強駆動方向にある。

底面壁(1,1)モード: 321 Hz ($\approx$ E4)、Q $\approx$ 33

壁面放射のリングダウン時間:

$$\tau = \frac{Q}{\pi f} \approx 33\ \text{ms}$$

#### C2: ABS壁面の振動 -> 箱内空気へ再放射 -> 穴から外部

壁の振動が箱の内側に音を戻す。この音がA1経路（穴からの放射）に混入して外部に出る。**穴から出る楽音に壁の共鳴が「乗る」**メカニズム。

#### C3: 閉じたバルブの振動 -> 外部空気へ放射

A3で振動を受けたバルブが外部に微小な音を放射する。

#### C4: 外装カバーの振動 -> 外部空気へ放射

B6でネジ経由の振動を受けた外装カバーが鳴る。

#### C5: パイプの振動 -> 外部空気へ放射

B系列でABS箱から伝わった振動、およびA4で内部を通過する音圧により、パイプ壁面が振動して放射する。

---

### D系列: フィードバック経路（3経路）

振動が音源（リード）に戻る帰還経路。自励振動の安定性に関わる。

#### D1: 箱内圧力変動 -> リード自身の振動に影響

自励振動の本質的なフィードバックループ。リードの振動が空気流を変調し、それが箱内の圧力変動を作り、その圧力がリードを駆動する。

#### D2: ABS壁面の振動 -> 箱内空気 -> リードに戻る

C2の逆方向。壁面が振動して箱内に再放射した音がリードに影響する。壁の共振周波数でリードの振動が変調される（二次フィードバック）。

#### D3: リードプレートの曲げ振動 -> 隣のリードのクランプ条件を変える

リードプレートは複数のリードが共通の金属板にリベットで取り付けられている。1枚のリードの振動がプレートの曲げモードを励振し、隣のリードのクランプ点の境界条件を微妙に変化させる。機械的連成。

---

## 4. テープ処置が影響する経路

メンディングテープを空気室内壁（リード以外）に全面貼付した場合、以下の経路が影響を受ける。

### 直接影響（壁面の振動特性変化）

| 経路 | テープの作用 | 効果 |
|------|------------|------|
| A2 | 壁のインピーダンス変化 | 壁の応答振動パターンが変わる |
| B4 | ABS壁の面密度・剛性・損失変化 | 壁内の振動伝導が変わる |
| C1 | **壁共振Q低下 (33->28)** | **壁面放射 -1.8 dB、リンギング -17%** |
| C2 | **壁の箱内再放射が変わる** | **穴から出る楽音の着色が変わる** |
| D2 | 壁->箱内->リードの帰還が変わる | リードの自励振動条件に微小な影響 |

### 間接影響（壁面変化の二次効果）

| 経路 | テープの作用 | 効果 |
|------|------------|------|
| A1 | C2経由で穴からの音質が変わる | エンベロープの変化 |
| C4 | B4->B6経由でカバー振動が変わる | 外装からの付帯音が微減 |

### 影響なし

| 経路 | 理由 |
|------|------|
| B1a, B1b | リードとリードプレートの結合は変わらない |
| B2, B3 | ネジ・パッキンの結合は変わらない |
| A3, A4, A5 | バルブ・パイプ・隣リード連成は変わらない |
| D1 | リード自身の自励振動ループは変わらない |
| D3 | リードプレート内の機械的連成は変わらない |

## 5. 「不快な周波数の共鳴を止める」の22経路的解釈

不快な共鳴の発生:

1. リードが振動（B1a + B1b でリードプレートを駆動）
2. リードプレートからABS壁に伝導（B2, B3, B4）
3. ABS壁がQ = 33 で共振（底面 321 Hz 付近が最も強い）
4. 壁の共振振動が箱内に再放射（C2）
5. 再放射された音が穴から出る楽音に着色を加える（A1に混入）
6. 同時に壁面が外部に直接放射（C1）

テープの処置:

1. 壁のQ低下（33 -> 28）で共振ピークが鈍る
2. 壁モードが +54 cent シフトし、楽音との「当たり」がずれる
3. リンギング短縮（33 ms -> 27 ms）でアタック/リリースがクリーンに
4. 穴からの楽音本体（A1）は変わらない

**結果: 穴から出る楽音はそのまま保ち、壁由来の不快な共鳴だけが選択的に抑えられる。**

---

## 6. 「エンベロープが変わった」の22経路的解釈

エンベロープ変化は主にC1とC2の時間応答変化に由来する。

テープなし:

- [Attack] リード起振(~20 ms) + 壁の励振立ち上がり(~30 ms)
- [Sustain] リード定常振動 + 壁の定常放射 + C2経由の着色
- [Release] リード停止 + 壁のリングダウン(~33 ms) がC1 + C2で残る

テープあり:

- [Attack] リード起振(~20 ms) + 壁の励振立ち上がり(~25 ms)
- [Sustain] リード定常振動 + 壁の定常放射（やや低レベル）
- [Release] リード停止 + 壁のリングダウン(~27 ms)

壁のリンギングが 17% 短縮されることで、音の切れが改善される。

## 7. プロ用アコーディオンとの対比

| 要素 | プロ用アコーディオン | ピアニカP-32D |
|------|---------------------|-------------|
| エンクロージャ | **硬材（メイプル、ビーチ）** | ABS 1.5mm |
| リード固定 | **蜜蝋（粘弾性ダンパー）** | リベット on 金属 |
| 内壁処置 | **ニス塗装（振動伝導制御）** | なし（テープで簡易対応） |
| 音色制御 | **cassotto（トーンチャンバー）** | なし |
| 木材選択 | **部位ごとに異材（職人の秘伝）** | 均一ABS |
| B1a経路の制御 | 蜜蝋の粘弾性で固体伝導を減衰 | リベット直結（無制御） |
| 壁のQ | 低い（硬材 + ニス） | 高い（Q $\approx$ 33） |

プロのアコーディオン職人は22経路のうち特にB1a, B4, C1, C2を**木材選択・蜜蝋・ニス**で精密に制御している。メンディングテープはこれらのごく簡易な近似操作。
