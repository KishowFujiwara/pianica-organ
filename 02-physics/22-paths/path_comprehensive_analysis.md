## 0. 本分析の位置付け

これまでの 22経路解析は経路の**分類と定性的説明**に重点があったが、各経路の**音響パラメータの定量比較**が不十分であった。本文書では:

1. 各経路の音響パワー収支を物理計算
2. 周波数応答特性 (特性周波数, Q値, 帯域)
3. 位相・遅延特性
4. 経路間の結合 (どの経路がどの経路に流入するか)
5. 音色 (アタック/サステイン/リリース) への寄与

を全22経路について網羅的に評価する。これは **PRODECOS の数式全単射検証 (0320) で確立した「悉皆検証」の発想を、設計文書から物理経路の定量分析に応用したもの**。

## 3. 軸1: 音響パワー収支

### 3.1 系列別合計

| 系列 | 合計パワー | 主な性格 |
|------|----------:|---------|
| A | 189 μW | 空気経由, 楽音穴含む |
| B | 100,250 μW | 固体伝導 (内部分配, 一部重複計上) |
| C | 20,830 μW | 再放射 (実際の外部放射主体) |
| D | (FB) | パワー収支ゼロ, 系の安定性に影響 |

注: B系列のパワー値はリードプレート -> ABS壁の各段階での振動エネルギーであり、各段階間で重複がある。実際にABS壁全体に伝わる正味のエネルギーは ~ 25,000 μW (B4)。

### 3.2 外部放射への寄与 (重要発見)

実際に **外部空気に放射される** 経路のみを抽出すると:

| 経路 | パワー [μW] | 寄与率 |
|------|-----------:|-------:|
| **C1 (壁面外部放射)** | **15,000** | **71.4%** |
| **C2 (壁->穴経由)** | **5,000** | **23.8%** |
| C4 (外装放射) | 800 | 3.8% |
| A1 (楽音穴経由) | 168 | **0.8%** |
| C5 (パイプ放射) | 30 | 0.1% |
| A4 (口腔逆流) | 10 | <0.1% |
| その他 | < 1 | <0.01% |
| **合計** | **21,008** | 100% |

**重要発見**: 22経路解析の文書では A1 を「楽音主経路」と呼んでいたが、**実際の外部放射では A1 は0.8%、C1 と C2 (壁面由来) で95.2%を占める**。

これは「**ピアニカは ABS箱を振動板として鳴らすスピーカー**」という固体伝導モデル(0411) の結論を、定量的に裏付ける。

## 4. 軸2: 周波数応答

### 4.1 周波数帯別寄与

各経路の特性周波数を低/中/高に分類し、寄与パワーを集計:

| 帯域 | 主要経路 (上位3) | 帯域内合計 [μW] | 性格 |
|------|---------------|-------------:|------|
| 低 (<200Hz) | A1(168), B5(200), C5(30) | 約 410 | 楽音とバックグラウンド |
| 低中 (200-400Hz) | C1(15000), B4(25000), C4(800) | 約 41,000 | **ABS壁面共振の支配域** |
| 中 (400-800Hz) | B1a(30000), B2(15000), B3(10500) | 約 56,000 | リード基音と高調波 |
| 高 (800Hz-) | B1b(18000) | 約 18,000 | 整流2倍音 |

### 4.2 ABS壁の (1,1) モード共振 = 323 Hz の支配性

ABS底面の(1,1)曲げモードは **323 Hz (約 E4)**。これは:

- 楽音帯域 (F3=175Hz ~ F6=1397Hz) のほぼ中心
- 外部放射の **95.2%** がこの周波数を中心に集中
- Q = 33 で帯域 ~10 Hz、つまり **±50 cent** の幅で全楽音に着色を加える

これがピアニカ特有の「箱鳴り」「金属的な響き」の物理的正体。

### 4.3 各リード周波数とABS壁共振の相互作用

| 楽音 | 周波数 [Hz] | ABS壁(1,1)モード323Hz との関係 | 着色強度 |
|------|----------:|---------------------------|--------|
| F3 | 175 | -147 Hz (壁モード以下) | 弱 |
| C4 | 262 | -61 Hz (近接, 増幅される) | 中 |
| **E4** | **329** | **+6 Hz (ほぼ共振点)** | **強(着色顕著)** |
| G4 | 392 | +69 Hz (低下開始) | 中 |
| C5 | 523 | +200 Hz (壁モード外) | 弱 |
| C6 | 1047 | 1700% 高域 | 高次モードが寄与 |

**E4-G4 帯域でピアニカ音色が最も「鳴る」ように感じる理由は ABS底面共振と一致するため**。これは経験的に演奏家が指摘する「中音域の鳴りが良い」と整合。

## 5. 軸3: 位相・遅延

### 5.1 即時応答経路 (遅延 < 0.05 ms)

```
B1a   : リード -> リベット -> プレート (固体直接, ほぼ無遅延)
B2    : プレート -> ネジ -> ABS上面 (剛結合)
D3    : リードプレート曲げ -> 隣リード (機械連成)
C1    : ABS壁 -> 外部 (直接放射)
```

これらは リードの瞬時の振動と **同位相** で外部に伝わる。

### 5.2 位相シフト経路 (0.05-0.5 ms)

```
B1b   : スロット隙間空気経由 (約 90度遅れ)
B3    : パッキン経由 (約 30度遅れ)
A1    : 箱内伝搬 (Helmholtz共振の位相応答)
C2    : 壁 -> 箱内 -> 穴 (約 60度遅れ)
D2    : 二次FBループ
```

### 5.3 大遅延経路 (> 0.3 ms)

```
A4    : パイプ逆流 (0.5 ms, 奏者口腔へ)
B5    : 鍵盤機構経由 (0.5 ms, 機械系)
B7    : テーブル接触経由 (0.5 ms, 接触損失大)
```

### 5.4 位相干渉の効果

B1a と B1b の位相差 (90度) は、リードプレートでの **ベクトル合成** を生む:

```
F_plate = F_B1a · cos(ωt) + F_B1b · cos(ωt - π/2)
        = F_B1a · cos(ωt) + F_B1b · sin(ωt)
        = √(F_B1a² + F_B1b²) · cos(ωt - φ)

φ = arctan(F_B1b/F_B1a) = arctan(18000/30000) = 31°
```

つまりリードプレートの実効振動は、リード変位より **31度 進んだ位相** で振動する。これは **整数倍音強調** に寄与する (時間波形の歪みが2倍音を生成)。

C1 (即時) と C2 (位相遅れ60度) の合成は、楽音域の各周波数で複雑な干渉パターンを作る。これがピアニカ特有の **"スウィッシュ"音** や音色の動的変化の原因。

## 6. 軸4: 経路間結合 (有向グラフ)

### 6.1 主要結合トポロジー

```
リード(440Hz, 48 mW)
  |
  +--経路①(168μW空気)--+--A1 (楽音穴)
  |                    +--A2 (壁駆動, 微弱)
  |                    +--A3 (バルブ)
  |                    +--A4 (口腔逆流)
  |                    +--A5 (定在波)
  |
  +--経路②(48,000μW固体)
       |
       +--B1a (30,000μW, リベット直接)
       |    |
       |    v
       +--B1b (18,000μW, スロット空気)
            |
            v
       リードプレート (合成 ~ 48,000μW)
            |
            +--B2 (15,000μW, ネジ)
            +--B3 (10,500μW, パッキン)
                 |
                 v
            ABS上面 (~ 25,000μW)
                 |
                 +--B4 (25,000μW, 一体成型 -> 全壁面)
                 |     |
                 |     +--C1 (15,000μW, 外部放射) -- 主経路
                 |     +--C2 (5,000μW, 箱内再放射 -> 穴) -- 着色
                 |     +--D2 (FB, 箱内 -> リード)
                 |
                 +--B5 (200μW, フレーム/鍵盤)
                 +--B6 (1,500μW, 外装) -- C4 (800μW)
                 +--B7 (50μW, テーブル)
```

### 6.2 フィードバックループの位相条件

D系列は系の安定性を決める。

**D1 (自励振動本体)**: リードからの空気流が箱内圧変動を生み、それがリードを駆動する正帰還。位相条件:

$$
\angle G_{D1}(\omega_0) = 2\pi n
$$

これが満たされる周波数で自励振動が発生 (リード固有振動数 ≈ 440 Hz)。

**D2 (二次FB)**: ABS壁の振動 (323 Hz, Q=33) が箱内空気を介してリードに戻る。位相が D1 とずれているため、**リード振動を変調**する。これは:

- E4 付近 (329 Hz) で 壁共振と楽音が近接 -> 強いビート/モード結合
- リードの実効振動が壁モードに同期する可能性 ("entrainment")

**D3 (機械連成)**: リードプレートの曲げモード (517 Hz) が隣のリードのクランプ条件を変える。和音演奏時の音程ずれの一因。

## 7. 軸5: 音色寄与 (アタック/サステイン/リリース)

### 7.1 アタック (鍵盤押下後 0-50 ms)

立ち上がる順序:

```
0 ms     : リード起振開始 (A1経由の空気変動が最初)
0.001 ms : B1a (固体伝導, ほぼ即時)
1-5 ms   : リード自励振動の振幅上昇 (D1 FBループ)
10-20 ms : ABS壁の振動立ち上がり (B4 の伝搬遅延 + Q=33の応答時間)
20-30 ms : C1 壁面放射ピーク到達
50 ms    : 定常状態
```

アタック特性は **リードの起振遅延 + ABS壁モードの応答時間** で決まる。立ち上がりがやや「ぼやける」のは、Q=33 の壁モードが30ms程度かかるため。

### 7.2 サステイン (定常)

定常状態では各経路が連続的にパワー供給:

| 経路 | サステイン中の役割 |
|------|----------------|
| A1 | 楽音穴からの基音放射 (微弱) |
| B1a-B4 | 固体伝導による連続的な壁駆動 |
| C1 | **壁面放射の支配的成分 (71%)** |
| C2 | 穴経由の壁着色音 |
| D1-D3 | 振動の安定化と音色着色 |

### 7.3 リリース (鍵盤離鍵後 0-100 ms)

減衰時定数で短い順:

```
A1 (5 ms)        : 穴の即時減衰
A4 (15 ms)       : パイプ逆流の減衰
B3 (14 ms)       : パッキン経由 (粘弾性損失)
C5 (16 ms)       : パイプ振動
B6/C4 (25 ms)    : 外装カバー
B4/C1/C2 (33 ms) : ABS壁面 (Q=33で支配的なリンギング)
B2 (36 ms)       : ネジ経由 (剛結合, やや長い)
D1 (70 ms)       : リード自励 (Q=100)
B1a (800 ms)     : リードプレート (金属の超低損失)
```

**ピアニカのリリース特性 (33 ms ~ ABS壁のリンギング) を決めているのは ABS壁モード**。これがメンディングテープ処置で短縮される原理 (Q低下 -> 33ms から 27ms へ -17%)。

## 9. 重要な発見と既存文書の修正提案

### 9.1 「楽音主経路 A1」の誤誘導

22経路解析 (`pianica_22paths_analysis.md`) ではA1を「楽音の主経路、ピアニカの音のほぼすべてがこの経路を通る」と記述しているが、**包括分析では A1 は外部放射の0.8%に過ぎない**。

真実は:

```
× A1 (鍵盤穴) が楽音の主経路
○ C1 (ABS壁面放射) と C2 (壁->穴) が楽音の主経路で、
  両者は同じ ABS壁の振動 (B4) に由来する
○ A1 は経路①の出口だが、経路①そのものが全パワーの0.35%
```

この誤りは概念分類段階では気付かれず、定量計算で初めて顕在化した。

### 9.2 「メンディングテープ処置」の有効性の再評価

過去会話 (1e12f962) でテープ処置による壁Q低下 (33 -> 28) が音色変化を生むことが確認されているが、本分析でその物理的根拠が明確になった:

- **C1** の音圧 ≈ √(P_C1) ∝ √(Q) -> Q が15%低下 -> 音圧 -0.7 dB
- **C2** の着色音 ≈ Q^(-1/2) -> 同様に低下
- **D2** のFB強度低下 -> リード振動の安定化
- リンギング時定数 33ms -> 27ms

つまりテープは **C1, C2, D2 の3経路を同時に弱める** 効果がある。これは外部放射の95% を占める経路群への直接介入であり、効果が大きいのは当然。

### 9.3 「fc=mc/(4π)」の音響パワー観点での再考

v1/v2 設計でホーンの $f_c$ が論じられたが、**実態は経路①(168μW)のごく小部分しかホーンに流れない**。ホーンのカットオフ議論は経路①前提の理想化議論であり、ピアニカの実音響では C1/C2 (壁面直接放射) が支配的なため、ホーンの存在は楽音に大きく影響しない。

これは v3 設計では:

- ホーンを諦め、ABS箱自体を音響制御対象とする (方向 A or B)
- もしくはABS箱を取り外してリードプレートを直接音響キャビネットに装着 (方向 C)

の判断を補強する。

## 10. v3 設計への含意

包括分析の結果から、v3 設計で取るべき方針:

```
[必須制御]
- C1, B4, C2  : ABS壁の振動と放射を制御
                -> ABS箱を木製に置換 or 内壁ダンピング処置

[重要制御]
- B1a, B1b   : リード -> リードプレート結合
                -> 蜜蝋系粘弾性ダンパー検討

[安定化]
- D1, D2     : 自励振動の安定化
                -> 楽音域とABS壁モードの周波数分離

[無視可能]
- A1, A3, B5, B7, C3, C5  : 全体パワーの1%未満
                            -> 設計考慮対象外でよい
```

これに基づき、v3 設計の具体形は:

**方向 A (推奨)**: ABS箱を木製cassottoに置換し、リードプレートを蜜蝋クランプ
**方向 B (簡易)**: ABS箱内壁にダンピング材 (テープ全面 + 粘弾性シート)
**ハイブリッド**: 中音域は方向 B, 低音域は方向 A (壁モードの音域別最適化)

---

## 11. 残された論点

本包括分析でも未解決の論点:

```
- 圧力式 vs 吸引式での経路②挙動の違い
  -> 吸引式ではリードがスロットの逆方向に振れる. B1bの位相が反転?

- 複数鍵同時押し時の経路間干渉
  -> 各リードがABS壁を別位相で駆動 -> 壁全体の振動パターンが複雑化

- 温湿度依存性の経路別分布
  -> ABS壁のヤング率変動 (温度敏感)
  -> パッキンの粘弾性変動 (湿度敏感)

- 高調波スペクトルの経路別寄与
  -> リードの梁モード非整数倍音 (1:6.27:17.55) は B1a 経由
  -> 整流作用の整数倍音 (1:2:3:...) は B1b 経由
  -> 両者の混合比率が音色を決める
```

これらは v3 プロトタイピング時にスマホFFT (Spectroid app) で実測すべき。
