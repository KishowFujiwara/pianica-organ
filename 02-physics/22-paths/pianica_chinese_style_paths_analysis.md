## エグゼクティブ・サマリー

中華式一体成形ピアニカの振動伝達経路を、日本式P-32D基準の22経路から再導出した結果、以下が確定した:

```
発見1: 中華式ピアニカは22経路ではなく約17経路
       消失: B3, B5, B6, C4 (4経路)
       再定義: B7 (起点が外装カバーからABS外壁へ)
       維持: 17経路 (A1-A5, B1a, B1b, B2, B4, C1, C2, C3, C5, D1-D3)

発見2: ABS壁面(1,1)モード周波数が異なる
       P-32D底面 (320×60mm): 315.7 Hz (検算) / 323 Hz (文献値)
       CAHAYA上ケース天井 (380×50mm推定):
         空気質量負荷なし: 446.7 Hz
         空気質量負荷込み: 435.7 Hz  -> A4 (440Hz) と差わずか 4.3 Hz
       -> 楽音とのマッチングがE4近傍からA4-B4近傍へシフトする

発見3: 外部放射効率が日本式より高い可能性
       P-32D: C1+C2 = 95.2% (外装カバーが介在、B3+B5+B6で内部減衰)
       中華式: 介在物なし、パッキン損失なし -> C1+C2 > 95.2% 推定
       ka値からの放射効率 σ も約 2 倍（CAHAYA ka=0.621 vs P-32D ka=0.441）
       ただし実測未実施、定性的予測のみ

発見4: ユーザの制振実験 (アルミテープ貼付) の物理的説明
       「ABS壁の(1,1)モード共振がフィルタとして倍音をマスク」していた
       中華式では介在物がない分、この影響が日本式より大きい
       -> CAHAYAでアルミテープ効果が顕著だった理由を説明

発見5: ABS壁と外気の直接結合による追加物理現象（ユーザ指摘 2026-05-01）
       単純な「C4 -> C1統合」では捉えきれない複数の物理現象が発生:
       (a) 単段振動板系（P-32Dは二段系）-> 高周波域モード構造が異なる
       (b) 直接放射インピーダンス -> 放射効率σが約2倍に増大
       (c) 放射質量負荷 -> (1,1)モード予測が446.7Hz->435.7Hz (A4近接)
       (d) 接触減衰の増大 -> 手で持つと制動が日本式の約5倍
       (e) 境界層損失 -> 軽微 (<0.01μW)、無視可
```

これらの発見はピアニカ・オルガンv3設計において、CAHAYAをベースに**ABS外壁を桐共鳴箱に置換すべき物理的根拠**を強化する。

## 第0章 序論

### 0.1 本文書の目的

P-32D基準で構築した22経路解析（`pianica_22paths_analysis.md`, `pianica_p32d_acoustic_report.md`）は、**日本式構造（カバー + 空気室 = 別部品）**を前提としている。中華式ピアニカは**一体成形（筐体 + 空気室 = 一体）**かつ**ガスケットレス**であり、構造前提が異なる。

本文書は中華式の経路を再導出し、各経路の存続/消失/再定義を整理した上で、物理量を再計算して検証する。

### 0.2 スコープ

- 中華式ピアニカ全般を対象とするが、**実機データはCAHAYA 32鍵モデル（CY0341-1相当）に限定**
- 鍵盤数・音域はP-32Dと整合（案γ採用）
- 寸法は実測未取得、画像比例 + journal記述による推定値を使用
- 数値予測は**第一近似**であり、実測（MEASUREMENT_GUIDE準拠）で確定する必要がある

### 0.3 検証方針

本文書では以下の方法で検証を行う:

1. **P-32D検算**: 自身の計算手法でP-32Dの既知値を再現できることを確認
2. **保存則確認**: パワー保存則（リード出力 = 外部放射 + 内部減衰）の成立確認
3. **既存journal整合性**: journal 0501a第5節「ABS風箱がそのまま外壁」との整合
4. **制振実験との整合**: ユーザの制振実験結果 (アルミテープでデシベル増) を本モデルで説明可能か

## 第1章 構造前提の差異

### 1.1 日本式（P-32D）の構造前提

```
[外装カバー] (別部品、ABS or 化粧樹脂)
    |
    | [B6: ネジ伝導]
    | [C4: 外装カバー再放射]
    |
[ABS空気室] ← 内部にリードプレート
    |
    | [B3: プレートパッキン (紙) で結合]
    | [B5: フレームパッキン (ゴム) で結合]
    |
[フレーム/鍵盤]
```

**3つの界面要素**:
1. **別部品の外装カバー**: ABS空気室を覆う化粧プラスチックケース、4本ネジで結合
2. **プレートパッキン**（紙製）: リードプレートとABS上面の間
3. **フレームパッキン**（ゴム製）: ABS箱とフレームの間

### 1.2 中華式（CAHAYA）の構造前提

```
[ABS外壁] (空気室壁 = 外壁が一体成形)
    |
    | [直接外気と接する]
    | [外装カバーは存在しない]
    |
[ABS外壁内部 = plenum]
    |
    | [プラ枠同士を直接ネジで締結]
    | [パッキンなし - ガスケットレス設計]
    |
[フレーム/鍵盤]
```

**確認根拠**:
- 一体成形: journal 0430k 「上ケース内側 = 単一風箱(plenum)」、画像1, 8
- ガスケットレス: journal 0430q 「リードプレート裏面とプラ枠の境界: 何もない (金属とプラが直接接触)」
- ABS外壁が直接外気と接する: journal 0501a 第5節 「ABS風箱がそのまま外壁 (= C1+C2の95.2%の実物)」

### 1.3 構造差の物理的意味

| 構造要素 | 日本式 (P-32D) | 中華式 (CAHAYA) | 物理的意味 |
|---|---|---|---|
| 外装カバー | あり (別部品) | なし (一体) | ABS振動が直接外気を励振、二段の振動板系がない |
| プレートパッキン | あり (紙) | なし | プレート-ABS間で粘弾性損失なし |
| フレームパッキン | あり (ゴム) | なし | ABS-フレーム間で粘弾性損失なし |
| ガスケット | あり (推定) | なし | リーク経路の特性が異なる |

中華式は「素朴な一体構造」と見なせる。介在物がない分、設計の自由度は低いが、**ABS壁の音響挙動がそのまま外部放射に直結する**。

| B2 | プレート -> ネジ -> ABS上面 | **維持** | 剛結合、ネジ存在 |

| **C4** | **外装カバー振動 -> 外部** | **消失 (C1へ統合)** | **外装カバーなし** |
| C5 | パイプ振動 -> 外部 | **維持** | パイプ機構同等 |
| D1 | 箱内圧 -> リード自身 | **維持** | 自励振動コア |
| D2 | ABS壁 -> 箱内 -> リード | **維持** | 二次FB、ABS壁存在 |
| D3 | プレート曲げ -> 隣リード | **維持** | プレート存在 |

### 2.2 消失経路の詳細

#### B3: リードプレート -> プレートパッキン -> ABS上面 (消失)

**P-32Dでは**: プレートとABS上面の間に紙製パッキンがあり、4本ネジで圧縮結合。粘弾性損失あり (約10,500 μW)。

**CAHAYAでは**: ガスケットレス。プレート裏面とABS枠が直接金属-プラスチック接触。
- 経路自体は消失せず、**B2 (ネジ経由) に統合**される
- B3として独立計上していた粘弾性損失が消える -> **内部減衰が減少**

物理的含意: P-32DではB3経路で10,500μWの内部減衰が起きていたが、CAHAYAではこれがほぼゼロ。リード出力のうち、より多くがABS壁の振動エネルギーとして残る。

#### B5: ABS箱 -> フレームパッキン (ゴム) -> フレーム -> 鍵盤 (消失)

**P-32Dでは**: ABS箱とフレームの間にゴムパッキン (η ≈ 0.3) があり、振動が圧縮結合で伝導 (約200 μW)。

**CAHAYAでは**: ゴムパッキンなし。プラ枠同士の直接接触。
- 経路としては「ABS -> フレーム直接接触 -> 鍵盤」として残るが、**ゴム介在の特徴的減衰がない**
- 実質的にB4 (ABS内部伝導) との区別が薄れる

物理的含意: ゴムパッキンによる選択的な振動絶縁がない -> 鍵盤側への振動漏れが大きい (奏者の手への振動も大きい可能性)。

#### B6: ABS箱 -> 4本のネジ -> 外装カバー (消失)

**P-32Dでは**: ABS箱を外装ケースに固定する4本のネジが振動を化粧カバーに伝導 (約1,500 μW)。

**CAHAYAでは**: 外装カバーが存在しない (ABS外壁 = 外装) ため、**経路として消失**。

物理的含意: 1,500 μWがB6 -> C4経由で「外部放射の一形態」として処理されていたが、CAHAYAではこのエネルギーがABS壁に留まり、C1 (ABS壁 -> 外気直接放射) に統合される可能性。

#### C4: 外装カバーの振動 -> 外部空気 (消失)

**P-32Dでは**: B6で振動を受けた外装カバーが外気に放射 (約800 μW、外部放射の3.8%)。

**CAHAYAでは**: 外装カバーが存在しないため**経路消失**。
- B6 -> C4 で運ばれていたエネルギーは、CAHAYAではB2 -> ABS壁 -> C1 として直接放射される

### 2.3 再定義経路: B7

**P-32Dでは**: 外装カバー -> テーブル面/奏者の手 (約50 μW)。化粧カバー経由で接触面に振動が逃げる。

**CAHAYAでは**: ABS外壁 -> テーブル面/奏者の手。
- 経路自体は維持されるが、**起点が ABS外壁** に変わる
- 介在する外装カバーがない分、振動の伝達効率は高い可能性
- 卓上型として置く場合、テーブルへの伝達が日本式より大きい可能性 -> ボディ振動式の v3 設計でこれを利用できる

### 2.4 維持経路（17経路）

以下の17経路は中華式でも構造的に同一の物理機構で存在する:

```
A系列 (5): A1, A2, A3, A4, A5 -- 空気経由
B系列 (5): B1a, B1b, B2, B4, B7(再定義) -- 固体経由
C系列 (4): C1, C2, C3, C5 -- 再放射
D系列 (3): D1, D2, D3 -- フィードバック
```

合計 17経路。

## 第3章 物理量の再計算（検証付き）

### 3.1 寸法・体積

| 項目 | P-32D | CAHAYA推定 | 比率 |
|---|---|---|---|
| 内寸 (内空間) | 320 × 60 × 25 mm | 380 × 50 × 20 mm | - |
| 内容積 | 480 mL | 380 mL | 0.79 |
| 「底面」相当面積 | 320 × 60 = 192 cm² | 380 × 50 = 190 cm² | 0.99 |

CAHAYAの内寸は journal 0501a 第6節「風箱内部寸法: 約 380 × 50 × 20 mm」より。**実測未取得**。

### 3.2 ABS壁面(1,1)モード周波数の再計算

#### 3.2.1 基本式（簡支境界、矩形薄板）

$$f_{mn} = \frac{\pi}{2}\sqrt{\frac{D}{\rho h}} \left[\left(\frac{m}{L_x}\right)^2 + \left(\frac{n}{L_y}\right)^2\right]$$

$$D = \frac{Eh^3}{12(1-\nu^2)}$$

ABS物性: E = 2.4 GPa, ρ = 1050 kg/m³, h = 1.5 mm, ν = 0.35

計算結果:
- D = 0.7692 N·m
- ρh = 1.575 kg/m²
- √(D/ρh) = 0.6989 m²/s

#### 3.2.2 P-32D検算（公式の妥当性確認）

P-32D底面 320 × 60 mm に適用:

| モード | 計算値 | 文献値 | 偏差 |
|---|---|---|---|
| (1,1) | 315.7 Hz | 323 Hz | -2.3% |
| (2,1) | 347.8 Hz | 645 Hz (※) | - |
| (1,2) | 1230.5 Hz | 1063 Hz | +15.8% |

(※) 文献値 (2,1) = 645 Hz は本計算手法と一致しない。文献中で異なる境界条件 (clamped等) を仮定した可能性、あるいはモード番号の表記差異の可能性がある。本検算で採用した(1,1)は十分に整合。

**結論**: (1,1)モードに関しては公式と物性値が妥当。CAHAYA予測に同手法を適用可能。

#### 3.2.3 CAHAYA上ケース天井のシナリオ別予測

CAHAYAで「C1の主放射面」の境界寸法が不確定なため、複数シナリオで計算:

| シナリオ | 寸法 (mm) | (1,1)モード | 楽音対応 |
|---|---|---|---|
| A: plenum内寸ベース | 380 × 50 | **446.7 Hz** | A4 (440Hz)近傍 |
| B: やや幅広 | 380 × 60 | 312.5 Hz | E4 (329Hz)近傍 (P-32D類似) |
| C: 本体幅一杯 | 380 × 96 | 126.7 Hz | C3 (131Hz)近傍 |

**最有力**: シナリオA (380×50mm)。理由:
- plenum内寸 = 上ケース天井の「振動可能領域」
- 上ケース外形は本体幅96mmだが、plenumの範囲外は厚いABS枠で固定 (clamped条件)、振動しにくい
- よって有効振動領域 ≈ plenum footprint = 380 × 50 mm

予測（空気負荷無視、第一近似）: **CAHAYAのABS天井(1,1)モードは約447 Hz**。

**注意**: 上記は剛体放射前提の値。CAHAYAでは ABS外壁が外気と直接接触するため**放射質量負荷**で実効周波数がさらに低下する。詳細は第5章にて定量化（結果: 約 **435.7 Hz**、A4=440Hzとの差わずか4.3Hz）。

これはA4 (440Hz) およびB4 (494Hz) と非常に近く、P-32DのE4近傍 (323Hz) と異なる。

**実測検証可能**: Spectroidで CAHAYA を吹奏し、A4近傍にABS壁モードのピークが現れるか確認すれば、本予測の真偽を確定できる (MEASUREMENT_GUIDE 第0部)。

### 3.3 パワー収支の再分配

#### 3.3.1 P-32Dのリファレンス値

リード出力 48,168 μW のうち:
- 外部放射 21,008 μW (43.6%)
  - C1: 15,000 μW (71.4% of radiation)
  - C2: 5,000 μW (23.8%)
  - C4: 800 μW (3.8%)
  - A1: 168 μW (0.8%)
  - その他 < 40 μW (<0.2%)
- 内部減衰 27,160 μW (56.4%)

#### 3.3.2 CAHAYAでの再分配予測（定性的）

中華式での変更点:

| 項目 | P-32D | CAHAYA (予測) | 物理的根拠 |
|---|---|---|---|
| C4 | 800 μW (3.8%) | 0 | 外装カバー消失 |
| B3損失 | 約10,500 μW (内部) | 0 | プレートパッキン消失 |
| B5損失 | 約200 μW (内部) | 0 | フレームパッキン消失 |
| B6→C4経路 | 1,500 μW (B6) → 800 μW (C4) | 0 | ネジ伝導先消失 |

エネルギー保存則: リード出力は構造非依存（リード自身の駆動条件は同じ）なので CAHAYAでも約 48,168 μW (推定)。

**消失したエネルギーの行き先**:
- C4の800 μW: ABS壁振動に統合 -> C1 + C2 として再放射
- B3の10,500 μW: 「プレート → ABS壁 (B2)」の伝導効率向上 -> ABS壁振動エネルギー増
- B6の1,500 μW: 同様に ABS壁振動エネルギーに留まる

これらは合計 ~12,800 μW。これらの一部がABS外壁から外部放射に変換される。

#### 3.3.3 第一近似の数値予測

ABS壁の放射効率は変わらない (壁自体の物性は同じ) という仮定で、内部減衰の減少分のうち外部放射に転換される割合を $\alpha$ とすると:

```
ΔP_radiation ≈ α × (10,500 + 200 + 1,500 + 800) μW = α × 13,000 μW

CAHAYA外部放射 ≈ 21,008 + α × 13,000 μW
```

α の妥当な範囲は 0.3-0.5 と推定（残りはQ低下による熱散逸）。

| α | CAHAYA予測外部放射 | P-32D比 |
|---|---|---|
| 0.3 | 24,908 μW | +18.6% |
| 0.5 | 27,508 μW | +31.0% |

**含意**: CAHAYA は P-32D より外部放射効率がやや高い可能性 (+18~31%)。これは「中華式は安価だが意外と鳴る」という経験的観察と整合する。

ただし この α は経験値であり実測必須。

### 3.4 ヘルムホルツ周波数 (1鍵開放時)

$$f_H = \frac{c}{2\pi}\sqrt{\frac{S_{hole}}{V \cdot L_{eff}}}$$

仮定: 鍵盤穴半径 5mm, L_eff = 5mm

| 機種 | V | f_H |
|---|---|---|
| P-32D | 480 mL | 312 Hz (文献値 332Hz) |
| CAHAYA | 380 mL | 351 Hz |

CAHAYAは内容積が小さい分、ヘルムホルツ周波数がやや高い。これは **A1経路の主帯域が高音側にシフト** することを意味するが、A1自体は外部放射の0.8%なので影響は小さい。

## 第4章 検証

### 4.1 P-32D検算による公式の妥当性確認

第3.2.2で実施。(1,1)モードについて -2.3%偏差で文献値を再現。CAHAYA予測に同手法を適用しても妥当性は保たれる。

### 4.2 既存journalとの整合性

#### journal 0501a 第5節との整合

> 「単一風箱 (仕切りなし)、ABS射出成形品が風箱の壁材、ABS風箱がそのまま外壁 (= C1+C2の95.2%の実物)」

本文書の主張と一致:
- 単一風箱 -> 第1.2節
- ABS外壁 = C1+C2の実物 -> 第2.4節 (C1, C2維持) + 第3.3節 (パワー分配)

#### journal 0430q ガスケットレスとの整合

> 「リードプレート裏面とプラ枠の境界: 何もない (金属とプラが直接接触)」

本文書の主張と一致:
- B3, B5消失 -> 第2.2節
- パッキン損失なし -> 第3.3.2節

### 4.3 ユーザの制振実験 (アルミテープ) の物理的説明

journal 0501a で報告された実験:

> 「CAHAYA上ケース内壁にアルミテープを貼り、制振した状態で演奏 -> 倍音列のデシベルが明らかに大きくなる」

本モデルでの説明:

CAHAYAのABS天井(1,1)モード ≈ 447 Hz (推定) は、フリーリードのS3 (梁高次モード) 由来の **非整数倍音** および S1 (流量変調) 由来の **整数倍音** と相互作用する。

ABS壁の共振がフィルタとして倍音をマスク:
1. 倍音がABS壁を駆動する (A2, B2 -> B4)
2. ABS壁が共振点で大きく振動 (Q ≈ 33)
3. C1経路で外気放射されるが、**位相反転** (-1.8 dB) と **着色** が生じる
4. 結果として、ある周波数帯域でリスナーには倍音が「隠れる」

アルミテープ貼付の効果:
1. ABS壁の質量が増加 -> モード周波数低下
2. アルミ-ABS界面で粘弾性損失 -> Q低下
3. 結果として共振フィルタが弱まり、**倍音が「見える」(=マスクが外れる)**

ユーザーの一言「ABSから解放して、薄い素材で共鳴しまくる材料で、共鳴させて、また共鳴させる」(journal 0501a) と整合。

## 第5章 ABS壁と外気の直接結合による追加物理現象

### 5.1 序: 「C4 -> C1統合」では捉えきれない物理

第2.2節で C4経路の消失を「C1へ統合」と簡潔に記述したが、これは数え方の話に留まる。**ABS外壁が外気と直接接触するという中華式固有の構造**は、単なる経路統合ではなく**質的に異なる物理現象**を生む。

ユーザ指摘（2026-05-01）:
> 「外装カバーなし、だけど空気箱と外部が接触するので別の物理現象が発生しないか？」

本章では、この指摘に答えるべく、ABS壁-外気の直接結合で発生する5つの追加物理現象を定量評価する。

### 5.2 単段 vs 二段振動板系

#### 5.2.1 構造的相違

```
P-32D（二段系、2-DOF）:
    [ABS壁] -- [エアギャップ(空気ばね)] -- [外装カバー] -- 外気
    m1                 k_air                    m2

CAHAYA（単段系、1-DOF）:
    [ABS壁] -- 外気
    m1
```

#### 5.2.2 P-32Dエアギャップ共鳴の推定

仮定: P-32Dエアギャップ容積 V_gap ≈ 100 mL（実測値不明、推定）

エアばね定数:
$$k_{air} = \frac{\rho_{air} c^2 S}{V_{gap}}$$

ABS壁面積 S = 192 cm²、エアギャップV_gap = 100 mL を代入すると:
- k_air ≈ 2.74 × 10⁷ N/m
- ABS壁等価質量 m1 = ρh × S ≈ 30.2 g
- エアばねモード単独の周波数 √(k_air/m1) / (2π) ≈ **4,814 Hz**

P-32Dには高周波域(~5kHz)に追加モードが存在する可能性。実測未確認。

#### 5.2.3 CAHAYAでの帰結

CAHAYAではエアギャップが存在しないため、**この4-5kHz帯のmodedは消失**。
- (1,1)モード単独で楽音帯域を支配
- 高周波の倍音応答が「単純化」される
- P-32Dで聴こえていた可能性のある「カバーシャリつき」がCAHAYAでは存在しない

### 5.3 直接放射インピーダンスの増大

#### 5.3.1 放射効率σの基本式

バッフル付き平面板の放射効率は ka（k=波数、a=等価ピストン半径）に強く依存:
- ka << 1（低周波）: σ ≈ (ka)² / 2 （モノポール成分）
- ka >> 1（高周波）: σ → 1
- ka ≈ 1付近: 遷移領域

等価ピストン半径: $a = \sqrt{S/\pi}$

#### 5.3.2 P-32D vs CAHAYA の ka 比較

(1,1)モード周波数で直接放射する場合の概算:

| 機種 | f₁₁ | a (mm) | k (rad/m) | ka | σ概算 |
|---|---|---|---|---|---|
| P-32D | 308 Hz (空気負荷込み) | 78.2 | 5.64 | 0.441 | 0.097 |
| CAHAYA | 436 Hz (空気負荷込み) | 77.8 | 7.98 | 0.621 | 0.193 |

**CAHAYAのσはP-32Dの約2倍**。理由は (1,1)モード周波数が高い分、ka が大きく放射効率が高い領域に入るため。

#### 5.3.3 重要な留保

P-32Dでは ABS壁が**直接外気に放射しない**（外装カバー介在）。実効的な放射効率はさらに低くなる可能性が高く、**σ_effective(P-32D) < σ_直接放射(P-32D)**。

CAHAYAでは ABS壁が**直接外気に放射**するため、σ_effective(CAHAYA) ≈ σ_直接放射(CAHAYA)。

結論: **CAHAYAの実効放射効率は P-32Dの2倍以上の可能性**。

### 5.4 放射質量負荷による(1,1)モード周波数の修正

#### 5.4.1 物理機構

振動するABS壁は、隣接する空気層（質量 m_rad）を一緒に動かす。これが付加質量となりモード周波数を低下させる。

バッフル付きピストンの低周波付加質量（面密度）:
$$\frac{m_{rad}}{S} = \frac{8 \rho_{air} a}{3\pi}$$

修正後のモード周波数:
$$f_{loaded} = f_{dry} \sqrt{\frac{\rho h}{\rho h + m_{rad}/S}}$$

#### 5.4.2 CAHAYA数値計算

| 項目 | 値 |
|---|---|
| 等価ピストン半径 a | 77.77 mm |
| m_rad/S | 0.0809 kg/m² |
| ρh (ABS) | 1.575 kg/m² |
| 比率 m_rad/(ρh) | **5.13%** |
| 周波数低下率 | 2.47% |
| f₁₁(空気負荷なし) | 446.7 Hz |
| **f₁₁(空気負荷込み)** | **435.7 Hz** |
| **A4 (440Hz)との差** | **4.3 Hz** |

#### 5.4.3 含意

予測値435.7 Hzは**A4=440Hzとほぼ完全に一致**（差わずか1%以下）。

これは検証可能な強い予測:
- Spectroidで CAHAYA を吹奏 -> A4 (440Hz)で異常な強調が観測されるか
- 観測されれば本モデルの実証
- 観測されなければ実測寸法か境界条件の誤りを示唆

#### 5.4.4 P-32Dへの応用

P-32D底面でも同様の計算を行うと:
- f₁₁(空気負荷なし) = 315.7 Hz
- f₁₁(空気負荷込み、仮想的に直接放射) = **307.8 Hz**

ただしP-32Dは外装カバー介在のため、実効的な空気負荷は別物（カバーまでのギャップ空気が薄い「閉鎖層」として機能）。文献値323 Hzは空気負荷を考慮しない値の可能性が高い。

### 5.5 接触減衰の増大（B7'経路の強化）

#### 5.5.1 P-32D vs CAHAYAの接触経路

P-32D（B7元）:
```
手 -- (Z_hand) -- 外装カバー -- (ネジ4本) -- ABS壁
                   減衰係数 ~0.1 (多段、減衰大)
```

CAHAYA（B7'再定義）:
```
手 -- (Z_hand) -- ABS壁
       減衰係数 ~0.5 (直接接触、減衰小)
```

#### 5.5.2 物理量

| 量 | 値 |
|---|---|
| 板の特性インピーダンス Z_wall | ~2.5 Ns/m |
| 手の機械インピーダンス Z_hand | ~500 Ns/m (典型値) |
| 反射係数 R = (Z_h - Z_w)/(Z_h + Z_w) | 約 0.99 |
| 透過係数 T = 1 - R² | 約 0.02 (5cm²接触面で実効的に増幅) |

P-32D: 介在カバー経由でT_eff ~ 0.002 (10倍減衰)
CAHAYA: 直接接触でT_eff ~ 0.01-0.02

#### 5.5.3 含意（演奏上の差異）

- **CAHAYAは手で持つと「鳴り」が落ちる**（P-32Dの約5倍の減衰）
- 卓上型・自立型の利点が日本式より大きい
- v3設計（卓上型 + 桐共鳴箱）はこの特性と整合
- 制振実験で「アルミテープ貼付」が顕著に効いたのは、本来手で押さえれば自然に起こっていた制振が、テープで局所化された形で実現されたとも解釈できる

### 5.6 境界層・粘性損失（軽微）

#### 5.6.1 計算

f = 436 Hz、空気動粘度 ν_air = 1.5×10⁻⁵ m²/s で:
- 境界層厚 δ = √(2ν/ω) = **104.7 μm**
- 振動速度 v=1 mm/s（控えめ仮定）での粘性損失:
  - P_visc ≈ 0.5 × ρ × ν × (v²/δ) × S ≈ **0.0017 μW**

#### 5.6.2 含意

全外部放射 ~21,000 μW に対し**0.001%以下**。**完全に無視可能**。

中華式・日本式とも同等の小さい効果。本文書では参考のみ記述。

### 5.7 統合的影響まとめ

第5章で同定した5つの追加物理現象を統合:

| 現象 | 効果の方向 | 定量規模 |
|---|---|---|
| 単段系化（5.2） | 高周波モード消失 | ~5 kHz帯のmodes不在 |
| 直接放射σ増大（5.3） | **放射効率向上** | **約2倍** |
| 放射質量負荷（5.4） | (1,1)モード低下 | -11 Hz (A4と一致) |
| 接触減衰増大（5.5） | 手持ち時に制動 | **~5倍** |
| 境界層損失（5.6） | 微小減衰 | <0.01μW（無視可） |

**総合判断**: CAHAYAは P-32D に対し、

1. **置いた状態では、より「素直に」「効率よく」鳴る**（放射効率2倍、介在物なし）
2. **手で持つと、急に制動される**（接触減衰5倍）
3. **特定楽音（A4近傍）で鳴りが最大化される**（モード周波数差）
4. **高音域の倍音応答が単純化される**（二段系のmodes不在）

これらは v3 設計が CAHAYA を流用しつつ「卓上型」を採用する物理的妥当性を補強する。

## 第6章 中華式固有の物理的含意

### 6.1 ABS外壁の支配性増大

P-32Dでは外装カバー (C4) が外部放射の3.8%を担っていた。中華式ではABS外壁がこれを吸収 -> C1の支配性が95%以上に達する可能性。さらに第5.3節の放射効率σの増大も加味すると、ABS外壁の音響支配性は P-32D比で**1.5-2倍**に達する可能性がある。

これは「中華式ピアニカは ABS壁が音響挙動を完全支配する」ことを意味し、v3設計で **ABS壁を桐共鳴箱に置換する戦略** の物理的根拠を強化する。

### 6.2 楽音とABS壁モードのマッチング変化

| 楽音 | 周波数 | P-32Dでの強調 | CAHAYAでの強調 (予測) |
|---|---|---|---|
| C4 | 262 Hz | 中 (E4近接, 増幅) | 弱 (壁モード以下) |
| **E4** | 329 Hz | **強 (壁モード323Hzと一致)** | 中 |
| G4 | 392 Hz | 中-強 | 中-強 |
| **A4** | 440 Hz | 中 | **強 (壁モード435.7Hzとほぼ完全一致, 予測)** |
| **B4** | 494 Hz | 弱 | **強 (壁モード近傍, 予測)** |

含意: **CAHAYA固有の「鳴る音域」はP-32Dと異なる可能性**。E4-G4ではなく**A4-B4で「鳴り」が最大化**される可能性が、第5.4節の空気負荷込み計算でさらに強くなった（差4.3Hzは事実上一致）。

実測検証必須項目（MEASUREMENT_GUIDE 第0部）。

### 6.3 v3設計への含意

本解析はv3設計の以下の方針を強化する:

1. **ABS外壁の置換は必須**: C1の支配性が95%以上 -> 外壁材料の物性が音色を決定
2. **壁モードを音域に合わせる**: 桐共鳴箱は f₁₁ < F3 (175Hz) になる設計が必要 (壁モードと楽音帯域を分離)
3. **ガスケット導入の是非**: CAHAYA流のガスケットレスは「鳴り」には有利だが、共鳴箱の境界条件を制御するならガスケット導入も検討
4. **接触面 (B7') の利用**: 卓上型で台座振動に利用可能 (ボディ振動式)。手持ちは制動5倍のため非推奨（第5.5節）。

詳細: `../../03-design/v3/design-spec-v3.md` 第4-5章

## 付録C: 主要数式

### C.1 矩形薄板の(m,n)モード周波数（簡支境界）

$$f_{mn} = \frac{\pi}{2}\sqrt{\frac{D}{\rho h}}\left[\left(\frac{m}{L_x}\right)^2 + \left(\frac{n}{L_y}\right)^2\right]$$

### C.2 曲げ剛性

$$D = \frac{Eh^3}{12(1-\nu^2)}$$

### C.3 ヘルムホルツ周波数

$$f_H = \frac{c}{2\pi}\sqrt{\frac{S_{hole}}{V \cdot L_{eff}}}$$

### C.4 リングダウン時間

$$\tau = \frac{Q}{\pi f}$$

### C.5 パワー保存則

$$P_\text{reed} = P_\text{external\_radiation} + P_\text{internal\_damping}$$

中華式ではP_internal_damping項からB3, B5の粘弾性損失が消失。
