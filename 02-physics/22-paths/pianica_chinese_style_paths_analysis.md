## 第0章 序論

### 0.1 本文書の目的

P-32D基準で構築した22経路解析（`pianica_22paths_analysis.md`, `pianica_p32d_acoustic_report.md`）は、**日本式構造（カバー + 空気室 = 別部品）**を前提としている。中華式ピアニカは**一体成形（筐体 + 空気室 = 一体）**かつ**ガスケットレス**であり、構造前提が異なる。

本文書は中華式の経路を再導出し、各経路の存続/消失/再定義を整理した上で、物理量を再計算して検証する。

### 0.2 スコープ

- 中華式ピアニカ全般を対象とするが、**実機データはCAHAYA 32鍵モデル（CY0341-1相当）に限定**
- 鍵盤数・音域はP-32Dと整合（案γ採用）
- 寸法は実測未取得、画像比例 + journal記述による推定値を使用
- 数値予測は**第一近似**であり、実測（MEASUREMENT_GUIDE準拠）で確定する必要がある

### 0.3 検証方針

本文書では以下の方法で検証を行う:

1. **P-32D検算**: 自身の計算手法でP-32Dの既知値を再現できることを確認
2. **保存則確認**: パワー保存則（リード出力 = 外部放射 + 内部減衰）の成立確認
3. **既存journal整合性**: journal 0501a第5節「ABS風箱がそのまま外壁」との整合
4. **制振実験との整合**: ユーザの制振実験結果 (アルミテープでデシベル増) を本モデルで説明可能か

## 第1章 構造前提の差異

### 1.1 日本式（P-32D）の構造前提

```
[外装カバー] (別部品、ABS or 化粧樹脂)
    |
    | [B6: ネジ伝導]
    | [C4: 外装カバー再放射]
    |
[ABS空気室] ← 内部にリードプレート
    |
    | [B3: プレートパッキン (紙) で結合]
    | [B5: フレームパッキン (ゴム) で結合]
    |
[フレーム/鍵盤]
```

**3つの界面要素**:
1. **別部品の外装カバー**: ABS空気室を覆う化粧プラスチックケース、4本ネジで結合
2. **プレートパッキン**（紙製）: リードプレートとABS上面の間
3. **フレームパッキン**（ゴム製）: ABS箱とフレームの間

### 1.2 中華式（CAHAYA）の構造前提

```
[ABS外壁] (空気室壁 = 外壁が一体成形)
    |
    | [直接外気と接する]
    | [外装カバーは存在しない]
    |
[ABS外壁内部 = plenum]
    |
    | [プラ枠同士を直接ネジで締結]
    | [パッキンなし - ガスケットレス設計]
    |
[フレーム/鍵盤]
```

**確認根拠**:
- 一体成形: journal 0430k 「上ケース内側 = 単一風箱(plenum)」、画像1, 8
- ガスケットレス: journal 0430q 「リードプレート裏面とプラ枠の境界: 何もない (金属とプラが直接接触)」
- ABS外壁が直接外気と接する: journal 0501a 第5節 「ABS風箱がそのまま外壁 (= C1+C2の95.2%の実物)」

### 1.3 構造差の物理的意味

| 構造要素 | 日本式 (P-32D) | 中華式 (CAHAYA) | 物理的意味 |
|---|---|---|---|
| 外装カバー | あり (別部品) | なし (一体) | ABS振動が直接外気を励振、二段の振動板系がない |
| プレートパッキン | あり (紙) | なし | プレート-ABS間で粘弾性損失なし |
| フレームパッキン | あり (ゴム) | なし | ABS-フレーム間で粘弾性損失なし |
| ガスケット | あり (推定) | なし | リーク経路の特性が異なる |

中華式は「素朴な一体構造」と見なせる。介在物がない分、設計の自由度は低いが、**ABS壁の音響挙動がそのまま外部放射に直結する**。

### 2.2 消失経路の詳細

#### B3: リードプレート -> プレートパッキン -> ABS上面 (消失)

**P-32Dでは**: プレートとABS上面の間に紙製パッキンがあり、4本ネジで圧縮結合。粘弾性損失あり (約10,500 μW)。

**CAHAYAでは**: ガスケットレス。プレート裏面とABS枠が直接金属-プラスチック接触。
- 経路自体は消失せず、**B2 (ネジ経由) に統合**される
- B3として独立計上していた粘弾性損失が消える -> **内部減衰が減少**

物理的含意: P-32DではB3経路で10,500μWの内部減衰が起きていたが、CAHAYAではこれがほぼゼロ。リード出力のうち、より多くがABS壁の振動エネルギーとして残る。

#### B5: ABS箱 -> フレームパッキン (ゴム) -> フレーム -> 鍵盤 (消失)

**P-32Dでは**: ABS箱とフレームの間にゴムパッキン (η ≈ 0.3) があり、振動が圧縮結合で伝導 (約200 μW)。

**CAHAYAでは**: ゴムパッキンなし。プラ枠同士の直接接触。
- 経路としては「ABS -> フレーム直接接触 -> 鍵盤」として残るが、**ゴム介在の特徴的減衰がない**
- 実質的にB4 (ABS内部伝導) との区別が薄れる

物理的含意: ゴムパッキンによる選択的な振動絶縁がない -> 鍵盤側への振動漏れが大きい (奏者の手への振動も大きい可能性)。

#### B6: ABS箱 -> 4本のネジ -> 外装カバー (消失)

**P-32Dでは**: ABS箱を外装ケースに固定する4本のネジが振動を化粧カバーに伝導 (約1,500 μW)。

**CAHAYAでは**: 外装カバーが存在しない (ABS外壁 = 外装) ため、**経路として消失**。

物理的含意: 1,500 μWがB6 -> C4経由で「外部放射の一形態」として処理されていたが、CAHAYAではこのエネルギーがABS壁に留まり、C1 (ABS壁 -> 外気直接放射) に統合される可能性。

#### C4: 外装カバーの振動 -> 外部空気 (消失)

**P-32Dでは**: B6で振動を受けた外装カバーが外気に放射 (約800 μW、外部放射の3.8%)。

**CAHAYAでは**: 外装カバーが存在しないため**経路消失**。
- B6 -> C4 で運ばれていたエネルギーは、CAHAYAではB2 -> ABS壁 -> C1 として直接放射される

### 2.3 再定義経路: B7

**P-32Dでは**: 外装カバー -> テーブル面/奏者の手 (約50 μW)。化粧カバー経由で接触面に振動が逃げる。

**CAHAYAでは**: ABS外壁 -> テーブル面/奏者の手。
- 経路自体は維持されるが、**起点が ABS外壁** に変わる
- 介在する外装カバーがない分、振動の伝達効率は高い可能性
- 卓上型として置く場合、テーブルへの伝達が日本式より大きい可能性 -> ボディ振動式の v3 設計でこれを利用できる

### 2.4 維持経路の構造的検証（手抜き禁止の個別検証）

**ユーザ指摘（2026-05-01）**: 「画像や画像から得られた構造を再構成して検証していないでしょ？」

正当な指摘。第2.1節で「維持（構造非依存）」とラベルしたが、それは怠慢であった。各経路に必要な構造要素を CAHAYA画像で個別確認し、P-32Dとの差異を明示する。

#### 2.4.1 検証フォーマット

各経路について以下4項目で記述:

1. 必要構造要素（経路成立に必須の物理構造）
2. CAHAYAでの実態（画像番号付きの観察事実）
3. P-32Dとの差異（構造的・幾何的相違）
4. 結論（経路の存続度合いと変化点）

#### 2.4.2 A系列の検証（5経路）

##### A1: リード -> 箱内空気 -> 開いた穴 -> 外部

1. **必要構造**: 鍵盤バルブ + plenum + 開口部
2. **CAHAYAでの実態**:
   - バルブ32個（白色プラスチック、画像2・6・10で確認）
   - plenum内寸 380×50×20mm（journal 0501a）
   - リードスロット（画像4でスロット内のリード可視、画像11で本体下端の開口列可視）
3. **P-32Dとの差異**:
   - plenum容積: P-32D 480mL vs CAHAYA 380mL（**0.79倍**）
   - 開口部位置: P-32D側面/上面 vs CAHAYA下面（推定、要実測）
   - ヘルムホルツ周波数 P-32D: 312Hz, CAHAYA: 351Hz（容積差による）
4. **結論**: **経路は存続するが幾何パラメータ変化**。放射方向（指向性）も差異あり。

##### A2: 箱内空気 -> ABS壁面（音圧駆動）

1. **必要構造**: plenum音圧 + ABS壁面
2. **CAHAYAでの実態**:
   - plenum: 画像5・6で確認、上ケース内部空間
   - 上ケース ABS壁（画像5、6、10）: 天井 + 4側面 = 5面
   - **plenumの「底面」はリードプレート（金属アルミ）、ABSではない**
3. **P-32Dとの差異**:
   - P-32D「ABS箱」は6面すべてABS連続（一体成型）
   - **CAHAYA は5面ABS + 1面金属（リードプレート）**
   - 音圧で駆動可能なABS壁面積: P-32D 約 720cm² (6面合計推定) vs CAHAYA 約 405cm² (天井190 + 側面215cm²推定)
4. **結論**: **経路存続、ただし駆動可能なABS壁構成が異なる（5面 vs 6面）**。リードプレート（金属）はモード周波数が桁違いに高く、音圧駆動下では実質的に剛体壁。

##### A3: 箱内空気 -> 閉じたバルブ振動

1. **必要構造**: 閉じたバルブ (押下されていない鍵)
2. **CAHAYAでの実態**: 32個のバルブ（画像2・10で2列交互配置）
3. **P-32Dとの差異**:
   - バルブ材質・色: P-32D黄色 vs CAHAYA白色（プラスチック組成微差の可能性）
   - 形状・寸法: 不明（要実測）
4. **結論**: **経路完全維持、定量値はバルブ物性で微差**。

##### A4: 箱内空気 -> パイプ -> マウスピース -> 奏者

1. **必要構造**: マウスピース口 + パイプ/ホース
2. **CAHAYAでの実態**:
   - マウスピース口: 上ケース側に円筒突起（画像7・8で確認）
   - 内部にダクト延長なし、単純貫通（画像7）
   - 外部ホース: 付属品の有無と寸法は画像で未確認
3. **P-32Dとの差異**:
   - P-32D は長い柔軟ホース付属が標準
   - CAHAYA は短い直管マウスピース口（外部ホース有無不明）
   - パイプ管内共鳴は管長で決まる -> 周波数特性異なる
4. **結論**: **経路存続、管長による定在波周波数が異なる**。

##### A5: 箱内定在波 -> 隣リードとの空気的連成

1. **必要構造**: plenum内の定在波 + 複数リード
2. **CAHAYAでの実態**:
   - plenum 380×50×20mm（journal 0501a）
   - 32リードが2列で plenum 内に位置
3. **P-32Dとの差異**: plenum形状の違いから定在波モードが異なる:

   | 方向 | P-32D (320×60×25) | CAHAYA (380×50×20) |
   |---|---:|---:|
   | 長手 (Lx)半波長共鳴 | 536 Hz | 451 Hz |
   | 中手 (Ly)半波長共鳴 | 2858 Hz | 3430 Hz |
   | 短手 (Lz)半波長共鳴 | 6860 Hz | 8575 Hz |

4. **結論**: **経路存続、定在波周波数は構造的に異なる（特に中手・短手で大きい差）**。

#### 2.4.3 B系列の検証（5経路、B7'は再定義済み）

##### B1a: リード -> リベット -> リードプレート

1. **必要構造**: リード + リベット/ネジ + プレート
2. **CAHAYAでの実態**: 画像4でリードがプレートにリベット/ネジで固定されている断面視認可
3. **P-32Dとの差異**:
   - **プレート材質: P-32D 真鍮 vs CAHAYA アルミ**
   - 真鍮 ρ ≈ 8400 kg/m³, E ≈ 100 GPa
   - アルミ ρ ≈ 2700 kg/m³, E ≈ 70 GPa
   - 機械インピーダンス Z ∝ √(ρE) -> 真鍮: ~916, アルミ: ~435
   - **アルミプレートは真鍮の約半分のインピーダンス**
4. **結論**: **経路存続、ただしリード→プレート伝達インピーダンスが約2倍異なる**。CAHAYAではプレート振動が大きくなる傾向。

##### B1b: リード -> スロット隙間空気 -> プレート面

1. **必要構造**: リードがスロット内で振動、スロット隙間（0.05-0.1mm）
2. **CAHAYAでの実態**: 画像4でリード-スロット構造確認可
3. **P-32Dとの差異**:
   - リード材質・寸法は実測未取得
   - スロット隙間幅は実測未取得（製造精度依存）
4. **結論**: **経路存続、定量値はリード/スロット幾何で変動**。実測必要。

##### B2: リードプレート -> ネジ -> ABS壁

1. **必要構造**: プレート + 固定ネジ + ABS壁
2. **CAHAYAでの実態**:
   - 画像8でリードプレート周囲に固定ネジ多数（プレート全周）
   - **プレートは鍵盤ユニットの黒色プラ枠に固定されている**（air chamber wall ではない）
   - 上ケースとの接触は別経路（プレート上面 -> 直接接触 -> 上ケース内縁）
3. **P-32Dとの差異**:
   - **P-32D**: プレート → ネジ4本 → ABS air chamber 上面（直接、1段）
   - **CAHAYA**: プレート → 多数のネジ → 鍵盤ユニット黒色枠 → 楽器周囲ネジ → 上ケースABS（**多段経路**）
   - + 副経路: プレート上面 → 直接接触圧 → 上ケース内縁（ガスケットレス、journal 0430q）
   - **伝達経路が根本的に異なる**
4. **結論**: **経路存続、ただし伝達チェーンが複雑化（1段→2段または並列2経路）**。実効伝達効率は実測必須。

##### B4: ABS上面 -> 側面 -> 底面（一体成型内伝導）

1. **必要構造**: 連続したABS箱
2. **CAHAYAでの実態**:
   - 上ケース は一体成型 ABS（画像5・6・10）
   - **上ケースの「底」は開放（リードプレートと接触するため）**
   - 鍵盤ユニットも別の一体成型ABS、上ケースとは別部品
3. **P-32Dとの差異**:
   - P-32D: ABS air chamber が天井・4側面・**底面まで連続**（6面ABS）
   - **CAHAYA: 上ケース ABS は天井 + 4側面の5面、底面は開放（リードプレートが蓋）**
   - **B4経路は「天井 → 4側面」で完結し、ABSの「底面」へは到達しない**
4. **結論**: **経路存続するが連続性が限定（5面のみ）**。最大の差異: P-32Dで「ABS底面（192cm²）」が C1 の主放射面だったが、CAHAYAではABS底面が存在しない。

##### B7' (再定義): ABS外壁 -> テーブル/手

第2.3節で詳述済み。CAHAYAのABS外壁が直接接触面に接触する。第5章で接触減衰約5倍として定量化。

#### 2.4.4 C系列の検証（4経路）

##### C1: ABS壁の振動 -> 外部空気へ直接放射

1. **必要構造**: ABS壁が外気に露出
2. **CAHAYAでの実態**:
   - 上ケース外壁すべてが外気と直接接触（外装カバーなし、journal 0501a）
   - 主放射面: 上ケース**天井**（外面、約190cm²）
   - 側面放射: 4側面（合計約215cm²推定）
3. **P-32Dとの差異**:
   - **放射面の物理的位置が異なる**:
     - P-32D: ABS air chamber の **底面** (192 cm²) + 4側面 が外装カバー経由で外部へ
     - CAHAYA: 上ケースの **天井** (190cm²) + 4側面 が直接外部へ
   - **CAHAYAではABS底面が存在しない**（リードプレートが蓋）
   - 放射効率σ: CAHAYA約2倍（第5.3節）
4. **結論**: **経路存続するが放射面の幾何が「底」から「天井」へ反転、外装カバー消失で直接放射、σ約2倍**。

##### C2: ABS壁振動 -> 箱内再放射 -> 開いた穴

1. **必要構造**: ABS壁振動 + plenum + 開口部
2. **CAHAYAでの実態**: A1, A2の構造要素を含む、すべて確認可
3. **P-32Dとの差異**: plenum形状によるカップリング効率差。定在波モード周波数差（A5節）。
4. **結論**: **経路存続、定量値は plenum形状で変動**。

##### C3: 閉じたバルブ振動 -> 外部放射

1. **必要構造**: 閉じたバルブの微小振動
2. **CAHAYAでの実態**: バルブ32個（画像2・10）
3. **P-32Dとの差異**: バルブ物性差は微小、放射効率は微小
4. **結論**: **経路完全維持、寄与は元々0.2μW未満で誤差**。

##### C5: パイプ振動 -> 外部放射

1. **必要構造**: パイプ
2. **CAHAYAでの実態**: マウスピース短直管（画像7・8）
3. **P-32Dとの差異**: 長いホース vs 短い直管 -> パイプ振動モード異なる
4. **結論**: **経路存続、パイプ長で定量値変動**。

#### 2.4.5 D系列の検証（3経路）

##### D1: 箱内圧 -> リード自身

1. **必要構造**: plenum + リード（自励振動コア）
2. **CAHAYAでの実態**: 全要素確認可
3. **P-32Dとの差異**: plenum容積差（480 vs 380 mL）でリードへの圧変動 envelope が変わる可能性
4. **結論**: **経路存続、自励振動の安定性は構造非依存**。圧変動は容積差で微変化。

##### D2: ABS壁振動 -> 箱内 -> リード

1. **必要構造**: ABS壁振動 + plenum + リード
2. **CAHAYAでの実態**: 全要素確認可
3. **P-32Dとの差異**:
   - ABS壁モード周波数差（323Hz vs 435.7Hz）
   - リードがこのモード周波数に応答する周波数帯域が異なる
   - **CAHAYAではA4=440Hz楽音と壁モードが共振 -> D2帰還がA4で最大化される可能性**
4. **結論**: **経路存続、二次FBの最大化周波数が変化**（E4近傍 → A4近傍）。

##### D3: リードプレート曲げ -> 隣リード機械的連成

1. **必要構造**: 1枚のリードプレートに複数のリード
2. **CAHAYAでの実態**:
   - アルミプレート + 32リード（画像4・6・8）
   - リードは2列交互配置
3. **P-32Dとの差異**:
   - **プレート材質**: P-32D 真鍮 vs CAHAYA アルミ
   - プレート曲げ剛性: 真鍮は密度3倍・ヤング率1.4倍 -> 曲げ剛性が異なる
   - リード配列: P-32D は dual-plate 構造の可能性（参考画像11_dual_reed_plates.png）、CAHAYA は単一プレート
4. **結論**: **経路存続、プレート剛性差で連成強度が異なる**。CAHAYAアルミプレートはより柔らかい -> 連成強度が大きい可能性。

#### 2.4.6 検証の総合評価

「維持」とラベルした17経路を個別検証した結果:

| 検証カテゴリ | 該当経路 | 数 |
|---|---|---|
| **完全維持**（構造・定量値ともほぼ同一） | A3, B1b, C3 | 3 |
| **存続するが定量値変化**（plenum/管/プレート寸法依存） | A1, A4, A5, B1a, C5, D1, D3 | 7 |
| **存続するが構造的差異あり**（経路チェーンや放射面が異なる） | A2, B2, B4, C1, C2, D2 | 6 |
| **再定義** | B7' | 1 |

**完全維持と言えるのは3経路のみ**（17経路のうち18%）。残り14経路は CAHAYA固有の構造で何らかの定量値・構造差異がある。

これは「P-32D基準22経路がそのまま適用」という当初の主張がいかに粗かったかを示す。本検証により、CAHAYA物理モデルは:

```
                        旧主張（v1.0）        正確な記述（v1.2）
経路数:                 22 (P-32D流用)         17（4消失+1再定義+17存続）
存続経路の状態:         そのまま適用可能       完全同一は3経路のみ
構造的差異の認識:       なし                   14経路で定量値・構造差異
```

物理モデルとしての精度が大きく向上。次節以降の数値予測（モード周波数、パワー収支等）は本検証を踏まえた値を採用する。

### 3.1 寸法・体積

### 3.2 ABS壁面(1,1)モード周波数の再計算

#### 3.2.1 基本式（簡支境界、矩形薄板）

$$f_{mn} = \frac{\pi}{2}\sqrt{\frac{D}{\rho h}} \left[\left(\frac{m}{L_x}\right)^2 + \left(\frac{n}{L_y}\right)^2\right]$$

$$D = \frac{Eh^3}{12(1-\nu^2)}$$

ABS物性: E = 2.4 GPa, ρ = 1050 kg/m³, h = 1.5 mm, ν = 0.35

計算結果:
- D = 0.7692 N·m
- ρh = 1.575 kg/m²
- √(D/ρh) = 0.6989 m²/s

#### 3.2.2 P-32D検算（公式の妥当性確認）

P-32D底面 320 × 60 mm に適用:

| モード | 計算値 | 文献値 | 偏差 |
|---|---|---|---|
| (1,1) | 315.7 Hz | 323 Hz | -2.3% |
| (2,1) | 347.8 Hz | 645 Hz (※) | - |
| (1,2) | 1230.5 Hz | 1063 Hz | +15.8% |

(※) 文献値 (2,1) = 645 Hz は本計算手法と一致しない。文献中で異なる境界条件 (clamped等) を仮定した可能性、あるいはモード番号の表記差異の可能性がある。本検算で採用した(1,1)は十分に整合。

**結論**: (1,1)モードに関しては公式と物性値が妥当。CAHAYA予測に同手法を適用可能。

#### 3.2.3 CAHAYA上ケース天井のシナリオ別予測

CAHAYAで「C1の主放射面」の境界寸法が不確定なため、複数シナリオで計算:

| シナリオ | 寸法 (mm) | (1,1)モード | 楽音対応 |
|---|---|---|---|
| A: plenum内寸ベース | 380 × 50 | **446.7 Hz** | A4 (440Hz)近傍 |
| B: やや幅広 | 380 × 60 | 312.5 Hz | E4 (329Hz)近傍 (P-32D類似) |
| C: 本体幅一杯 | 380 × 96 | 126.7 Hz | C3 (131Hz)近傍 |

**最有力**: シナリオA (380×50mm)。理由:
- plenum内寸 = 上ケース天井の「振動可能領域」
- 上ケース外形は本体幅96mmだが、plenumの範囲外は厚いABS枠で固定 (clamped条件)、振動しにくい
- よって有効振動領域 ≈ plenum footprint = 380 × 50 mm

予測（空気負荷無視、第一近似）: **CAHAYAのABS天井(1,1)モードは約447 Hz**。

**注意**: 上記は剛体放射前提の値。CAHAYAでは ABS外壁が外気と直接接触するため**放射質量負荷**で実効周波数がさらに低下する。詳細は第5章にて定量化（結果: 約 **435.7 Hz**、A4=440Hzとの差わずか4.3Hz）。

これはA4 (440Hz) およびB4 (494Hz) と非常に近く、P-32DのE4近傍 (323Hz) と異なる。

**実測検証可能**: Spectroidで CAHAYA を吹奏し、A4近傍にABS壁モードのピークが現れるか確認すれば、本予測の真偽を確定できる (MEASUREMENT_GUIDE 第0部)。

#### 3.3.2 第2.4節構造差異のパワー係数

##### (1) B1a伝達効率（プレート材質差）

機械インピーダンス比較:
- 真鍮（P-32D）: Z = √(ρE) = √(8400 × 100×10⁹) ≈ 28.98 × 10⁶ Pa·s/m
- アルミ（CAHAYA）: Z = √(2700 × 70×10⁹) ≈ 13.75 × 10⁶ Pa·s/m
- **比 Z_alu / Z_brass = 0.474**

リード→プレート伝達効率（Z_reed << Z_plateの極限）:
- P_transfer ∝ 1/Z_plate
- **CAHAYA / P-32D ≈ 2.11倍**

ただし完全に効率2.11倍は非物理的（リード自体の駆動条件で頭打ち）。実効的なプレート振動エネルギーの増加率は **+30%** と見積もる（パラメータ範囲: +20~+50%）。

##### (2) B2伝達効率（多段化と並列副経路）

P-32D（1段、ガスケット介在）:
- プレート → 4本ネジ → ABS air chamber上面
- 効率推定 η_p32d ≈ 0.80

CAHAYA（多段 + 並列副経路）:
- 主経路（多段）: プレート → 多数のネジ → 鍵盤ユニット黒色枠 → 周囲ネジ → 上ケース
  - 各段 η ≈ 0.80, 2段で η² = 0.64
- 副経路（直接接触）: プレート上面 → 直接接触圧 → 上ケース内縁（ガスケットレス）
  - η ≈ 0.80（金属-プラ直接、低損失）
- 並列合成（実効）: max(η_main, η_sub) ≈ **0.80**

**B2効率比 (CAHAYA/P-32D) ≈ 1.00**（多段化の損失をガスケットレス副経路がほぼ相殺）。

##### (3) B4面積（ABS壁の連続性）

P-32D: 6面ABS（天井 + 4側面 + 底面、合計約 720 cm²）
CAHAYA: 5面ABS（天井 + 4側面、合計約 405 cm²）

**面積比 = 0.56**

##### (4) C1実効放射効率（σ doubling）

第5.3節で導出: ka差により **σ比 ≈ 2.0倍**（CAHAYA/P-32D）

##### (5) 実効C1放射比

```
実効C1放射比 = (面積比) × (σ比) = 0.56 × 2.0 ≈ 1.12
```

**含意**: CAHAYAのC1経路は、面積こそ少ないが直接放射と高σで実効的にP-32Dの約 **1.12倍** の放射効率を持つ。

##### (6) 消失経路の散逸量（再掲）

- B3損失（プレートパッキン）: 約 10,500 μW（内部減衰）
- B5損失（フレームパッキン）: 約 200 μW（内部減衰）
- B6伝導（ネジ→外装）: 1,500 μW（うち C4 として 800μW 外部放射、残り内部）

**消失経路により、合計約 12,200 μW がCAHAYAでは「内部減衰されない」状態で利用可能**。

#### 3.3.3 段階的パワー再分配（5ステップ）

##### Step 1: リード総出力（構造非依存）

```
P_reed = 48,168 μW （P-32Dと同等、リード駆動条件同一仮定）
```

検証: リード自身の物理（リン青銅C5210、寸法）はP-32Dと同等想定。リード総出力は構造非依存とする。

##### Step 2: プレート振動エネルギー（B1a効率増を反映）

P-32D: 35,000 μW
CAHAYA: 35,000 × 1.30 = **45,500 μW** （B1a効率係数 +30%）

物理的根拠:
- アルミプレートは Z が真鍮の半分 -> リードからの伝達効率増（§3.3.2(1)）
- ただしプレート内部損失も増えるため実質+30%が現実的（理論値+111%は非物理的）

##### Step 3: ABS壁到達エネルギー

P-32D: 25,000 μW（プレート振動の 71%が ABS到達）
CAHAYA: B3消失（プレート内部損失 10,000 → 5,000 μW程度）+ B2効率同等 を考慮:

```
プレート振動 (CAHAYA) = 45,500 μW
プレート内部損失 (CAHAYA, B3消失後) = 5,000 μW
プレート→ABS伝達分 = 45,500 - 5,000 = 40,500 μW
B2効率 0.80 を掛けて: 40,500 × 0.80 = 32,400 μW
```

```
P_ABS_CAHAYA ≈ 32,400 μW （+29.6% vs P-32D 25,000 μW）
```

**注記**: P-32D元解析の数値は「クランプ部損失」と「プレート内部損失」の正確な区別が曖昧で、内部整合性に約±10%の誤差を含む。本予測値32,400μWも同程度の不確実性を持つ。

##### Step 4: ABS到達後の分配

P-32D（リファレンス、ABS到達 25,000 μWの分配）:
- C1: 60% (= 15,000/25,000)
- C2: 20% (= 5,000/25,000)
- 内部減衰: 17% (= 4,200/25,000)

## 第5章 ABS壁と外気の直接結合による追加物理現象

### 5.1 序: 「C4 -> C1統合」では捉えきれない物理

第2.2節で C4経路の消失を「C1へ統合」と簡潔に記述したが、これは数え方の話に留まる。**ABS外壁が外気と直接接触するという中華式固有の構造**は、単なる経路統合ではなく**質的に異なる物理現象**を生む。

ユーザ指摘（2026-05-01）:
> 「外装カバーなし、だけど空気箱と外部が接触するので別の物理現象が発生しないか？」

本章では、この指摘に答えるべく、ABS壁-外気の直接結合で発生する5つの追加物理現象を定量評価する。

### 5.2 単段 vs 二段振動板系

#### 5.2.1 構造的相違

```
P-32D（二段系、2-DOF）:
    [ABS壁] -- [エアギャップ(空気ばね)] -- [外装カバー] -- 外気
    m1                 k_air                    m2

CAHAYA（単段系、1-DOF）:
    [ABS壁] -- 外気
    m1
```

#### 5.2.2 P-32Dエアギャップ共鳴の推定

仮定: P-32Dエアギャップ容積 V_gap ≈ 100 mL（実測値不明、推定）

エアばね定数:
$$k_{air} = \frac{\rho_{air} c^2 S}{V_{gap}}$$

ABS壁面積 S = 192 cm²、エアギャップV_gap = 100 mL を代入すると:
- k_air ≈ 2.74 × 10⁷ N/m
- ABS壁等価質量 m1 = ρh × S ≈ 30.2 g
- エアばねモード単独の周波数 √(k_air/m1) / (2π) ≈ **4,814 Hz**

P-32Dには高周波域(~5kHz)に追加モードが存在する可能性。実測未確認。

#### 5.2.3 CAHAYAでの帰結

CAHAYAではエアギャップが存在しないため、**この4-5kHz帯のmodedは消失**。
- (1,1)モード単独で楽音帯域を支配
- 高周波の倍音応答が「単純化」される
- P-32Dで聴こえていた可能性のある「カバーシャリつき」がCAHAYAでは存在しない

### 5.3 直接放射インピーダンスの増大

#### 5.3.1 放射効率σの基本式

バッフル付き平面板の放射効率は ka（k=波数、a=等価ピストン半径）に強く依存:
- ka << 1（低周波）: σ ≈ (ka)² / 2 （モノポール成分）
- ka >> 1（高周波）: σ → 1
- ka ≈ 1付近: 遷移領域

等価ピストン半径: $a = \sqrt{S/\pi}$

#### 5.3.2 P-32D vs CAHAYA の ka 比較

(1,1)モード周波数で直接放射する場合の概算:

| 機種 | f₁₁ | a (mm) | k (rad/m) | ka | σ概算 |
|---|---|---|---|---|---|
| P-32D | 308 Hz (空気負荷込み) | 78.2 | 5.64 | 0.441 | 0.097 |
| CAHAYA | 436 Hz (空気負荷込み) | 77.8 | 7.98 | 0.621 | 0.193 |

**CAHAYAのσはP-32Dの約2倍**。理由は (1,1)モード周波数が高い分、ka が大きく放射効率が高い領域に入るため。

#### 5.3.3 重要な留保

P-32Dでは ABS壁が**直接外気に放射しない**（外装カバー介在）。実効的な放射効率はさらに低くなる可能性が高く、**σ_effective(P-32D) < σ_直接放射(P-32D)**。

CAHAYAでは ABS壁が**直接外気に放射**するため、σ_effective(CAHAYA) ≈ σ_直接放射(CAHAYA)。

結論: **CAHAYAの実効放射効率は P-32Dの2倍以上の可能性**。

### 5.4 放射質量負荷による(1,1)モード周波数の修正

#### 5.4.1 物理機構

振動するABS壁は、隣接する空気層（質量 m_rad）を一緒に動かす。これが付加質量となりモード周波数を低下させる。

バッフル付きピストンの低周波付加質量（面密度）:
$$\frac{m_{rad}}{S} = \frac{8 \rho_{air} a}{3\pi}$$

修正後のモード周波数:
$$f_{loaded} = f_{dry} \sqrt{\frac{\rho h}{\rho h + m_{rad}/S}}$$

#### 5.4.2 CAHAYA数値計算

| 項目 | 値 |
|---|---|
| 等価ピストン半径 a | 77.77 mm |
| m_rad/S | 0.0809 kg/m² |
| ρh (ABS) | 1.575 kg/m² |
| 比率 m_rad/(ρh) | **5.13%** |
| 周波数低下率 | 2.47% |
| f₁₁(空気負荷なし) | 446.7 Hz |
| **f₁₁(空気負荷込み)** | **435.7 Hz** |
| **A4 (440Hz)との差** | **4.3 Hz** |

#### 5.4.3 含意

予測値435.7 Hzは**A4=440Hzとほぼ完全に一致**（差わずか1%以下）。

これは検証可能な強い予測:
- Spectroidで CAHAYA を吹奏 -> A4 (440Hz)で異常な強調が観測されるか
- 観測されれば本モデルの実証
- 観測されなければ実測寸法か境界条件の誤りを示唆

**ユーザ実演観測（v1.4で追記、2026-05-01）**:
ユーザは「日本製と中華製でピッチ一致、たまにウルフ」と観測。
- ピッチ一致 -> 壁モードはリード周波数を引き込まない（リード高Q支配、§6.2で詳述）
- たまにウルフ -> 壁モード共振帯（FWHM 13.2Hz）に入る音 (CAHAYAではA4近傍) で局所干渉
- これは本予測値435.7HzをA4=440HzにほぼマッチさせるFWHM内の現象として整合
- ただし「異常な強調」ではなく「subtleなウルフリスク」として現れる

#### 5.4.4 P-32Dへの応用

P-32D底面でも同様の計算を行うと:
- f₁₁(空気負荷なし) = 315.7 Hz
- f₁₁(空気負荷込み、仮想的に直接放射) = **307.8 Hz**

ただしP-32Dは外装カバー介在のため、実効的な空気負荷は別物（カバーまでのギャップ空気が薄い「閉鎖層」として機能）。文献値323 Hzは空気負荷を考慮しない値の可能性が高い。

### 5.5 接触減衰の増大（B7'経路の強化）

#### 5.5.1 P-32D vs CAHAYAの接触経路

P-32D（B7元）:
```
手 -- (Z_hand) -- 外装カバー -- (ネジ4本) -- ABS壁
                   減衰係数 ~0.1 (多段、減衰大)
```

CAHAYA（B7'再定義）:
```
手 -- (Z_hand) -- ABS壁
       減衰係数 ~0.5 (直接接触、減衰小)
```

#### 5.5.2 物理量

| 量 | 値 |
|---|---|
| 板の特性インピーダンス Z_wall | ~2.5 Ns/m |
| 手の機械インピーダンス Z_hand | ~500 Ns/m (典型値) |
| 反射係数 R = (Z_h - Z_w)/(Z_h + Z_w) | 約 0.99 |
| 透過係数 T = 1 - R² | 約 0.02 (5cm²接触面で実効的に増幅) |

P-32D: 介在カバー経由でT_eff ~ 0.002 (10倍減衰)
CAHAYA: 直接接触でT_eff ~ 0.01-0.02

#### 5.5.3 含意（演奏上の差異）

- **CAHAYAは手で持つと「鳴り」が落ちる**（P-32Dの約5倍の減衰）
- 卓上型・自立型の利点が日本式より大きい
- v3設計（卓上型 + 桐共鳴箱）はこの特性と整合
- 制振実験で「アルミテープ貼付」が顕著に効いたのは、本来手で押さえれば自然に起こっていた制振が、テープで局所化された形で実現されたとも解釈できる

### 5.6 境界層・粘性損失（軽微）

#### 5.6.1 計算

f = 436 Hz、空気動粘度 ν_air = 1.5×10⁻⁵ m²/s で:
- 境界層厚 δ = √(2ν/ω) = **104.7 μm**
- 振動速度 v=1 mm/s（控えめ仮定）での粘性損失:
  - P_visc ≈ 0.5 × ρ × ν × (v²/δ) × S ≈ **0.0017 μW**

#### 5.6.2 含意

全外部放射 ~21,000 μW に対し**0.001%以下**。**完全に無視可能**。

中華式・日本式とも同等の小さい効果。本文書では参考のみ記述。

### 5.7 統合的影響まとめ

第5章で同定した5つの追加物理現象を統合:

| 現象 | 効果の方向 | 定量規模 |
|---|---|---|
| 単段系化（5.2） | 高周波モード消失 | ~5 kHz帯のmodes不在 |
| 直接放射σ増大（5.3） | **放射効率向上** | **約2倍** |
| 放射質量負荷（5.4） | (1,1)モード低下 | -11 Hz (A4と一致) |
| 接触減衰増大（5.5） | 手持ち時に制動 | **~5倍** |
| 境界層損失（5.6） | 微小減衰 | <0.01μW（無視可） |

**総合判断**: CAHAYAは P-32D に対し、

1. **置いた状態では、より「素直に」「効率よく」鳴る**（放射効率2倍、介在物なし）
2. **手で持つと、急に制動される**（接触減衰5倍）
3. **特定楽音（A4近傍）で鳴りが最大化される**（モード周波数差）
4. **高音域の倍音応答が単純化される**（二段系のmodes不在）

これらは v3 設計が CAHAYA を流用しつつ「卓上型」を採用する物理的妥当性を補強する。

## 第6章 中華式固有の物理的含意

### 6.1 ABS外壁の支配性増大

P-32Dでは外装カバー (C4) が外部放射の3.8%を担っていた。中華式ではABS外壁がこれを吸収 -> C1の支配性が95%以上に達する可能性。さらに第5.3節の放射効率σの増大も加味すると、ABS外壁の音響支配性は P-32D比で**1.5-2倍**に達する可能性がある。

これは「中華式ピアニカは ABS壁が音響挙動を完全支配する」ことを意味し、v3設計で **ABS壁を桐共鳴箱に置換する戦略** の物理的根拠を強化する。

### 6.2 楽音とABS壁モードのマッチング（v1.5、フリーリード工業精密性の観点で再校正）

#### 6.2.1 ユーザの実演観測ファクトと意図（2026-05-01、再解釈）

ユーザは日本製ピアニカ（P-32D系）とCAHAYAを同時演奏した結果:

> 「ピッチは合っている。たまにウルフが発生するぐらい。」

ユーザの意図（再確認、2026-05-01）:

> 「ウルフはリードの個体差もあるから、ウルフ自体より、稀なウルフの発生として、フリーリードの工業的精密さが言いたかっただけ」

これは本プロジェクトの基本哲学「**ピアニカ = 工業品音源 + 工芸品筐体**」の実証的根拠である。

v1.4では本ファクトを「壁モード理論の校正データ」として狭く解釈したが、ユーザの本来の意図は**フリーリード製造の工業的精密性**であった。本v1.5でこの理解を反映する。

#### 6.2.2 ファクトの再解釈: 工業的精密性の3つの証拠

##### (1) ピッチ一致 = リード製造精度の高さ

異なる製造者（ヤマハ vs 中華系）、異なるリード材質（真鍮 vs アルミ）、異なる筐体構造（日本式 vs 中華式）でも、**同じ鍵盤ラベルが同じピッチを発する**。これは:

- リード自身の高Q（材料Q=500-3000、playing Q=30-60）
- **リード製造の高精度**（寸法・厚み・テーパーの均一性）
- 壁モード等の二次効果はピッチを動かさない（リード支配）

の組み合わせ。なかでも**製造精度**が機種間ピッチ整合の主要因。

##### (2) 「ウルフが稀」 = 各要因の総合的な精密性

ウルフ（不安定な共鳴）の潜在的原因は複数ある:

| 原因 | 説明 |
|---|---|
| **リード個体差** | 製造公差、特に低価格品で影響しうる |
| ABS壁モードとの局所結合 (D2) | 壁モード近傍音で subtle な干渉 |
| plenum内の局所音場ムラ | 定在波による楽音間の音場差 |
| ガスケット/接触圧の個体差 | 組立公差による境界条件の変動 |

**これらすべての要因が重なる可能性があるにも関わらず「ウルフが稀」であることは、各要因の影響が抑制されていることを意味する**。最も支配的と推定される**リード個体差が小さい**ことが、その背景にある。

##### (3) フリーリード工業精密性の意味（量的）

フリーリードは機械加工で製造される。一般的な公差:

| 寸法 | 量産品の公差 |
|---|---|
| リード厚み | ±0.005~0.010 mm |
| リード長さ | ±0.05 mm |
| ピッチ調整 | ±2~5 cents（量産品でも）|
| スロット幅 | ±0.01 mm |

これらの精度が、機種間でピッチ一致 + ウルフ稀を実現している。**価格2,500円のCAHAYAでも十分な品質を持つ**ことが、本プロジェクトのCAHAYA流用戦略の正当化となる。

#### 6.2.3 v3 設計への含意（工業品 + 工芸品の分業）

「ピアニカ = 工業品音源 + 工芸品筐体」の分業は、v3設計の戦略を物理的に正当化する:

| 要素 | カテゴリ | 設計戦略 | 根拠 |
|---|---|---|---|
| リード（音源） | **工業品** | **流用** | 機械加工精度で品質確保 |
| プレート（リード支持） | **工業品** | **流用** | アルミ32枚分、CAHAYAから |
| 鍵盤ユニット（バルブ） | **工業品** | **流用** | 32バルブ機構、コスト効率良 |
| 共鳴箱（外殻） | **工芸品** | **新設** | 桐 + X-brace、設計自由度大 |
| 接続界面 | **工芸品** | **新設** | 境界条件の精密制御に重要 |

**v3設計は安価で精密な工業品音源 + 設計可能な工芸品筐体の組み合わせ**。CAHAYAの2,500円で得られるリード品質は、設計v3の16,000円目標予算内で十分。

#### 6.2.4 壁モード物理の位置づけ（参考、過大解釈しないこと）

壁モード周波数の機種間差は構造解析として依然として存在:

| 機種 | (1,1)モード | 楽音帯域での潜在的影響 |
|---|---:|---|
| P-32D | 323 Hz (文献値) | E4 (329Hz) 近傍に subtle な timbre 影響の可能性 |
| CAHAYA | 435.7 Hz (本文書予測、§5.4) | A4 (440Hz) 近傍に subtle な timbre 影響の可能性 |

これらは**潜在的な要因の一つ**であり、ウルフのような顕在的現象として観測されるとは限らない。リード個体差や組立精度と区別困難。

##### v1.4の過大予測の撤回

v1.4の§6.2.3で以下のように記述していた:

> | E4 | 329 Hz | **ウルフリスク（壁モード一致）** | 通常 |
> | A4 | 440 Hz | 通常 | **ウルフリスク（壁モード一致、+4.3Hz差）** |

これは**ウルフを壁モードに過大帰属**した予測。実態は:

- ウルフは複数要因の総合（リード個体差、壁モード、plenum音場、組立）
- 機種固有の壁モード周波数 ≠ 機種固有のウルフ音域とは限らない
- 系統的な実機測定なしに断定できない

##### 検証可能な実験設計（参考）

「壁モードがウルフに寄与する」を検証するには大規模な統計実験が必要:

1. 同一機種の複数個体（10台以上）でウルフ音を比較 -> リード個体差の評価
2. 同一個体で温湿度を変化させた時のウルフ音変化 -> 組立精度の評価
3. 機種間（P-32D vs CAHAYA各10台）でウルフ音の統計的差異があるか検定

これは大規模測定プログラムを要し、本プロジェクトの範囲外。**本v1.5では壁モード周波数の差異は理論計算の結果として残すが、実用的な楽音への影響は subtle で工業精密性に隠れるレベル**と位置づける。

#### 6.2.5 反省: 5回目の同種バイアス（新型）

過去の指摘パターンを整理:

| 回 | 指摘箇所 | バイアスの形 |
|---|---|---|
| 1 | 「P-32D 22経路がそのまま適用」 | 構造前提を確認せず適用 |
| 2 | 「C4 → C1統合」 | 簡略化で新物理を考えず |
| 3 | 「維持（構造非依存）」 | 個別検証せず一括ラベル |
| 4 | 「強い鳴りが最大化」 | 理論を過大評価 |
| 5 | 「ウルフリスクA4と断定」 | **ユーザの意図を自分のモデル枠で狭く解釈** |

##### 第5回の特徴（新型バイアス）

第1-4回はすべて「**構造を仮説で埋める**」型のバイアスだった。

第5回は新型: ユーザの empirical statement を「自分のモデルへの校正データ」として扱い、**ユーザの本来の意図（工業精密性）を読み取れなかった**。これは「ユーザの言葉を自分の理論枠に押し込める」バイアス。

##### 教訓

- ユーザの言葉は「私のモデルへの入力」ではなく「**ユーザが伝えたい論点**」として理解する
- 短い empirical statement の背後にある哲学的意図を尊重する
- 「FACT」と強調された言葉は、その意図まで含めて校正データとする
- 自分のモデル/章立てに合うように解釈を曲げない



### 6.3 v3設計への含意

本解析はv3設計の以下の方針を強化する:

1. **ABS外壁の置換は必須**: C1の支配性が95%以上 -> 外壁材料の物性が音色を決定
2. **壁モードを音域に合わせる**: 桐共鳴箱は f₁₁ < F3 (175Hz) になる設計が必要 (壁モードと楽音帯域を分離)
3. **ガスケット導入の是非**: CAHAYA流のガスケットレスは「鳴り」には有利だが、共鳴箱の境界条件を制御するならガスケット導入も検討
4. **接触面 (B7') の利用**: 卓上型で台座振動に利用可能 (ボディ振動式)。手持ちは制動5倍のため非推奨（第5.5節）。

詳細: `../../03-design/v3/design-spec-v3.md` 第4-5章

## 付録C: 主要数式

### C.1 矩形薄板の(m,n)モード周波数（簡支境界）

$$f_{mn} = \frac{\pi}{2}\sqrt{\frac{D}{\rho h}}\left[\left(\frac{m}{L_x}\right)^2 + \left(\frac{n}{L_y}\right)^2\right]$$

### C.2 曲げ剛性

$$D = \frac{Eh^3}{12(1-\nu^2)}$$

### C.3 ヘルムホルツ周波数

$$f_H = \frac{c}{2\pi}\sqrt{\frac{S_{hole}}{V \cdot L_{eff}}}$$

### C.4 リングダウン時間

$$\tau = \frac{Q}{\pi f}$$

### C.5 パワー保存則

$$P_\text{reed} = P_\text{external\_radiation} + P_\text{internal\_damping}$$

中華式ではP_internal_damping項からB3, B5の粘弾性損失が消失。
