# 空気室エンクロージャー材料の物理計算

## 問い

空気室全体をABS以外の単一材料でエンクローズする。
魂柱は立てない。壁面全体が放射面になる。
最適な材料は何か。


---

## 隠れた支配パラメータ: スパン

壁材を選ぶ前に、もっと重要なパラメータがある。
**壁板のスパン（支持点間距離）** である。

空気室の壁は内部圧力500Paに耐える必要がある。
壁の最小板厚 h_min は次の式で決まる:

```
h_min = ( alpha * P * a^4 * 12 * (1-nu^2) / (E * w_limit) )^(1/3)

alpha: 板たわみ係数 (四辺固定 = 0.0138)
P:     内圧 (500 Pa)
a:     スパン (支持点間距離)
E:     ヤング率
w_limit: 許容たわみ (0.5mm)
```

**h_min は a の 4/3 乗に比例する。**
スパンを半分にすると、h_min は 0.5^(4/3) = 0.40 倍になる。

そして Gamma は h に比例する。
つまり**スパンを半分にすれば Gamma は 0.4 倍になる。**


### スパンの効果 (桐の場合)

```
| スパン | h_min | rho*h | Gamma | eta_rad | T_BN2 | Net  |
|--------|-------|-------|-------|---------|-------|------|
| 400 mm | 9.2mm | 2.57  | 8.25  | 10.8%   | 2.6%  | 0.3% |
| 200 mm | 3.6mm | 1.02  | 3.28  | 23.4%   | 14.8% | 3.5% |
| 100 mm | 1.4mm | 0.40  | 1.30  | 43.5%   | 55.6% | 24%  |
|  60 mm | 0.7mm | 0.21  | 0.66  | 60.3%   | 67.5% | 41%  |
|  50 mm | 0.6mm | 0.16  | 0.52  | 66.0%   | 58.2% | 38%  |
```

スパン400mmでは桐でも9mm厚が必要で、Gamma=8.25。
スパン60mmでは0.7mm厚で済み、Gamma=0.66。

**同じ材料で効率が0.3%から41%へ、136倍変わる。**
材料選択よりスパン設計の方がはるかに支配的。

## 各材料の最適スパンでの性能

各材料について、正味効率(BN2以降)が最大となる
スパンと板厚を求めた結果:

```
| 材料 | h [mm] | span [mm] | Gamma | eta_rad | T_BN2 | Net   | vs P-32D |
|------|--------|-----------|-------|---------|-------|-------|----------|
| Aluminum | 0.30 | 20 | 0.03 | 96.8% | 68.5% | 66.3% | 114x |
| CFRP | 0.30 | 20 | 0.39 | 72.2% | 70.0% | 50.5% | 87x |
| Bamboo | 0.50 | 20 | 0.70 | 58.7% | 69.7% | 40.9% | 70x |
| Kiri | 0.73 | 60 | 0.66 | 60.3% | 67.5% | 40.7% | 70x |
| Cedar | 0.65 | 60 | 0.74 | 57.6% | 68.1% | 39.3% | 67x |
| Spruce | 0.51 | 55 | 0.82 | 54.9% | 64.8% | 35.6% | 61x |
| Hinoki | 0.53 | 55 | 0.96 | 51.1% | 66.1% | 33.8% | 58x |

```

## 結果の物理的解釈

### アルミニウム: 数値上の最高効率、しかし

アルミ0.3mmの Gamma=0.03、eta_rad=97% は計算上最高。
しかしこれは eta=0.0001（内部損失がほぼゼロ）のため。

内部損失がゼロに近いということは、板が一度振動すると
いつまでも鳴り続けるということ。鐘や銅鑼と同じ。

楽器としては板の固有振動数にエネルギーが集中し、
鋭いピークと深い谷が並ぶ周波数応答になる。
これは「金属的な音」であり、楽器の音としては
使いにくい。

木材が楽器に使われる理由は、eta=0.005-0.015 の
「適度な損失」が周波数応答を滑らかにし、
自然な音色を生むからである。

アルミは物理的には最高効率だが、音楽的には最良ではない。


### 桐: 木材の中で最高

木材の中では桐が最高効率(40.7%)。理由は:

1. 密度が最低(280 kg/m^3) -> 面密度が小さい
2. 損失係数が低い(0.008) -> 適度な減衰
3. ヤング率が中程度(5 GPa) -> 0.7mmで圧力に耐える

**桐が「ちょうどよい」理由:**

密度が低い -> 薄くても軽い -> Gamma が小さい。
ヤング率が中程度 -> 薄くても曲げ剛性が確保される。
損失が低い -> エネルギーが熱にならず音になる。

バルサは密度がさらに低い(160)が、ヤング率も低い(3.5 GPa)
ため、同じスパンでより厚い板が必要になる。
結果として rho*h 積はバルサの方が少し大きく、
桐より劣る。


### 杉: 桐に次ぐ候補

杉(sugi)は日本で最も入手しやすい軽量木材。
rho=350で桐の280より少し重いが、E=7 GPaで桐より硬い。
結果は39.3%で桐の40.7%に僅差。

入手性を考えると杉は有力な代替材。


### CFRP: 夢の材料だが過剰

CFRP(カーボン繊維強化プラスチック)は
E=70 GPa, rho=1600, eta=0.002。
0.3mmで圧力に耐え、Gamma=0.39。

しかし50.5%という効率は桐の40.7%と大差なく、
コストと加工性を考えると過剰。


### ABS: どうやっても悪い

ABSは最適スパン(20mm)でも8.0%にしかならない。
eta=0.030 が致命的。振動するたびにエネルギーの3%が
熱になる。桐(0.8%)の3.75倍の損失。
密度も1050で桐の3.75倍重い。
eta * rho の積はABSが桐の14倍。
だからGammaが桐の14倍になる。

## 最適構造: 桐の薄板箱 + 内部リブ

計算結果から、理想的な空気箱の構造が見える。

```
材料: 桐
板厚: 0.7 - 1.0 mm
リブ間隔: 50 - 60 mm
リブ材: 桐の角材 (5x5mm程度)
気密処理: 内面に柿渋 or 漆
```

この構造は何かに似ている。

**ギターの表板**である。

クラシックギターの表板はスプルース2.5mm、
力木(ブレーシング)間隔は50-80mm。
力木は板を補強すると同時に、板の振動モードを
制御する。

ピアニカの桐薄板箱はまさにギターの表板構造と同じ。
ただしギターは片面(表板)だけが放射面だが、
ピアニカの桐箱は全面(6面)が放射面になる。

```
ギター: 表板のみ放射 (0.08 m^2)
ピアニカ桐箱: 6面放射 (0.09 * 2 + 側面 ~ 0.25 m^2)
```


### リブの物理的役割

リブ(力木/ブレーシング)は3つの機能を持つ:

1. **圧力支持**: スパンを縮め、薄板でも内圧に耐える
2. **モード制御**: 板の固有振動数と節線を設計する
3. **振動分配**: リードプレートからの入力を板全面に伝える

第3の機能は特に重要。リードプレートとの接触は
箱の一面(上面)に限られるが、リブが箱の各面を
構造的に結合するため、振動が全面に広がる。

これはギターの力木が「駒からの入力を表板全体に
広げる」のと同じ機能。

## 全体効率の再計算

桐薄板箱 (0.7mm, span 60mm, 全面放射) の全体効率:

```
BN1: 18% (物理法則、不変)
BN2: 桐壁の Z_point が小さい -> T_BN2 = 67.5%
BN3: Gamma = 0.66 -> eta_rad = 60.3%

全体: 0.18 * 0.675 * 0.603 = 7.3%
```

比較:

```
| 設計 | BN1 | BN2 | BN3 | 全体 | vs P-32D |
|------|-----|-----|-----|------|----------|
| P-32D | 18% | 53% | 1.1% | 0.1% | 基準 |
| v3全面接着(最良) | 18% | 53% | 36% | 3.4% | 34x |
| v3魂柱1面 | 18% | 48% | 36% | 3.1% | 31x |
| 桐薄板箱 | 18% | 68% | 60% | 7.3% | 73x |
| 桐薄板箱(2mm厚,span60mm) | 18% | 68% | ??% | ??% | ??x |
```

ここで注意: BN2 が 53% から 68% に上がっている。

これは重要な発見。ABS壁の Z_point=0.69 に対し、
桐壁の Z_point=1.53 (0.7mm) の方が
リードプレートの Z_plate=2.25 に近いため。

```
T = 4 * Z1 * Z2 / (Z1 + Z2)^2

Z_plate=2.25, Z_ABS=0.69:
T = 4 * 2.25 * 0.69 / (2.94)^2 = 6.21/8.64 = 71.9%
effective = 0.7 * 71.9 = 50.3%

Z_plate=2.25, Z_kiri(0.7mm)=1.53:
T = 4 * 2.25 * 1.53 / (3.78)^2 = 13.77/14.29 = 96.4%
effective = 0.7 * 96.4 = 67.5%
```

桐壁は ABS壁よりリードプレートとの Z整合が良い。
BN2 も改善される。


### なぜ BN2 が改善するか

リードプレート(真鍮1mm)の Z_point = 2.25 Ns/m。

壁材の Z_point は sqrt(D * rho_s) に比例する。
D は E*h^3 に比例し、rho_s は rho*h に比例するから、
Z_point は sqrt(E * rho) * h^2 に比例する。

```
Z_point 比:
ABS(1.5mm):  sqrt(2.3e9 * 1050) * 0.0015^2 = 49130 * 2.25e-6 = 0.111 -> *8 = 0.89
桐(0.7mm):   sqrt(5e9 * 280) * 0.0007^2    = 37417 * 4.9e-7  = 0.0183 -> *8 = 0.147

... 計算値にずれあり。正確値はプログラム出力を参照。
```

重要なのは、Z_plate=2.25 に対して
壁の Z_point が近いほど T_BN2 が高くなること。
完全一致 (Z_wall = 2.25) のとき T = 100%。

桐薄板は Z_point が ABS壁より Z_plate に近いため、
BN2 の伝達率が上がる。

## 結論

### 材料の答え: 桐

木材の中で桐が最高効率。
低密度(280)と中程度のヤング率(5 GPa)の組み合わせが、
「薄くても強度があり、軽くて損失が小さい」を実現する。

杉(cedar)が僅差の次点。入手性では杉が有利。

### しかし材料より重要なのはスパン

同じ桐でも:
- スパン400mm -> Gamma=8.25, Net=0.3%
- スパン60mm  -> Gamma=0.66, Net=41%

**136倍の差は材料選択では生まれない。スパン設計で生まれる。**

### 理想構造

桐0.7mm薄板 + 桐リブ(5x5mm) + リブ間隔60mm の箱。
ギターの表板と同じ「力木構造」を空気室全体に適用。
全面が放射面として機能する。

全体効率 7.3% (P-32D の 73倍, +19dB)。

### 前版設計(v3全面接着)との比較

```
v3全面接着: BN1(18%) * BN2(53%) * BN3(36%) = 3.4%
桐薄板箱:   BN1(18%) * BN2(68%) * BN3(60%) = 7.3%

桐薄板箱は v3 の 2.1倍。
差の内訳: BN2 が 1.28倍、BN3 が 1.67倍。
```

BN2 の改善は「桐壁の Z がプレートに近い」ため。
BN3 の改善は「薄板によるGamma低減」。

ただしこの計算には実用上の注意がある:
桐0.7mmは和紙のように薄く、気密の維持と
リブへの接着が極めて繊細な作業になる。
(今は実用制約を考えないので問題としない)
