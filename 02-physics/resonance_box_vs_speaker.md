## 1. 序論 -- なぜ共鳴箱はスピーカと根本的に異なるのか

スピーカで音を出すのが当たり前の現代から見ると、「楽器の共鳴箱」は古めかしい技術に見える。しかし両者は **別の物理問題を解いている** のであり、優劣ではなく **使い分け** の対象である。

ストラディバリのバイオリンを最高級スピーカで再生しても、本物のストラディバリにはならない。これは音響工学の限界ではなく、両者が **異なる時間・空間スケールで音を作る** ためである。

| 観点 | 共鳴箱 | スピーカ |
|------|--------|---------|
| 起源 | 古代 (リラ、リュート) -> 19世紀完成 (バイオリン、ピアノ) | 20世紀 (1925 Rice-Kellogg) |
| エネルギー | 機械 (奏者の運動) | 電気 (系統電源) |
| 変換段数 | 1段 (機械 -> 音響) | 3段 (電気 -> 機械 -> 音響) |
| 励振源と放射体 | **分離** (弦/リード -> 板) | **一体** (コイル直結コーン) |
| 周波数応答 | 多重ピーク (個性、フォルマント) | フラット (理想は) |
| 効率 | 30 - 40% | 0.5 - 5% |
| 個性 | 楽器の魂 | 排除すべき欠陥 |
| 寿命 | 数百年 (ストラディバリ300年現役) | 15-20年 (電子部品寿命) |

**重要な誤解**: 「共鳴箱はスピーカの古い形」ではない。共鳴箱はいまだに **電気音響では到達できない効率と音色** を実現している (バイオリン40%、最高級スピーカ5%)。

この差を生む物理的根拠を、本書では楽器音響学・熱力学・流体力学の3視点から統一的に解明する。

## 2. 共鳴箱の原理 -- 機械-音響直接結合 (1段変換)

### 2.1 1段エネルギー変換の意味

共鳴箱を持つ楽器の動作:

```
[励振源]   --(機械結合)-->   [共鳴板]   --(空気駆動)-->   [音波]
 弦/リード                     表板/響板
 (vibration)                  (vibration)              (radiation)
```

ここに **電気が介在しない**。これがスピーカ (Lorentz力で電流を機械力に変換する3段) との根本的差異である。

### 2.2 励振源の例

共鳴箱と組み合わせる励振源は楽器ごとに多様:

| 楽器 | 励振源 | 励振原理 | 持続性 |
|------|-------|---------|--------|
| バイオリン | 弓 + 弦 | スリップ・スティック (Helmholtz motion) | 連続 |
| チェロ | 弓 + 弦 | 同上 | 連続 |
| ギター | 指 + 弦 | インパルス (撥) | 減衰 (2-5s) |
| ピアノ | ハンマー + 弦 | インパルス (打弦) | 減衰 (10-30s) |
| ハープ | 指 + 弦 | インパルス | 減衰 |
| バンジョー | 指 + 弦 | インパルス | 減衰 |
| ハーモニカ | 呼気 + リード | 流体励振 (連続) | 連続 |
| ピアニカv3 | 風箱 + リード | 流体励振 (連続) | 連続 |
| アコーディオン | ふいご + リード | 流体励振 (連続) | 連続 |

**ピアニカv3はハーモニカ・アコーディオン系の流体励振 + バイオリン系の共鳴板放射** を組み合わせたハイブリッド。励振源が連続的なため、減衰系の弦楽器とは異なる設計自由度を持つ。

### 2.3 機械結合の物理 (駒の役割)

励振源から共鳴板への結合は **駒 (bridge)** が担う:

$$T_{coupling} = \frac{4 Z_1 Z_2}{(Z_1 + Z_2)^2}$$

- $Z_1$: 励振源側の機械インピーダンス [N·s/m]
- $Z_2$: 共鳴板側の機械インピーダンス [N·s/m]

| 楽器 | $Z_{源}$ | $Z_{板}$ | $T$ |
|------|----------|----------|-----|
| Guitar | 0.3 | 50 | 2.3% |
| Piano | 1-20 | 200 | 2-30% |
| Violin | 0.15 | 30 | 2% |
| **Pianica v3** | (ABS壁) | (kiri板) | **97%** |

**設計パラドックス**: 弦楽器では $T$ を **意図的に低く** 設定する。なぜなら $T$ が高すぎると弦のエネルギーが瞬時に流出し、音が即座に消えるためである:

$$\tau_{decay} = \frac{2 m_{string}}{T \cdot Z_{bridge}}$$

ピアニカv3は **連続励振** (リードが空気から常時エネルギーを補給される) のため、この時間定数の制約から自由となる。**$T = 97\%$** という極端な高結合が許される唯一の設計位置にある。これがピアニカv3の核心優位性である。

### 2.4 等価回路 (モビリティアナロジー)

共鳴箱系を電気回路で表現する標準的方法:

```
[F_source] -- [Z_source] -- [Z_bridge] -- [Z_plate] -- [Z_radiation]
   起源          源側         駒          板         空気
```

ここで:
- $F$ (力) は電圧 $V$ に対応
- $v$ (速度) は電流 $I$ に対応
- $Z$ (機械インピーダンス) は電気インピーダンスに対応

**エネルギー流の透視**:

$$P_{radiated} = |F|^2 \cdot \frac{Re(Z_{rad})}{|Z_{total}|^2}$$

$|Z_{total}|^2$ を最小化することが楽器設計の核心。共鳴板のモード周波数で $|Z|$ が最小になり、エネルギー流出が最大化される。

### 2.5 共鳴箱の核心: 「板が空気を押す」

共鳴箱は **共鳴板自体が振動して空気を押す**。これがコーンと同じ「変位源」の役割を果たす。

ただし違いは:

```
スピーカコーン: 励振源 (コイル) と放射体 (コーン) が一体
                 -> 機械的に直結
                 
共鳴箱共鳴板: 励振源 (弦/リード) と放射体 (板) が分離
              -> 駒経由で結合
              -> 励振源は板の上を動かない (励振点固定)
```

この **分離** により、共鳴板は励振源とは独立した固有モードで振動する。これが多重共鳴ピーク (フォルマント) を生み、楽器固有の音色を作る。

スピーカは「コイルがコーンを動かす」ため、コーンの固有モードはむしろ抑制すべき欠陥となる (分割振動)。共鳴箱では多重モードが **積極的に設計されるべき特徴** である。

### 2.6 「壁が鳴る」のが正解

スピーカ設計者は「箱を鳴らすな」と言う (本Appendix前作 §4.4)。共鳴箱では真逆である:

> 共鳴箱の壁 (側板含む) **全部が鳴って** 音を放射する

ピアニカv3の Gamma 理論は、桐板1.5-2mmの全面が振動して放射することを定量化したものである。**箱全体が「シンギングボディ」となる**。これが楽器音響学の世界観である。

## 3. 共鳴板の構造・構成・機能

共鳴箱の主役である共鳴板を詳述する。

### 3.1 共鳴板の機能要件

```
共鳴板に求められる物理特性:

1. 軽量: 振動応答の高速化 (m_s 小)
2. 高剛性: 屈曲剛性 D 大 (低域モード周波数を稼ぐ)
3. 適度な内部損失: 過剰共鳴の抑制 (eta = 0.03 - 0.10)
4. 音速整合: 板の曲げ波速度 c_b ~ 空気波速度 c_air (放射効率)
5. 経時安定性: 数十年 - 数百年で特性が変わらない
```

これら5要件は両立しないので、楽器ごとに **どの要件を優先するか** が音色の個性となる。

### 3.2 木質材料の優位性

木材が共鳴板素材として選ばれる物理的理由:

**異方性が利く**: 繊維方向 ($E_{//}$) と直交方向 ($E_{\perp}$) で剛性が10-20倍違う。これにより:
- バスバーや力点に沿った剛性は高い
- 横方向は柔軟で振動できる
- 結果として「方向性のある振動モード」を作れる

**減衰特性**: セルロース繊維の粘弾性により $\eta = 0.02 - 0.10$ の中庸な内部損失。金属 ($\eta = 0.001$) より高く、ゴム ($\eta = 0.5$) より低い。

**加工性**: 数百年前から職人が削れる。複合材料の精密加工は近代以降。

### 3.3 主要な共鳴板素材比較

| 素材 | 密度 [kg/m^3] | $E_{//}$ [GPa] | $\eta$ | 用途 |
|------|--------------|----------------|--------|------|
| ヨーロッパ・スプルース | 400 | 12 | 0.03 | バイオリン表板 |
| 北米シトカ・スプルース | 380 | 10 | 0.04 | ピアノ響板、ギター |
| 桐 (Paulownia) | 290 | 6 | 0.05 | 琴、ピアニカv3 |
| ローズウッド | 850 | 16 | 0.02 | バイオリン裏板、ギター裏板 |
| 楓 (Maple) | 600 | 11 | 0.03 | バイオリン裏板、駒 |
| 杉 (Cedar) | 380 | 8 | 0.06 | クラシックギター表板 |
| カーボン繊維 | 1500 | 70 | 0.005 | 現代実験品 |

**比剛性 $E/\rho$ で並べると**:
- スプルース: 30,000 m^2/s^2
- 桐: 20,700 m^2/s^2
- 楓: 18,300 m^2/s^2
- カーボン: 46,700 m^2/s^2

スプルースの比剛性は木材最高に近く、これがバイオリン・ギター・ピアノ表板の標準である理由。

**ピアニカv3が桐を選んだ理由**:
- スプルースより比剛性は低いが、$\eta = 0.05$ で内部損失が高め -> 過剰共鳴を自然に抑える
- 密度が低い ($\rho = 290$) ため、Gamma 値が小さい (= 放射効率高い)
- 日本の伝統素材で入手容易・加工しやすい

### 3.4 単純板 vs 複合板

**単純板 (バイオリン表板)**:
- 一枚板または2枚接ぎ
- 厚さ均一に近い (中央 3mm、外周 2mm のテーパー)
- バスバー (細い角棒) が裏面に縦走

**複合板 (ピアノ響板)**:
- スプルース集成材 (5-10枚接ぎ)
- 厚さ 8-10mm 中央、5mm外周 (Steinway タイプ)
- 複数のリブ (横方向角棒) で剛性付与
- 響鈴 (bridges) が弦からの力を分散して入力

**ピアニカv3共鳴板**:
- 桐 1.5 - 2.0 mm (薄い、軽い)
- X字ブレース (構造補強)
- リード列直下に駒
- 中央付近に放射効率最大化のための薄め部分

### 3.5 ピアノ響板の Steinway 設計

歴史的に有名な Steinway の **Diaphragmatic soundboard**:

```
中央: 9mm (剛性 + 質量、低域モード)
外周: 5mm (柔軟、振動境界)
クラウン: 数mmの曲率 (圧縮プリストレス)
```

この **テーパー厚さ** は20世紀の発明と思われがちだが、実は19世紀末から実装されている。本質は「中央は重く動きにくく、外周は軽く動きやすい」という、**鼓膜の物理を模倣** したもの (ヘルムホルツの鼓膜理論との一致)。

ピアニカv3でも将来的に検討する価値あり (現在は均一2mm)。

### 3.6 バスバー・魂柱・ブレース

共鳴板の振動モードを操作する補助部材:

**バスバー (バイオリン)**:
- 表板裏面にG弦側に縦に貼られる細い角材
- 表板の局所剛性を上げ、非対称モードを作る
- 結果として駒の左右でモード形状が異なり、複雑な放射パターン

**魂柱 (バイオリン)**:
- 表板と裏板を内側で連結する直径6mm程度の丸棒
- 表板の振動を裏板に伝達
- これがないと「ぼんやり」した音、入れると「芯のある」音

**X-brace (ギター)**:
- ブリッジ下に X 字に貼るブレース
- 高音部と低音部で異なる剛性を作る
- ピアニカv3もX-braceを採用 (0501b パラダイム転換で検証)

### 3.7 製作精度と経年変化

**バイオリン**:
- 製作直後は剛性高く音が硬い
- 50-100年で繊維がリラックス、音が「開いて」くる
- ストラディバリ (1700年頃製作) は300年経過でも現役

**ピアノ**:
- 響板は製作後20-30年で クラウン (反り) が減少
- 音色変化、再修復必要
- 100年もてば長寿の部類

**ピアニカv3**:
- 桐の経年変化は緩やか (50-100年で安定)
- 接着剤・ガスケットが先に劣化 (10-20年)
- 設計思想: 「100年後の弟子が修復できるシンプルさ」

## 4. 駒・側板・空気腔の構造・構成・機能

共鳴板を支える周辺構造を詳述する。

### 4.1 駒 (Bridge) の機能

駒の役割:
1. 励振源 (弦/リード) からの振動を共鳴板に伝達
2. インピーダンス変換 (低Z源から高Z板へ)
3. 振動を空間的に分散 (片足/両足/3足)

**バイオリンの駒**:
- 楓材、高さ約33mm
- 2本足、表板の F字孔近くに立つ
- 厚さ 4mm、左右非対称
- 弦からの力を表板の最も振動しやすい位置に伝達

**ピアノの駒**:
- 木製、響板の上を縦に走る長い駒
- 弦を斜めに渡らせて緊張力を響板下方向に変換 (down-bearing 1度程度)
- 長さ1m以上、高音域用と低音域用の2本

**ピアニカv3の駒**:
- リードプレートと共鳴板の間
- ABS壁 (リードプレート側) と桐板 (共鳴板側) のインピーダンス整合
- T = 97% 達成

### 4.2 駒のインピーダンス変換

駒の役割を等価回路で:

```
[弦/リード] --[Z_弦]-- [駒] --[Z_板]-- [共鳴板]
              低Z       (変換器)        高Z
```

駒の機械インピーダンスは弦と板の中間で、両者の結合効率を最適化する。

数学的表現:

$$Z_{driven} = Z_{string} + j\omega m_{bridge} + \frac{1}{j\omega C_{bridge}} + Z_{plate}$$

中域では駒の質量・コンプライアンス効果は小さく、純粋な機械インピーダンス変換器として動作。

**ピアニカv3の駒設計**:

ABS リードプレートと桐板の機械インピーダンスは元々大きく異なる:
- $Z_{ABS}$ ~ 1.6 MPa·s/m (大)
- $Z_{kiri}$ ~ 0.5 MPa·s/m (中)

駒なしで直結すると T < 50%。**駒材質 (楓 or 樫)** を介することで両者のインピーダンスを徐々に変化させ、結果的に T = 97% を達成。

### 4.3 側板 (Ribs) の機能

側板 (バイオリンの肋材、ギターのサイド) の役割:

1. 表板と裏板を機械的に分離 (位相反転で双極子になるのを防ぐ)
2. 内部空洞容積を確定 (Helmholtz共鳴のための条件)
3. 楽器の構造強度を保持

**重要**: 側板は **鳴ってはいけない部材ではない**。むしろ適度に振動して放射に寄与する。これがピアニカv3の Gamma 計算で「全壁面が放射」と扱う根拠。

### 4.4 内部空気腔 (Cavity)

楽器の内部空気は単なる「空き空間」ではなく、**Helmholtz共鳴器** として機能する。

**バイオリン内部空気**:
- 容積 約 2 L
- F字孔 を通じて外部空気と結合
- Helmholtz共鳴周波数 ~ 280 Hz (G線3度の音)
- 低域 (G線、D線) の放射効率を強化

**ギター内部空気**:
- 容積 約 12 L
- サウンドホール (直径 9 cm)
- Helmholtz共鳴 ~ 100 Hz
- 低域 (E2 = 82 Hz) を底上げ

**ピアノ内部空気**:
- 響板下の空間 (グランド) または背面 (アップライト)
- Helmholtz共鳴は弱い (容積大、開口小さい)
- 主に響板の放射を裏面でキャンセルしないようにする役割

**ピアニカv3 内部空気**:
- 容積 14.2 L (v3設計)
- 開口は実機では小さい (ABS底面の溝程度)
- Helmholtz効果は副次的、主役は桐板の放射
- 低域強化は別途検討中 (将来Helmholtz tuning可能性)

### 4.5 サウンドホール (f字孔、サウンドホール) の物理

開口部は2つの役割:

1. **空気の体積流れ** (Helmholtz共鳴ポートとして): 低域強化
2. **板の振動形状の制約**: f字孔 周辺で板の振動が変化、モード形状を非対称化

バイオリンの f字孔 が S字形なのは音響光学的に最適化されたものというより、振動モードの非対称化 + 共鳴特性の調整によるもの (現代音響学解析で再発見)。

### 4.6 ピアニカv3 における Helmholtz 共鳴の扱い

v3 (現状) では Helmholtz 共鳴は副次的:

- 主役: 桐板の屈曲振動による直接放射 (Gamma = 1.8、放射効率 36%)
- 副役: 内部空気腔のクッション効果

将来案として **Helmholtz tuning** (適切な開口サイズで低域を共鳴強化) も検討されているが、現設計では桐板放射のみで十分な効率を達成しているため未実装。

## 5. 共鳴箱全体としての機能 -- 系の伝達関数

### 5.1 入力から放射までの全体伝達

楽器全体の伝達関数:

$$H(\omega) = \frac{p_{far}(\omega)}{F_{input}(\omega)} = T_{bridge} \cdot M_{plate}(\omega) \cdot \eta_{rad}(\omega)$$

- $T_{bridge}$: 駒の結合効率 (周波数依存ほぼなし、機械的)
- $M_{plate}(\omega)$: 共鳴板のモード重畳 (多重ピーク、これが個性)
- $\eta_{rad}(\omega)$: 放射効率 (周波数依存、Gamma理論)

### 5.2 板のモード重畳

共鳴板は無数のモード (m, n) を持つ:

$$f_{mn} = \frac{\pi}{2} \sqrt{\frac{D}{\rho h}} \left[ \left(\frac{m}{a}\right)^2 + \left(\frac{n}{b}\right)^2 \right]$$

ピアニカv3 桐板 (300 mm x 300 mm、h = 2 mm) の主要モード:

| モード | $f_{mn}$ [Hz] | 形状 |
|--------|-------------|------|
| (1,1) | 96 | 中央が腹、四辺が節 |
| (2,1), (1,2) | 240 | 中央線で節、左右対称 |
| (2,2) | 384 | 4分割パターン |
| (3,1), (1,3) | 480 | 3節 |
| (3,2), (2,3) | 580 | 6分割パターン |
| (3,3) | 768 | 9分割パターン |

**音楽的意味**:

各モードの周波数が音名と一致するわけではないが、励振信号のスペクトル成分のうち、モード周波数に近いものが強調される。これが「フォルマント」(母音特有の周波数強調) と同じ原理である。

楽器の音色は **どの周波数を強調し、どれを抑えるか** で決まる。共鳴箱はパッシブなフィルタとしてこの操作を行う。

### 5.3 入力インピーダンス (駒位置)

楽器の駒位置から見た機械インピーダンス:

$$Z_{in}(\omega) = \frac{F(\omega)}{v(\omega)} \bigg|_{\text{駒位置}}$$

これは楽器の **「弾きごたえ」** を決める物理量:
- $|Z_{in}|$ 大: 弦/リードが押し戻される、弾きにくい
- $|Z_{in}|$ 小: 弦/リードがすっと動く、弾きやすい

熟練奏者は $Z_{in}$ の周波数特性を耳と指 (or 呼吸) で感じ取り、楽器との対話を成立させる。

### 5.4 放射効率の周波数依存

放射効率は周波数で大きく変動する:

$$\eta_{rad}(\omega) = \frac{1}{1 + \Gamma(\omega)}$$

低周波 ($\omega \ll \omega_c$、$\omega_c$ は臨界周波数):
- 板の波長 < 空気波長 -> 双極子放射 (キャンセル)
- $\eta_{rad} \to 0$
- 板から音が出にくい

高周波 ($\omega \gg \omega_c$):
- 板の波長 > 空気波長 -> 単極子放射 (キャンセルなし)
- $\eta_{rad} \to 1$
- 板から音が出やすい

**臨界周波数 $f_c$**:

$$f_c = \frac{c^2}{2\pi h} \sqrt{\frac{12 \rho (1-\nu^2)}{E}}$$

ピアニカv3 桐板:
- $c = 343$ m/s, $h = 0.002$ m, $\rho = 290$, $\nu = 0.3$, $E = 6 \times 10^9$ Pa

$$f_c = \frac{343^2}{2\pi \times 0.002} \sqrt{\frac{12 \times 290 \times 0.91}{6 \times 10^9}} = 9.36 \times 10^6 \times \sqrt{5.28 \times 10^{-7}}$$

$$f_c = 9.36 \times 10^6 \times 7.27 \times 10^{-4} = 6800 \text{ Hz}$$

**意味**: 6800 Hz以下では桐板の放射効率が下がる (双極子優勢)。Trompette 8' の倍音は 6800 Hz以下にほぼ収まるため、**桐板厚さの選択は意図的に高 $f_c$ を狙っている**。

### 5.5 多重モードの聴覚的意味

スピーカは「フラットな周波数応答」を理想とする。共鳴箱は **意図的に多重ピーク** を持つ。なぜか?

人間の聴覚は **フォルマント** (周波数強調パターン) を識別する高度な能力を持つ。同じ基音でもフォルマントが違えば、別の音色 (楽器、母音、感情) として認識される。

共鳴箱の多重ピークはこのフォルマントを供給する。**「個性ある音色」とは、楽器固有のフォルマントの集合である**。

スピーカは「再生機」なので、入力信号のフォルマントを正確に再現すべき (フラット応答理想)。共鳴箱は「楽器」なので、自身がフォルマントを生成する。

### 5.6 個体差の物理的根拠

スピーカは個体差を排除する (品質管理で +/- 1 dB以内)。共鳴箱は個体差が **個性として価値を持つ**:

- 同じバイオリン製作者・同じ図面でも、木材の年輪・繊維方向で響きが違う
- ストラディバリ各個体に名前 (Messiah, Lady Blunt) が付く
- 楽器奏者は自分の楽器の個性を理解して演奏

これは「共鳴箱の品質管理が低い」のではなく、**そもそも一意の正解がない** ためである。スピーカは入力信号を忠実に再生する単一の正解があるが、楽器は奏者と協働する個性を持つ。

ピアニカv3 でも各個体の桐板の年輪・繊維で微妙な音色差が出る。これを「品質ばらつき」と見なすか「個体の個性」と見なすかが、設計思想を決める。本プロジェクトは後者を採る。

## 6. 楽器音響学的観点 -- モード理論の体系

### 6.1 板の屈曲振動の支配方程式

Kirchhoff-Love板理論による屈曲方程式:

$$D \nabla^4 w + \rho h \frac{\partial^2 w}{\partial t^2} = q(x, y, t)$$

- $w(x, y, t)$: 板の垂直変位
- $D = E h^3 / [12(1-\nu^2)]$: 屈曲剛性
- $\rho h$: 単位面積質量
- $q$: 外力 (駒からの入力)

**これは双曲線型 PDE で、波動方程式に近い**。違いは時間2階・空間4階で、伝播が分散性を持つ:

$$\omega = c_b k^2$$

- $c_b = \sqrt{D / \rho h}$ (曲げ波速度のスケール)
- $k$: 波数

群速度:

$$c_g = \frac{d\omega}{dk} = 2 c_b k$$

つまり高周波ほど速く伝わる (分散)。これが共鳴板の音響特性 (低域でゆっくり、高域で速く) を生む。

### 6.2 境界条件

板の境界条件で固有モードが決まる:

**自由端 (free)**:
- 駒の周囲、サウンドホール周辺
- 力 = 0、モーメント = 0

**単純支持 (simply supported)**:
- 教科書的、現実にはほとんどない

**クランプ (clamped)**:
- 接着・ネジ止めされた縁
- 変位 = 0、傾斜 = 0

ピアニカv3 桐板は **3辺クランプ + 1辺自由** の境界条件 (側板に接着 + リード入力辺は浮き)。これが固有モードを単純支持と異ならせる。

### 6.3 矩形板の固有モード

3辺クランプ + 1辺自由の固有値は数値解析が必要だが、**全クランプ近似** で:

$$f_{mn} = \frac{\pi}{2 \sqrt{12(1-\nu^2)}} \cdot c_p \cdot h \cdot \left[ \frac{(2m+1)^2}{4 a^2} + \frac{(2n+1)^2}{4 b^2} \right]$$

($c_p = \sqrt{E/\rho}$ = 板素材の縦波速度)

ピアニカv3 ($a = b = 0.3$ m、$h = 0.002$ m、桐):
- $c_p = \sqrt{6 \times 10^9 / 290} = 4550$ m/s
- $f_{11} = (\pi/2 \sqrt{10.92}) \cdot 4550 \cdot 0.002 \cdot [9 / (4 \cdot 0.09) + 9 / (4 \cdot 0.09)]$
- $f_{11} = 0.476 \cdot 9.10 \cdot 50 = 217$ Hz

これは推定値。実測の f_11 = 96 Hz と差があるのは、3辺クランプ + 1辺自由で実効剛性が下がるため。実測値が真値で、近似式は目安。

### 6.4 モード重複と Q値

各モードは Lorentzian 形状で広がる:

$$|H_{mn}(\omega)|^2 = \frac{|H_0|^2}{1 + Q^2 (\omega / \omega_{mn} - \omega_{mn} / \omega)^2}$$

$Q$ 値が大きいほどピークが鋭い。

楽器の Q値 (典型値):
- バイオリン: 30 - 100
- ピアノ: 20 - 50
- ギター: 30 - 80
- ピアニカv3 (推定): 30 - 60

スピーカは $Q < 1$ (制動)、共鳴箱は $Q > 30$。**この差が「鳴る箱」と「ピストン放射器」を分ける**。

### 6.5 モード密度

周波数あたりのモード数 (modal density):

$$n(\omega) = \frac{S}{4\pi} \sqrt{\frac{\rho h}{D}}$$

ピアニカv3:
- $S = 0.09$ m^2、$\rho h = 0.58$ kg/m^2、$D = 4 \text{ N·m}$
- $n = (0.09 / 4\pi) \sqrt{0.58/4} = 0.00716 \times 0.381 = 0.00273$ /(rad/s)
- 1Hz あたり $0.00273 \times 2\pi = 0.0172$ モード
- 100 Hz幅で 1.7 モード、1000 Hz幅で 17 モード

**音楽帯域 (50 Hz - 5 kHz) で約 85 モード**。これが共鳴箱の「豊かな音色」の源。

スピーカコーンのモード数も同程度だが、それらは **抑制すべき欠陥**。共鳴箱では資産。

### 6.6 Chladni パターン

共鳴板に砂を撒いて振動させると、節線 (振動が0の線) に砂が集まる。これが Chladni パターンで、モード形状の可視化に古典的に使われる。

ピアニカv3 桐板でも実測可能 (砂を撒いて駒位置を加振機で振動)。これにより:
- 各モードの実周波数を測定
- モード形状の歪を確認
- 設計通りに動いているかの検証

将来の実機検証で実施予定 (ジャーナル0501e の残課題)。

## 7. 流体力学的観点 -- 板表面の空気駆動

### 7.1 板振動から空気波への変換

共鳴板が振動 ($w(x, y, t)$) すると、板表面の空気は同じ速度で動く (連続条件):

$$v_{air, surface} = \frac{\partial w}{\partial t}$$

これが空気内部に圧力波として伝播する。

### 7.2 体積速度 vs 音圧

各点の音圧は速度ポテンシャル経由で:

$$p(x, y, z, t) = -\rho_0 \frac{\partial \phi}{\partial t}$$

板からの音波は **重ね合わせ**:

$$p_{far}(\vec{r}) = \frac{j \omega \rho_0}{2 \pi r} \int_S v(\vec{r'}) e^{-jk|\vec{r} - \vec{r'}|} dS$$

これは板上の各点がイメージング源として加算される (Rayleigh integral)。

### 7.3 双極子と単極子のキャンセル

板が屈曲振動すると、板の前面・後面で逆位相の音波が出る:

```
[共鳴板]
   |       前面: 圧縮
   v       後面: 膨張
   |
=========  板  =========
   |
   ^       
   |       両面が独立に放射
```

**低周波** (波長 >> 板サイズ):
- 前後の音波が空気中で回り込み、相殺
- 双極子放射、$\eta_{rad} \to 0$

**高周波** (波長 << 板サイズ):
- 前後で干渉せず、独立放射
- 単極子的、$\eta_{rad} \to 1$

ここで「板サイズ」は実際は **モード形状の節間距離** (= 半波長 $\pi/k$):
- $k_{plate} > k_{air}$: 双極子優勢、$\eta_{rad}$ 低
- $k_{plate} < k_{air}$: 単極子優勢、$\eta_{rad}$ 高

臨界周波数 $f_c$ は両者が一致する点 (§5.4)。

### 7.4 Gamma 理論の流体力学的解釈

Gamma 理論 (内部参照: 0430b):

$$\Gamma = \frac{\eta \cdot \rho_{mat} \cdot h}{\rho_{air} \cdot c}$$

これは「**板単位面積の音響インピーダンス比**」と解釈できる:

- 分子: 板の機械損失 + 単位面積質量
- 分母: 空気の特性インピーダンス

$\Gamma$ が大きい = 板が「重い」 -> 空気を効率的に押せない
$\Gamma$ が小さい = 板が「軽い」 -> 空気を効率的に押す

ピアニカv3 (Gamma = 1.8):
- $\eta = 0.05$, $\rho h = 0.58$ kg/m^2, $\rho_{air} c = 415$ kg/m^2·s
- $\Gamma = 0.05 \times 0.58 / 415 = 1.8 \times 10^{-4}$ (s^{-1})

無次元化のために $\omega$ を含めるべきだが、典型周波数 1 kHz で:
$\Gamma_{normalized} = 1.8$ (中域での値)

### 7.5 板の境界層 (粘性損失)

共鳴板表面では空気が粘性で板に追従する境界層が形成される。境界層厚さ:

$$\delta_{viscous} = \sqrt{\frac{2 \nu}{\omega}}$$

$\nu = 1.5 \times 10^{-5}$ m^2/s (空気)、$\omega = 2\pi \times 1000$ Hz:

$$\delta = \sqrt{\frac{3 \times 10^{-5}}{6283}} = 6.9 \times 10^{-5} \text{ m} = 0.069 \text{ mm}$$

板厚 (2 mm) に対し境界層は薄く、**バルク放射が支配的**。境界層効果は周辺壁面のみ重要。

### 7.6 開口部 (Helmholtz ポート) の流体力学

楽器のサウンドホールやf字孔での空気流れ:

**線形領域** (小振幅):
- 正弦波振動、Helmholtz共鳴

**非線形領域** (大振幅):
- 渦剥離 (vortex shedding)
- 開口端で渦が形成、エネルギー損失
- これが **大音量での音色変化** (ギターの大音量歪み)

レイノルズ数による判定:

$$Re = \frac{v_{port} \cdot d_{port}}{\nu}$$

ピアニカv3 (現設計、Helmholtz効果は小):
- $v_{port}$ < 1 m/s, $d = 0.01$ m -> $Re < 700$ (層流)
- 線形領域、渦は出ない

ギター (大音量演奏):
- $v_{port}$ ~ 5 m/s, $d = 0.09$ m -> $Re = 30,000$ (乱流)
- 渦剥離発生、サウンドホール「鳴き」

### 7.7 板の屈曲波とせん断波

板の振動は厳密には屈曲波 + せん断波 + 圧縮波の重畳。Kirchhoff板理論 (薄板) では屈曲波のみ扱うが、厚板 (Mindlin理論) ではせん断波が無視できない。

ピアニカv3 桐板 (h/a = 0.002/0.3 = 0.0067) は薄板の領域 (h/a < 0.01) で、Kirchhoff理論で十分。

### 7.8 板から音波への変換効率

板の振動エネルギーが音波に変換される割合:

$$\eta_{rad} = \frac{P_{radiated}}{P_{vibration}}$$

楽器全体での値:
- バイオリン: 30 - 40% (低域)、60% (高域)
- ピアノ: 35 - 45% (中域)
- ギター: 20 - 30% (中域)
- ピアニカv3: 36% (中域、設計値)

**残り (60-70%) は板内部に蓄積され、結果的に内部摩擦で熱化**。これが共鳴箱の主損失 (§8で詳述)。

## 8. 熱力学的観点 -- なぜ共鳴箱は熱に逃げないのか

### 8.1 エネルギー流路の対比

スピーカ (本前作 §6 から):

```
電気エネルギー 100%
  +-- 90-99% -> Joule熱 (コイル抵抗)
  +-- 1-5%   -> 機械損失 (粘性)
  +-- 0.5-5% -> 音響放射
```

共鳴箱:

```
機械エネルギー 100%
  +-- 30-40% -> 音響放射 (主)
  +-- 30-40% -> 板内部の屈曲振動エネルギー (蓄積)
  +-- 20-30% -> 板内部摩擦 -> 熱 (微小)
  +-- 0%     -> 電気的損失 (なし)
```

**熱化率は 20-30%** (スピーカの 90% に対し圧倒的に低い)。これが効率差の本質。

### 8.2 内部摩擦の物理

木材の内部摩擦 (内部減衰) は粘弾性に由来する:

$$\eta = \frac{E_{loss}}{E_{stored}}$$

$E_{loss}$: 1サイクル中に失われるエネルギー (熱化)
$E_{stored}$: 1サイクル中に蓄えられるエネルギー (弾性)

桐の $\eta = 0.05$ -> 1サイクルで 5% 損失 -> 20サイクルで $1 - (1-0.05)^{20} = 64\%$ が熱化。

しかし楽器演奏中は **連続的に新しいエネルギーが入力** されるため、定常的には:

$$P_{heat} = \eta \cdot \omega \cdot E_{stored}$$

板の振動エネルギー (1J想定) で 1kHz演奏:
$P_{heat} = 0.05 \times 6283 \times 1 = 314$ W

これは大きいように見えるが、振動エネルギー1J自体が極めて大きく、現実は $E_{stored} \sim 10^{-4}$ J程度なので $P_{heat} \sim 30$ mW。

### 8.3 温度上昇のスケール

共鳴板の温度上昇推定:

板質量 $m = \rho \cdot S \cdot h = 290 \times 0.09 \times 0.002 = 0.052$ kg
比熱 $c_w = 1700$ J/kg·K (木材)
熱容量 $C = m c_w = 88$ J/K

入力 30 mW を熱化、空気・接触で同量を放熱と仮定すると、定常で:

$$\Delta T = \frac{P_{heat}}{h_{conv} \cdot S}$$

$h_{conv}$ ~ 5 W/m^2·K (自然対流)

$$\Delta T = \frac{0.030}{5 \times 0.09} = 0.067 \text{ K}$$

**0.07度の温度上昇**。実質感じられない。

つまり共鳴箱は **演奏中もほぼ室温** で動作する。これがスピーカ (大電力時にコイル温度 +100K以上) との対照。

### 8.4 「共鳴箱は冷たい」の意味

赤外線サーモグラフィで共鳴箱を撮影すると、演奏中も温度が上がらない (誤差範囲)。スピーカ (特に大入力時) は熱画像で明らかに発光。

**この物理的事実**:
- 共鳴箱: 入力エネルギー -> 大半が音 (30-40%) + 板内部の振動 (50-60%) -> 振動の一部が緩慢に熱化
- スピーカ: 入力エネルギー -> 大半が即座に熱 (90%以上) + 一部が音

スピーカは本質的に「ヒーター」、共鳴箱は「振動器」。

### 8.5 室温との等温過程

共鳴箱の温度上昇が小さいため、内部空気の状態方程式は等温で扱える:

$$pV = nRT, \quad T = const$$

スピーカエンクロージャ内では大入力時に空気が温度上昇し、静圧変動が起こる (前作 §6.2)。共鳴箱内ではほぼ等温で、純粋な機械的圧縮のみ考慮すればよい。

これが共鳴箱の **静的安定性** に貢献している。100年保管しても物理特性が変わらない (温度サイクルがないため)。

### 8.6 経時変化の遅さ

熱応力がほぼ無いため、共鳴箱の経時変化は化学的・物理的緩和のみ:
- 木材の繊維リラックス (50-100年)
- 接着剤の老化 (10-30年)
- 漆/ニス層の硬化 (50-200年)

スピーカ (電解コンデンサ寿命 10-15年、エッジ加水分解 5-15年) より圧倒的に長寿。

ストラディバリ300年現役、Steinwayピアノ100年現役は、この熱化抑制の帰結である。

### 8.7 環境条件の影響

共鳴箱は熱変動 (季節、室温) より **湿度変動** に敏感:
- 木材含水率変化で剛性 ($E$) が変動
- 含水率1% 増 -> 剛性 5% 低下、共振周波数下がる
- バイオリンは演奏前の調律で補償

ピアニカv3 でも同様。**温度より湿度管理** が長期保守の鍵となる。

### 8.8 熱力学第二法則との関係

共鳴箱は熱力学第二法則に違反しない:

- エネルギー入力: 機械 (奏者の労力)
- エネルギー出力: 音波 + 微小熱
- 全体としてエントロピー増加 (奏者の代謝熱を含めて)

しかし **局所的には** 共鳴箱は「ほぼ冷たい変換器」として動作する。これは:
- 入力エネルギーが既に整列済み (機械的な振動)
- 出力エネルギーが整列度の低い空気波 (拡散)
- 整列度差が小さい -> エントロピー生成が少ない -> 熱化少ない

スピーカは入力エネルギー (電気の整列度高い) と出力 (空気波) の整列度差が大きく、整列度の劣化が大量の熱を生む。

これが **「電気を介さない直接変換」が高効率である** 物理的根拠である。

## 10. 各楽器への含意

### 10.1 バイオリン -- Cremerの理論

Cremer (1984, *The Physics of the Violin*) の到達点:

- 駒結合 T = 2%、表板 S = 0.035 m^2、$f_{11}$ = 460 Hz
- Gamma = 2.0、放射効率 33%
- 高Q値 (50 - 100) -> 多重ピーク -> 強烈な個性
- 表板 (スプルース) + 裏板 (楓) で異素材の組合せ
- f字孔 + 内部空気 (Helmholtz 280Hz) で低域強化

**バイオリンの音色形成**:
- 弓-弦相互作用 (Helmholtz motion) -> 鋸歯波スペクトル
- 表板モード重畳 -> フォルマント形成
- 個体ごとの板特性 -> 個性

**400年間の進化**:
- 1500年代 ヴィオラ・ダ・ガンバ (前身)
- 1600年代 アマティ (現代型完成)
- 1700年代 ストラディバリ・グァルネリ (頂点)
- 以降は基本設計変わらず -> 物理的最適点に到達

### 10.2 ピアノ -- Fletcher-Rossing

Fletcher & Rossing (1998, *Physics of Musical Instruments*):

- 響板 S = 2 m^2、厚さ 8 mm 中央 -> 5 mm 外周
- 88鍵分の弦からの入力を駒で集約
- $f_{11}$ = 45 Hz (低域から)
- 放射効率 40% (中域)、低域は弱い

**ピアノ独特の物理**:
- 打弦による衝撃励振 (impulse)
- 弦の非調和性 (鋼のヤング率有限) -> 倍音が整数比からズレる
- インハーモニシティが音色の特徴 ("piano stretch tuning")
- 響板の経年変化 (クラウン低下) で音色変化

**Steinway vs ヤマハの設計差**:
- Steinway: クラウン強い、温かい音 (ニューヨーク派 vs ハンブルク派でも違う)
- ヤマハ: 制御性高い、明快な音

これら違いも結局 **共鳴板の設計差** に帰する。

### 10.3 ピアニカv3 -- 19世紀物理学の実装実験

ピアニカv3 (本プロジェクト) の位置付け:

- 桐 1.5-2 mm、$S$ = 0.09 m^2、$f_{11}$ = 96 Hz
- Gamma = 1.8、放射効率 36%
- T = 97% (連続励振の利点を最大化)
- 多重モード ($f_{11}$ から $f_{33}$ まで楽音帯域) でフォルマント

**バイオリンとの類似性** (0430f より):
- 同じ Gamma スケール ($\sim 2$)
- 同じ放射効率スケール (35%前後)
- 連続励振 (バイオリンは弓、ピアニカはリード)

**バイオリンとの差**:
- 連続励振の質: 弓は奏者制御、リードは流体駆動
- 放射体の境界条件: バイオリンは魂柱で表裏結合、ピアニカv3は片面駆動
- 個性の起源: 木材選択 vs 桐の年輪

**プロジェクトの意義**:
ピアニカv3 は「19世紀末までに完成した楽器音響学を、現代の素材・工具で再現する実装実験」と位置付けられる (0501b ジャーナル)。

ストラディバリの製作技法は失われたが、**物理は残っている**。物理を理解し、現代素材 (桐 + アクリル接着剤 + 加工機械) で再構築するのが本プロジェクトの研究的意義である。

### 10.4 アコーディオン・ハーモニカ -- フリーリード仲間

ピアニカv3 と最も近い楽器:

- **ハーモニカ**: 共鳴箱なし or 小さい -> 効率 5-10%、音圧低い
- **アコーディオン**: ふいご + リード + 大きな共鳴箱 -> 効率 20-30%

## Appendix A: 主要数式まとめ (共鳴箱主体)

### 共鳴板の物理

$$\text{板の屈曲方程式 (Kirchhoff-Love):} \quad D \nabla^4 w + \rho h \frac{\partial^2 w}{\partial t^2} = q$$

$$\text{屈曲剛性:} \quad D = \frac{E h^3}{12(1-\nu^2)}$$

$$\text{固有モード周波数:} \quad f_{mn} = \frac{\pi}{2}\sqrt{\frac{D}{\rho h}}\left[\left(\frac{m}{a}\right)^2 + \left(\frac{n}{b}\right)^2\right]$$

$$\text{曲げ波速度:} \quad c_b = \sqrt[4]{\frac{D}{\rho h}} \cdot \sqrt{\omega}$$

$$\text{臨界周波数:} \quad f_c = \frac{c_{air}^2}{2\pi h}\sqrt{\frac{12\rho(1-\nu^2)}{E}}$$

### 駒の結合

$$\text{結合効率:} \quad T = \frac{4 Z_1 Z_2}{(Z_1 + Z_2)^2}$$

$$\text{時間定数 (弦楽器):} \quad \tau_{decay} = \frac{2 m_{string}}{T \cdot Z_{bridge}}$$

### 放射効率

$$\text{Gamma 理論:} \quad \Gamma = \frac{\eta \rho_{mat} h}{\rho_{air} c}$$

$$\text{放射効率:} \quad \eta_{rad} = \frac{1}{1 + \Gamma}$$

$$\text{放射インピーダンス (低周波 ka << 1):} \quad Z_{rad} \approx \rho_0 c S \cdot \frac{(ka)^2}{2}$$

### Helmholtz共鳴

$$\text{共鳴周波数:} \quad f_H = \frac{c}{2\pi}\sqrt{\frac{S_{port}}{V \cdot l_{eff}}}$$

$$\text{実効ポート長:} \quad l_{eff} = l + 1.7\sqrt{S_{port}/\pi}$$

### モード密度 (高周波)

$$n(\omega) = \frac{S}{4\pi}\sqrt{\frac{\rho h}{D}}$$

### Q値とパワー流れ

$$Q = \frac{\omega \cdot E_{stored}}{P_{loss}}$$

$$P_{radiated} = |F|^2 \cdot \frac{Re(Z_{rad})}{|Z_{total}|^2}$$

### 比較: スピーカの主要数式 (前作 Appendix Aより)

$$\text{Lorentz力:} \quad F = Bli$$

$$\text{機械共振:} \quad f_s = \frac{1}{2\pi\sqrt{m C_m}}$$

$$\text{効率:} \quad \eta = \frac{B^2 l^2 \rho_0 S_d^2}{2\pi c R_e m^2}$$

$$\text{Hofmann鉄則:} \quad \eta \propto V_b \cdot f_3^3$$

両系の対比:
- 共鳴箱: $\eta_{rad} = 1/(1+\Gamma)$、素材決定論
- スピーカ: $\eta = B^2 l^2 \rho_0 S_d^2 / ...$ 、磁石・コイル・コーン全体

## Appendix B: ピアニカv3 共鳴箱設計の思考過程具体化

前作 Appendix B はスピーカ設計を扱った。本Appendix は対称的に **共鳴箱設計の思考過程** を扱う。設計対象は同じ用途 (家庭の8畳礼拝室 + パイプオルガン Trompette 8' 音色) だが、設計言語は楽器音響学に切り替わる。

### B.1 要求仕様の導出と設計目標の設定

#### B.1.1 序 -- 共鳴箱設計の上流工程

スピーカ設計と共通の方法論 (要求仕様の導出は設計の最上流) を適用する。違いは:

```
スピーカ設計の支配制約:    Hofmann鉄則 (eta ∝ V_b f_3^3)
共鳴箱設計の支配制約:      Gamma理論 (eta_rad = 1/(1+Gamma))
```

両者は数学的に独立で、設計言語そのものが違う。共鳴箱では「材料選択 -> 形状決定 -> モード配置」の流れが核心。

#### B.1.2 音響環境の解析

前作 Appendix B と同じ家庭の8畳礼拝室を対象:
- 容積 100 m^3 (典型的な家庭の小礼拝室)
- 残響時間 1.2 s
- 暗騒音 NC-30 (32 dBA)
- 聴取距離 2-6 m

**共鳴箱固有の追加考慮**:
- 聴取者は奏者の至近 (奏者自身が聴く) + 礼拝者全員 (1-6 m)
- 部屋の音響に共鳴箱が「呼応」する (吸音率の少ない部屋では響きが豊か)

#### B.1.3 音源プロファイル -- フリーリードの音響特性

ピアニカv3 はフリーリード (37枚、A4=440Hz、平均律):

- 周波数範囲: F2 (87 Hz) - C6 (1047 Hz) -- 基音
- 倍音: 第30倍音まで保持 (リード固有のスペクトル)
- 全帯域: 87 Hz - 31 kHz (大部分は不可聴、楽音的には - 15 kHz)
- ダイナミック範囲: 連続励振なので狭い (10-15 dB)

**Trompette 8' との対応**:
- パイプオルガン Trompette 8' は同じ「金管的フリーリード音色」
- 音域 C2 (65 Hz) - C7 (2093 Hz) -- ピアニカv3 より広い
- 倍音構造はほぼ同等

ピアニカv3 は Trompette 8' より高音域に偏るが、典型的な讃美歌帯域 (F3-C5) はカバー。

#### B.1.4 聴取者要求 -- 奏者と礼拝者

奏者 (足踏み演奏者): 手元 (鍵盤) と耳元 (頭部) で同時に音を聞く
- 演奏感覚 = 板からの直接音 + 部屋反射音
- 周波数バランス重要 (倍音が痩せていると演奏が萎える)

礼拝者: 数 m 離れた位置で聞く
- 基音 + 倍音のバランスが評価対象
- 文化的記憶 (パイプオルガン) との一致が重要

**音色要求 (Trompette 8' 整合)**:
- 第2倍音 -5 dB 以内
- 第3-5倍音 -10 dB 以内
- 第6-15倍音 -20 dB 程度
- 第16倍音以上 -30 dB 以下

これらが崩れると「ピアニカっぽい」or 「リコーダっぽい」音色になる。

#### B.1.5 SPL要求の導出

ピアニカv3 は連続励振で、ダイナミクスは奏者の足踏み速度のみ:
- 弱奏 (slow pumping): 70 dB SPL @ 1m
- 中奏: 80 dB SPL
- 強奏 (fast pumping): 88 dB SPL

部屋全体 (4 m半径) で 70-78 dBA 平均、これが讃美歌の合唱 (80 dBA) と調和する範囲。

**仕様**: 1m 地点で 80 dB SPL @ 中奏。これは:
$\eta_{rad} \cdot P_{vibration} = P_{acoustic}$
$0.36 \cdot P_{vibration} = 10^{(80-94)/10} = 0.04$ mW (1m球面波)
$\Rightarrow P_{vibration} \approx 0.11$ mW

リードからの機械エネルギー入力 (連続励振):
- 風量 50 L/min、圧力 100 Pa -> 力学的エネルギー入力 約 80 mW
- 有効伝達 (リード -> 板): 1-2 mW (大半は風損失)
- これが板に入って 0.36 が放射: 0.36-0.7 mW
- これが充分 (0.04 mW 必要) で大幅マージン

**仕様達成は設計余裕あり**。

### B.2 材料選択の思考過程

#### B.2.1 共鳴板素材 (桐の正当化)

候補比較:

```
比較表:
  ヨーロッパ・スプルース  : 比剛性 30,000 m^2/s^2、eta=0.03、価格 8,000円
  シトカ・スプルース      : 比剛性 26,300 m^2/s^2、eta=0.04、価格 6,000円
  桐 (Paulownia)         : 比剛性 20,700 m^2/s^2、eta=0.05、価格 5,000円
  楓 (Maple)             : 比剛性 18,300 m^2/s^2、eta=0.03、価格 7,000円
  カーボン繊維            : 比剛性 46,700 m^2/s^2、eta=0.005、価格 30,000円

判定: 桐
理由:
  - 内部損失 eta=0.05 で過剰共鳴を自然に抑える (中庸な響き)
  - Gamma 計算で最適 (eta_rad = 36%、本仕様の核)
  - 日本の伝統素材、入手・加工容易
  - 軽量で運搬性良好
  - 経年変化が緩やか (50-100年)
  - 価格適正
```

スプルースが楽器素材の標準だが、桐はピアニカv3 の **連続励振** という特殊条件で最適化されている。

#### B.2.2 リード材料 (フリーリード)

CAHAYA P-32D 互換のフリーリード:
- 素材: リン青銅 (C5210)
- スケーリング則: $f \propto h/L^2$
- $C_f = 571.1$ Hz·m (詳細はジャーナル0430d)

37枚のリードが平均律で配置。

#### B.2.3 駒材料

駒は高インピーダンスが必要:
- 楓: バイオリン駒の伝統素材
- 樫: 高密度、バイオリン駒の代替
- ABS (現状CAHAYA): 工業樹脂、適切なインピーダンス

ピアニカv3 では CAHAYA の ABS 駒を流用、桐板への結合効率を最適化。

#### B.2.4 側板材料

側板の機能: 共鳴板の境界条件、内部空気腔の確定

候補:
- 桐: 軽量で外部放射に貢献 (ピアニカv3採用)
- アルミ薄板: 軽量、剛性高、Gamma 高い
- ラワン合板: 安価、加工容易

判定: 桐 4mm
理由:
- 共鳴板と同素材で熱膨張係数一致
- 経年変化が同期
- 全壁面が放射に寄与 (Gamma 計算前提)

#### B.2.5 接着剤・仕上げ

伝統的選択:
- ニカワ (動物性接着剤): 楽器の標準、修復容易
- ニス (Cremer由来): バイオリンの仕上げ

現代的選択:
- アクリル接着剤: 強度高く、修復はやや困難
- ポリウレタンクリア: 現代の仕上げ

ピアニカv3 (試作段階): アクリル接着剤 + 無塗装 (素材音優先)
将来的な仕上げ: 桐特有の柔らかい光沢を出すための薄塗装 (検討中)

### B.3 機械的設計の思考過程

#### B.3.1 寸法決定のプロセス

```
モード周波数目標 (B.1.6 より):
  - f_11 ≈ 100 Hz (基音域強調)
  - f_22 ≈ 300 Hz (中域)
  - f_33 ≈ 800 Hz (プレゼンス)

板寸法 a × b、厚さ h で:
  f_11 = (π/2) × √(D/(ρh)) × ((1/a)² + (1/b)²)

桐: ρ = 290 kg/m³、E = 6 GPa、ν = 0.3
   D = E h³ / (12(1-ν²)) = 6e9 × h³ / 10.92 = 5.49e8 h³

f_11 = 100 Hz要求から逆算:
  100 = (π/2) × √(5.49e8 h³ / (290 h)) × (1/a² + 1/b²)
       = (π/2) × √(1.89e6 h²) × (1/a² + 1/b²)
       = 0.785 × 1375 h × (1/a² + 1/b²)
       = 1080 h × (1/a² + 1/b²)

a = b = 0.3 m, h = ?
  100 = 1080 h × (2/0.09) = 24000 h
  h = 0.00417 m = 4.17 mm  -- 厚すぎ

a = b = 0.4 m, h = ?
  100 = 1080 h × (2/0.16) = 13500 h
  h = 7.4 mm  -- もっと厚い

逆算: h = 2 mm (実装寸法) → a = b = ?
  100 = 1080 × 0.002 × (2/a²) = 4.32/a²
  a² = 0.0432 → a = 0.208 m = 208 mm

修正: f_11 = 96 Hz実測ベース → a = ?
  96 = 1080 × 0.002 × (2/a²)
  a² = 0.045 → a = 0.212 m
```

実装寸法 250-300 mm 角と理論最適 210 mm の差は、3辺クランプ + 1辺自由境界条件の補正。

#### B.3.2 重量の最適化

板質量 = $\rho \cdot S \cdot h$ = 290 x 0.09 x 0.002 = 52 g
側板 + 駒 + リード等 = 約 1 kg

合計 約 1 kg (内部のみ)。外装含めて 8 kg は余裕。

#### B.3.3 構造剛性

桐は薄いので **X-brace** で曲げ剛性を補強:
- 共鳴特性を壊さず、構造強度を上げる
- 楽器音響の伝統 (ギターの X-brace と同じ)
- ピアニカv3 では中央交差で1組

これにより板の経時的な歪を防止。

### B.4 流体・空気力学的設計の思考過程

#### B.4.1 風箱とふいご

ふいご (足踏み): 風圧 100 Pa一定、流量 50 L/min

風箱の設計:
- 容積: 約 5 L (リード37個の同時鳴動に必要)
- 内面積: リードプレート + 駒経由で板へ

#### B.4.2 リード前後の圧力差

リードはチャネル内の圧力差で振動:
- $\Delta P = 100$ Pa (足踏み運用時)
- リード長 5-30 mm (37個)
- 振動エネルギー = $\Delta P \times \Delta V$

#### B.4.3 空気腔 (共鳴箱内部)

容積 14.2 L (v3)、開口は小さい (ABS底面の溝程度)。

Helmholtz共鳴は副次的:
- 主役: 桐板放射
- Helmholtz は低域強化のサプリ的役割

将来 Helmholtz tuning を導入する場合、開口サイズ調整で:
$f_H = c/(2\pi) \cdot \sqrt{S/(V l)}$
$f_H = 100$ Hz狙い → $S \cdot l$ 比から逆算可能

### B.5 音響的設計の思考過程

#### B.5.1 Gamma 1.8 の達成

設計値:
$\Gamma = \eta \rho h / (\rho_{air} c)$
$\Gamma = 0.05 \times 290 \times 0.002 / (1.2 \times 343)$
$\Gamma = 0.029 / 411.6$
$\Gamma_{normalized} = 1.8$ (周波数1kHzで)

放射効率: $\eta_{rad} = 1/(1+1.8) = 0.357 \approx 36\%$

これは設計値 (要実機検証)。

#### B.5.2 モード配置の戦略

Trompette 8' フォルマント目標 (B.1.6) と桐板モード:

```
目標:                             実装:
  200-400 Hz (基音温かさ)    f_11 = 96 Hz、f_21 = 240 Hz、f_22 = 287 Hz
  800-1200 Hz (中域輝き)     f_31 = 480 Hz、f_32 = 580 Hz
  2-4 kHz (プレゼンス)       f_42, f_44, ... 多数の高次モード
  5-8 kHz (エッジ)           臨界周波数 6800 Hz と一致
```

**80% カバー** = 設計成功の指標。

#### B.5.3 駒位置の最適化

駒位置を $(x_0, y_0)$ にすると、励振が各モードに分配される:

$$F_{mn} = F_{input} \cdot \sin(m\pi x_0/a) \cdot \sin(n\pi y_0/b)$$

理想は **どのモードも均等に励振** すること。これを達成するには $x_0/a \approx 1/3, 2/3$ を避け、$1/4$ 付近に。

ピアニカv3 では駒位置をリードプレートに合わせて配置 (構造制約)、これが結果として黄金比近似 ($y/b \approx 0.382$、(1,1)モードの 11/12) で良好な励振を実現。

#### B.5.4 高次モードの活用

板は無数のモードを持つが、楽音帯域に入る数は限定:
- 100-1000 Hz: 約 17 モード
- 1000-5000 Hz: 約 70 モード

これら全てが寄与してフォルマントを構成。**多重モード重畳がスピーカの「フラット応答」とは別物の音色** を作る。

### B.6 統合とトレードオフ

共鳴箱設計の三大トレードオフ:

```
1. 板を厚く (剛性大、低域モード上がる、Gamma 大)
   vs
   板を薄く (柔軟、低域モード下がる、Gamma 小、放射効率高)

2. 板を大きく (低域強化、運搬性悪化)
   vs
   板を小さく (運搬性良好、低域弱体化)

3. 内部損失高 (過剰共鳴抑制、Gamma 大、放射効率低)
   vs
   内部損失低 (鋭いピーク、Gamma 小、放射効率高)
```

ピアニカv3 の選択:
- 薄め (h = 2 mm) -> 低域 100 Hz クリア
- 中サイズ (a = b = 0.3 m) -> 運搬可能
- 中庸の損失 (eta = 0.05) -> 過剰でも不足でもない

これら3要素のバランスが **桐 1.5-2 mm + 300 x 300 mm の最適点** を導く。スピーカで言う Hofmann 鉄則の共鳴箱版。

## Appendix C: 主聖堂規模への思考実験 -- 共鳴箱パラダイムの物理限界

前作 Appendix C は神戸中央教会主聖堂 (4,500 m^3、300人) のSRシステム設計を扱った。本Appendix は対称的に **「もし共鳴箱だけで主聖堂を満たせるか?」** という思考実験を行い、**物理的不可能性を証明** する。これが共鳴箱パラダイムの適用範囲を画定する。

### C.0 序 -- 共鳴箱で主聖堂を満たせるか

主聖堂の音響要求 (前作 Appendix C より):
- 容積 4,500 m^3
- 聴衆 300人 (ピーク)
- 最遠聴取距離 25 m
- 残響 3.3 s (満席)
- 必要 SPL: 90 dBA at 全聴衆位置 (平均)、95 dBA ピーク

共鳴箱で同じ用途を実現するには? 物理的に何が起こるか?

### C.1 共鳴箱を「主聖堂サイズ」にスケールアップ

仮想的に共鳴箱の各次元を巨大化する:

```
基準: ピアニカv3 (0.3 m × 0.3 m × 2 mm 板、放射 36%)
仮想: ピアニカ巨大 (例 3 m × 3 m × 20 mm 板)

スケーリング:
  寸法 a, b: 10倍
  厚さ h: 10倍 (板の比率維持)
  面積 S: 100倍
  質量 m: 1000倍
  Gamma: 不変 (素材依存、寸法に依存しない)
  放射効率 eta_rad: 不変 (約36%)
```

そのまま放射効率は維持されると仮定。

### C.2 物理限界の解析

#### C.2.1 励振エネルギーの限界

共鳴箱の音響出力:

$$P_{acoustic} = \eta_{rad} \cdot P_{vibration}$$

$P_{vibration}$ = 板の振動エネルギー、これは励振源 (リード等) からの入力に等しい (定常)。

ピアニカv3 (足踏み演奏):
- 入力 80 mW (足踏みエネルギー、有効伝達)
- 板への伝達: 1-2 mW
- 放射 (36%): 約 0.5 mW

主聖堂 (90 dBA at 全聴衆位置 = 25 m半径領域):

放射音響パワー (聴衆位置で 90 dBA = 1 mW/m^2、エネルギー流):
$$P_{required} = 1 \text{ mW/m}^2 \times 4\pi \times r^2$$

最遠 25m での球面: $4\pi \times 625 = 7854$ m^2
$P_{required} = 1 \times 7854 = 7.85$ W (10倍マージン込み)

部屋の残響 3.3s で実際は1/3でいい:
$P_{required} \approx 2.6$ W (連続音響パワー)

**共鳴箱で 2.6W 出すには**:
- 入力機械エネルギー = 2.6 / 0.36 = 7.2 W (連続)
- これを足踏みで出す? -> 人体筋力では困難
- 人体最大連続労力: 約 100W (機械)、有効伝達 10% -> 10W
- ぎりぎり可能だが、奏者は数分でバテる

**結論**: 励振源を増やせばOK。リード数を 37枚 -> 1000枚にすれば10W入力可能。しかし...

#### C.2.2 板表面積の物理限界

$\eta_{rad}$ = 36% は **小規模板** ($S$ < 0.1 m^2) で成立する近似。スケールアップで:

**(1) 低周波 (基音帯域、100 Hz)**:
- 板の波長 $c_b/f$ vs 板サイズ
- 大型板 (3 m) では波長より大きく、双極子相殺は起こらない -> 良い

**(2) 中周波 (1 kHz)**:
- 板上に多数の節 (mode (10,10) 以上)
- 節と節の間で位相反転 -> 自己相殺 (self-cancellation)
- 大型板の中域 $\eta_{rad}$ は **逆に低下**

**(3) 高周波 (10 kHz)**:
- 板の臨界周波数 ($f_c$) 付近で振動損失大
- 大型板ではこれが下がる ($f_c \propto 1/h$)、しかし相対的な放射は維持

**結論**: 巨大板は **モード密度が高すぎ、自己相殺で実効効率が下がる**。実測例:
- 大型ピアノ響板 (2 m^2): $\eta_{rad}$ = 35-40%
- 仮想 9 m^2 板: 推定 $\eta_{rad}$ = 15-20% (自己相殺で半減)

スケールアップしても効率は線形ではない。

#### C.2.3 音色 (フォルマント) の崩壊

巨大共鳴板はモード密度 $n(\omega)$ が大きく:

$$n(\omega) = \frac{S}{4\pi}\sqrt{\frac{\rho h}{D}}$$

100倍の面積なら 100倍のモード密度。1Hzあたり 17 モードから 1700 モード。

これは:
- フォルマントが平均化されてフラット応答に近づく
- 楽器の個性が失われ、「機械的な音」になる
- そもそも共鳴箱の魅力 (多重ピーク = 個性) が消失

**結論**: 巨大共鳴箱は「巨大スピーカ」として動作するが、楽器音色は失う。

#### C.2.4 構造強度の限界

3 m × 3 m × 20 mm の桐板の質量:
$m = 290 \times 9 \times 0.02 = 52.2$ kg

これを支える側板・X-brace の重量:
- 側板 (3 m x 3 m x 0.5 m箱の4面 = 36 m^2 x 4 mm 桐): 41 kg
- X-brace (重量 5 kg)

合計 約 100 kg。これを駒/共鳴板から励振するエネルギー:
- 共振振幅 100 mm レベルが必要 (低域モードを十分鳴らす)
- 加速度 = $\omega^2 \cdot a$ = $(2\pi \cdot 100)^2 \cdot 0.1$ = 39,500 m/s^2 = 4030 G
- 板にかかる力 = $m \cdot a$ = $52.2 \cdot 39,500$ = **2,000,000 N (200トン)**

これは桐板の引張強度 (約 50 MPa) を桁違いに超える。**板が割れる**。

#### C.2.5 励振源の現実

仮想的に1000リードで10W入力を実現したとして、各リード10mWの入力が必要。

ピアニカv3 のリードは 1-2 mW入力。10倍は無理ではないが、リードが大型化、構造が複雑になる。さらに駒/共鳴板への伝達効率が95% -> 50%以下に落ちる (大型化での結合損失)。

**結論**: 励振源を増やすと結合効率が下がるトレードオフ。線形スケールでない。

### D.0 序 -- なぜ「魂柱なし」を独立に検討するか

```

   なぜ魂柱を導入しないか? その物理的帰結は何か?
   バリエーション設計として、Appendix B と並列に位置付ける
```

**結論先取り**: ピアニカv3 は **連続励振** (リードによる空気駆動) と **片面駆動** (表板単独) を組み合わせた設計であり、これは **バイオリン系 (双面 + 魂柱) ではなく、ピアノ響板系 (単面)** に近い。本Appendix はこれを物理的に正当化する。

### D.1 魂柱の物理的役割 -- 楽器ごとの位置付け

#### D.1.1 バイオリンの魂柱 (Sound Post)

バイオリンにおける魂柱は **楽器の魂** と呼ばれるほど重要な部材:

```
魂柱の物理的構成:
  位置:  表板と裏板の間、駒の右足下より少し離れた位置
  寸法:  直径 6mm、長さは表板-裏板の間隔に正確に合わせる
  素材:  スプルース (表板と同じ繊維方向)
  固定:  接着なし、両端の摩擦と圧力のみで保持
  調整:  製作後も奏者が微調整可能 (1mm単位で音色変化)
```

魂柱の機能:

1. **表板-裏板の機械的結合**: 表板の振動を裏板に伝達
2. **インピーダンス整合**: 異種素材 (スプルース表板 + 楓裏板) の振動結合
3. **モード形状の制御**: 表板の振動パターンを非対称化 (G線側とE線側で異なる応答)
4. **双極子放射の改善**: 表裏が同期振動 -> 単極子的放射成分が増加
5. **音色の決定**: あるなしで「芯のある音」vs「ぼんやりした音」が変わる

#### D.1.2 楽器ごとの「魂柱」の有無

魂柱に類する部材を持つ楽器・持たない楽器:

| 楽器 | 表裏結合の有無 | 結合方法 | 設計思想 |
|------|--------------|---------|---------|
| バイオリン | 有 (魂柱) | 摩擦式丸棒 | 表裏同期、双極子改善 |
| チェロ | 有 (魂柱) | 同上、寸法大 | 同上 |
| コントラバス | 有 (魂柱) | 同上、寸法大 | 同上 |
| ヴィオラ | 有 (魂柱) | 同上 | 同上 |
| ガンバ系 | 有 | 同上 | 同上 |
| **ギター** | **無** | (側板のみ) | **表板単独放射** |
| **マンドリン** | **無** | (側板のみ) | 同上 |
| **ピアノ (グランド)** | **無** | (背面開放) | **響板単独、背面は空間** |
| **ピアノ (アップライト)** | **無** | (背面壁) | 響板単独、背面は壁面 |
| **チェンバロ** | **無** | (側板のみ) | 表板 + 内部空気 |
| **ハープシコード** | **無** | (側板のみ) | 同上 |

**重要観察**:
- 弦楽器のうち **擦弦楽器 (バイオリン族) のみ魂柱を持つ**
- 撥弦・打弦楽器 (ギター、ピアノ、チェンバロ等) は魂柱を持たない
- フリーリード楽器 (アコーディオン、ピアニカ系) もすべて持たない

#### D.1.3 「魂柱を持つ条件」の物理的解釈

魂柱を持つのはバイオリン族だけ。なぜか?

仮説: **励振の時間特性** が魂柱の要否を決定する。

```
擦弦 (continuous bowing):
  特性: 持続音、短時間で安定状態
  問題: 表板の局所振動が安定する前にエネルギーが減衰する可能性
  解決: 魂柱で表裏結合 -> 大きな振動系 (表板 + 裏板)
        慣性大、減衰時定数長
        持続音が安定する

撥弦・打弦 (impulse excitation):
  特性: インパルス + 自由減衰
  問題: 急速な減衰は楽器の特徴 (音価が決まる)
  解決: 表板単独で減衰させる (魂柱不要)
        サスティン制御は弦の質量と張力で

連続励振 (free reed):
  特性: 連続持続、空気駆動で自己安定
  問題: なし、励振が連続的なので自己安定する
  解決: 表板単独で十分
        魂柱は不要、むしろ自由振動を阻害
```

**ピアニカv3 はフリーリード**:
- リードからの励振は連続的
- 表板の振動は連続的にエネルギー補給される
- 魂柱で表裏結合する **必要がない**
- むしろ表板の自由振動を維持した方が音色豊か

これがピアニカv3 が魂柱を持たない物理的根拠。

### D.2 ピアニカv3 が魂柱を持たない物理的根拠

#### D.2.1 連続励振による自己安定性

リードによる連続励振の特徴:

$$F(t) = F_0 \sin(\omega t) + \text{higher harmonics}$$

これが駒経由で表板に常時入力される。表板の運動方程式:

$$m_s \ddot{w} + R_s \dot{w} + K_s w = F(t)$$

定常解 (連続励振):

$$w(t) = \frac{F_0}{Z(\omega)} \sin(\omega t - \phi)$$

ここで $Z(\omega)$ は表板の機械インピーダンス。

**魂柱を入れた場合の変化**:

魂柱で表板に追加質量 $m_{post}$ + 追加抵抗 $R_{post}$ + 裏板側結合 $K_{back}$ が加わる:

$$Z'(\omega) = Z(\omega) + j\omega m_{post} + R_{post} + \frac{K_{back}}{j\omega}$$

連続励振では $|F_0|$ が一定なので、$|Z|$ が増える -> 振幅 $|w|$ が減る。これは音圧低下を意味する。

**結論**: 連続励振では魂柱は **音圧を下げる方向** に作用する。バイオリンの場合は擦弦の不安定性を魂柱で安定化する利点が音圧低下を上回るが、ピアニカv3 では励振が既に自己安定的なので、魂柱の追加は不利。

#### D.2.2 構造的に表裏結合が不要な配置

ピアニカv3 の構造:

```
[空気箱 (リード列)]
      |
      v (リードプレートから駒経由)
[桐 表板 1.5-2mm]  <-- 主放射体
      |
[内部空気 14.2 L]
      |
[桐 底板 4mm]      <-- 構造支持、副放射
      |
[側板 + 内部構造]
```

**バイオリンとの構造比較**:

```
[バイオリン]                    [ピアニカv3]
  弦                              リード
  |                               |
  駒                              駒
  |  (魂柱で連結)                |
  +-表板  +-裏板                  +-表板
            |                     |
          側板                  側板 + 底板
```

バイオリンは表板と裏板を **同等の振動体** として設計する。両方が放射に寄与し、魂柱で同期させる。

ピアニカv3 は表板を主放射体、底板を構造支持として **役割分担** する。両者を結合する魂柱は不要。

#### D.2.3 桐板の振動特性

桐 1.5-2mm 表板の振動モード (本書 §5.2):
- $f_{11}$ = 96 Hz、$f_{22}$ = 287 Hz、$f_{33}$ = 768 Hz
- 高Q値 (30-60)、多重ピーク

これらが **魂柱なしで自由振動** することで:
- モード形状が綺麗 (拘束点なし)
- フォルマント特性が清浄
- 製作個体差が小さい (魂柱位置の精密調整不要)

魂柱を入れると:
- モード形状が魂柱位置で歪む
- 高次モードが乱れる
- 個体差が大きくなる (魂柱位置1mm差で音色変化)

これは **量産性** を考慮した設計選択でもある。

### D.3 表板単独振動の物理

#### D.3.1 単一面板の運動方程式

表板単独 (魂柱なし) の運動は §6.1 の Kirchhoff-Love板理論で記述される:

$$D \nabla^4 w + \rho h \frac{\partial^2 w}{\partial t^2} = q(x, y, t)$$

境界条件:
- 4辺: 側板に接着 (クランプ条件)
- 駒位置: リードからの入力

$D = E h^3 / [12(1-\nu^2)]$ なので、桐 ($E$ = 6 GPa、$\nu$ = 0.3、$h$ = 0.002 m):

$$D = \frac{6 \times 10^9 \times 8 \times 10^{-9}}{12 \times 0.91} = 4.4 \text{ N·m}$$

#### D.3.2 表板のモード解析

クランプ境界 (4辺固定) の正方形板 ($a = b$) の主要モード:

| モード | $f_{mn}$ [Hz] (h=2mm, a=300mm) | 形状 |
|--------|------------------------------|------|
| (1,1) | 96 | 中央が腹、四辺が節 |
| (2,1)=(1,2) | 196 | 中央線で節 |
| (2,2) | 288 | 4分割 |
| (3,1)=(1,3) | 358 | 3節 |
| (3,2)=(2,3) | 484 | 6分割 |
| (3,3) | 615 | 9分割 |
| (4,1)=(1,4) | 588 | |
| (4,2)=(2,4) | 720 | |
| (4,3)=(3,4) | 851 | |
| (4,4) | 1024 | 16分割 |

**音楽帯域 50 Hz - 5 kHz** に約 80-100 個のモードが入る。これがフォルマントの源。

#### D.3.3 駒位置によるモード励振

駒位置 $(x_0, y_0)$ にすると、各モードへの励振振幅:

$$A_{mn} \propto \sin(m\pi x_0/a) \cdot \sin(n\pi y_0/b)$$

最適駒位置 (黄金比 $\approx 0.382$):
- $(x_0, y_0) = (0.382 a, 0.382 b)$
- 全モードがほぼ均等に励振される

ピアニカv3 では駒位置がリード列に合わせて決まるため、自由には選べない。実装制約と音響最適性の妥協。

#### D.3.4 表板の総合周波数応答

各モードの和:

$$|H(\omega)|^2 = \sum_{m,n} \frac{|A_{mn}|^2}{(1 - (\omega/\omega_{mn})^2)^2 + (1/Q_{mn})^2}$$

これが **多重ピークの周波数応答** (フォルマント)。連続励振の倍音スペクトルがこれにかけられて、最終放射スペクトルが決まる。

魂柱があれば各 $\omega_{mn}$ がシフトし、$Q_{mn}$ が低下 (魂柱の摩擦損失で減衰)。ピアニカv3 では魂柱なしで $Q$ を高く保ち、ピーク鮮明な音色を実現。

### D.4 双極子相殺問題と対策

#### D.4.1 表板単独放射の問題

魂柱なしで表板単独が振動する場合、表板の前面と後面で逆位相の音波が放射される (本書 §7.3):

```
[前面 (奏者・聴衆側)]: 圧縮波
       |
[表板 (振動)]
       |
[後面 (内部空気)]: 膨張波
```

低周波で波長が大きいと、前後の音波が回り込んで相殺 (双極子放射)。これが魂柱なしの最大の弱点。

**対策はバイオリンの場合**: 魂柱で表裏結合 -> 裏板も同位相で放射 -> 単極子的に音が広がる -> 双極子相殺改善。

**ピアニカv3 の場合 (魂柱なし)**:
- 後面音は **内部空気** に閉じ込められる
- 内部空気の Helmholtz共鳴で位相反転
- 前面音と同位相で放射 (バスレフ的)
- 結果: 双極子相殺を回避

#### D.4.2 内部空気による位相反転

ピアニカv3 内部空気 14.2 L:

Helmholtz共鳴周波数:

$$f_H = \frac{c}{2\pi}\sqrt{\frac{S_{port}}{V \cdot l_{eff}}}$$

開口面積 $S_{port}$ = 5 cm^2 (典型例)、ポート長 $l_{eff}$ = 5 cm として:

$$f_H = \frac{343}{2\pi}\sqrt{\frac{5 \times 10^{-4}}{0.0142 \times 0.05}} = 54.6 \times \sqrt{0.704} = 45.8 \text{ Hz}$$

これは表板 $f_{11}$ = 96 Hz より低い。**$f_H < f_{11}$** の関係:
- $f_H$ 以下: ポート空気が振動、表板と同位相 -> 単極子強化
- $f_H$ 〜 $f_{11}$ 間: 過渡領域、位相が180度回転
- $f_{11}$ 以上: ポート寄与小、表板単独放射

**設計の含意**: 内部空気は **低域** の双極子相殺を防ぐ機能を持つ。中高域は表板単独放射で十分。

#### D.4.3 底板の役割

桐 4mm 底板は:
- 質量大 ($\rho h$ = 1.16 kg/m^2、表板の2倍)
- 剛性大 ($D \propto h^3$、表板の8倍)
- 振動振幅小 -> 放射弱い
- 内部空気を反射 -> 表板に振動エネルギーを集中

これにより **底板は静的な「壁」** として機能し、表板に振動エネルギーが集中する。バイオリンの裏板 (動的、放射体) とは正反対の役割。

#### D.4.4 設計トレードオフ

魂柱あり vs 魂柱なし、ピアニカv3 の場合:

| 項目 | 魂柱あり (仮想) | 魂柱なし (採用) |
|------|---------------|--------------|
| 双極子相殺 | 表裏同期で改善 | 内部空気で改善 (低域のみ) |
| モード形状 | 魂柱位置で歪む | 自由振動、清浄 |
| 個体差 | 魂柱位置で大 | 小 |
| 製作難度 | 高 (魂柱調整) | 低 |
| 音色 | 「芯のある」 | 「自由な響き」 |
| 連続励振との整合 | 不利 (Z増加) | 有利 (Z適正) |
| 量産性 | 低 | 高 |

**ピアニカv3 は連続励振 + 量産前提** なので、魂柱なしが最適選択。

### D.5 内部空気との結合 -- Helmholtz活用

#### D.5.1 表板 + 内部空気の連成系

表板単独ではなく **表板 + 内部空気** の連成振動を扱う必要がある。

連成方程式 (簡略):

$$\begin{cases} m_s \ddot{w} + R_s \dot{w} + K_s w = F(t) - S_d \cdot p_{int} \\ V \dot{p}_{int} / (\rho c^2) + S_p v_p = -S_d \dot{w} \\ m_p \dot{v}_p + R_p v_p = S_p p_{int} - S_p p_{ext} \end{cases}$$

- $w$: 表板の平均変位
- $p_{int}$: 内部空気圧
- $v_p$: ポート空気速度
- $S_d$: 表板有効面積、$S_p$: ポート面積

これは表板 - 内部空気 - ポート の3自由度連成系。

#### D.5.2 連成共振の特性

連成系の特性方程式から固有周波数 (近似):

$$\omega_{1,2}^2 = \frac{1}{2}\left[(\omega_s^2 + \omega_H^2) \pm \sqrt{(\omega_s^2 - \omega_H^2)^2 + 4\alpha^2}\right]$$

ここで $\alpha$ は連成強度。$\omega_s$ (表板) と $\omega_H$ (Helmholtz) が近いと連成が強く、両者が「割れる」。

ピアニカv3 (推定値):
- $\omega_s$ (f_{11}) = 96 Hz
- $\omega_H$ = 46 Hz
- 比 = 0.48 (やや離れているが連成あり)

連成後の2モード:
- 低域: 約 38 Hz (Helmholtzが下にシフト)
- 高域: 約 116 Hz ($f_{11}$ が上にシフト)

これがバスレフ効果で、低域の放射効率を高める。

#### D.5.3 設計上の最適化

Helmholtz チューニングを意図的に設計するなら:
- $f_H$ を $f_{11}$ より少し低く設定 (バイオリン的: $f_H = 280$ Hz < $f_{11}$ = 460 Hz)
- ポート開口・長さで調整
- 過剰な連成は音色を不自然に -> 適度に

ピアニカv3 では現状 Helmholtz は副次的。将来的に意図的チューニングを検討する価値あり (本書 §4.6)。

### D.6 連続励振との整合性

#### D.6.1 持続音における魂柱の不利

連続励振 (フリーリード) で持続音を出す場合、魂柱の存在は不利:

```
擦弦 (バイオリン) - 短時間励振:
  弓-弦相互作用が瞬時的に起こり停止
  魂柱で表裏結合 -> 振動系が大きい -> 慣性大 -> 持続音延長
  これがバイオリンに必要な機能

連続励振 (リード) - 持続的励振:
  リードが空気で常時駆動される
  振動エネルギーが連続的に供給される
  慣性大は不要、むしろ応答速度を遅らせる
  魂柱は不利
```

#### D.6.2 アタック特性

リードのアタック (音の立ち上がり):
- 魂柱なし: 速い (表板の慣性のみ)
- 魂柱あり: 遅い (表板 + 裏板 + 魂柱の慣性)

楽器演奏で「アタックの明瞭さ」は重要。連続音楽 (聖歌、Trompette 8') でもアタックが甘いと演奏感が悪い。魂柱なしがアタック特性で優れる。

#### D.6.3 サスティン特性

サスティン (音の持続):
- 連続励振なので、魂柱の有無に関わらず励振が続く限り持続
- 魂柱なしでも問題なし

#### D.6.4 リリース特性

リリース (音の消え方):
- 魂柱あり: 表裏結合で減衰時定数長 -> ゆっくり消える
- 魂柱なし: 表板単独で減衰 -> 速やかに消える

教会音楽では速やかなリリースが望ましい (次の音と混ざらない)。これも魂柱なし有利。

### D.9 結論 -- ピアニカv3 はピアノ響板系設計

#### D.9.1 楽器系統樹におけるピアニカv3 の位置

魂柱の有無で楽器を分類:

```
弦楽器:
  擦弦 (魂柱あり):
    バイオリン、ヴィオラ、チェロ、コントラバス、ガンバ族
  
  撥弦・打弦 (魂柱なし):
    ギター、マンドリン、リュート -- 表板 + 内部空気
    ハープ -- 駒経由共鳴板
    ピアノ、チェンバロ -- 響板 (単面)

管楽器: 共鳴箱なし、空気柱が振動

フリーリード楽器 (魂柱なしのみ):
    ハーモニカ -- 共鳴箱小
    アコーディオン -- 共鳴箱中、ふいご駆動
    ピアニカ系 -- 共鳴箱変動
    
    ピアニカv3 -- 共鳴箱大 + 表板単面 + 内部空気
                <-- ピアノ響板系に最も近い
```

ピアニカv3 は系統樹上:
- フリーリード励振 (アコーディオン系)
- 表板単面放射 (ピアノ系)
- 内部空気活用 (ギター系)

の **3系統のハイブリッド**。これらすべてが「魂柱なし」を選択する楽器群である。

#### D.9.2 「ピアノ響板系」としての再定義

本書 §10.3 で「ピアニカv3 はバイオリン物理に最も近い」と書いた。これは Gamma 値 (放射効率パラメータ) の数値的類似性に基づく。

しかし **構造的には「ピアノ響板系」** に近い。本Appendix の議論を踏まえて整理:

| 観点 | バイオリン系 | ピアノ響板系 | ピアニカv3 |
|------|-----------|------------|-----------|
| 励振 | 擦弦 (短時間) | 打弦 (短時間) | リード (連続) |
| 表板 | 単面 | 単面 | 単面 |
| 裏板 | 振動体 | なし or 静的 | 静的 (底板) |
| 魂柱 | あり | なし | なし |
| 内部空気 | 重要 (Helmholtz) | あまり重要でない | 重要 (Helmholtz候補) |
| Gamma | 2.0 | 1.5-2.0 | 1.8 |
| 放射効率 | 33% | 35-45% | 36% |

**結論**: 数値的特性 (Gamma) はバイオリンに近いが、**構造的設計はピアノ響板系**。両者は数学的に類似だが哲学が異なる:
- バイオリン: 表裏結合 + 全体共鳴
- ピアノ: 単面振動 + 力学集約
- ピアニカv3: ピアノ流の単面振動 + 連続励振

#### D.9.3 設計の最終確認

ピアニカv3 (Appendix D 仕様) の物理的位置付け:

```
励振:    フリーリード (アコーディオン系統)
放射:    桐表板単面 + X-brace (ピアノ響板系)
共鳴:    内部空気 14.2 L (オプションでHelmholtz)
構造:    桐 1.5mm 表板 + 4mm 底板 + 5mm 側板
結合:    駒経由 T = 97%
魂柱:    なし (連続励振 + 単面駆動の最適選択)
```

これが **本プロジェクトのピアニカv3 の物理的アイデンティティ** である。

#### D.9.4 19世紀物理学との整合

ピアニカv3 設計は 19世紀末までに完成した楽器音響学 (Helmholtz、Rayleigh、Cremer) を踏まえている:

- バイオリンの魂柱の物理 (Cremer, *The Physics of the Violin*)
- ピアノ響板の動作 (Fletcher-Rossing, *Physics of Musical Instruments*)
- フリーリードの空気力学 (Tarnopolsky, Fletcher)
- Helmholtz共鳴 (Helmholtz, *On the Sensations of Tone*)

これら古典的知見の **正しい組み合わせ** がピアニカv3 設計を導く:
- 「フリーリード + 連続励振」 -> 魂柱不要
- 「単面放射 + 内部空気」 -> ピアノ響板系
- 「桐の高 Gamma特性」 -> 効率最適

魂柱を入れるかどうかは「楽器の宗派 (バイオリン系か ピアノ系か)」の選択であり、ピアニカv3 はピアノ系を採用した。これが本Appendix の核心結論である。

### E.1 部屋を共鳴箱とみなす物理

#### E.1.1 板振動モードと室内音響モードの数学的対応

板の屈曲振動 (Kirchhoff-Love、本書§6) と部屋の音響モード (3D波動方程式) は数学的に類似:

**板 (2D)**:
$$D \nabla^4 w + \rho h \frac{\partial^2 w}{\partial t^2} = 0$$

固有モード:
$$f_{mn}^{plate} = \frac{\pi}{2}\sqrt{\frac{D}{\rho h}}\left[\left(\frac{m}{a}\right)^2 + \left(\frac{n}{b}\right)^2\right]$$

**部屋空気 (3D)**:
$$\nabla^2 p - \frac{1}{c^2}\frac{\partial^2 p}{\partial t^2} = 0$$

剛壁矩形室の固有モード:
$$f_{lmn}^{room} = \frac{c}{2}\sqrt{\left(\frac{l}{L_x}\right)^2 + \left(\frac{m}{L_y}\right)^2 + \left(\frac{n}{L_z}\right)^2}$$

**対応関係**:

| 板 (本書主役) | 部屋 |
|------|------|
| 板の境界 | 壁 |
| 駒 (励振点) | 楽器/声/**スピーカ (= 魂柱)** |
| 板表面の振動 | 空気の圧力変動 |
| モード形状 (節線) | 圧力モード形状 (節面) |
| Q値 (内部摩擦) | Q値 (壁面吸音) |
| 放射効率 | 「結合効率」(壁の振動への変換) |
| **魂柱 (表裏結合)** | **スピーカ (壁駆動の励振点)** |

つまり共鳴箱の設計言語が、室内音響設計にそのまま転用できる。「**部屋は2階建ての共鳴箱、スピーカはその魂柱**」と言ってもよい。

#### E.1.2 部屋振動の3軸分類

部屋の固有モードは3軸 ($l, m, n$) のインデックスで分類:

```
軸モード (Axial mode):     1軸のみ非ゼロ
  例: (1,0,0), (0,1,0), (0,0,1)
  特性: モード密度小、ピーク鋭い、最も強い

接線モード (Tangential):   2軸非ゼロ
  例: (1,1,0), (2,1,0)
  特性: 中程度のモード密度

斜めモード (Oblique):      3軸全て非ゼロ
  例: (1,1,1), (2,1,1)
  特性: モード密度大、ピーク弱い、高周波で支配的
```

低周波で軸モードが個別に識別され、高周波では斜めモードが密集して連続スペクトルとなる。

#### E.1.3 Schroeder 周波数 -- 離散と連続の境界

部屋の音響特性が「離散的モード」から「統計的残響」に移行する境界周波数:

$$f_S = 2000 \sqrt{\frac{RT_{60}}{V}}$$

主聖堂 (満席、$RT_{60} = 3.3$ s、$V = 4500$ m^3):

$$f_S = 2000 \sqrt{\frac{3.3}{4500}} = 54 \text{ Hz}$$

**意味**:
- $f < 54$ Hz: 個別モードが識別可能、配置と励振点 (= 魂柱位置) が支配的
- $f > 54$ Hz: モード密集、統計的扱い (残響時間)

楽音帯域はほぼ全て統計的領域。魂柱配置の物理は **低周波で本質的、高周波では平均化** される。

### E.9 「魂柱=スピーカ」の物理 (本Appendix Eの核心)

#### E.9.1 魂柱の物理的役割の再考

バイオリンの魂柱 (本書§3.6 + Appendix D):
- 直径6mm丸棒、表板と裏板を機械的に結合
- 摩擦のみで保持、接着なし
- 振動を表板から裏板に伝達

**物理的本質**: **「分離した2つの振動体 (表板と裏板) を同期させる励振点」**

魂柱がない場合:
- 表板と裏板は独立に振動
- 表板の振動位相と裏板の振動位相がランダム
- 両面からの放射が相殺 (双極子) -> 効率低下

魂柱がある場合:
- 表板と裏板が同期 (位相整合)
- 両面が同位相で放射 -> 単極子的、効率向上

これを抽象化すると:
$$\text{魂柱の機能} = \text{「複数振動体の位相同期」}$$

#### E.9.2 スピーカを魂柱として捉える

主聖堂を共鳴箱として捉えたとき、その「振動体」は何か?

```
主聖堂の振動体一覧:
  - 入口側壁 (祭壇から見て後方の白煉瓦壁)
  - 祭壇側壁 (前方の白煉瓦壁)
  - 左右側壁 (白煉瓦)
  - 床 (石床)
  - 天井 (緑青銅板大屋根)
  - 内部空気 4500 m^3
```

これらは **物理的に分離した振動体**。各々が独立に振動すれば、放射の位相がバラバラになり、聴衆位置で相殺する。

**スピーカの役割を魂柱として解釈**:

スピーカは部屋に音波を放射する。この音波は:
1. 直接音として聴衆に到達
2. 各壁面に到達 -> 壁面を振動させる (極微小だが)
3. 反射波として部屋を満たす
4. 各壁面が **同位相で振動** -> 単極子的に音場形成

つまり **スピーカは「部屋の各壁面を同期させる励振点」** として動作する。これは **魂柱の機能の電気音響的実装**。

```
バイオリン:           スピーカ in 部屋:
[弓] -> [弦]          [電力] -> [アンプ]
       |                       |
      [駒]                    [ボイスコイル]
       |                       |
   [表板]                  [コーン]
       |                       |
   [魂柱] = 表裏結合      [スピーカ自体] = 壁面結合
       |                       |
   [裏板]                  [部屋全壁面] (同期駆動)
```

数学的に同型 (isomorphic):
- 機械的魂柱: 力 → 機械振動 (棒経由)
- 電気音響的魂柱: 電力 → 機械振動 (空気経由)

#### E.9.3 数学的同型性

魂柱とスピーカの伝達関数の対応:

**バイオリン魂柱**:
$$F_{back}(\omega) = T_{post}(\omega) \cdot F_{front}(\omega)$$

$T_{post}(\omega)$ は魂柱の周波数依存伝達特性 (機械インピーダンス比)。

**主聖堂スピーカ**:
$$p_{wall_i}(\omega) = T_{spk \to wall_i}(\omega) \cdot V_{spk}(\omega)$$

$T_{spk \to wall_i}(\omega)$ はスピーカから壁面 $i$ への伝達 (空気経路 + 距離)。

**両者の本質的同等性**:
- 入力: 単一駆動信号 (魂柱の場合は表板の振動、スピーカの場合は電圧)
- 出力: 複数振動体の位相整合された振動
- 物理: 線形伝達 (小振幅近似)

差異は媒体 (機械 vs 空気) のみで、**設計言語 (位相整合、伝達関数最適化) は共通**。

#### E.9.4 主聖堂SRの「魂柱配置」としての再評価

Appendix C で設計したSRシステム:
- 主アレイ x 2 (祭壇両側)
- ディレイフィル x 4 (中段、後段)
- サブウーファー x 2 (祭壇下)

これらを **魂柱配置として再解釈**:

```
SRシステム = 多重魂柱システム:

  主魂柱 (主アレイ x 2):
    バイオリンの魂柱に相当
    部屋の主モード (l,0,0)~(l,m,n) を励振
    祭壇両側 = 部屋の対称性に整合

  補助魂柱 (ディレイフィル x 4):
    バイオリンには存在しない (1本のみ)
    主聖堂の規模 (4500 m^3) では局所モード励振が必要
    距離による位相整合 = ディレイ補正

  低域魂柱 (サブウーファー x 2):
    バイオリンの「裏板共鳴」(280Hz) に相当
    祭壇下 = (l,0,1) モードの腹寄り
    床面結合で部屋全体に低域伝達
```

**重要観察**: バイオリンには魂柱が **1本のみ**。これは表板と裏板の単純な結合で済むため。主聖堂は **8つの主要振動体 (6壁 + 床 + 天井 + 空気)** があるので、複数魂柱が必要。これが多重スピーカ配置の物理的根拠。

#### E.9.5 ピアニカv3 (Appendix D) と主聖堂 (本Appendix E) の対比

両者は **「魂柱の有無」** の両極端を実装する:

| 項目 | ピアニカv3 (Appendix D) | 主聖堂 (本Appendix E) |
|------|---------------------|---------------------|
| 共鳴箱規模 | 0.0142 m^3 (14.2 L) | 4,500 m^3 (約32万倍) |
| 振動体 | 桐表板 (単面) | 部屋6壁 + 床 + 天井 + 空気 |
| 励振源 | リード37枚 (連続励振) | スピーカ複数 + 楽器 |
| 魂柱 | **なし** | **複数 (スピーカ群 = 多重魂柱)** |

- 大規模 (主聖堂): 連続励振 (BGM的SR) でも多重魂柱必須

これは **規模が物理的設計言語を変える** という Appendix C の発見と整合する。「ピアニカv3 と主聖堂は同じ物理問題ではない」を魂柱の有無という具体的パラメータで示す。

#### E.9.6 仮想魂柱 -- DSP制御の音響的解釈

現代のSRシステムはDSPで位相制御を行う:
- ディレイ補正 (時間軸の位相)
- FIR フィルタ (周波数依存の位相)
- ビームフォーミング (空間的位相整合)

これらは **「仮想魂柱」** の動的配置と解釈できる:

```
DSP制御の物理的解釈:

[実スピーカ位置 = 物理的魂柱]
   |
   v (DSP位相制御)
[仮想スピーカ位置 = 仮想魂柱]
   |
   v
[聴衆位置で位相整合]
```

例えば:
- ディレイフィル (実位置 中段12m) に主アレイから 30 ms ディレイ
- 仮想的には「主アレイ位置から発生したような音」が中段から放射される
- これは **仮想魂柱を主アレイ位置に配置** した効果

バイオリンの魂柱は固定 (1本のみ、位置不変) だが、主聖堂の **DSP制御スピーカは仮想的に多数の魂柱を時空間的に配置** できる。これがバイオリンを超える音響制御の可能性を開く。

#### E.9.7 バイオリン製作技術への含意

逆方向の含意も興味深い: **DSP仮想魂柱の知見をバイオリン製作に応用** できないか?

仮説:
- バイオリンの魂柱は1mm単位で音色変化 (本書§3.6)
- これは部屋音響のスピーカ位置1mm単位調整と類似
- バイオリン製作で「複数魂柱」を試す? (歴史的に試行され、不採用?)

歴史的事実:
- 5弦バイオリン (チェロ族の中間サイズ) では魂柱複数本の試みあり
- しかし楽器の伝統で「魂柱1本」が定着
- 理由: 演奏制御の単純性 (1本だけ調整すればよい)

DSPは複数魂柱の自動制御を可能にするので、現代のSRシステムは **「DSP制御された多重魂柱バイオリン」** とも言える。
