# スピーカと共鳴箱の本質的違い -- 工学・熱力学・流体力学からの統一的考察

**目的**: ピアニカv3設計における「共鳴箱」の物理的位置付けを明確化するため、電気音響工学の代表である「スピーカ」と対比し、両者の原理・構造・機能の違いを定量的に整理する。

**結論先取り**:

```
スピーカ : 能動的放射体    (コーン自体が空気を押す変位源)
共鳴箱   : 受動的増幅器    (励振源は別、板が振動して放射)
```

両者を区別する核心パラメータは **「放射体が自前のエネルギー源を持つか否か」** にある。

---

## 1. 序論 -- なぜ両者を混同してはならないか

人は音が出る箱を見ると区別なく「スピーカ」と呼びがちだが、楽器の共鳴箱とオーディオスピーカは **エネルギー流路の構造そのものが異なる**。

| 観点 | スピーカ | 共鳴箱 (バイオリン/ピアノ/ピアニカv3) |
|------|---------|-------------------------------------|
| エネルギー源 | 電力 (AC電流) | 機械的励振 (弦・リード) |
| 変換段数 | 電気 -> 機械 -> 音響 (3段) | 機械 -> 音響 (1段) |
| 放射体 | コーン (能動) | 共鳴板 (受動) |
| 周波数応答 | 広帯域フラット (設計目標) | 共鳴ピーク多数 (個性) |
| 効率 | 0.5 - 5% | 0.1 - 1% |
| 設計指標 | 忠実度 (THD, 周波数特性) | 音色 (倍音バランス, sustain) |

**重要な誤解**: 「共鳴箱はスピーカの代用品ではない」。両者は異なる物理問題を解いている。

## 2. スピーカの原理 -- 3段エネルギー変換

### 2.1 電気 -> 機械変換 (Lorentz力)

ボイスコイルに電流 $i(t)$ が流れると、磁石の磁束密度 $B$ との相互作用により Lorentz力が発生する:

$$F(t) = B \cdot l \cdot i(t)$$

ここで:
- $B$: 磁気回路ギャップ内の磁束密度 [T]
- $l$: ギャップ内のコイル巻線長 [m]
- $i(t)$: 駆動電流 [A]

積 $Bl$ を **力係数 (force factor)** と呼び、スピーカの基本特性を決める。典型値: $Bl = 5 - 20$ [T·m]。

### 2.2 機械系の運動方程式

コーン+ボイスコイル系は、質量・コンプライアンス・抵抗を持つ振動系として記述される:

$$m \frac{d^2 x}{dt^2} + R_m \frac{dx}{dt} + \frac{x}{C_m} = F(t) - S \cdot p(t)$$

- $m$: 振動系の機械質量 (コーン + コイル + 空気付加質量) [kg]
- $R_m$: 機械抵抗 (エッジ・ダンパーの粘性損失) [Ns/m]
- $C_m$: コンプライアンス (サスペンションのバネ) [m/N]
- $S$: コーンの実効振動面積 [m^2]
- $p(t)$: 前面音圧 (放射インピーダンスからの反作用) [Pa]

機械共振周波数 (フリーエア):

$$f_s = \frac{1}{2\pi \sqrt{m \cdot C_m}}$$

これがスピーカ最低再生周波数の物理限界となる。

### 2.3 機械 -> 音響変換

コーンの速度 $v = dx/dt$ が空気を押し、体積速度 $U = S \cdot v$ を生む。これが音波として放射される:

$$p_{far}(r) = \frac{j \omega \rho_0 U}{2 \pi r} e^{-jkr}$$

(球面波近似、$k = \omega/c$、$\rho_0$ = 空気密度)

**Thiele-Smallパラメータ**:

スピーカの全特性は以下7パラメータで記述される (T-S parameters):

| 記号 | 意味 | 単位 |
|------|------|------|
| $f_s$ | フリーエア共振周波数 | Hz |
| $Q_{ms}$ | 機械系Q値 | - |
| $Q_{es}$ | 電気系Q値 | - |
| $Q_{ts}$ | 全Q値 ($Q_{ms} \| Q_{es}$) | - |
| $V_{as}$ | 等価コンプライアンス容積 | L |
| $S_d$ | 実効振動面積 | m^2 |
| $Bl$ | 力係数 | T·m |

これら7つで、任意のエンクロージャ条件下の周波数応答が計算可能。

## 3. コーン部分 (発声体) の構造・構成・機能

### 3.1 構造の階層

```
+-------------------------------------+
| コーン (paper / poly / metal / kevlar) |
+-------------------------------------+
| ボイスコイル (Cu / Al wire on bobbin) |
+-------------------------------------+
| ダストキャップ (中心保護)              |
| サスペンション:                       |
|   - エッジ (外周): ゴム / 布 / フォーム |
|   - ダンパー (内周): 樹脂含浸布 (蜘蛛) |
+-------------------------------------+
| 磁気回路 (磁石 + ヨーク + ポールピース) |
+-------------------------------------+
| フレーム (バスケット, 鋼板 / アルミ)    |
+-------------------------------------+
```

### 3.2 コーン素材の工学トレードオフ

理想コーンは「軽くて剛性が高くて内部損失が大きい」もの。これら3要素は両立しないため設計選択が生まれる:

| 素材 | 比剛性 $E/\rho$ [m^2/s^2] | 内部損失 $\eta$ | 用途 |
|------|---------|---------|------|
| パルプ紙 | 中 (5e6) | 高 (0.05) | 汎用、自然な音色 |
| ポリプロピレン | 低 (2e6) | 高 (0.1) | フラット応答 |
| アルミ | 高 (26e6) | 低 (0.001) | ハイエンドツイーター |
| カーボン繊維 | 極高 (50e6) | 中 (0.01) | 高性能ウーファー |
| ケブラー | 高 (16e6) | 中 (0.02) | 中域用 |

**分割振動 (cone breakup)**: 周波数が上がると、コーン全体が一様に動かず、波動が広がる。発生周波数:

$$f_{breakup} \approx \frac{1.62 \cdot h}{a^2} \sqrt{\frac{E}{12 \rho (1-\nu^2)}}$$

(円板薄肉理論、$a$ = 半径、$h$ = 厚さ)

これ以上では応答にピーク・ディップが生まれ、設計者は素材選択で位置を制御する。

### 3.3 コーンの「発声体としての機能」

スピーカコーンは **励振源 = 放射体** が同一物体である。Lorentz力でコイルが動き、コイル直結のコーンが空気を押す。この一体性は楽器の共鳴箱と決定的に異なる。

**機能の整理**:

| 機能 | 担当部品 | 物理 |
|------|---------|------|
| 駆動力発生 | 磁気回路 + ボイスコイル | Lorentz力 $F = Bli$ |
| 力 -> 変位 | サスペンション + コーン質量 | 2階線形ODE |
| 変位 -> 体積速度 | コーン表面積 | $U = Sv$ |
| 体積速度 -> 音圧 | 空気の慣性 | 球面波放射 |

## 4. エンクロージャの構造・構成・機能

### 4.1 なぜエンクロージャが必要か -- 双極子放射の問題

裸のコーンは前面と背面で **逆位相の音波** を放射する。低周波では波長が大きく、両者が回り込んで干渉相殺する:

$$p_{net} \approx p_{front} - p_{back} \to 0 \quad (\text{when } \lambda \gg a)$$

これを「音響短絡 (acoustic short-circuit)」という。エンクロージャの第一の役割は **背面波の制御** である。

### 4.2 主要エンクロージャ方式

#### (a) 密閉型 (Closed Box / Acoustic Suspension)

背面の空気をバネとして利用。エンクロージャ容積 $V_b$ がコンプライアンスを変える:

$$f_c = f_s \sqrt{1 + \frac{V_{as}}{V_b}}$$

- 利点: 過渡応答が良い、設計が単純
- 欠点: 効率が低い (背面エネルギーは熱に)

#### (b) バスレフ型 (Bass Reflex / Vented)

背面波を Helmholtz共鳴器でπ反転させ、前面波と同位相にして放射する:

$$f_b = \frac{c}{2\pi} \sqrt{\frac{S_p}{V_b \cdot l_{eff}}}$$

- $S_p$: ポート断面積 [m^2]
- $l_{eff}$: 実効ポート長 ($l_{eff} = l + 1.7\sqrt{S_p/\pi}$、開口端補正)

**Helmholtz共鳴の物理**:

$$\omega_H^2 = \frac{c^2 S_p}{V_b l_{eff}} = \frac{\text{空気スプリング}}{\text{ポート空気の慣性}}$$

ポート内の空気塊が質量、エンクロージャ内の空気が圧縮性スプリングとして動作。

- 利点: 低域効率が約2倍 (3dB増)、$f_b$以下で急峻ロールオフ
- 欠点: 過渡応答悪化、ポートノイズ

#### (c) バックロードホーン (Back-loaded Horn)

背面波を指数関数的に拡張するホーンに通し、放射インピーダンスを整合する。

$$S(x) = S_0 \cdot e^{m x}, \quad f_{cutoff} = \frac{m c}{4\pi}$$

- 利点: 高効率 (5-10%、業務用で15%超も)
- 欠点: 巨大、低域カットオフ高め

#### (d) トランスミッションライン

背面波を1/4波長ラインに通し、出口で吸音または放射。

### 4.3 エンクロージャ構造要素

```
エンクロージャ要素一覧:
+ 筐体壁 (MDF / 合板 / アルミ): 振動防止 + 音漏れ防止
+ 補強材 (brace): 壁面共振の分割
+ 吸音材 (グラスウール / フェルト): 内部定在波の減衰
+ ポート (バスレフのみ): Helmholtz共鳴管
+ ガスケット (取付部): 空気漏れ防止
```

### 4.4 エンクロージャの「機能」

エンクロージャは **音源ではない**。空気の音響境界条件を整える「容器」である:

| 機能 | 物理的役割 |
|------|---------|
| 背面波制御 | 双極子 -> 単極子化 |
| 共振制御 | 壁の振動を音波として漏らさない |
| 定在波抑制 | 内部キャビティ共鳴の吸収 |
| インピーダンス整合 | (ホーン型) 空気との結合改善 |

**重要**: エンクロージャ自体は鳴ってはならない。箱が鳴るのは欠陥である。これが楽器の共鳴箱と最も異なる点である。

## 5. スピーカ全体としての機能

### 5.1 システム伝達関数

入力電圧 $V_{in}$ から放射音圧 $p_{out}$ までの周波数応答 (密閉型):

$$\frac{p_{out}(\omega)}{V_{in}(\omega)} = \frac{Bl \cdot S_d \cdot \omega^2}{R_e \cdot Z_m(\omega)} \cdot \frac{\rho_0}{2\pi r}$$

これは2次のハイパスフィルタ特性を持つ:
- 低域: 12 dB/oct ロールオフ
- 中域: フラット
- 高域: コーン分割振動 + 質量制御で減衰

### 5.2 効率とエネルギー流路

**効率の定義**:

$$\eta = \frac{P_{acoustic}}{P_{electrical}} = \frac{B^2 l^2 \rho_0 S_d^2}{2\pi c R_e m^2}$$

典型値:
- 民生用 (ホーム): 0.3 - 1%
- スタジオモニタ: 1 - 3%
- 業務用 (PA): 5 - 15%
- ホーン型: 15 - 30%

**エネルギー流路の図解**:

```
電力 100% 入力
  |
  +-- ボイスコイル抵抗発熱 (Joule loss): 90 - 99%   -> 熱
  |
  +-- 機械損失 (エッジ粘性): 1 - 5%                 -> 熱
  |
  +-- 音響放射 (実効出力): 0.5 - 5%                 -> 音
       |
       +-- 前面放射 (有効): ~50%
       +-- 背面処理 (吸音 or バスレフ): ~50%
```

つまりスピーカは **本質的にヒーター** である。音になるのは僅か数%。

## 6. 熱力学的観点

### 6.1 ボイスコイルの発熱とパワーコンプレッション

入力電力の大部分はコイル抵抗で熱になる:

$$P_{heat} = i^2 R_e \approx (1 - \eta) P_{in}$$

コイル温度上昇により銅の抵抗が増加 (温度係数 $\alpha = 0.00393 / K$):

$$R(T) = R_0 [1 + \alpha (T - T_0)]$$

これにより駆動電流が減り、出力が圧縮される:

$$\Delta L_p \approx -10 \log_{10}\left(\frac{R(T)}{R_0}\right) \quad [\text{dB}]$$

100Wクラスでコイル温度が +200K上昇すると、抵抗1.8倍、出力 -2.5dB。これが **パワーコンプレッション** で、業務用機器の重要設計事項。

### 6.2 熱伝達経路

```
ボイスコイル熱源
  |
  +-- 伝導: ボビン -> ポールピース (磁気回路)  ~70%
  +-- 対流: ボビン背面 -> エンクロージャ内空気 ~20%
  +-- 放射: コーン裏面 -> エンクロージャ壁面    ~10%
```

エンクロージャ内空気は密閉時に温度上昇し、空気の状態方程式から内部圧力が変動:

$$\frac{P_1 V_1}{T_1} = \frac{P_2 V_2}{T_2}$$

長時間大入力で内部温度 +30K -> 静圧が約10%上昇 -> サスペンションのバイアス偏移。

### 6.3 断熱・等温過程の境界

低周波では空気の圧縮は **等温的** (熱が筐体壁に逃げる時間がある):

$$pV = nRT \Rightarrow B_{iso} = p_0$$

高周波では **断熱的** (圧縮が速すぎて熱が拡散しない):

$$pV^\gamma = const \Rightarrow B_{adi} = \gamma p_0$$

ここで $\gamma = c_p/c_v = 1.4$ (空気の比熱比)。境界周波数:

$$f_{boundary} \approx \frac{\alpha_{thermal}}{2\pi L^2}$$

($\alpha_{thermal}$ = 空気の熱拡散率 = 22e-6 m^2/s、$L$ = 代表長さ)

エンクロージャ内では数百Hzで切り替わり、これがバスレフ計算で空気を断熱として扱う根拠。**音速 $c = \sqrt{\gamma p_0/\rho_0} = 343$ m/s** はこの断熱条件で導出される。

## 7. 流体力学的観点

### 7.1 コーンによる空気の駆動 -- ピストン運動と圧縮性

コーンの動作は **線形音響近似** で記述される。条件:

1. **小振幅**: マッハ数 $M = v/c \ll 1$
2. **線形圧縮**: 圧縮率 $|\Delta p/p_0| \ll 1$
3. **層流**: レイノルズ数が乱流境界以下

この近似下で波動方程式が成立:

$$\nabla^2 p - \frac{1}{c^2}\frac{\partial^2 p}{\partial t^2} = 0$$

### 7.2 放射インピーダンス

コーン前面の空気は質量と抵抗を持つ「負荷」として作用:

$$Z_{rad}(\omega) = \rho_0 c S_d \cdot [R_1(2ka) + j X_1(2ka)]$$

$ka \ll 1$ (低周波) の極限:

$$R_1 \to \frac{(ka)^2}{2}, \quad X_1 \to \frac{8ka}{3\pi}$$

これは:
- **抵抗成分**: $\propto \omega^2$ (放射効率は周波数の2乗で増加)
- **リアクタンス**: 空気付加質量 $m_{air} = (8/3) \rho_0 a^3$

低周波で放射効率が下がる根本原因がこの $\omega^2$ 依存。

### 7.3 ポートでの流れ -- 線形/非線形遷移

バスレフポートでの空気速度:

$$v_p = \frac{S_d \cdot v_c}{S_p}$$

(連続の式: コーン側体積速度 = ポート側体積速度)

低レベルでは Helmholtz共鳴が働き正弦波運動だが、大入力で速度が上がるとポート出口で **渦剥離** が発生:

$$Re = \frac{v_p \cdot d}{\nu}$$

($\nu = 1.5 \times 10^{-5}$ m^2/s = 空気動粘性)

$Re > 10^4$ で乱流化、ポートが「鳴る」(ポートノイズ、chuffing)。これが大入力時のバスレフの音質劣化要因。

設計基準: ポート空気速度を $5\%$ of音速 (約17 m/s) 以下に保つ。

### 7.4 内部定在波

エンクロージャ内では3軸方向の定在波が発生:

$$f_{lmn} = \frac{c}{2}\sqrt{\left(\frac{l}{L_x}\right)^2 + \left(\frac{m}{L_y}\right)^2 + \left(\frac{n}{L_z}\right)^2}$$

最低モードは長辺方向で、典型エンクロージャ (40cm) では約 $c/(2 \cdot 0.4) = 430$ Hz。これを抑えるために吸音材を充填する。

### 7.5 コーンの空力騒音

高速のコーン振動は周辺空気で渦を生む可能性があり、特にコーン端の不連続部で **エッジ回折** が起こる:

$$p_{diff}(\theta) \propto p_{cone} \cdot \frac{1}{\sqrt{kr}} \cdot D(\theta)$$

これがバッフルステップ (バッフル境界での6dB低下) の原因。

## 8. 共鳴箱との本質的違い -- 統一比較

### 8.1 構造の対比図

```
[スピーカ]
   電源 -> アンプ -> ボイスコイル -> [コーン] -> 空気
                                     ^
                                     +-- 励振源 = 放射体 (一体)
                          +--------- エンクロージャ (背面波制御のみ)
                          |
   電気エネルギー        変換段        受動的容器


[共鳴箱楽器 (バイオリン/ピアニカv3)]
   弾力/呼気 -> [リード/弦] -> 駒 -> [共鳴板] -> 空気
                  ^                    ^
                  励振源              放射体 (分離)
                                       +-- 共鳴箱 (それ自身が振動して鳴る)
   機械エネルギー   変換無し       共鳴板 = 能動的放射体
```

### 8.2 物理パラメータ比較

| 比較項目 | スピーカ (8インチウーファー) | バイオリン共鳴箱 | ピアニカv3 (設計目標) |
|---------|---------------------|----------------|---------------------|
| 励振源 | ボイスコイル (Lorentz力) | 弓 + 弦 | リード (空気駆動) |
| 励振源-放射体 結合 | 直結 (機械接続) | 駒 (機械接続) | 駒経由 ($T = 97\%$) |
| 放射体の質量 [g] | 30 - 100 (コーン) | 60 (表板) | 約100 (桐板1.5mm) |
| 放射面積 [m^2] | 0.02 - 0.05 | 0.035 | 0.09 |
| 放射効率 | 0.5 - 5% | 30% | 36% (設計目標) |
| 共振周波数 [Hz] | $f_s$ = 20 - 80 | $f_{11}$ = 460 | $f_{11}$ = 96 |
| 周波数応答 | フラット (狙い) | 多重ピーク (個性) | 多重ピーク (狙い) |
| Q値 | $Q_{ts} = 0.3 - 0.7$ (制動) | 各モード 30 - 100 | 30 - 100 |
| 設計指標 | THD < 1%, 平坦応答 | 音色, sustain | 音色, 倍音バランス |

### 8.3 エネルギー流路の違い

**スピーカ**: 電気エネルギーが大半 (>90%) **熱として捨てられる**。残り数%が音。

**共鳴箱**: 機械エネルギーの大半が **共鳴板の運動 -> 空気放射** に流れる (放射効率 30%以上)。電気的損失なし。

スピーカの低効率はLorentz変換の本質的限界 ($Bl$ が大きくなれば効率上がるが磁石・コイルが重くなり過渡応答悪化)。共鳴箱は機械-音響を直接結合するため高効率を達成できる。

### 8.4 設計哲学の違い

| 観点 | スピーカ | 共鳴箱 |
|------|---------|--------|
| 目標 | 入力信号を忠実に音波化 | 励振源の音色を増幅・色付け |
| エンクロージャの理想 | 完全に静かな箱 | 全体が鳴り響く箱 |
| 共振 | 抑制すべきもの (Q<1) | 設計するもの (Q=30-100) |
| 周波数特性 | フラット | 多重ピーク (フォルマント) |
| 個体差 | 排除すべき | 個性として尊重 |

**スピーカ設計者**: 「箱を鳴らすな、コーンだけ鳴らせ」
**楽器製作者**: 「箱を鳴らせ、それが楽器の声」

## 9. ピアニカv3への含意 -- 設計判断の正当化

### 9.1 v3が「スピーカではなく共鳴箱」である根拠

ピアニカv3設計 (桐1.5mm + X-brace + 14.2L箱) は **明確に共鳴箱パラダイム** に立脚している:

1. **励振源 (リード) と放射体 (桐板) が分離**: 駒経由で結合 ($T = 97\%$)
2. **共鳴板が能動的に鳴る**: ガンマ値 $\Gamma = 1.8$ で放射効率 36%
3. **多重共振モード**: 96Hz (1,1) を起点に個性的な周波数応答
4. **電気的構成要素ゼロ**: 純粋な機械-音響系

これは バイオリン物理 ($\Gamma = 2.0$, 効率33%) に最も近い ($f_{11}$ は低い)。

### 9.2 「もしスピーカを使ったら」の思考実験

仮にリードの音をマイクで拾い、アンプ・スピーカで再生する設計を考えると:

- リード -> 空気波 -> マイク (機械->電気)
- アンプ (電気増幅、損失~5%)
- スピーカ (電気->機械->音響、効率1%)

合計効率 ~0.05% に対し、共鳴箱直結 36% で **720倍の効率差**。さらにスピーカの周波数応答はフラットなので、リード固有の倍音バランスが歪まないという保証もない。

**結論**: ピアニカv3にスピーカは不要であり、共鳴箱の方が物理的に優れる。

### 9.3 共鳴箱とエンクロージャの似て非なる関係

注意点として、ピアニカv3の「共鳴箱」は スピーカでいう「エンクロージャ」とは **役割が逆** である:

| 部位 | スピーカ | ピアニカv3 |
|------|---------|----------|
| 主放射体 | コーン | 共鳴板 (桐) |
| 容器の役割 | 背面波遮蔽 | 板の振動を支持 + 内部空洞共振 |
| 容器の振動 | 抑制 (補強剤) | 活用 (X-brace で「壊れにくく、よく鳴る」) |
| 内部空気 | 圧縮スプリング (バスレフ) | Helmholtzアシスト + 板結合 |

桐板はコーンに対応するのではなく、バイオリン表板に対応する。

## Appendix B: スピーカ設計の思考過程具体化 -- 3段エネルギー変換の実装

本章 (本文§2) で導出したスピーカの3段エネルギー変換 (電気 -> 機械 -> 音響) を、具体的な設計目標を設定して材料・電気・電子・制御の各層で実装する。これは「ピアニカv3と同じ用途をスピーカで実現するなら、どう設計するか」という直接比較可能な思考実験でもある。

### B.1 要求仕様の導出と設計目標の設定

#### B.1.1 序 -- 要求仕様の導出は設計の最上流工程である

「定量仕様 (要求書)」を **既知の入力** として扱うのは、設計プロセスとしては不完全である。実際には、仕様の各数値 (周波数応答、感度、THD、寸法、価格) はそれ自体、上流工程での **設計判断の集積** である:

```

    |
    +-- 音響環境解析 (使用空間の物理)
    +-- 音源解析 (再生対象の音響特性)
    +-- 聴取者解析 (心理音響学)
    +-- 制約条件解析 (物理・経済)
    +-- 規格参照
    |

    |

    +-- 制御工学的設計 (B.5)
```

**本セクションB.1.2 - B.1.10 は工程#1を担う**。具体的に言えば、「なぜ50 Hzなのか」「なぜ88 dBなのか」「なぜ30 Wなのか」を、上流の物理・心理学から論証する作業である。これを欠いた設計は、仕様値が変動した際に再設計の根拠を失う。

#### B.1.2 音響環境の解析 -- 教会小礼拝室の物理特性

使用空間 (教会小礼拝室) を音響的に定量化する。

**幾何学的パラメータ (典型的小礼拝室)**:
- 床面積: 5 m x 6 m = 30 m^2
- 天井高: 3.3 m
- 容積: $V = 99$ m^3 (約100 m^3 とする)
- 表面積: $S = 2(30) + 2(5 \times 3.3) + 2(6 \times 3.3) = 60 + 33 + 39.6 = 132.6$ m^2

**残響時間と吸音特性**:

教会内装は石壁・木製ベンチ・カーペット・木製天井の混合。Sabineの式から逆算:

$$RT_{60} = 0.161 \frac{V}{A}$$

$A$ は等価吸音面積 (sabin)、目標 $RT_{60} = 1.2$ s (音楽再生に適した中庸):

$$A = \frac{0.161 \times 100}{1.2} = 13.4 \text{ m}^2 \text{(sabin)}$$

平均吸音率: $\bar{\alpha} = A/S = 13.4/132.6 = 0.10$

これは「やや反響気味だが音楽演奏に適する」レベル (家庭リビングは $\bar{\alpha} \approx 0.25$、コンサートホールは $\bar{\alpha} \approx 0.15$)。

**部屋定数**:

$$R = \frac{S \bar{\alpha}}{1 - \bar{\alpha}} = \frac{132.6 \times 0.10}{0.90} = 14.7 \text{ m}^2$$

**臨界距離 (direct-reverb 等価点)**:

$$r_c = 0.141 \sqrt{R} = 0.141 \sqrt{14.7} = 0.54 \text{ m}$$

聴取距離 2-6 m はすべて **残響場優勢** であり、距離減衰は理論値 (6 dB/octave) より緩やか。実測上 $r=1 \to 4$ m で約 8 dB 低下が典型。

**暗騒音レベル**:

教会の典型暗騒音: NC-25 - NC-30 (30-35 dBA)。HVACなし時はNC-20 (25 dBA) も可能。設計時はNC-30 (32 dBA) を想定。

**聴取距離分布**:
- 最前列: 1.5 m
- 中央列: 3 m
- 最後列: 5 m
- 最遠座席 (角): 6 m

設計基準: 最遠 6 m での聴感を確保しつつ、最近 1.5 m で過大とならない。

#### B.1.3 音源プロファイル -- Trompette 8' の音響特性

再生対象の音響をプロファイリングする。

**Trompette 8' の楽器分類**:

パイプオルガンのリードストップ (リード管系)。チャンバー内の真鍮舌 (シャロー) が空気力学的に振動し、共鳴管 (レゾネータ) で倍音を整形する。「8'」 は最低音管が 8 フィート長で C2 (65.4 Hz) 基音であることを示す。

**音域 (典型的なオルガンの場合)**:
- 最低音: C2 = 65.4 Hz
- 最高音: C7 = 2093 Hz (一部オルガンはG7 = 3136 Hzまで)

**倍音構造**:

リード管は **第30倍音までエネルギーを保持** する金管的音色。FFT解析の典型例:
- 第1倍音 (基音): 0 dB
- 第2-5倍音: -5 〜 -10 dB
- 第6-10倍音: -10 〜 -20 dB
- 第11-20倍音: -20 〜 -35 dB
- 第21-30倍音: -35 〜 -50 dB

最低音 65 Hz の第30倍音 = 1.95 kHz、最高音 2093 Hz の第7倍音 = 14.6 kHz。**音楽的に意味ある成分は 50 Hz - 15 kHz** に集中する。

**ダイナミック特性**:
- Trompette単独: 風圧一定 -> 音圧変動なし (連続音)
- 演奏中: ストップ組合せ・速度・複声で 25-30 dB 変化
- クレストファクター: 録音物の場合 15-18 dB (クラシック音楽の標準)

#### B.1.4 聴取者要求 -- 心理音響学からの規定

**快適聴取レベル (Fletcher-Munson および ISO 226)**:

オルガン音楽の演奏会では、聴取者位置で:
- 最小知覚レベル: 暗騒音 + 15 dB ($\to$ 47 dBA)
- 快適平均レベル: 65-75 dBA
- 上限ピーク: 90-95 dBA (聴覚保護観点)

クラシック音楽のクレストファクター 18 dB を考慮すると:
- 快適平均 70 dBA で運用 -> ピーク 88 dBA
- これを **設計ピーク基準** とする

**信号対雑音比 (SNR)**:

NC-30 (32 dBA) の暗騒音に対し、ピアニッシモ部分 (典型 -25 dB from peak) が 88 - 25 = 63 dBA。SNR = 63 - 32 = 31 dB 確保。これは音楽聴取の最低基準 (30 dB) を満たす。

**歪み知覚閾値**:

ITU-R BS.1116 および Geddes-Lee の研究に基づく:
- 純音 (1 kHz): THD < 1.0% で不知覚
- 楽音 (倍音マスキング): THD < 0.5% で不知覚
- 訓練聴取者 (Hi-Fi): THD < 0.1% で識別可能
- 教会用途 (一般聴取者 + 残響): THD < 0.5% を目標とする

**周波数応答均一性**:

Toole の研究 (Sound Reproduction, 2008) によれば、20 Hz - 20 kHz で +/- 3 dB 以内なら一般聴取者は「色付け」を感じない。Hi-Fi標準 (DIN 45500) も +/- 3 dB を採用。

#### B.1.5 SPL要求の導出計算

聴取者要求 (B.1.4) からスピーカ感度を算出する。

**設計ピーク条件**:
- 聴取位置 (最遠 6m) でのピーク SPL: 90 dBA (5 m を典型距離として再設計)
- 実用最遠 6 m に対し基準距離 4 m を採用 (ほとんどの座席をカバー)

**Step 1: ペア -> シングル**

ステレオペア駆動の場合、相関信号で +3 dB、無相関で +3 dB 以下。設計余裕として:

$$L_{single} = L_{pair} - 3 \text{ dB} = 90 - 3 = 87 \text{ dB SPL}$$

**Step 2: 4m -> 1m (距離補正)**

残響場優勢な部屋では半自由音場式が使えない。Schultz の経験式 (室内音場)、または部屋定数を用いた式:

$$L_p(r) = L_W + 10\log_{10}\left(\frac{Q}{4\pi r^2} + \frac{4}{R}\right)$$

無指向性 ($Q=1$) として:

$1m$地点: $\frac{1}{4\pi} + \frac{4}{14.7} = 0.0796 + 0.272 = 0.352$, $10\log = -4.5$ dB
$4m$地点: $\frac{1}{64\pi} + \frac{4}{14.7} = 0.005 + 0.272 = 0.277$, $10\log = -5.6$ dB

つまり 1m -> 4m で約 1.1 dB しか低下しない (純粋に残響場)。

これは理論値だが、実際には指向性 $Q \approx 4-8$ (中域以上) のため直接音成分が大きく、典型 6-8 dB の距離減衰が観測される。**実用設計値: 1m -> 4m で 7 dB 減衰** とする。

$$L_{1m} = L_{4m} + 7 = 87 + 7 = 94 \text{ dB SPL}$$

**Step 3: ピーク -> 連続入力 (パワー要求)**

スピーカのピーク再生能力は連続定格より高い (典型 +3 dB)。30W RMS定格 -> 60W ピーク。

ピーク 60W 入力時に 94 dB を出すには:

$$\eta_{required} = L_{1m} - 10\log_{10}(P_{peak}) = 94 - 10\log_{10}(60) = 94 - 17.8 = 76.2 \text{ dB SPL/W/m}$$

これは非常に低い要求。**マージンが過剰** なので、より厳しい条件を再設定:
- 最遠距離 6 m で 90 dBA を達成 (典型 4m ではなく)
- ペア寄与は無相関 (+0 dB安全側)
- 距離減衰 1m -> 6m: 9 dB (実測式 $20\log(r) - $室内補正)

$$L_{1m, conservative} = 90 + 0 + 9 = 99 \text{ dB SPL}$$

**Step 4: 連続平均出力に対する要求**

連続定格 30 W で平均 SPL を達成する場合:
- 平均レベル 70 dBA at 6m -> 79 dB at 1m
- 連続 30 W 入力: 必要感度 = 79 - 14.8 = **64.2 dB SPL/W/m**

ピーク条件:
- ピーク 88 dBA at 6m -> 97 dB at 1m
- ピーク 60 W: 必要感度 = 97 - 17.8 = **79.2 dB SPL/W/m**

**Step 5: 設計マージンの設定**

- 部屋音響の不確定性: +/-2 dB
- 個体差: +/-1 dB
- 経年劣化: -1 dB
- リスニング位置のずれ: +/-3 dB

合計マージン: 6-8 dB

**最終要求感度**:

$$\eta_{spec} = 79.2 + 8 = 87.2 \approx 88 \text{ dB SPL/W/m}$$

「88 dB SPL/W/m」 は (B.1.4の心理音響、B.1.2の部屋音響、B.1.3のクレストファクター) から **算術的に導出される値** である。

#### B.1.6 周波数応答要求の導出

**下限 (低域カットオフ)**:

源音は Trompette 8' の最低音 C2 = 65.4 Hz (B.1.3)。これより 1/3 オクターブ下 (-3 dB点を確保) で:

$$f_{low} = 65.4 \times 2^{-1/3} = 65.4 \times 0.794 = 51.9 \text{ Hz}$$

**仕様値: 50 Hz** (切りの良い数字、上記とほぼ一致)

これより低い設計 (40 Hz, 32 Hz) は:
- 16' ストップ再生時に有効だが Trompette 8'では不要
- エンクロージャ容積を $f_3^3$ 比例で増加させる (Hofmann鉄則, B.6)
- Trompette専用機としては過剰

**上限 (高域カットオフ)**:

Trompette の音楽的最高成分は最高音 2093 Hz の第7倍音 = 14.6 kHz (B.1.3)。聴覚上限 (成人 16 kHz, 訓練聴取 18 kHz) を考慮:

**仕様値: 18 kHz** (1/3 オクターブ余裕で 16 kHz の第7倍音まで保護)

**許容偏差**:

DIN 45500 Hi-Fi規格に準拠:

**仕様値: +/- 3 dB**

これは知覚閾値 (Toole) と整合する産業標準。

#### B.1.7 歪・ノイズ要求の導出

**THD要求**:

聴取者の知覚閾値 (B.1.4) は「楽音、訓練無し聴取者」で 0.5% 以下。教会用途は訓練無し聴取者中心:

**仕様値: THD < 0.5% @ 90 dB SPL (100 Hz以上)**

100 Hz未満は心理音響的にTHDが感知されにくく (低域マスキング)、また小型スピーカで 0.5% を下回るのは技術的に困難。実用上の妥協。

**アイドルノイズ要求**:

暗騒音 NC-30 (32 dBA) より十分小さい必要がある。設計基準: 暗騒音より -5 dB以上下:

**仕様値: アイドルノイズ < 27 dBA at 1m (= 25 dBA at 6m)**

これがアンプ選択 (Class AB vs D) を制約する (B.4.1)。

#### B.1.8 物理・経済制約の導出

**寸法**:

教会内の卓上配置 (祭壇横、奥行 350mm 以下の天板) -> 奥行制限 300mm。
人手運搬可能 -> 重量 < 8 kg/本。
ペアでステレオ配置 -> 幅 350 mm 以下 (物理的余裕確保)。

**仕様値: 350 x 220 x 280 mm, < 8 kg/本**

(ピアニカv3 は 550 x 350 x 80 mm = 15.4 L、スピーカは 21.6 L で同等オーダ)

**電源**:

教会の単相100V 15Aコンセント。聴取空間で同時使用 (照明等) を考慮 -> < 200 W per channel。

**仕様値: 定格電源消費 < 100 W/ch (連続)**

これがアンプ効率 (Class A 不可、AB or D 可) を制約する (B.4.1)。

**価格制約**:

教会備品予算 -> 8万円 (ペア)。これは Hi-Fiエントリー機の中位レンジ。

ピアニカv3との比較:
- ピアニカv3: 1.6万円 (材料費)
- スピーカ (本件): 8万円 (材料費 + 電子部品)
- **比 5倍** -> エネルギー変換段数 (1段 vs 3段) と電子部品の対価がほぼ表現されている

**仕様値: ペア材料費 8万円**

**寿命要求**:

教会備品 -> 15年以上の継続使用。これが:
- エッジ素材 (NBRゴム、加水分解しないこと)
- コンデンサ (フィルム型、電解避ける)
- 接着剤 (UV劣化対策)
を制約する。

**仕様値: 期待寿命 15年 (主要部品交換無しで)**

派生仕様の正当性を業界標準で裏付ける:

| 規格 | 関連項目 | 本仕様との対応 |
|------|---------|---------------|
| DIN 45500 (Hi-Fi) | 周波数応答 +/-3 dB、THD < 1% | 50 Hz - 18 kHz +/-3 dB を満たす |
| IEC 60268-5 | スピーカ測定法 | 感度測定 2.83V/1m を採用 |
| ITU-R BS.1116 | リスニング条件 | THD < 0.5% を満たす |
| AES/EBU Tech 3276 | モニタースピーカ | 88 dB SPL/W/m は推奨範囲内 |
| ANSI S1.4 | 騒音レベル測定 | アイドルノイズ < 27 dBA を満たす |

これらの規格は本仕様の **後付けの正当化** ではなく、**設計思想の整合性チェック** として機能する。仕様値が業界標準と矛盾していたら、それは導出のどこかに誤りがある。

#### B.1.10 派生仕様 -- 各値のトレーサビリティ

B.1.2 - B.1.9 の解析を集約した最終仕様。各値は導出根拠を持ち、変更時には根拠ごと再評価する:

| 項目 | 仕様値 | 導出根拠 |
|------|--------|---------|
| 周波数応答 | 50 Hz - 18 kHz, +/-3 dB | B.1.3 (Trompette 8' 音域) + B.1.6 (許容偏差) |
| 感度 (1m, 2.83V) | 88 dB SPL/W/m | B.1.5 (聴取者ピーク 88 dBA + 距離補正 + マージン) |
| 最大SPL @ 1m | 100 dB (連続)、105 dB (ピーク) | B.1.4 (クレストファクター 18 dB) |
| 定格入力 | 30 W RMS / 60 W peak | B.1.5 (連続/ピーク要求から逆算) |
| THD | < 0.5% @ 90 dB SPL | B.1.4 (知覚閾値) + B.1.7 |
| アイドルノイズ | < 27 dBA at 1m | B.1.2 (NC-30) + B.1.7 (-5 dB余裕) |
| 公称インピーダンス | 8 ohm (最低 6.4 ohm) | 一般アンプ駆動可能性 (規格デフォルト) |
| 寸法 | 350 x 220 x 280 mm | B.1.8 (卓上配置制約) |
| 重量 | < 8 kg/本 | B.1.8 (人手運搬性) |
| 電源消費 | < 100 W/ch | B.1.8 (教会電源容量) |
| 期待寿命 | 15年 | B.1.8 (備品要求) |
| ペア価格 | 8万円 | B.1.8 (予算制約) |

**この表が以下のB.2-B.5の入力となる**。実装設計はこの仕様を「与件」として扱う。ただし、B.6 (統合検証) で Hofmann鉄則による物理的不可能性が判明した場合は、本表に戻って **どの根拠から緩和できるか** を再交渉する (例: 価格緩和? 寸法緩和? 帯域緩和?)。これが設計の反復構造である。

**設計の反復ループ**:

```
要求仕様 (B.1.10)
    |
    v
実装設計 (B.2-B.5)
    |
    v
統合検証 (B.6)  ----+
    |              |
    +-- OK -> 確定  |
    |              |
    +-- NG -> 仕様緩和の交渉 (B.1に回帰)
                   |
                   +---+
```

ピアニカv3の場合、要求仕様 (パイプオルガン Trompette 8' 音色を 8畳礼拝室で) -> 物理 (リード37枚 + 桐共鳴箱14.2L) で、Hofmann鉄則ではなく **共鳴板物理 ($\Gamma$, 表面積、剛性比)** が支配的だった。スピーカと共鳴箱で「上流から下流への翻訳規則」自体が異なることが、ここでも示される。



### B.2 材料選択の思考過程

#### B.2.1 ウーファーコーン素材

**判断ツリー**:

```
Trompette 8' の音色特性
  -> 倍音が豊富、エッジの効いた金管的音色
  -> ミッドバンド (300Hz - 3kHz) の解像度が肝
  -> コーン分割振動による着色は致命的

候補比較:
  パルプ紙 (treated)    : eta=0.05, E/rho=5e6, 価格 3,000円  -> 中庸
  ポリプロピレン        : eta=0.10, E/rho=2e6, 価格 4,000円  -> 解像度不足
  アルミニウム          : eta=0.001, E/rho=26e6, 価格 8,000円 -> 中域リンギング
  ケブラー              : eta=0.02, E/rho=16e6, 価格 12,000円 -> 高解像度
  カーボン繊維複合     : eta=0.01, E/rho=50e6, 価格 18,000円 -> オーバースペック

判定: 強化紙コーン (treated paper)
理由:
  - eta = 0.05 で分割振動を自然に減衰
  - クロスオーバー 2.5kHz以下では分割振動領域に入らない
  - 価格と音色のバランスが教会用途に適合
  - パルプ紙の経年変化 (10-15年) は許容
```

実装パラメータ: 直径 165mm (6.5インチ)、有効振動面積 $S_d = 132$ cm^2、質量 12g。

#### B.2.2 ボイスコイル

**Lorentz力 $F = Bli$ を最大化したい一方、コイル質量はコーン応答性を悪化**

```
線材選択:
  銅線 (Cu)              : 抵抗低 (1.7e-8 ohm.m)、密度 8.9 g/cm^3
  CCAW (Cu被覆Al)        : 抵抗 2.7e-8 ohm.m、密度 3.6 g/cm^3 (1/2.5)
  アルミ線 (Al)          : 抵抗 2.7e-8 ohm.m、密度 2.7 g/cm^3、終端処理難

判定: CCAW
  - 銅と同等の表皮特性 (高域での表皮効果に対抗)
  - 重量が銅の40%なので動的応答が向上
  - ボビン外径への熱伝達も向上 (アルミ芯)

ボビン素材:
  カプトン           : 耐熱 300°C、薄肉可、軽量
  アルミ             : 熱伝導 237 W/mK (最高)、ただし渦電流損
  ガラス繊維         : 安価、耐熱200°C

判定: アルミニウムボビン
  - ウーファー用途なので渦電流損は許容 (磁束密度の時間微分が小さい)
  - 熱を磁気回路へ逃がす経路を確保
  - パワーコンプレッション (本文§6.1) を抑制
```

仕様: CCAW巻線 0.18mm線径、ターン数 80T、ボビン高さ 12mm、$R_e = 6.2$ ohm、$L_e = 0.5$ mH。

#### B.2.3 磁気回路

```
寸法制約: エンクロージャ奥行 280mm、ウーファー奥行 100mm以下

磁石種別:
  フェライト (Y30)   : Br=0.4 T、コスト 500円、必要体積大
  アルニコ           : Br=0.7 T、温度安定、価格 5,000円
  ネオジム (N42)     : Br=1.3 T、コスト 2,500円、温度減磁(-0.12%/K)
  ネオジム (N52H)    : Br=1.45 T、高温対応、4,000円

判定: ネオジム N42 ディスク 80mm OD x 12mm (約400g)
  - 100mm奥行制約下で目標 Bl=6 T.m を達成可能
  - 温度減磁は熱保護回路で対応
  - 漏れ磁束対策にコッパーキャップ (歪低減)
```

磁気回路設計:

$$B_{gap} = \frac{B_r \cdot V_{magnet}}{V_{gap} \cdot k_{leak}}$$

ギャップ 1.2mm幅、コイル巻線占有率 0.7、これにより $Bl = 6$ T.m を実現。

#### B.2.4 サスペンション

**コンプライアンス $C_{ms}$ と Q値 $Q_{ms}$ のバランス調整**

```
エッジ素材:
  ゴム (NBR)     : 線形 +/-5mm、寿命20年、コスト 200円
  布 (含浸)     : Q高い、寿命15年、低コスト
  ウレタンフォーム: 線形性高、寿命5-7年で加水分解

判定: NBRゴムロールエッジ
  - 寿命が教会備品として許容
  - ピストン運動範囲 +/-5mm を線形に確保

スパイダー (内側ダンパー):
  フェノール含浸綿布 (蜘蛛形状)
  -> 経年でスティフネス変化 < 5%/10年
  -> $C_{ms}$ = 1.0 mm/N をターゲット
```

これらにより:
- 振動系総質量 $m_{ms} = 12$ (コーン) + 1.5 (コイル) + 1.5 (空気付加) = 15 g
- $f_s = 1/(2\pi \sqrt{m_{ms} C_{ms}}) = 1/(2\pi\sqrt{0.015 \times 0.001}) \approx 41$ Hz
- $Q_{ms} = 3.0$ (機械損失低)
- $Q_{es} = B^2 l^2 / (R_e \omega_s m_{ms}) = 36/(6.2 \times 258 \times 0.015) = 1.5$
- $Q_{ts} = (Q_{ms} \cdot Q_{es})/(Q_{ms} + Q_{es}) = 1.0$
- $V_{as} = \rho_0 c^2 S_d^2 C_{ms} = 1.2 \times 343^2 \times (0.0132)^2 \times 0.001 = 24$ L

#### B.2.5 ツィーター

**クロスオーバー 2.5 kHz以上を担当**

```
方式比較:
  ソフトドーム (シルク)   : 自然な高域減衰、リンギング少、感度 89-91 dB
  ハードドーム (アルミ)  : 解像度高、共振 25kHz、感度 90-92 dB
  リボン                  : 超低質量、指向性鋭い、コスト高
  AMT (Air Motion Trans) : 超高解像、コスト極大

判定: 25mm シルクドーム + フェロフルイド冷却
  - 教会音響 (残響長い) では尖った高域は過剰
  - シルクの自然な減衰がオルガン倍音と整合
  - フェロフルイドで $Q$ を制御 + 熱伝達向上
```

#### B.2.6 エンクロージャ材料

**剛性 + 内部損失 + 外観の三要素**

```
素材選択:
  MDF 18mm         : 内部損失 eta=0.02、加工容易、見た目要塗装
  バーチ合板 18mm  : 内部損失 eta=0.01、剛性高、自然な木目
  HDF 15mm         : MDFより高密度、価格 +30%
  竹集成材 18mm    : 高剛性、環境訴求、加工難

  - 教会内観に木目が調和
```

### B.3 電気的設計の思考過程

#### B.3.1 ボックスアラインメント決定

T-Sパラメータが揃ったので、エンクロージャ容積と方式を決める。

```
密閉型 vs バスレフ:
  密閉型:
    - $f_c = f_s \sqrt{1 + V_{as}/V_b}$
    - $V_b = 21$ L で $f_c = 41 \times \sqrt{1 + 24/21} = 60$ Hz
    - 過渡応答良好、F3 = 60Hz
    - Trompette 8' のF2 (65Hz) はギリギリ
  バスレフ型:
    - 同 $V_b$ で F3 = 50Hz 達成可能 (3dB低い)
    - ポート設計が必要

判定: バスレフ型
  - F3 = 50 Hz は要求仕様 (50Hz +/- 3dB) を満たす最低条件
  - 5Hz差は1オクターブの 1/4、知覚上有意
```

ポート計算:

$$f_b = \frac{c}{2\pi}\sqrt{\frac{S_p}{V_b \cdot l_{eff}}}$$

目標 $f_b = 48$ Hz ($f_s$ よりやや高め、Q=0.7のSBB4アラインメント)

選定: ポート直径 50 mm ($S_p = 1963$ mm^2)、

$$l_{eff} = \frac{c^2 S_p}{(2\pi f_b)^2 V_b} = \frac{343^2 \times 0.001963}{(301.6)^2 \times 0.021} = 121 \text{ mm}$$

開口端補正 $\Delta l = 1.7\sqrt{S_p/\pi} = 42$ mm を引いて、物理ポート長 $l = 79$ mm。

**ポート流速チェック** (本文§7.3):
- 最大入力時のコーン速度: $v_{max} = X_{max} \cdot \omega = 0.005 \times 301.6 = 1.5$ m/s
- ポート流速: $v_p = v_{max} \cdot S_d / S_p = 1.5 \times 132/19.6 = 10.1$ m/s
- 設計基準 (音速の5% = 17 m/s) 以下 -> ポートノイズ許容範囲

#### B.3.2 クロスオーバー設計

```
方式比較:
  Butterworth (BW) 2次  : ピーク+3dB、位相90度
  Linkwitz-Riley (LR) 2次: フラット和、位相 0度
  LR 4次                 : フラット和、急峻24dB/oct、位相 360度

判定: 4次 Linkwitz-Riley
  - スロープが急峻でドライバ各々の動作帯域が清浄
  - ウーファー高域の分割振動を確実に除去
  - ツィーター低域 (機械共振 1kHz) を保護
```

クロスオーバー周波数選定:

```
ウーファー使用上限: 分割振動開始 ~ 4kHz、推奨上限 3kHz
ツィーター使用下限: 機械共振 1kHz、推奨下限 1.5kHz
重複帯域: 1.5 - 3 kHz
中心: 2.2 kHz だが、Trompette 8'倍音密度ピーク (2-3kHz) を避けて

判定: 2.5 kHz
  - ウーファーは余裕、ツィーターは1.7倍 (推奨5倍まで)
  - 倍音密度の谷に配置 (Trompette はオクターブ毎にピーク)
```

LC値計算 (8 ohm系、$f_c = 2500$ Hz、LR4):

$$L_1 = \frac{R}{\sqrt{2}\pi f_c} = \frac{8}{\sqrt{2}\pi \cdot 2500} = 0.72 \text{ mH}$$

$$C_1 = \frac{1}{\sqrt{2}\pi f_c R} = \frac{1}{\sqrt{2}\pi \cdot 2500 \cdot 8} = 11.3 \mu\text{F}$$

(LR4は2次BWのカスケードで実装、コンポーネント数 4 ずつ)

#### B.3.3 インピーダンス補正 (Zobel network)

ボイスコイルインダクタンスにより、高周波で公称インピーダンスが上昇する。これがクロスオーバーカーブを歪める。

Zobel ネットワーク (コーン抵抗を見かけ上純抵抗化):

$$R_z = 1.25 R_e = 7.75 \Omega$$

$$C_z = \frac{L_e}{R_e^2} = \frac{0.5 \times 10^{-3}}{6.2^2} = 13 \mu\text{F}$$

ウーファー端子に並列接続。これによりインピーダンスが10kHzまで8ohm近傍を維持。

#### B.3.4 感度計算

スピーカ全体の参照感度 (1W入力時、1m距離):

$$\eta_{ref} = \frac{B^2 l^2 \rho_0 S_d^2}{2\pi c R_e m_{ms}^2}$$

数値代入:

$$\eta_{ref} = \frac{36 \times 1.2 \times (0.0132)^2}{2\pi \times 343 \times 6.2 \times (0.015)^2} = \frac{7.53 \times 10^{-3}}{3.0} = 2.5 \times 10^{-3}$$

つまり 0.25% の効率。SPL換算:

$$L_p = 112 + 10\log_{10}(\eta_{ref}) = 112 + 10\log_{10}(0.0025) = 112 - 26 = 86 \text{ dB SPL/W/m}$$

(測定基準では 2.83V/8ohm = 1W で同じ)

**目標88 dBに対し-2dB**。許容範囲だが、改善するなら $Bl$ 増 (磁石強化) または $S_d$ 増 (8インチ化)。今回は86 dB を仕様変更として確定 (要求書改訂)。

### B.4 電子的設計の思考過程

#### B.4.1 アンプ方式選択

```
方式比較:
  Class A     : THD<0.001%、効率 25%、発熱大
  Class AB    : THD<0.01%、効率 50%、最も一般的
  Class D     : THD<0.05%、効率 90%、スイッチノイズ
  Class H     : Class AB + 電源切替、効率 70%、複雑

判定: Class AB (パッシブクロスオーバー駆動)
  - 教会の静粛環境ではClass Dの残留ノイズが問題化しうる
  - 30W程度なら発熱も管理可能 (放熱体 0.5 K/W)
  - Hi-Fi基準のTHD < 0.01% を満たしやすい
```

電源容量計算:
- 出力 30W、効率50% -> 入力60W
- ピーク 60W、効率45% -> 入力133W
- トランス容量 200VA選定 (余裕含み)

#### B.4.2 DSP導入の判断

```
DSPあり:
  + 部屋補正 (1/3オクターブEQで残響補正)
  + 厳密なクロスオーバー (パッシブの誤差吸収)
  + Klippel式線形化 (B.5参照)
  - 5ms程度のレイテンシ
  - コスト +1.5万円/ch
  - A/D-D/A変換による音質劣化リスク

DSPなし:
  + 純アナログのシンプルさ
  + コスト最小
  - 部屋固有のディップ・ピークが残る

判定: DSPあり (オプション機能として)
  - 教会固有の100Hz定在波対策に必須
  - ADIのSHARC (32bit/96kHz) で透明性確保
  - スイッチで Bypass可能にしてA/B比較可能
```

DSP構成:
- ADC: 24bit/96kHz (AKM AK5388)
- DSP: ADAU1452 (294MHz)
- DAC: 32bit/192kHz (ESS ES9038Q2M)
- レイテンシ: 4.2 ms (許容)

#### B.4.3 保護回路

**過熱保護**:
- 磁気回路裏面にNTCサーミスタ
- 70°C で警告LED、85°Cでミュート
- パワーコンプレッション (本文§6.1) を間接的に管理

**DCオフセット保護**:
- 出力DCを差動増幅器で監視
- |V_DC| > 50 mV で 200ms以内にスピーカ切離
- ボイスコイル焼損防止 (DC電流はLorentz力にならず純発熱)

**過大入力保護**:
- ソフトクリップリミッタ (アタック1ms、リリース100ms)
- 定格 30W RMSの 95%でリミット開始

### B.5 制御工学的設計の思考過程

スピーカは本質的に非線形な機械系。線形T-Sパラメータが成立するのは小信号領域 (約 $X_{max}/3$以下) のみ。大信号での歪を抑える制御戦略を検討する。

#### B.5.1 開ループ vs 閉ループ vs 信号処理

```
方式比較:
  開ループ (Open-loop):
    - 特性をドライバ素性に依存
    - 大入力時THD 3-5% (典型)
    - シンプル、最もポピュラー

  動作フィードバック (MFB):
    - 加速度センサーをコーンに貼付
    - 誤差を増幅器に戻す (Velocity負帰還)
    - THD < 0.5% 達成可
    - 機械的取付が困難、コスト高
    - 1980年代のPhilips MFBで実用化

  サーボ制御 (Servo):
    - レーザー変位計などで $x(t)$ 観測
    - 高帯域フィードバック
    - 業務用サブウーファーで実装例あり
    - 民生では複雑すぎ

  Klippel式 DSP線形化:
    - 事前測定で Bl(x), C_{ms}(x), L_e(i) を関数化
    - リアルタイムDSPで逆変換 (前置歪)
    - THD < 0.5% 達成可、追加センサー不要
    - 個体差に弱い (経時変化に追従できない)

  適応制御 (Adaptive):
    - ドライバモデルをオンライン同定
    - 経時変化に追従
    - 計算負荷大、教会用途には過剰
```

判定: **B.4.2でDSPを採用したので、Klippel式線形化を併用**。

#### B.5.2 Klippel非線形モデル

ドライバの3つの主要非線形性:

**(1) Bl(x): 力係数の変位依存**

ボイスコイルがギャップ中央から外れると $Bl$ が低下:

$$Bl(x) = Bl_0 + Bl_2 x^2 + Bl_4 x^4 + ...$$

(対称性により奇数次は通常小)

**(2) C_{ms}(x): スティフネスの変位依存**

サスペンションがストロークエンドで硬化:

$$\frac{1}{C_{ms}(x)} = K_{ms,0} + K_{ms,2} x^2 + K_{ms,4} x^4$$

**(3) L_e(i): インダクタンスの電流依存**

磁気飽和とポール磁化により:

$$L_e(i) = L_{e,0} + L_{e,1} i + L_{e,2} i^2$$

これらの結果、運動方程式が非線形ODEとなる:

$$m\ddot{x} + R_m\dot{x} + \frac{x}{C_{ms}(x)} = Bl(x) i - S p(t)$$

$$L_e(i)\frac{di}{dt} + R_e i + Bl(x)\dot{x} = V_{in}$$

#### B.5.3 前置歪 (Pre-distortion) の設計

DSP内で逆変換を行う:

```
入力信号 V_target(t)
    |
    +--- 線形ドライバモデルで X_target(t)を計算
    |
    +--- 非線形補正項を計算:
    |       Delta_F_Bl  = (Bl_0 - Bl(X_target)) * I_target
    |       Delta_F_Kms = (Kms_0 - Kms(X_target)) * X_target
    |
    +--- 補正電圧 V_correction を加算
    |
    +--- パワーアンプへ出力
```

実装時:
- 個体測定: Klippel R&D テストシステムで Bl(x), Kms(x), Le(i) を1Hz - 20kHzでスキャン
- 多項式フィット: 4次 - 6次が実用的
- DSPでテーブル参照 + 補間

**補正効果の予測**:
- 50Hz、$X = 4$mmで開ループTHD 4% -> 補正後 0.4%
- 1次倍音 (高調波) を主に低減、相互変調歪も改善

#### B.5.4 実装の階層性

```
教会設置時の信号処理チェーン:

[音源] -> [USB DAC]
            |
            v
       [DSPブロック]
            |
            +-- (1) 部屋補正EQ (測定マイクで較正)
            |
            +-- (2) Klippel前置歪 (ウーファーのみ)
            |
            +-- (3) リミッタ (ピーク制御)
            |
            +-- (4) クロスオーバー (LR4 @ 2.5kHz)
            |    |
            |    +-- LP出力 -> ウーファーアンプ
            |    +-- HP出力 -> ツィーターアンプ
            |

       [パワーアンプ x 4ch]
            |

```

レイテンシ集計: 4.2ms (DSP) + 0.5ms (アンプ) = 4.7 ms。映像同期は不要 (純音声) なので許容。

### B.6 統合とトレードオフ -- Hofmannの鉄則

スピーカ設計の根本トレードオフ:

$$\eta \propto V_b \cdot f_3^3$$

これが「Hofmannの鉄則 (Hofmann's Iron Law)」。3変数のうち 2 を固定すると 1 が決まる。

- 効率 $\eta$ を上げる -> 容積 $V_b$ または低域カットオフ $f_3$ を犠牲に
- 低域 $f_3$ を伸ばす -> 容積 $V_b$ 増 または効率 $\eta$ 犠牲

今回の設計位置:
- $V_b$ = 21 L (中庸)
- $f_3$ = 50 Hz (要求準拠)
- $\eta$ = 0.25% (= 86 dB SPL/W/m、目標 88dBから-2dB)

**改善経路の選択肢**:

```
(A) 大型化       : V_b 40Lに -> eta +3dB、ただし寸法逸脱
(B) 帯域犠牲     : f_3 60Hzに -> eta +5dB、Trompette F2犠牲
(C) より強い磁石 : Bl 8 T.mに -> eta +2.5dB、コスト+8000円
(D) 大口径化     : 8インチに  -> eta +2dB、ツィーター見直し必要

選択: (C) 磁石強化 + (D) 部分採用 (7インチ)
理由: 寸法・帯域は変えず、コストインパクトが容認範囲
```

最終仕様:
- 7インチウーファー、$Bl = 7$ T.m、$\eta = 0.4\%$ -> 88 dB SPL/W/m 達成

### B.7 ピアニカv3との実装比較

設計プロセスを終えて、ピアニカv3 (本文§9.1) と並べる:

| 設計層 | スピーカ (本Appendix) | ピアニカv3 (共鳴箱) |
|--------|---------------------|-------------------|
| 励振源 | ボイスコイル + アンプ + DSP + 電源 | リード37枚 + 風箱 + 足踏みふいご |
| 励振原理 | Lorentz力 (電磁) | 空気動圧 (空気力学) |
| 励振制御 | DSP前置歪、Klippel補正 | 風圧調整のみ (奏者の足) |
| 放射体素材 | 強化紙コーン (12g) | 桐板 1.5mm (約100g) |
| 放射体駆動 | コイル直結 (硬結合) | 駒経由 ($T = 97\%$) |
| 容器役割 | 双極子相殺の防止 (受動) | 共鳴・放射の主体 (能動) |
| 容器素材 | バーチ合板 + 制振 | 桐 + X-brace |
| 内部空気 | バスレフ Helmholtz共鳴 | 同 + 板共振との結合 |
| 周波数応答 | フラット (DSPで強制) | 多重ピーク (個性) |
| 効率 | 0.4% (=88 dB SPL/W/m) | 36% (放射効率) |
| 部品点数 | 約300点 (電子部品含む) | 約100点 |
| エネルギー源 | 系統電源 (60-200W) | 人力 (足踏み 10-30W) |
| 故障要因 | コンデンサ寿命15年、エッジ加水分解5-15年 | 桐板割れ、ガスケット劣化 |

**この比較から得られる洞察**:

1. **「シンプルさ」の方向性が逆**: スピーカは電気部品の精度で性能を作り込む。共鳴箱は素材と幾何で性能を決める。
2. **エネルギー源の差**: スピーカは系統電源 (停電で無音)、ピアニカは人力 (どこでも演奏可能)。礼拝堂用としては後者の利点もある。
3. **個性の扱い**: スピーカは個体差を排除する (DSPで補正)。共鳴箱は個体差を「個性」として保存する。
4. **効率の桁差**: 共鳴箱36% vs スピーカ0.4% -> **90倍**の差。これはエネルギー変換段数の差 (1段 vs 3段) と各段損失の累積による必然。

### B.8 まとめ -- 設計プロセスの構造

スピーカ設計は以下の階層的決定の連鎖である:

```
要求仕様 (B.1)
    |
    v
材料選択 (B.2): T-S目標 -> Bl, m_ms, R_e, C_ms 確定
    |
    v
電気的アラインメント (B.3): V_b, f_b, クロスオーバー 確定
    |
    v
電子系統 (B.4): アンプ、DSP、保護 確定
    |
    v
制御戦略 (B.5): 線形化、適応性 決定
    |
    v
統合検証 (B.6): Hofmann鉄則の中で位置決め
    |
    v
最終仕様確定
```

各層は前層の出力を入力とし、選択肢が制約により絞り込まれる。**これは「電気エネルギーを音響エネルギーに変換する機械」の設計言語**であり、楽器の共鳴箱を設計する言語とは語彙も文法も異なる。

ピアニカv3の設計プロセス (機械励振 + 共鳴板選択 + 容積調整) は、本Appendixのスピーカ設計プロセスとは独立した、楽器音響学の体系に属する。両者を混同しないことが、それぞれの最良設計を導く出発点である。

## Appendix C: 主聖堂 (カトリック神戸中央教会) 用音響強化システム設計

Appendix B はピアニカv3に対応する規模 (容積100 m^3、聴衆 25人) の小礼拝室を扱った。本Appendix C では、同教会の **主聖堂** (推定容積 4,500 m^3、聴衆 300人、白煉瓦壁、残響3-4秒) を対象とし、同じ設計プロセス (要求仕様の導出 -> 実装) を適用する。

両者の比較から、容積が約45倍、聴衆が約12倍、残響時間が約3倍となる主聖堂では、設計の **質的パラダイム** が「Hi-Fiスピーカ」から「音響強化システム (Sound Reinforcement)」へ移行することが示される。

### C.0 序 -- 主聖堂と小礼拝室の質的差異

設計目標を立てる前に、両者がなぜ違う設計言語を要求するかを示す:

| 観点 | 小礼拝室 (Appendix B) | 主聖堂 (Appendix C) |
|------|--------------------|-------------------|
| 容積 | 100 m^3 | 4,500 m^3 (45倍) |
| 残響時間 RT60 | 1.2 s | 3.5 s (3倍) |
| 聴衆数 | 25人 | 300人 (12倍) |
| 最遠聴取距離 | 6 m | 25 m (4倍) |
| 主用途 | 音楽再生 (Trompette 8') | ミサ典礼 (発話 + 聖歌 + オルガン) |
| 言語数 | 1 (録音) | 6 (日本語/英語/ポルトガル語/インドネシア語/韓国語/タガログ語) |
| 設計支配パラメータ | 周波数応答 + 感度 | **STI (発話明瞭度)** + 均一度 |
| アーキテクチャ | 単一ペアブックシェルフ | 分散ライン アレイ + DSP + ディレイフィル |
| 予算 | 8万円 | 500-800万円 |

主聖堂では「音を出す」ことより「**残響に勝って言葉を伝える**」ことが支配的。これがシステム設計の根本を変える。

### C.1 要求仕様の導出と設計目標の設定

#### C.1.1 序 -- 主聖堂特有の設計入力

Appendix B で導入した「要求仕様の導出は設計の最上流工程」の原則は本章でも維持する。ただし、主聖堂では以下の追加入力が支配的となる:

```
追加入力 (主聖堂固有):
  + 典礼様式 (カトリック・ミサ式典)
  + 多言語運用 (6言語)
  + 建築意匠との両立 (白煉瓦・ステンドグラス・大屋根)
  + 補聴器対応 (高齢聴衆対応)
  + 残響制御の物理限界 (建築固定後の音響改善)
```

これら入力が、Appendix B では現れなかった **STI (発話伝達指数)** や **音場均一度** を設計指標として要求する。

#### C.1.2 音響環境の解析 -- 主聖堂の物理特性

設計事務所公開情報 + 一般的なカトリック中規模教会の典型値から推定:

**幾何学的パラメータ (推定)**:
- 平面: 25 m (奥行) x 18 m (幅) = 450 m^2 (扇形 or 矩形)
- 天井高: 8 m (側壁) - 12 m (中央峰、緑青銅板大屋根)
- 容積: $V \approx 4{,}500$ m^3
- 表面積: $S \approx 1{,}800$ m^2 (壁 + 天井 + 床)
- 座席: 約 300席 (固定木製ベンチ、聖堂家具は方圓館製)

**素材別吸音率 (聖堂内装)**:

| 素材 | 表面積 m^2 | $\alpha$ (中域 500Hz) | 吸音 sabin |
|------|-----------|---------------------|-----------|
| 白煉瓦壁 | 600 | 0.04 | 24 |
| 木製天井 (内側) | 450 | 0.10 | 45 |
| 石床 | 450 | 0.02 | 9 |
| ステンドグラス (16枚) | 50 | 0.04 | 2 |
| 木製ベンチ (空席) | 80 | 0.10 | 8 |
| 装飾 + 残部 | 170 | 0.05 | 8.5 |
| **小計 (空席時)** | 1800 | **平均 0.054** | **96.5** |

聴衆300名満席時 (1人あたり追加吸音 0.4 sabin):

$$A_{satou} = A_{empty} + 0.4 \times 300 = 96.5 + 120 = 216.5 \text{ sabin}$$

**残響時間**:

Sabineの式により:

- 空席時: $RT_{60} = 0.161 \times 4500 / 96.5 = 7.5$ s (**過剰**)
- 満席時: $RT_{60} = 0.161 \times 4500 / 216.5 = 3.3$ s
- 半席 (150名): $RT_{60} \approx 4.5$ s

**この大きな変動 (3.3 - 7.5s) が音響設計の最大の困難** である。空席時の練習・録音は別問題、ミサ運用時は満席を基準とする。

**部屋定数**:

$$R = \frac{S \bar{\alpha}}{1 - \bar{\alpha}}$$

満席時: $\bar{\alpha} = 216.5/1800 = 0.12$

$$R = \frac{1800 \times 0.12}{1 - 0.12} = \frac{216}{0.88} = 245 \text{ m}^2$$

**臨界距離**:

$$r_c = 0.141 \sqrt{R \cdot Q}$$

無指向性スピーカ ($Q=1$): $r_c = 0.141 \sqrt{245} = 2.2$ m
指向性高 ($Q=10$、線アレイ的): $r_c = 0.141 \sqrt{2450} = 7.0$ m

**設計示唆**: 聴取距離 5-25 m に対し、無指向性スピーカでは全聴衆が深い残響場 -> 明瞭度崩壊。指向性 $Q \geq 10$ のスピーカで初めて中央席までは直接音優勢にできる。**指向性制御がC設計の根本要請** となる。

**暗騒音レベル**:

中山手通沿いの立地 (神戸市中央区)、外部交通騒音が侵入。教会としては典型的:
- 静寂時: NC-30 (32 dBA)
- ミサ前後: NC-35 (37 dBA、聴衆の動きによる)
- 設計基準: 35 dBA

#### C.1.3 音源プロファイル -- ミサ典礼の音響

主聖堂で扱う音源は3種:

**(1) 発話 (聖職者・朗読者・聴衆応答)**

- 男性声 基本周波数 $F_0 = 100-150$ Hz、女性声 200-300 Hz
- 周波数帯域 (音声明瞭度に必要): 250 Hz - 4 kHz (ITU-T G.711 が電話帯域として採用)
- 拡張帯域 (自然な音声): 100 Hz - 8 kHz
- 動的範囲: 25-30 dB
- 母音/子音SNR: 子音は母音より 30 dB 低い -> 雑音に弱い

**設計示唆**: 発話帯域 (1-4 kHz) の歪・響きが最重要。

**(2) 聖歌 (聴衆 + 聖歌隊 + オルガン)**

- 周波数: 60 Hz (パイプオルガン 16') - 12 kHz
- ダイナミックレンジ: 30-40 dB (ピアニッシモ - フォルティッシモ)
- スペクトル: オルガン基音 + 倍音、人声フォルマント
- 持続音 + クレッシェンド: 発話より長時間応答が要求される

**(3) パイプオルガン (もし設置されていれば)**

主聖堂にパイプオルガンが導入されている場合:
- Trompette 8'は本Appendix B と同じ条件
- 16' ペダルストップを含む場合: 32 Hz まで
- 音響強化対象外 (パイプオルガンは自身で十分な音圧を発生)

ただし、ミサ録音や同時通訳音声等で **オルガン音の電子的補強** が必要な場合は対象。本設計では補強対象外と仮定。

**多言語ミサ運用**:

時刻表から読み取れる典型運用:
- 9:30 日本語ミサ (主聖堂)
- 11:15 英語ミサ (主聖堂)
- バイリンガルミサ 10:00 (時々)
- 第1日 ポルトガル語、第3日 インドネシア語、第3土 韓国語、第5日 タガログ語

**6言語に対し均一な明瞭度を確保することが運用要件**。各言語の周波数特性は微妙に異なる:
- 日本語: 母音優勢、エネルギー 200-2000 Hz集中
- 英語: 子音重要、4-8 kHz の伸びが必要
- ポルトガル語: 鼻母音、500 Hz - 3 kHz
- 韓国語/タガログ語: 摩擦子音多い、4-6 kHz

**設計示唆**: 8 kHz までフラットな再生が必要 (英語・摩擦音が最も厳しい)。

#### C.1.4 聴取者要求 -- 多言語明瞭度の心理音響

**STI (Speech Transmission Index) の重要性**:

ミサでは聖職者の発話が聴衆全員に届かないと典礼が成立しない。STI はこれを定量化する指標:

| STI | 評価 | 用途 |
|-----|------|------|
| 0.00 - 0.30 | Bad | 単語識別不能 |
| 0.30 - 0.45 | Poor | 困難、推測必要 |
| 0.45 - 0.60 | Fair | 一般教会の最低基準 |
| 0.60 - 0.75 | Good | 多言語運用の最低基準 |
| 0.75 - 1.00 | Excellent | 教育・劇場相当 |

主聖堂 (RT60=3.3s) の **音響強化なし STI** は 0.25-0.35 (Bad-Poor)。これでは外国語朗読は機能しない。

**6言語運用の場合**: 母語話者には文脈推測が効くが、第二言語話者・非母語話者には推測補完が困難。よって **STI > 0.60 (Good)** を全座席で確保する必要がある。

**心理音響的補正**:

Houtgast-Steeneken の研究によれば、STI と音声了解度 (intelligibility) の関係は:
- STI = 0.45: 単語了解度 70%、文章了解度 90%
- STI = 0.60: 単語了解度 90%、文章了解度 99%
- STI = 0.75: 単語了解度 95%以上、ほぼ完全

母語話者が対象なら STI = 0.45 でも文章は理解できるが、外国語朗読 (例: 韓国語ミサに参加する日本人信徒) では 0.60 必須。

**音声 SNR 要求**:

聴衆位置での音声レベル: 暗騒音 (35 dBA) に対し +25 dB 以上 -> **60 dBA 以上**。実際の運用では平均 75-80 dBA でピーク 90 dBA。

**ダイナミックレンジ要求**:

ミサの一部 (沈黙の祈り、聖変化等) では音圧 50 dBA まで下がる。一方、聖歌・オルガン強奏時は 95 dBA。**45 dB のダイナミックレンジ** を歪なく再生する必要。

#### C.1.5 SPL要求の導出計算

**Step 1: 聴衆位置での要求**

- 発話 (平均): 75 dBA
- 発話 (ピーク): 85 dBA
- 聖歌 (平均): 85 dBA
- 聖歌 (ピーク): 95 dBA
- 設計基準: **ピーク 95 dBA at 全聴衆位置**

**Step 2: スピーカ位置からの距離**

主聖堂の音響強化は **分散システム** が前提 (単一スピーカでは指向性 + 距離で破綻)。

```
配置案 (典型的なカトリック教会):

  +------------+
  |  祭壇      |     <-- 主スピーカ (左右ライン アレイ)
  |  +-+  +-+  |
  +--|S|--|S|--+
     +-+  +-+
       |    |
      [中央通路]
       |    |
  +-+  通路  +-+
  | |        | |   <-- ディレイフィル (中段)
  | |        | |
  | | 座席   | |
  | |        | |
  +-+        +-+
       |    |
  +-+  通路  +-+
  | | (後段)| |   <-- ディレイフィル (後段)
  | |        | |
  +-+        +-+
```

各聴衆位置から最寄スピーカまで:
- 前方席: 主アレイから 5-8 m
- 中段: 中段ディレイフィルから 4-6 m
- 後段: 後段ディレイフィルから 4-6 m

**設計目標**: 全聴衆位置で「最寄スピーカから 8 m 以内」。

**Step 3: スピーカ感度要求**

最寄スピーカから 8m地点でピーク 95 dBA を達成するには:
- 距離減衰: 1m -> 8m で純自由音場 18 dB低下、しかし残響場では 6-9 dB低下
- 設計値: 8 dB低下とする
- スピーカ 1 m地点で必要: 95 + 8 = **103 dB SPL**

ピーク入力 200 W (PA級) で:
$$\eta_{required} = 103 - 10\log_{10}(200) = 103 - 23 = 80 \text{ dB SPL/W/m}$$

しかし PA級スピーカは 95-100 dB SPL/W/m が標準 (高感度ホーン使用)。マージン 15-20 dB あり、これが **歪低減** と **長期信頼性** に充当される。

**仕様値**: スピーカ感度 95 dB SPL/W/m、定格入力 300 W RMS。

**音場均一度要求**:

聴衆位置間の SPL差 < 6 dB (前方-後方、左右)。これが「均一度 6 dB」要件で、ディレイフィル配置で達成する。

#### C.1.6 周波数応答要求の導出

**下限**: 聖歌・オルガン補強用 -> 60 Hz (パイプオルガン 16' は補強対象外、聖歌の男声低音 100 Hz が実用下限)

仕様: **70 Hz以下を確保 (-3dB点)**。サブウーファ追加で 40 Hz まで拡張可能 (オプション)。

**上限**: 多言語明瞭度のため 8 kHz以上必要 (B.1.3, 摩擦子音再生)

仕様: **15 kHz (-3dB)**。20 kHz は教会用途には過剰、ホーンの高域劣化を許容範囲に。

**許容偏差**: STI への影響を最小化するため、200 Hz - 4 kHz (発話帯域) を **+/-2 dB** にタイト管理。それ以外は +/-3 dB。

#### C.1.7 STI (発話伝達指数) 要求の導出

これは Appendix B には存在しなかった新しい要求項目。

**STI計算式 (簡略)**:

$$STI = \sum_{f, F} w(f, F) \cdot M(f, F)$$

ここで:
- $f$: 7オクターブバンド (125 Hz - 8 kHz)
- $F$: 14変調周波数 (0.63 - 12.5 Hz)
- $M(f, F)$: 変調伝達関数 (Modulation Transfer Function)

簡易式 (RASTI):

$$M = \frac{1}{\sqrt{1 + (2\pi F \cdot RT_{60}/13.8)^2}} \cdot \frac{1}{1 + 10^{-SNR/10}}$$

**残響成分**: $M_{rev} = \frac{1}{\sqrt{1 + (2\pi F \cdot RT_{60}/13.8)^2}}$
**雑音成分**: $M_{noise} = \frac{1}{1 + 10^{-SNR/10}}$

主聖堂の数値検証:
- $RT_{60} = 3.3$ s (満席)
- 変調周波数 $F = 4$ Hz (典型音節率)
- $SNR = 25$ dB (発話 60 dBA、暗騒音 35 dBA)

$$M_{rev}(4 \text{Hz}) = \frac{1}{\sqrt{1 + (2\pi \times 4 \times 3.3/13.8)^2}} = \frac{1}{\sqrt{1 + 36}} = 0.164$$

$$M_{noise}(25 \text{dB}) = \frac{1}{1 + 0.0032} = 0.997$$

$M_{total} = 0.164 \times 0.997 = 0.164$、これに応じる **STI ≈ 0.30 (Bad)**。**音響強化なしでは典礼成立しない**。

**指向性制御による改善**:

スピーカに指向性 $Q = 10$ を持たせると、聴衆位置での残響成分は約半減:
- 直接/残響比改善で実効 RT$_{60} \to 1.5$ s 相当

$$M_{rev}'(4 \text{Hz}) = \frac{1}{\sqrt{1 + (2\pi \times 4 \times 1.5/13.8)^2}} = \frac{1}{\sqrt{1 + 7.45}} = 0.342$$

**STI ≈ 0.55 (Fair)** -- 改善はするが目標 0.60 には届かない。

**分散ディレイシステムの効果**:

ディレイフィルにより最寄スピーカ距離 4-6 m に短縮、直接音圧 +6 dB:

$$STI \approx 0.65 \text{ (Good)}$$

**仕様値**: 全聴衆位置で **STI > 0.60 (Good)** 達成。これが「分散ディレイシステムを採用する物理的根拠」となる。

#### C.3.2 ライン アレイ垂直カバレッジの計算

各素子 6.5インチ、4素子で構成、素子間隔 d = 200 mm。

**主ローブ幅** (ニアフィールド条件):

$$\theta_{-6dB} = 2 \arcsin\left(\frac{0.43 \lambda}{N \cdot d}\right)$$

中域 1 kHz、$\lambda = 0.343$ m、$N=4$、$N \cdot d = 0.8$ m:

$$\theta = 2 \arcsin(0.184) = 21.2 \text{度}$$

**サイドローブ抑制**: シェーディング (各素子の電圧重み付け) で-15 dB以下。

**全聴衆カバレッジ**:

主アレイ高さ 4 m、聴衆距離 5-12 m。垂直カバ角度:
- 最近 (5m, 高低差 3m): 31度
- 最遠 (12m, 高低差 3m): 14度

合計垂直角 17度、ライン アレイ -6dB幅 21度に収まる。**設計適合**。

#### C.3.3 ディレイ計算

主アレイ ($d_0 = 0$ m位置) から各ディレイフィルまでの距離:
- 中段フィル: 12 m
- 後段フィル: 22 m

音速 343 m/s (室温 20°C):

$$\tau_{mid} = 12/343 = 35.0 \text{ ms}$$
$$\tau_{rear} = 22/343 = 64.1 \text{ ms}$$

**Haas効果による聴感**: 5-30 ms以内なら定位は先行音源 (主アレイ) に統合。35 ms はこれをやや超え、音源が分裂しがち。

**対策**: 後段フィルに **わずかに早い** 設定 (実距離より -5 ms) すると、聴感的には主アレイ起点に統合される (心理音響的微調整)。

**最終ディレイ設定**:
- 中段: 30 ms (実 35 ms - 5 ms)
- 後段: 59 ms (実 64.1 ms - 5 ms)

### C.4 電子的設計の思考過程

#### C.4.1 自動マイクミキサ (Automixer)

複数の聖職者・朗読者マイクが同時アクティブだとフィードバック・残響増大の元。**自動ゲーティング** で発話中マイクのみ開く:

```
方式:
  Gain-sharing Automixer (Dan Dugan型):
    - 全マイクで音圧レベル比較
    - 最大1本のみ全ゲイン、他を-20 dB減衰
    - 切り替えスムース (ramp 50ms)

  Threshold Gate型:
    - 各マイクに閾値設定
    - 閾値超えで開、低下で閉
    - 急峻 (ノイズ拾いリスク)

判定: Dan Dugan型 (Biamp/QSC内蔵機能)
理由:
  - 切り替え音が出ない
  - 同時複数発話に対応 (ゲイン共有)
  - 教会用途で実績多数
```

#### C.4.2 フィードバック抑制

主聖堂のような広帯域・高Q残響空間ではフィードバック発生周波数が変動する。

**自動抑制 (Automatic Feedback Suppression)**:
- 24個のナローノッチフィルタ (1/24 oct)
- 増幅前のFFT解析でピーク検出
- ピーク検出 -> 0.5秒以内にノッチ挿入

**従来運用**: ミサ前にスタッフが手動 EQ調整。AFS導入で **無人運用が可能** に。

**ゲインビフォアフィードバック (GBF)**:

理論限界:

$$GBF = 30 - 10\log_{10}(N \cdot D)$$

($N$ = 開いているマイク数、$D$ = マイク-スピーカ距離 m)

主聖堂 (主アレイ -> アンボマイク 4m、聖歌隊マイク 8m) で $N=2$ 時:
$GBF = 30 - 10\log_{10}(2 \times 4) = 30 - 9 = 21$ dB

AFSで +6 dB を稼ぐ -> 27 dB の実効増幅可能。**多くの実用条件をカバー**。

#### C.4.3 部屋音響補正

満席/空席で残響が 7.5 -> 3.3 s と変動。常時補正のため:

**測定マイク**: 中段に2本のリファレンスマイク
**補正アルゴリズム**: FIR適応フィルタ、毎15分更新
**補正対象**: 200 Hz - 4 kHz の発話帯域 +/-3 dB をフラットに

これはAppendix B (DSPはあれば良いが必須でない) と異なり、**主聖堂では運用必須**。

#### C.4.4 緊急放送対応

EN 60849 準拠:
- 緊急マイク入力時、全他入力 -20 dB
- 全スピーカ強制ON、95 dBA以上
- 自己診断機能 (毎日テストトーン)

これも Appendix B には無い要件。

### C.5 制御工学的設計の思考過程

主聖堂の制御課題は Appendix B (個別ドライバの非線形性) より広範:
- 室温・湿度による音速変化 (ディレイずれ)
- 観衆数変動による残響変化
- 多マイク同時運用のフィードバック
- 全システムの自己診断・保護

#### C.5.1 適応ディレイ補正

音速 $c$ は温度 $T$ で変化:

$$c = 331.3 \sqrt{1 + T/273.15}$$

冬 5°C: 334.5 m/s、夏 30°C: 349.1 m/s。差 4.4%。

22m地点のディレイ:
- 冬: 22/334.5 = 65.8 ms
- 夏: 22/349.1 = 63.0 ms
- 差 2.8 ms

これは Haas window (5-30 ms) 内では問題ないが、聖歌隊マイクとオルガンの位相整合では識別される (1ms = 1度at 1kHz)。

**対策**: 室温計連動でディレイ自動補正 (5°C毎、0.5 ms単位)。

#### C.5.2 適応 FIR (聴衆数による残響補正)

満席 RT=3.3s、空席 RT=7.5sの中間 (200名なら 4.5s)。これに対し:

**戦略**:
- 入口センサで聴衆数推定 (or マニュアル入力)
- 残響推定値からFIR係数選択 (3つのプリセット: 空席/半席/満席)
- 切替はミサ開始時に行う (途中変更しない)

#### C.5.3 自己診断・保護

- 各スピーカへインピーダンス監視 (異常検出)
- アンプ温度監視 (ドライバプロテクション)
- 停電時バッテリーバックアップ (緊急放送 30分)
- ログ記録 (障害発生時に解析)

これらは Hi-Fi では過剰、PA運用では必須。

### C.6 統合とトレードオフ

主聖堂設計の Hofmann 鉄則類似式は SPL・カバレッジ・コストの3変数:

$$\text{Cost} \propto \text{SPL} \cdot \text{Coverage Volume}^{1/2}$$

ここで:
- SPL: 最大音圧能力
- Coverage Volume: カバー対象空間体積

主聖堂 (4500 m^3、95 dB ピーク) は小礼拝室 (100 m^3、88 dB ピーク) に対し:
- Volume 比 45倍 -> sqrt = 6.7倍
- SPL 比 +7 dB -> 2倍

予測コスト比: 6.7 x 2 = **13倍**

実際のコスト比: 569万 / 8万 = **71倍**

「物理スケーリングだけでは説明できない約5倍の追加コスト」は:
- DSP の必要性 (+2倍)
- 多マイク運用 (+1.5倍)
- 緊急放送 + 補聴器対応 (+1.3倍)
- 業務用品質要求 (信頼性、長寿命) (+1.5倍)

これらは Appendix B と Appendix C の **質的設計差** に由来する追加コスト。

### C.7 神戸中央教会・固有要件への適応

#### C.7.1 白煉瓦壁の音響的扱い

白煉瓦は $\alpha = 0.04$ と極めて低吸音。建築意匠を変えずに音響改善する手段:

**(1) 壁面吸音材は不可** (建築意匠維持)
**(2) 天井・床下に隠蔽吸音 (改修時のみ)**: 木製天井裏にロックウール100mm
**(3) 主対策はスピーカ指向性で残響を「避ける」**: 本Appendix採用方針

#### C.7.2 ステンドグラス前を遮らない配置

16枚のステンドグラスは視覚意匠上重要。スピーカ配置ルール:
- ステンドグラスの正面は完全クリア
- 主アレイは祭壇 **両側柱** 取付 (ステンドグラスから 1m以上離隔)
- ディレイフィルは内壁 **柱埋込** で目立たず

#### C.7.3 多言語ミサ運用への適応

日本語・英語・ポルトガル語・インドネシア語・韓国語・タガログ語の6言語。

**プリセット設計**:
- 日本語ミサ: 200 Hz - 3 kHz重視 (母音優勢)
- 英語ミサ: 4-8 kHz +1 dB ブースト (子音強調)
- バイリンガル: 中庸セッティング

これらをプリセットボタン1-6に割当て。

#### C.7.4 神戸の地震リスクへの適応

阪神淡路大震災 (1995) の被災地に建つ教会。**地震時に天井/壁取付物が落下しない設計**:

- ライン アレイ: 落下防止セカンダリワイヤ (耐震)
- DSPラック: 床固定アンカー
- 緊急放送: 停電後30分動作 (避難誘導用)

これは設計事務所 (村上晶子アトリエ) の建築設計思想 (震災復興教会) と整合する。

### C.8 Appendix B との比較 -- スピーカ設計のスケール則

最後に Appendix B (小礼拝室) と Appendix C (主聖堂) を13項目で対比:

| 項目 | Appendix B (小礼拝室) | Appendix C (主聖堂) | 差比 |
|------|--------------------|-------------------|-----|
| 容積 | 100 m^3 | 4,500 m^3 | 45x |
| 残響時間 RT60 | 1.2 s | 3.3 s (満席) | 2.8x |
| 聴衆 | 25人 | 300人 | 12x |
| スピーカ数 | 2個 (Hi-Fiペア) | 8個 (主+フィル+サブ) | 4x |
| 入力チャンネル数 | 2 (ステレオ) | 16 (マルチマイク+ライン) | 8x |
| 出力チャンネル数 | 2 | 4 (主左右+フィル+サブ) | 2x |
| アンプ容量 | 60 W (30W x 2) | 2,000 W (500W x 4) | 33x |
| DSP の役割 | オプション (部屋補正) | 必須 (Automixer + AFS + FIR) | 質的差 |
| 設計支配指標 | 周波数応答 | **STI > 0.60** | 質的差 |
| 感度要求 | 88 dB SPL/W/m | 95 dB SPL/W/m | +7 dB |
| 部品点数 | 約 300点 | 約 2,500点 | 8x |
| 設置工事 | DIY可能 | 専門業者必須 (耐震・配線・調整) | 質的差 |
| 予算 | 8万円 | 569万円 | 71x |

**スケーリング則の観察**:

```
容積 45倍 -> アンプ容量 33倍 (Hofmann ~)
残響 3倍 -> DSP の役割 オプション -> 必須 (質的相転移)
聴衆 12倍 -> 入力チャンネル 8倍 (マルチマイク必須)
予算 71倍 (純物理スケーリング 13倍 x 質的因子 5倍)
```

主聖堂では「スピーカ」というより「**音響強化システム (Sound Reinforcement System)**」 と呼ぶべきもの。これはピアニカv3 (共鳴箱) -> 小礼拝室スピーカ (Hi-Fi) -> 主聖堂SR の3段階のうち、最も後者 2段階の差が大きいことを示す。

### C.9 まとめ -- 規模が変える設計言語

主聖堂と小礼拝室は「どちらもスピーカ」だが、設計言語は完全に異なる:

```
小礼拝室 (Appendix B): Hi-Fi スピーカ設計
  目標: 入力信号を忠実に音波化
  支配指標: 周波数応答、THD、感度
  設計言語: Thiele-Smallパラメータ、エンクロージャ理論
  類比: 静寂な室で音楽鑑賞

主聖堂 (Appendix C): Sound Reinforcement 設計
  目標: 残響に勝って明瞭な発話を全聴衆に届ける
  支配指標: STI、音場均一度、フィードバック余裕
  設計言語: 指向性制御、ディレイ整合、自動制御
  類比: 集会場・劇場で発話を伝達
```

ピアニカv3 (共鳴箱) を加えた 3段階構造で、**設計言語の相転移点が2つ存在する**:

```
共鳴箱 (ピアニカv3)
   |  <-- 相転移1: 機械音響 -> 電気音響
   v
Hi-Fi スピーカ (Appendix B)
   |  <-- 相転移2: Hi-Fi -> SR (規模・残響・多様性)
   v
SR システム (Appendix C)
```

各段階で「上位の設計言語が下位を吸収する」のではなく、**それぞれが独立した最適解** を持つ。共鳴箱はHi-Fiスピーカの代替ではなく、Hi-FiスピーカはSRシステムの小規模版ではない。

設計者はまず「どの段階の問題か」を識別する必要があり、これが本シリーズ Appendix B + C の最大の教訓である。
