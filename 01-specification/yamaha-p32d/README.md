# YAMAHA Pianica P-32D

**ステータス**: ヤマハ公式仕様書の調査要

## 識別根拠

本リポジトリでP-32Dと識別している分解写真12枚 (`images/`) について、識別根拠は以下:

- リードプレートが**真鍮製 (金色)** - CAHAYAの銀色アルミと区別
- 筐体が**水色** - ヤマハのブルー/ピンク/オレンジ系統と整合 (P-32E系の色展開)
- バルブが**黄色プラスチック** - 日本式の典型
- 鍵盤数およそ32 - P-32Dの規格と整合

## 公式仕様 (P-32E系として参考、P-32Dは要調査)

参考: ヤマハ P-32E/EP/EO の公式仕様 (2026年現在の関連モデル)

| 項目 | 値 |
|---|---|
| 鍵盤数 | 32 |
| 音域 | f - c''' (F3-C6, 175Hz - 1047Hz) |
| 本体寸法 | 425 × 96 × 49 mm |
| ケース寸法 | 456 × 173 × 58 mm |
| 質量 | 600 g (ケース含まず) |
| 色展開 | ブルー (P-32E) / ピンク (P-32EP) / オレンジ (P-32EO) |

P-32Dは旧モデルであり、上記P-32E系と同等規格と推定されるが、**正式な公式仕様の確認が必要**。

## 22経路解析での扱い

P-32Dは本プロジェクトの**22経路解析の主対象機**。以下の主要発見はP-32Dを基準としている:

- 外部放射の95.2%はABS壁面由来 (C1+C2)
- 鍵盤穴経由 (A1) はわずか0.8%
- ABS底面(1,1)曲げモード 323Hz (E4近傍) が音色の核
- リード→プレート→ABS壁→外部放射 の3層構造 (B系)

詳細: `../../02-physics/22-paths/`

## CAHAYAとの対比

| 項目 | P-32D (ヤマハ) | CAHAYA 32鍵 |
|---|---|---|
| 音域・鍵数 | F3-C6, 32鍵 | F3-C6, 32鍵 (同一) |
| リード材質 | 真鍮 (金色) | アルミ (銀色) |
| バルブ色 | 黄色 | 白色 |
| 筐体構造 | カバー+空気室 別部品 (日本式) | 一体成形 (中華式) |
| リード設計思想 | 弦楽器型2段駒 (鈴木と同系) | グランドピアノ型1段駒 |
| 価格帯 | ~5,000円 | ~2,500円 |
