## 識別根拠

本リポジトリでCAHAYA 32鍵モデルと識別している分解写真11枚 (`images/`) について、識別根拠は以下:

- リードプレートが**銀色アルミ製** - ヤマハP-32Dの真鍮（金色）と区別
- バルブが**白色プラスチック** - ヤマハP-32Dの黄色と区別

## 内部構造（実機分解観測）

### 構造の階層

```
[マウスピース挿入口]   バス端側面、円形断面の単純貫通
    |
    v 息
[上ケース内側 = 単一風箱(plenum)]
    - 中央に縦リブ2本（補強用、見た目は仕切に見えるが実機では単一plenum）
    - マウスピース直下にネジボス（ケース結合用、仕切りではない）
    - ネジボス計4個（角4個）+ 中央2個（白い樹脂カバー）
    - spit valve (水抜き) 機構が金属スライドで1か所
    - ユーザ後付けの透明制振フィルム（CAHAYA出荷時にはなし）
    |
    v 息が拡散
[32個のバルブ穴 (鍵盤裏)]
    - 白色プラスチックバルブが2列に交互配置
    - 鍵を押すとバルブが開く
    |
    v
[リードプレート (32鍵分のフリーリード)]
    - 銀色アルミプレート + 銅色リード
    - リードはプレート下面側に固定、スロットを通って振動
    - 各リードに固定ネジ/リベット1個
    - リード長は段階的（低音=長、高音=短）
    |
    v
[外気へ放射] (C系: 底面 + 側面、22経路解析でC1+C2が95.2%)
```

### 機構の確定事項（Step 1観測 + ジャーナル統合）

| 項目 | 内容 | 確定根拠 |
|---|---|---|
| 空気経路 | マウスピース口 -> 単一plenum -> 該当バルブ -> 該当リード -> 外気 | 画像5・7・10 |
| マウスピース口の位置 | 本体長手のバス端側面（CAHAYAロゴ側）、円形断面の単純貫通 | 画像7・11 |
| spit valve | 上ケース内、金属スライド式1か所 | 画像3・5 |
| 中央リブ | 構造リブ（補強用）、風路は単一plenum | 実機観測（journal 0430k） |
| リードプレート構造 | 銀色アルミ + 銅リード32枚、2列交互配置 | 画像6・8 |
| バルブ機構 | 白色プラスチック32個、2列交互、押鍵で開放 | 画像2・10 |
| ガスケット | **なし**（ガスケットレス設計、プラ枠同士の直接ネジ締結） | 実機観測（journal 0430q） |
| 筐体方式 | 中華式一体成形（筐体 + 空気室一体） | 画像1・8 |
| 制振フィルム | ユーザ後付け（CAHAYA出荷時にはなし） | ユーザ証言（journal 0430q） |

### 重要な構造特徴

- **単一風箱構造**: 仕切りなし、空気は風箱全体に拡散
- **中華式一体成形**: 筐体 + 空気室 = 一体（日本式は別部品）
- **マウスピース直下はネジボス**: 仕切りや音響的機能はなし
- **ガスケットレス**: プラ枠同士を直接ネジで締結のみ

## 22経路解析での扱い（構造差で経路数が異なる）

**重要な訂正**: CAHAYAはP-32D基準の22経路がそのまま適用される **わけではない**。中華式一体成形 + ガスケットレス構造のため、P-32D前提の数経路が消失または再定義される。

### 構造差による経路の変化

P-32D前提（日本式）が依存する3つの構造要素:
1. **別部品の外装カバー**（ABS箱を覆うプラスチックケース、4本ネジで結合）
2. **プレートパッキン**（リードプレートとABS上面の間の紙製パッキン）
3. **フレームパッキン**（ABS箱とフレームの間のゴムパッキン）

CAHAYAではこれら3要素がすべて欠落:
- 一体成形のため**外装カバーが存在しない**（ABS外壁が直接外気と接する）
- ガスケットレス設計（journal 0430q）のため**プレートパッキン・フレームパッキンとも欠落**

### 経路別の状態（22 -> 約17経路）

| 系列 | P-32D基準の22経路のうち | CAHAYAでの状態 |
|---|---|---|
| 消失 | B3 (プレートパッキン経由), B5 (フレームパッキン経由), B6 (外装カバーへのネジ伝導), C4 (外装カバー再放射) | 4経路が物理的に存在しない |
| 再定義 | B7 (外装カバー -> テーブル/手) | 起点が「ABS外壁 -> テーブル/手」に変更（経路自体は維持）|
| 維持 | A1-A5, B1a, B1b, B2, B4, C1, C2, C3, C5, D1-D3 | 17経路はそのまま適用可能 |

参考: `../../02-physics/22-paths/pianica_22paths_analysis.md` の付録分類表
詳細: `../../02-physics/22-paths/pianica_chinese_style_paths_analysis.md` （中華式の17経路を物理的に再導出した分析）

### 維持される主要発見（CAHAYAでも継承可能）

CAHAYAで構造的に維持される17経路は、22経路解析の主要発見の物理的核心を担う:

- 外部放射の **95.2% は ABS壁面由来 (C1+C2)** -> CAHAYAでも維持（C1の支配性はむしろ増大）
- 鍵盤穴経由 (A1) はわずか **0.8%** -> CAHAYAでも維持
- ABS底面(1,1)曲げモード **323Hz (E4近傍)** が音色の核 -> CAHAYAでも維持
- リード -> プレート -> ABS壁 -> 外部放射 の3層構造（B系） -> CAHAYAでも維持

### CAHAYA固有の物理的含意

- 中華式一体成形のため、**ABS外壁が直接「C1+C2の95.2%の実物」**（journal 0501a 第5節）
  -> P-32Dでは外装カバーがABS壁と外気の間に介在するが、CAHAYAでは介在物なし
- ユーザの制振実験（アルミテープ貼付）でABS 323Hz共振がフィルタとして倍音列をマスクしていたことが判明
  -> 「制振で倍音が見える」のではなく「ABSが邪魔だった」という解釈に転換（journal 0501a-b）
- v3設計でABS外壁を桐共鳴箱に置換することで、C1経路の物性（材料・板厚・モード）が根本的に変わる

詳細: `../../02-physics/22-paths/`

## v3設計での扱い

| 部品 | 扱い | 備考 |
|---|---|---|
| リードプレート | **流用** (32鍵分そのまま) | アルミ + 銅リードの音源を保持 |

- 共鳴箱 14.2L (桐1.5mm + X-brace) + ボディ放射 + ピアニカ原理駆動
- 32鍵 F3-C6 (CAHAYA/P-32D流用)
- 卓上型 550×350×80mm、約16,000円
