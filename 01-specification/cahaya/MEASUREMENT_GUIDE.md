# CAHAYA 32鍵 実測手順ガイド (L5 製作前検証)

**目的**: v3設計のリードプレート流用前提を確定させるため、CAHAYA実機の必須寸法と倍音特性を取得する。
**作業時間目安**:
- 第0部 倍音計測 (分解前): 15-20分
- 第1-3部 寸法計測・気密観察 + 第4部 写真記録: 60-90分
- 再組立 + 第5部 倍音計測 (再組立後): 45-50分
- **合計: 約2-2.5時間**

**必要工具**:
- ノギス (デジタルが望ましい)、定規 300mm
- スマホ + Spectroid (Android) または n-Track Tuner (iOS)
- 三脚 or スマホホルダー (倍音計測の再現性のため)
- 静かな部屋 (周囲音 < 40dB SPL推奨)

---

## 測定の基本ルール

1. **写真と数値を必ず両方残す**: ノギスを当てた写真 + 数値メモ
2. **不明な箇所は飛ばしてOK**: 測れない or 測り方が分からない項目はメモして次へ
3. **誤差±0.5mm程度で十分**: 仕様書側に余裕を持たせる
4. **複数回測定して平均**: 同じ箇所を3回測って中央値

## 全体フロー

```
[Step 0] CAHAYA分解前 倍音計測 (15分)        <- 第0部
    |
    v
[分解作業] 上ケース・下ケース・リードプレート分離
    |
    v
[Step 1-3] 寸法計測・気密インターフェース観察 (60-90分)  <- 第1-3部
    |
    v
[Step 4] 写真記録                            <- 第4部
    |
    v
[再組立] 分解の逆順で復元
    |
    v
[Step 5] CAHAYA再組立後 倍音計測 (15分)       <- 第5部 (新規)
    |
    v
[Step 6] 既設計値とのギャップ確認 + 倍音再現性評価  <- 第6部
```

倍音計測を分解前後で実施する理由:
- 分解前 = CAHAYA出荷状態の音響品質ベースライン
- 再組立後 = ユーザの組立精度を反映した倍音
- 差分が大きければ「分解操作で何かが変わった」、小さければ「組立精度で本機製作可能」の証明

## 第0部: CAHAYA分解前 倍音計測 (新規追加)

### 目的

**ユーザ仮説の事前検証**:
> 「ピアニカ = 工業品音源 + 工芸品筐体。倍音列が大きくなれば成功」

この仮説の前提となる「リード単体の音色 (倍音構造) は CAHAYA時点で確定している」を、**実機で確認** する。

倍音がリッチであれば、共鳴箱で増幅すれば確実に成功。倍音が貧弱であれば、共鳴箱で増幅しても限界あり。**作る前に分かる**情報。

### 工具

- スマホ (iPhone or Android)
- アプリ: **Spectroid** (Android、無料) または **n-Track Tuner** (iOS) または **AudioTool** (任意のスペアナアプリ)
- 静かな部屋 (周囲音 < 40dB SPL推奨)

### 計測項目

#### Z1. 代表音7点の倍音スペクトル

CAHAYA を分解せずに以下の代表音を吹奏し、Spectroid でピーク周波数を記録:

| 音名 | 基音 f1 | 計測手順 |
|---|---|---|
| F3  | 175 Hz  | 最低音、左端の鍵 |
| A3  | 220 Hz  | F3から+5鍵 |
| C4  | 262 Hz  | 中央C、F3から+8鍵 |
| F4  | 349 Hz  | T1=345Hzと一致候補 |
| C5  | 523 Hz  | 1オクターブ上のC |
| F5  | 698 Hz  | 推定T2近傍 |
| C6  | 1047 Hz | 最高音、右端の鍵 |

各音について、**1秒以上吹奏して安定後**にスペクトルを撮影:
- スマホをCAHAYAから30cm離した位置に固定
- 一定の息圧で吹く (不安定だとピークがふらつく)
- 吹奏中の Spectroid 画面をスクリーンショット

#### Z2. 倍音強度の記録

各音について、Spectroid から以下を読み取り:

```
F3 (基音 175Hz):
  f1 = ___ Hz (測定値)、振幅 ___ dB
  f2 = ___ Hz (倍音2次)、振幅 ___ dB
  f3 = ___ Hz (倍音3次)、振幅 ___ dB
  f4 = ___ Hz (倍音4次)、振幅 ___ dB
  f5 = ___ Hz (倍音5次)、振幅 ___ dB

  THD (全高調波歪) = (f2-f5の総和振幅) - f1の振幅 = ___ dB
  -> THDが大きいほど倍音リッチ
```

(同様にA3, C4, F4, C5, F5, C6 も記録)

#### Z3. 倍音特性のメモ

直感的な印象を記録:

- [ ] **倍音リッチ** (整数倍音が5次以上まで明確に見える)
- [ ] **倍音中程度** (3次倍音まで明確、4-5次は減衰)
- [ ] **倍音貧弱** (基音と2次倍音のみ、それ以上は不明確)
- [ ] **非整数倍音あり** (リードの曲げモード由来、6.27 × f1 近傍にピーク)

### 計測のコツ

1. **静かな環境**: 室内のエアコン・冷蔵庫の音は40Hz-200Hzにピークがあり、F3/A3の倍音と重なる
2. **マイク位置固定**: 三脚 or ホルダーで固定すると再現性が高まる
3. **複数回吹奏**: 同じ音を3回吹いて、ピーク位置がぶれないか確認
4. **Spectroid設定**: 「Logarithmic frequency」「Decibel scale」推奨、解像度は中程度で可

### 計測時間目安

- 7音 × 各2-3分 = 15-20分

### 結果の解釈 (事前ガイド)

| 状況 | 解釈 | アクション |
|---|---|---|
| 全7音で倍音5次以上明確 | リード単体で十分リッチ | 共鳴箱増幅で確実に成功、安心して製作へ |
| F3-F4 で倍音リッチ、C5以上で減衰 | 中音域に強み | 教会用に最適、製作OK |
| 全音で倍音貧弱 | リード品質が低い | リード材変更検討 (P-32D流用、P-32E、ハルモニウム用etc.) |
| F3 (175Hz) の基音より倍音350Hzが強い | T1=345Hzと一致して有利 | 共鳴箱で完全結合、最良ケース |

## 第1部: リードプレート (最重要)

### 工具配置

CAHAYAから取り外したリードプレートを、リード側 (銅片が見える側) を上にして平面に置く。

### 測定項目

#### A1. プレート全長 [mm]

**測り方**: プレートの長辺を端から端まで。32鍵が並んでいる方向。

```
  +--+--+--+--+--+ ... +--+
  |R1|R2|R3|R4|R5|     |R32|     <- 各リード
  +--+--+--+--+--+ ... +--+
  |<---------- L1 -------->|
```

| 測定値 | 単位 |
|---|---|
| L1 = ___ | mm |

#### A2. プレート幅 [mm]

**測り方**: プレートの短辺。リードの長さ方向に直交。

```
  +-------------------+ ---
  |                   |  ↑
  | リード並び        |  W1
  |                   |  ↓
  +-------------------+ ---
```

| 測定値 | 単位 |
|---|---|
| W1 = ___ | mm |

#### A3. プレート厚 [mm]

**測り方**: ノギスでプレート本体 (リード除く) の厚みを測る。リードがある部分は避け、端の余白部分で測定。

| 測定値 | 単位 |
|---|---|
| t1 = ___ | mm |

#### A4. リードピッチ [mm]

**測り方**: 隣接する2つのリードの中心間距離。**低音側と高音側で違う可能性**があるため、3箇所で測定:
- 最低音側 (R1-R2 の間)
- 中央 (R16-R17 の間)
- 最高音側 (R31-R32 の間)

| 位置 | 測定値 | 単位 |
|---|---|---|
| 低音側 P_low = ___ | mm |
| 中央  P_mid = ___ | mm |
| 高音側 P_high = ___ | mm |

注: ピッチは32鍵均等とは限らない。ピアニカは標準で**音域内で連続変化**する場合がある。

#### A5. 最低音リードの長さ [mm]

**測り方**: F3 = 175Hz のリード (最も長い1本)。スロットの根元から先端まで。

```
  +---根元
  |   ↓
  | === === === ←リード片
  |  
  +---スロット端
  |<-- L_R1 -->|
```

| 測定値 | 単位 |
|---|---|
| L_R1 = ___ | mm |

#### A6. 最高音リードの長さ [mm]

**測り方**: C6 = 1047Hz のリード (最も短い1本)。

| 測定値 | 単位 |
|---|---|
| L_R32 = ___ | mm |

#### A7. ビス位置 (4箇所) [mm]

**測り方**: プレートをCAHAYA本体に固定するビスの位置。プレートの左下角を原点 (0,0) として、各ビスの中心座標を取得。

```
       y
       ↑
  +---*------*---+
  |              |
  |   リード並び  |
  |              |
  +---*------*---+ → x
  (0,0)
```

| ビス | x [mm] | y [mm] |
|---|---|---|
| 左下 | ___ | ___ |
| 右下 | ___ | ___ |
| 左上 | ___ | ___ |
| 右上 | ___ | ___ |

ビスのサイズも記録: **M___ × ___mm長** (例: M3 × 8mm)

### 測定項目

#### B1. 鍵盤ユニット全長 [mm]

## 第3部: 気密インターフェース (L6 で重要)

### 測定項目

#### C1. リードプレート周辺のシール構造

CAHAYA本体からリードプレートを外したときに、リードプレートと本体の間に **何が挟まっていたか**を記録:

- [ ] 黒色ゴムパッキン (シート状)
- [ ] 白色シーリングテープ
- [ ] 何も無し (直接接触)
- [ ] その他: ___

シール材があれば、その厚みも測定: **t_seal = ___ mm**

#### C2. ネジボス位置の精度

A7 で測ったビス位置と、CAHAYA本体側のネジボス位置の関係を記録:

- [ ] 完全に一致 (プレート流用時にそのまま使える)
- [ ] わずかにズレ (新規共鳴箱で位置決め必要)
- [ ] 大きくズレ

写真記録: 本体側のネジボス4箇所を撮影

#### C3. プレート押え方向

リードプレートは:

- [ ] 下から押し上げて固定 (中華式典型)
- [ ] 上から押さえつけて固定 (日本式典型)

これによりv3共鳴箱への組込方向が決まる。

## 第5部: CAHAYA再組立後 倍音計測 (新規追加)

### 目的

**組立精度の検証**:
分解前の倍音スペクトル (第0部) と比較し、ユーザの組立技能で本機製作が可能かを判定する。差が小さければ「ユーザの組立は精度十分」、大きければ「組立工程にリスク要因あり」と切り分けられる。

### 計測手順

第0部とまったく同じ手順を繰り返す:

1. CAHAYAを分解の逆順で完全再組立
2. ネジは元の位置・順序で締める (4箇所均等に少しずつ、応力集中を避ける)
3. 7音 (F3, A3, C4, F4, C5, F5, C6) の倍音スペクトル計測
4. 第0部と同じスマホ位置・距離・吹奏圧を再現

## 第6部: 既設計値とのギャップ確認 + 倍音再現性評価

### 6.1 寸法ギャップ確認

実測値が出揃ったら、design-spec-v3.md の以下の値とのギャップを確認:

| 設計値 (v3) | 実測予測 (本ガイドで取得) |
|---|---|
| リードプレート: 200×40mm | A1=___, W1=___ |
| ブリッジパッチ: 200×60×3mm | A1+周辺余裕 |
| 鍵盤ユニット: 300×80×60mm | KB_L=___, KB_D=___, KB_H=___ |

ギャップが10mm以上あれば、共鳴箱寸法を再評価する必要がある。

### 6.2 倍音再現性評価 (新規追加)

第0部 (分解前) と第5部 (再組立後) の倍音スペクトルを比較:

| 音名 | f1差 [Hz] | f2差 [Hz] | f3差 [Hz] | THD差 [dB] | 評価 |
|---|---|---|---|---|---|
| F3  | ___ | ___ | ___ | ___ | [ ] OK [ ] WARN [ ] NG |
| A3  | ___ | ___ | ___ | ___ | [ ] OK [ ] WARN [ ] NG |
| C4  | ___ | ___ | ___ | ___ | [ ] OK [ ] WARN [ ] NG |
| F4  | ___ | ___ | ___ | ___ | [ ] OK [ ] WARN [ ] NG |
| C5  | ___ | ___ | ___ | ___ | [ ] OK [ ] WARN [ ] NG |
| F5  | ___ | ___ | ___ | ___ | [ ] OK [ ] WARN [ ] NG |
| C6  | ___ | ___ | ___ | ___ | [ ] OK [ ] WARN [ ] NG |

#### 判定基準

| 状況 | 評価 |
|---|---|
| 周波数差 ±2Hz以内、振幅差 ±2dB以内 | OK (組立精度十分、本機製作可) |
| 周波数差 ±5Hz以内、振幅差 ±5dB以内 | WARN (許容範囲、注意して製作) |
| それ以上の差 | NG (組立工程に問題あり、原因究明要) |

#### 全7音の評価サマリー

- [ ] **全OK**: 組立精度問題なし、本機製作へ進めて良い
- [ ] **一部WARN**: 製作はOKだが、該当周波数帯で組立に注意
- [ ] **NGあり**: 原因究明が先 (例: ガスケット劣化、ネジトルク不均一、リード位置ズレ)

### 6.3 倍音特性とユーザ仮説の検証

第0部で取得した倍音特性が「倍音列が大きくなれば成功」の仮説を支持するか:

| 第0部の結論 (Z3) | 本機製作への含意 |
|---|---|
| 全7音で倍音リッチ | 仮説支持、製作で確実に成功 |
| F3-F4 で倍音リッチ、C5以上で減衰 | 教会用音域 (C4-G5) に最適、製作推奨 |
| 全音で倍音貧弱 | **要再検討**: リード単体の品質に限界あり、共鳴箱だけでは克服不能 |
| 非整数倍音 (6.27×f1) が強い | リードの曲げモードが目立つ、共鳴箱でフィルタするとリッチに変化可能性あり |

**判定**: ユーザ仮説は [ ] 支持される [ ] 部分支持 [ ] 棄却される
