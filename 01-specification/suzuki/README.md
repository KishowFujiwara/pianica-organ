# 鈴木楽器 メロディオン (W-37C系)

**ステータス**: 仕様詳細不明、ジャーナル0430bからの抽出のみ

## 概要

鈴木楽器製作所は、ヤマハと並ぶ日本の鍵盤ハーモニカ大手メーカー。本プロジェクトでは**設計思想の比較対象**として参照する。

## 確認できている情報

| 項目 | 値 |
|---|---|
| メーカー | 鈴木楽器 (Suzuki) |
| 参照モデル | W-37C |
| 鍵盤数 | 37 (推定、要確認) |
| 構造分類 | 日本式 (カバー + 空気室 = 別部品) |

## 設計思想の特徴

ジャーナル0430bでの分析:

- **弦楽器型2段駒** の構造を採用 (CAHAYA改造で目指した「グランドピアノ型1段駒」と対比)
- **カバー + 空気室 = 別部品** の構成 (中華式CAHAYAの一体成形と対比)
- **壁面放射の物理を経験的に理解した設計** と評価

```
鈴木W-37C (推定):  リード → プレート → 空気室 → カバー → 外部
                              [駒1]          [駒2]    (2段駒)

CAHAYA改造目標:    リード → プレート → ABS側壁 → 木底面(共鳴板) → 空気
                                         [駒]              (1段駒)
```
