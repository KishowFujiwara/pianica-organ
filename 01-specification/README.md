# 01-specification: 楽器の仕様調査

## 章の目的

本章は、本プロジェクトで参照・流用する**既存の鍵盤ハーモニカ製品の仕様情報**を集約する。
22経路解析の対象機材、設計v3の流用部品候補、比較対象としての先行機種を、メーカー別に整理する。

## メーカー別の構成

| ディレクトリ | メーカー | モデル | 役割 |
|---|---|---|---|
| `yamaha-p32d/` | ヤマハ | P-32D (32鍵) | **22経路解析の主対象機**。リードプレートが真鍮製。 |
| `cahaya/` | CAHAYA (中華) | CY0341相当 (32鍵) | **v3設計のリードプレート流用候補**。リードプレートが銀色アルミ。 |
| `suzuki/` | 鈴木楽器 | W-37C など | **比較・参照用**。日本式（カバー+空気室の別部品構造）の代表。 |

## 主要発見の要約

### ヤマハ P-32D (22経路解析の対象)

- 鍵数 32、音域 F3-C6 (175 Hz - 1047 Hz)
- リードプレート: 真鍮製
- 22経路解析の結果、**外部放射の95.2%はABS壁面由来 (C1+C2)**、鍵盤穴経由 (A1) はわずか0.8%
- ABS底面(1,1)曲げモード 323Hz (E4近傍) が音色の核

### CAHAYA 32鍵 (v3流用候補)

- 鍵数 32、音域はP-32Dと同一規格 (案γ採用、0430k参照)
- リードプレート: 銀色アルミ
- 上ケースは**単一風箱(plenum)**、内部に仕切りなし
- マウスピース直下はネジボス (空気は風箱全体に拡散)
- v3設計でリードプレートと鍵盤ユニットを流用、上ケースは廃棄して新規共鳴箱を製作

### 鈴木 W-37C (比較対象)

- 日本式: カバー + 空気室 = 別部品 (中華式は一体成形)
- 「弦楽器型2段駒」設計 (CAHAYA改造は「グランドピアノ型1段駒」)
- 壁面放射の物理を経験的に取り込んだ設計
