## 1. 物理構造 (写真検証済み)

### 1.1 部品構成 (5部品)

写真#1, #9で確認。元モデルの3層構造は誤りで、実際は4層+端部キャップ。

```
Layer 1: 白色ABSカバー (~1mm) [写真#1上, #7, #8]
         パッキンを介してリードプレートに圧着(ラッチ固定)。
         単なる「蓋」ではなく、気密境界の上壁。
         バルブ開放時に音圧を直接受ける。
         Gamma=20.6 (放射効率4.6%) -- ABS底面(1.1%)より高効率。

Layer 2: 空気室+リードプレート一体ユニット [写真#1中, #5, #10-12]
         白色ABS射出成型。楽器の心臓部。
         - リードプレート2枚 (真鍮+リン青銅) がネジ止め
         - 黒ゴムパッキンがプレート縁の上面に載る (カバーとの気密用)
         - ABS仕切り壁31枚でプレート下側に個別気室形成
         - 鍵バルブ+コイルバネ (32鍵、プレート上面側)
         - マウスピース管 (ABS, 直径10-12mm)
         構造: カバー--[パッキン]--プレート--[仕切り壁+ネジ]--ABSフレーム

Layer 3: 青色ABS底面ケース [写真#1下, #2]
         ネジ4本でLayer 2に結合。
         放射用スリット多数。黒スポンジ帯がLayer 2との間に挟まる。

Parts:   端部キャップ2個 (青色ABS) [写真#3, #4]
```

### 1.2 リードプレート (写真#10, #11, #12)

2枚構成。写真#11で明確に確認:

| 位置 | 色 | 推定材質 | E (GPa) | rho (kg/m^3) | eta | Q |
|------|-----|---------|---------|-------------|------|-----|
| 高音側 (左) | 金色 | 真鍮 C2600 | 100 | 8500 | 0.003 | 333 |
| 低音側 (右) | 銅色 | リン青銅 C5191 | 110 | 8800 | 0.002 | 500 |

低音リードにQが高い材料を使用 = 自励振動維持のための設計。

### 1.3 リードの構造 (写真#12)

- 片持ち梁: 基端をリベット/溶接でプレートに固定
- 先端は自由端: スロットから0.1-0.3mm浮上
- 長さは高音(~8mm)から低音(~25mm)に徐々に変化
- 一部のリードに先端質量付加(ハンダ)あり
- リードとプレートは同種金属: Z_reed ~ Z_plate -> ほぼ無損失伝達

### 1.4 バルブ機構 (写真#5)

- 各鍵にコイルバネ(金属)が1本
- バネが鍵を上方に付勢(常時閉じ)
- 鍵を押すとバルブパッド(白色円盤、フェルト貼り)がリフト
- パッド直径 ~8mm、面積 ~50mm^2
- オルガンの風箱バルブと同原理

## 2. エネルギー経路マップ (22+4経路)

### 2.1 全体フロー

```
息圧 P_blow (1000 Pa)
  |
  v
マウスピース管 [写真#6] -> 空気室プレナム (Layer 2内)
  |
  v  鍵を押す [写真#5] -> バルブ開放
  |
  v
リードスロット [写真#12]
  |
  +--[BN1] DC->AC変換: 自励振動 (82%損失, 不可避)
  |   リードに結合: 48 mW (18.1%)
  |   渦散逸(ボルダ・カルノー): 217 mW (81.8%)
  |
  +-- [S1] 流量音源 -> A1経路 (穴->鍵隙間->外部)
  |   [S2] ダイポール -> A2経路 (壁面駆動)
  |
  v
リード舌の振動 (48 mW)
  |
  +--[B1a] リベット/溶接 -> リードプレート (金属-金属, ~100%)
  |   リード内部減衰: 22.5 mW (Q=333-500で熱に)
  |
  v
リードプレートの振動 (25.5 mW)
  |   プレートは3つの役割を持つ:
  |   (a) 曲げ波の導波路
  |   (b) 空気室の蓋(気密)
  |   (c) 質量によるエネルギー蓄積体
  |
  +--[B2a] ネジ15-20本 (点接触, T=18%)   寄与  8%
  +--[B2b] ゴムガスケット (面接触, 高損失)  寄与  4%
  +--[B2c] 仕切り壁31枚 (面接触, T=18%)   寄与 88% [支配的]
  |
  v
白ABS空気室フレーム (Layer 2)
  |   壁面振動 -> 2つの競合過程:
  |   (a) 放射(音): P_rad = sigma * Z_air * S * <v^2>
  |   (b) 散逸(熱): P_diss = eta * omega * m_s * S * <v^2>
  |   Gamma = P_diss/P_rad = 93 (ABS 1.5mm) -> 99%が熱
  |
  +--[C2] 箱内へ再放射 -> A1穴経由で外部 [写真#5, #8]
  |
  +--[B4b] 黒スポンジ帯 (eta=0.3-0.5) [写真#2]
  |         大幅な振動遮断
  |
  v
青ABS底面ケース (Layer 3) [写真#1下]
  |   二重壁の第2壁。さらにGamma損失。
  |
  +--[C1] 壁面放射 -> 外部空気 [スリット経由]
  +--[C1b] スリット開口の直接放射
  |
  v
外部空気 -> 音 (0.22 mW, 0.085%)
```

## 3. ボトルネック理論 (BN1/BN2/BN3)

### 3.1 BN1: DC->AC結合 (82%損失)

```
真因: 自励振動のDC気流->AC振動変換
機構: ボルダ・カルノー損失 + F(t)-v(t)位相ずれ
結合効率: ~18% (全リード楽器共通)
改善可否: 不可 (物理法則)
外部検証: Tarnopolsky(2000), Ricot(2005), 戸村・熊谷(2021)
```

### 3.2 BN2: リードプレート -> ABS壁面 (47%損失)

写真検証で大幅に再定義:

```
旧定義: リード -> 壁面 (Z_metal vs Z_ABS = 20:1)
新定義: リード -> [リベット] -> リードプレート -> [3並列接触] -> ABS

伝達チェーン:
  (i)   リード -> プレート: 金属-金属、ほぼ無損失
  (ii)  プレート内: 曲げ波伝搬、Q=333-500で減衰 (~10-15%損失)
  (iii) プレート -> ABS: 3つの並列接触
        B2a ネジ15-20本 (点接触, T=18%): 寄与  8%
        B2b ゴムパッキン (面接触, プレート->カバー, 高損失): 寄与  4%
        B2c 仕切り壁31枚 (面接触, T=18%): 寄与 88% [支配的]

改善可否: Z不整合は材料物性で固定。接合方式で漸進的改善のみ。
```

仕切り壁31枚の面接触(B2c)がBN2の支配的経路であるという発見は、
元モデルでは完全に見落としていた。

### 3.3 BN3: 壁面 -> 空気 (99%損失)

写真検証で二重壁構造が判明:

```
旧モデル: ABS壁面(単壁) -> 外部空気
          Gamma = eta * omega * m_s / (2 * sigma * Z_air) = 93
          放射効率 1/(1+93) = 1.1%

実際の構造: 白ABS壁 -> [黒スポンジ] -> 青ABS壁 -> 外部空気
          二重壁 + スポンジ遮断 -> 実効BN3はさらに悪化
          放射効率 < 1% (二重壁効果で追加遮音)

改善可否: 唯一劇的に改善可能
          桐2mm一体構造: Gamma = 1.8, 放射効率 36%
```

### 3.4 総合効率

```
P-32D型:  BN1(18%) x BN2(53%) x BN3(<1%) = < 0.085%
v3桐箱:   BN1(18%) x BN2(~53%) x BN3(36%) = ~3.4%  (+15dB以上)
```

## 4. Gammaパラメータ

壁面のエネルギーバランスから導出された無次元数:

```
Gamma = eta * omega * m_s / (2 * sigma * Z_air)

eta:   材料損失係数
omega: 角周波数 (= 2*pi*f)
m_s:   面密度 [kg/m^2]
sigma: 放射効率
Z_air: 空気の比音響インピーダンス (= 420 Rayl)
```

放射効率 = 1 / (1 + Gamma)

| 材料 | eta | m_s (kg/m^2) | Gamma@C4 | 放射効率 |
|------|------|-------------|----------|---------|
| ABS 1.5mm | 0.030 | 1.575 | 93 | 1.1% |
| シナベニヤ 4mm | 0.015 | 2.000 | 12 | 7.8% |
| スプルース 4mm | 0.006 | 1.760 | 4.1 | 19.5% |
| 桐 3mm | 0.009 | 0.780 | 2.8 | 26.7% |
| 桐 2mm | 0.009 | 0.520 | 1.8 | 35.3% |

Gammaは周波数に比例 -> 共鳴板が自然に倍音フィルタとなる。
Gamma < 2 で Trompette 8' 音色が実現する。

## 5. v3設計への示唆

写真検証から得られたv3設計への教訓:

1. **二重壁の排除**: 桐箱一体構造で白ABS+スポンジ+青ABSの3層遮音を解消
2. **仕切り壁の材料変更**: ABSからの桐に変えればBN2のT値は変わらないが、仕切り壁自体が放射面になる
3. **ゴムガスケットの排除**: 桐箱が空気室を兼ねれば振動伝達経路からゴムを排除
4. **ブリッジ集中伝達**: 31枚仕切り壁の「分散伝達」をブリッジ1本の「集中伝達」(黄金分割位置)に置換
5. **2材質リードプレート**: 低音にリン青銅(Q=500)、高音に真鍮(Q=333)の音域別最適化を踏襲
