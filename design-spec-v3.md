## v3 の位置づけ

v1 (37本ラッパ) と v2 (楔形ホーン1本) はいずれもリード**下流**にホーンを置く構成だった。v3 はフラットトップアコースティックギターの 3DOF 結合系を翻訳し、**ボディ放射 + サウンドホール** によって全音域を共通の共鳴箱で支援する構成へ転換する。

| 項目 | v1 (前期) | v2 (中期) | **v3 (後期、本書)** |
|---|---|---|---|
| 共鳴方式 | 37本ラッパ (各音独立) | 楔形ホーン1本 | **共鳴箱 14.2L + ボディ放射** |
| 放射形態 | 管楽器型 | スピーカー型 | **弦楽器(ギター)型** |
| 学ぶべき先行楽器 | パイプオルガン | スピーカーホーン | **フラットトップアコースティックギター** |
| 駆動 | 呼気→ブロワ→足踏み3段階 | (同左) | **ピアニカ原理 (呼気/簡易加圧)** |
| サイズ | 家具級1m | 1500x600x1200mm | **小型 580x350x140 mm** |

### v1/v2 から v3 への転換根拠

1. **0428h** の 22経路解析: 音の99%は壁面放射 → 管共鳴は支配的でない
2. **0428i** スピネット形状の検討: 共鳴箱の物理が決定的
3. **0430f** 3楽器(ギター・ピアノ・バイオリン)共鳴構造の物理モデル比較: 弦楽器型が最も無駄が少ない
4. **0430h** T0/T1 パラダイム: ヘルムホルツ単独でなく、結合系として設計すべき
5. **0430i** フラットトップギター 3DOF 適用: 桐0.7mmの限界 → 1.5mm + X-brace へ転換
6. 本セッション (0430j 相当): 容積整合性検証 + 14.2L 確定、ピアニカ原理駆動への回帰

## 第1章 設計要件

### 1.1 用途

教会の小礼拝室での聖歌伴奏・独奏・合奏を主目的とする。

| 環境変数 | 値 |
|---|---|
| 部屋容積 V_room | 100 m³ (20-30人収容) |
| 残響時間 RT | 1.5 s |
| 目標 SPL | 70-80 dB at 1m |
| 必要音響パワー | 数 mW 程度 |

### 1.2 駆動: ピアニカ原理

**フットポンプ・大型リザーバーは採用しない**。本機はピアニカと同じく呼気駆動 (または軽量加圧) を採用する。

理由:
- 入力空気パワーは 数百 mW 程度で十分 (上記計算)
- リードプレートは P-32D / CAHAYA 流用、ピアニカの空気回路をそのまま継承
- 楽器全体の構成が「共鳴箱 + 鍵盤ユニット + 吹き口」のみとなり、家具級ではなく **可搬式・卓上設置型** になる

| 項目 | 値 |
|---|---|
| 駆動方式 | 呼気駆動 (Phase 1) または ハンドポンプ加圧 (Phase 2) |
| 静圧 | 200-500 Pa |
| 流量 | 単音時 0.2-0.5 L/s |
| 入力空気パワー | 50-250 mW |
| 出力音響パワー | 5-25 mW (変換効率 5-10%) |

### 1.3 音域・音色

| 項目 | 値 |
|---|---|
| 音域 | F3 - C6 (32音、約2.5オクターブ + 半音7) |
| 最低音 F3 | 175 Hz |
| 最高音 C6 | 1047 Hz |
| 音色 | アメリカン・オルガンのフルートストップ系 |

### 2.1 寸法・容積

| 項目 | 値 | 備考 |

| 外寸 | 550 x 350 x 80 mm | 矩形 + 内部曲面壁 |

### 2.2 板組

| 面 | 寸法 (mm) | 厚 (mm) | 材 | 役割 |
|---|---|---|---|---|
| トップ | 550 x 350 | 1.5 | 桐 (柾目) | 主放射、X-brace支持 |
| バック | 550 x 350 | 2.0 | 桐 | 副放射 |
| サイド (長辺) | 550 x 80 | 4.0 | ラワン | 連結・剛性 |
| サイド (短辺) | 350 x 80 | 4.0 | ラワン | 連結・剛性 |
| ライニング | 全周 20x8 | - | 桐 | kerfed lining |
| エンドブロック | 40x60x76 | - | 松 | 短辺中央2個 |
| 内部曲面壁 | R150 x 76 | 5 | 桐 | 定在波対策、両エンド内側 |

板厚は 0430i 「桐0.7mmは柔らかすぎる」の結論から **1.5mm** へ増。バックとサイドはトップより厚くして剛性を確保 (κ²=13.6 を抑える方向)。

### 2.3 ブレース構造

Martin 流の X-brace を踏襲し、Bass bar と Tone bar、Bridge patch を組み合わせる。剛性ファクタ目標 12 倍 (文献値 8-15)。

| 部材 | 寸法 (mm) | 材 | 配置 |
|---|---|---|---|
| X-brace (×2) | 8 x 12 x 300 | スプルース | 90度交差、サウンドホール直下 |
| Bass bar | 8 x 12 x 400 | スプルース | 中心から低音側へ100mmオフセット、長手 |
| Tone bar | 8 x 12 x 300 | スプルース | 中心から高音側へ80mmオフセット、長手 |
| Bridge patch | 200 x 60 x 3 | 桐 | リードプレート直下 |
| サウンドホール補強 | 外径60 内径38 t=2 | 桐 | SH縁の割れ防止 |

### 2.4 サウンドホール

| 項目 | 値 |
|---|---|
| 形状 | 円形単一 |
| 位置 | トップ板中央、上端から 100 mm |
| 直径 | 38 mm |
| 板厚 | 1.5 mm + 補強パッチ 2mm = 実効 3.5 mm |

### 2.5 リードプレート配置

| 項目 | 値 |
|---|---|
| 位置 | トップ板中央、上端から 220 mm (X-brace 交点の下方120mm) |

## 第3章 物理計算結果

### 3.1 ヘルムホルツ A0

```
A0 = (c / 2π) × √(S / (V × L_eff))

V = 14.2 L
ポート: d=38mm 単一
S = π(d/2)² = 11.34 cm²
L_eff = t_板 + 1.7×(d/2) = 36.8 mm

→ A0 = 80 Hz
```

E2 (82Hz) 直下、F3 (175Hz) より約1オクターブ低い。バスレフ動作で F3-F4 帯域に +3-6 dB の支援を期待。

### 3.2 トップ板基本モード

```
板単体 (1,1) モード:
  D = E h³ / (12(1-ν²))
  f_p = (π/2) √(D/(ρh)) × (1/a² + 1/b²)
       = 47.2 Hz  (ブレース無し、桐1.5mm、振動域502x302mm)

X-brace + Bass bar + Tone bar (剛性12倍):
  f_p_braced = f_p × √12 = 164 Hz
```

### 3.3 結合系 T0/T1 (Christensen 2DOF)

```
入力: f_p = 164 Hz, f_a (= A0) = 80 Hz, V = 14.2L

板コンプライアンス比: κ² = C_p / C_a = 13.6
  (5以下なら理想的な結合、>20で板が柔らかすぎ)

結合解:
  T0 = 38 Hz   (低音側結合モード、聞こえない)
  T1 = 345 Hz  (高音側結合モード、放射主体、F4近傍)
```

### 3.4 内部空気モード (定在波)

| 順位 | 周波数 [Hz] | モード(nx,ny,nz) | 方向 | 対策 |
|---|---|---|---|---|
| 1 | 316 | (1,0,0) | 長手 | 内部曲面壁 R150 で分散 |
| 2 | 501 | (0,1,0) | 奥行 | 同上 |
| 3-4 | 593, 633 | (1,1,0)(2,0,0) | 複合 | 同上 |
| 5以上 | >700 | 高次 | - | 自然減衰 |

低音域 (F3=175Hz) 付近にモードが無いため、ブーミーになるリスクは低い。

### 3.5 整合性チェック

| 項目 | 値 | 目標 | 判定 |
|---|---|---|---|
| 内容積 | 14.2 L | 12-15 L | OK |
| A0 | 80 Hz | 80 Hz | OK |
| 剛性ファクタ | 12 倍 | 8-15 倍 | OK |
| κ² | 13.6 | <10理想 | WARN (許容) |
| T1 | 345 Hz | F3-E4 (175-330Hz) | WARN (F4近傍、許容) |
| 空気1次 | 316 Hz | 音域中央 | OK |

### 4.2 吹き口

| 項目 | 値 |
|---|---|
| 径 | 12 mm |
| 長さ | 50 mm |
| 材 | ABS 管 |
| 配置 | 鍵盤ユニットの低音側端 |

### 4.3 共鳴箱との接続

鍵盤ユニットのリードプレート出口を、共鳴箱内部 (Bridge patch の位置) に直接マウントする。リードから出る空気振動が共鳴箱の14.2L空間を励振し、トップ板・バック板・空気の3DOF結合振動を起こす。

```
[空気の流れ]

吹き口 -> 鍵盤ユニット内空気室 -> 鍵バルブ -> リードプレート -> リード振動
                                                              |
                                                              v
                                共鳴箱内部空気 (14.2L) -> トップ板駆動
                                                              |
                                                              v
                                              結合振動 (T0/T1)
                                                              |
                                                              v
                                            サウンドホール + ボディ板放射
```

## 第6章 残課題と検証項目

### 6.1 製作前検証

- [ ] CAHAYAリードプレートの実測寸法確認
- [ ] 桐1.5mm板の入手 (柾目品)
- [ ] スプルース ブレース材の入手

### 6.2 製作後検証

- [ ] T0/T1 の実測 (Spectroid)
- [ ] バスレフ動作の確認 (A0 ≈ 80Hz)
- [ ] 内部空気モードの実測 (316Hz 付近のピーク)
- [ ] 教会小礼拝室での実測 SPL (目標 70-80 dB at 1m)

### 6.3 既知のリスク

1. **κ² = 13.6** は理想値 (5以下) より大きい。板を厚くする (1.5→2.0mm) 余地あり。
2. **T1 = 345Hz** が F4 (349Hz) と接近。wolf-note 的不安定の可能性。実測後にブレース調整で対応。
3. **内部空気1次モード 316Hz (D#4近傍)** が音域内。曲面壁・吸音材で対策必須。

## 第7章 0423原点との対応

design-spec-v1 (0423) で宣言された目標に対する達成度:

| 0423目標 | v3達成度 | 備考 |
|---|---|---|
| 音域 F3-F6 (37鍵) | **F3-C6 (32鍵)に縮小** | CAHAYA/P-32D流用のため32鍵に整合 (5鍵分は将来拡張余地) |
| 音色 フルートストップ系 | 検証必要 | 製作後評価 |
| 音量 小ホール30-50人 | 教会20-30人で十分 | 規模合わせ |
| サイズ 120×50×100cm 以内 | **大幅達成** | 58×35×8cm、可搬式に |
| コスト 7万円以下 | **大幅達成** | 約16,000円 |
| 駆動 電動ブロワ→足踏みペダル | 不採用 | ピアニカ原理に回帰 |

「7万円・家具級」という当初の規模感は、ボディ放射への転換とフットポンプ廃止により大幅に小型化された。**機能は維持しつつ、構造は単純化**。
