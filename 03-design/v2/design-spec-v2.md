# ピアニカ・オルガン 設計仕様 v2 -- 全37鍵共用 楔形ホーン + 縦潰し折り返し

**日付**: 2026-04-23  
**モデル**: Claude Opus 4.7  
**状態**: 物理設計確定、プロトタイピング前  
**前版**: [design-spec-v1.md](./design-spec-v1.md) (37個の個別トランペット共鳴管構成)

## v1 からの主要転換

v2 は v1 と **根本的に異なる音響設計思想** に基づく。

| 項目 | v1 (2026-04 前半) | v2 (2026-04-23) |
|---|---|---|
| 共鳴管構成 | 37個の個別トランペット | **全37鍵共用の1本楔形ホーン** |
| ホーン形状 | 指数ホーン (個別) | **線形楔形 (2D sectoral horn)** |
| 折り畳み | 各ベル独立 | **1本を3段水平折り畳み** |
| 筐体利用 | ベルを並べる | **筐体底辺・天辺を楔形の壁に兼用** |
| 折り返し処理 | なし (短い直管) | **縦潰し (h=60mm) で曲率半径制約回避** |
| 総軸長 | 各鍵 0.2-4.2m | 2,260 mm (単一) |
| 想定 f_min | 各鍵個別 | 264 Hz (≈ C4) |
| V=F/C | 高 | **極めて高い** |

## 転換の根拠

### [1] フリーリードの独立性
フリーリードは**自己発振器**であり、管共鳴に依存しない。したがって各鍵に個別ベルを用意する物理的必然性は低い。ピアニカ内部で既に37リードが共通気室を共有しているため、下流のホーンも共用できる。

### [2] V=F/C の最大化
- F (機能): 音色の統一性が増す、音域 C4〜C6 を確保
- C (コスト): 部品数が 37分の1近くに削減、全て平面加工で済む

### [3] 筐体構造の再利用
楔形ホーンのもう1枚の壁を**筐体の底辺・天辺が兼ねる**ことで、音響壁を新規に作る必要がない。これは Klipschorn が部屋の角を壁として使うのと同じ思想。

### [4] 縦潰し折り返しの発見
折り返し部で断面のアスペクト比を変える (高さ h を下げ、幅 w を上げて断面積 S = h×w を保存) ことで、曲げ半径制約を奥行方向ではなく幅方向で吸収できる。これにより 3段折り畳みが筐体 1500×600×1200 mm に収まる。

**総軸長 L_eff ≈ 2,260 mm**  
**開口断面 S_mouth = 200 × 650 = 130,000 mm²**  
**下限周波数 f_min ≈ c / (2·w_mouth) = 343 / 1.30 ≈ 264 Hz (C4)**

## 主要な物理式

### フリーリード発振条件
m·ẍ + c·ẋ + k·x = F_aero(x, ẋ, p₀)  
c_aero > c となれば実効減衰が負になり発振持続

### Websterホーン方程式
∂²p/∂t² = (c²/S(x))·∂/∂x [S(x)·∂p/∂x]

### 2D sectoral horn
S(x) = h · w(x),  w(x) = w₀ + 2x·tan(θ/2)  
f_min ≈ c / (2·w_mouth)

### 折り畳み制約
- 平面波近似: k·a < 3.83
- 曲率半径: R_min = max(3a, 3c/(2π·f))
- 断面積連続性: S(x) は C¹ 級
- 軸長保存: L_eff = ∫ ds_axis

## v1 との共存方針

v1 (個別37トランペット構成) は **別の設計哲学** として保持する。どちらを最終採用するかは、プロトタイピング結果による:

- v1 の強み: 音色の個別最適化、鍵毎のカットオフ調整、既製部品 (トイトランペット) 活用
- v2 の強み: 構造の単純さ、音色の統一性、極めて低い製作コスト

両方のプロトタイプを小型で作って比較することで最終方針を決める、あるいはハイブリッド案を検討する余地がある。
