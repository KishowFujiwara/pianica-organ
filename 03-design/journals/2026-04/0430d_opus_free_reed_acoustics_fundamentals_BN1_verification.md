## 作業ログ

### Phase 1: 音響の物理的定義

- 音 = 弾性媒質の縦波(粗密波)
- 波動方程式: d2p/dt2 = c^2 * nabla^2(p)
- 空気: Z = rho*c = 420 Rayl
- 聴覚閾値 20uPa (粒子変位 7.6pm = 水素原子の1/10)
- ダイナミックレンジ 120dB (圧力100万倍, パワー1兆倍)

### Phase 2: 楽器の音響パワー

| 楽器 | 音響パワー | SPL@1m |
|------|-----------|--------|
| 会話 | 10 uW | 59 dB |
| ピアニカ P-32D | 220 uW | 72 dB |
| トランペット (ff) | 300 mW | 104 dB |
| パイプオルガン(全開) | 25 W | 123 dB |

フルオーケストラでも10W。LED電球1個(10W)と同じ。
楽器の仕事 = ごく微量のエネルギーを「正しい形」で空気に渡すこと。

### Phase 3: 楽器の3要素モデル

全ての楽器: 励振源 -> 共鳴系 -> 放射系

フリーリードの異質さ:
- 励振源: 自励振動 (DC気流 -> AC振動)
- 共鳴系: [不要] -- 周波数はリードが自己決定
- 放射系: 筐体壁面 (意図されていない)

フリーリード = 「共鳴なき放射の楽器」
楽器の3要素モデルへの反例。

### Phase 4: フリーリードの3つの根本的特性

**特性1: 周波数の独立性**
```
f = C * h / L^2
C = (1.875)^2 * sqrt(E/rho) / (4*pi*sqrt(3))
```
- 息圧に依存しない。共鳴管に依存しない。箱の形状に依存しない。
- リードの寸法(h, L)だけで決まる。
- クラリネット(空気柱に従う)とは根本的に異なる。

計算値 vs 実測: +91%の乖離
- 原因: 先端質量(ハンダ)、付加質量(スロット空気)、テーパ
- 構造関数形 f ~ h/L^2 は正しい。定数Cを実測校正(C=283)。

**特性2: 倍音の自律生成**
- 梁の高次モードは非整数(6.27, 17.6, 34.4...)
- しかし実際のフリーリードは整数倍音列を持つ
- 原因: スロット開閉の非線形が整数倍音を自動生成
- 非正弦波の周期関数 = フーリエ級数 = 整数倍音列
- 倍音振幅分布は開閉波形の幾何学(デューティ比、遷移時間)で決まる

**特性3: 音量の非効率性**
- P_rad ~ P_blow^alpha (alpha ~ 2)
- 息圧2倍 -> 音量4倍(+6dB)だが大部分は渦散逸
- 総合効率 0.085%

### Phase 5: 0.085%の真因分析

3つのボトルネックは全て同じ根本原因: **インピーダンス不整合**

**BN1: 気流 -> リード (82%損失) -- 変えられない**
- 自励振動にはDC気流が不可避
- スロットのボルダ・カルノー損失: Delta_p = rho/2 * v^2 ~ 1000Pa
- Re = 1548 (スロット流、剥離+渦放出で散逸)
- F(t)とv_reed(t)の位相ずれで結合効率が部分的
- 結合効率 ~18% (Tarnopolsky実測と整合)
- アナロジー: クラスA増幅器のDCバイアス損失

**BN2: リード(金属) -> 壁面(ABS) (47%損失) -- 改善余地小**
- Z_metal = 31.1 MRayl vs Z_ABS = 1.6 MRayl (20倍不整合)
- 伝達率 T = 4*Z1*Z2/(Z1+Z2)^2 = 18.1%
- 反射された82%がリード内部(Q=500)で熱に
- Qが高いのに散逸が大きい理由: 出口がインピーダンス不整合で堰き止め

**BN3: 壁面(ABS) -> 空気 (99%損失) -- 唯一変えられる**
- Z_wall = 1.6 MRayl vs Z_air = 420 Rayl (3700倍不整合)
- Gamma = eta * omega * m_s / (2 * sigma * Z_air) = 93
- 放射効率 1/(1+93) = 1.1%
- + 音響的短絡 (モード干渉で正負が打消)
- + コインシデンス周波数 29kHz >> 楽音帯域

総合効率 = 0.181 x 0.531 x 0.009 = 0.00085 = 0.085%

v3桐: 0.181 x 0.531 x 0.285 = 0.027 = 2.7% (+15dB)

**検証ソース一覧:**

**検証結果:**

BN1(DC->AC結合 ~18%)は全てのリード楽器に共通する物理法則:
- [OK] 全リード楽器で結合効率10-25%の範囲
- [OK] フルーパイプ(ジェット)はさらに低い(0.1-1%)
- [OK] ビーティングリードとフリーリードは連続体(戸村・熊谷)
- [OK] 同じリードが振動長変化でビーティング<->フリーに遷移
- [OK] 圧力損失(ボルダ・カルノー)が実測で確認

**追加発見:**
パイプオルガンがピアニカより大きい理由はBN1の差ではなく共鳴管の有無:
```
ピアニカ:  BN1(18%) x BN2(53%) x BN3(1%)   = 0.085%
オルガン:  BN1(18%) x [共鳴管で直接放射]      ~ 1-5%
```
共鳴管はBN2+BN3を完全にバイパスする。

### Phase 8: 戸村・熊谷論文の特に重要な知見

1. 同一リードの振動長変化でビーティング<->フリーリード遷移を実測
2. ビーティング時: 「はりを叩いたような波形」= 高調波豊富
   フリー時: 高調波成分が低下 = 開閉非線形の減少
3. 実機Trompette管の共鳴管あり/なし比較 = BN2+BN3の実験的証拠
4. ABS製シャロット vs 黄銅で音色差を確認 = BN3(材料損失)
5. 「意外なほどに参考となる文献が少ない」= この分野は未開拓

## 本セッションで確立された理解

1. 音響とは弾性媒質の粗密波。楽器とは励振-共鳴-放射のエネルギー変換系
2. フリーリードは3要素モデルの反例: 共鳴系不要の「共鳴なき放射の楽器」
3. 周波数はリード寸法で自己決定。倍音はスロット開閉の非線形で自律生成
4. 0.085%の真因は3段のインピーダンス不整合(BN1 x BN2 x BN3)
5. BN1は物理法則(全リード楽器共通)、BN3だけが設計で改善可能
6. 楽器設計 = インピーダンス不整合の克服工学
7. パイプオルガンは共鳴管でBN2+BN3をバイパス。ピアニカとの差はBN1ではない
