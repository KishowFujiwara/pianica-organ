## 2. 作業の経緯

### Phase 1: Webster方程式による設計原理の再構築

過去セッションで積み上げてきた「V字ホーン」「フリーリード」「松天板」の議論を、Webster方程式とカットオフ周波数 fc = mc/(4π) を基軸に再編成。

- カットオフは広がり率 m のみで決定、管長 L は独立に設計可能
- 規範: fc = f0/√2(基音の1/√2オクターブ下)を採用
- 過去に直感的に決めた F3=300mm等の寸法が、この規範から導出される物理最適解の帯に偶然収まっていたことを確認

### Phase 2: Phase A(等価回路設計)の開始

「Phase A(共鳴室+松天板の等価回路)→ Phase B,D(評価)→ Phase C(実装)」の進行順を合意。Phase A Step 1(観測)に着手し、松物性、板振動、音響容量を整理。

### Phase 3: ユーザーからの新規構造提示

ユーザーから3枚の図が順次提示され、設計の前提が揺さぶられる:

1. 柱バッフル型(加圧型、リード露出): 空間放射整形系として V字ホーンとの比較に
2. 吸引型(柱あり): アメリカン・オルガンの骨格構造
3. 柱の役割 = 気流制御+構造支持(音響寄与ほぼゼロ)

柱の物理的役割を「波長との比較(λ/柱幅=40倍)+ ヘルムホルツ条件不成立 + 真空室吸音」の3根拠から「音響ではなく気流制御」と同定。

### Phase 4: 発音体3方式の整理

エアリード(リコーダー、フルート)と金属フリーリード(オルガン)の倍音・ノイズ・アタック特性を体系的に比較。

- リコーダー: 整数倍音のみ、基音優勢、純音的、息ノイズあり
- フルート: 整数倍音、やや豊富、吹き込みノイズ
- フリーリード: 整数倍音(流量変調)+ 非整数倍音(梁モード 6.27·f0)+ 高次残留

「リコーダーのような音」を目指す場合、吸引型フリーリードが物理的に最適との結論。

### Phase 5: ピアニカ実物の観測

ヤマハの断面図、実物分解写真、バルブ拡大図の3枚を分析:

- ピアニカ = 加圧型フリーリード(呼気駆動)
- バルブ = 逆止弁(ウィンドセーバー)、気密性+リリース+吸込保護
- 空気室 = 脈動平滑リザーバ(容積約800 cm³)
- リードプレート = 差し込み式カートリッジ(モジュール化済)

### Phase 6: おもちゃのラッパ流用案

「空気室にホーンを付ける」という新発想が提示され、上流ホーン(共鳴管)+ 下流ホーン(放射)の二重構造に到達。上流ホーンがあれば楽器がフリーリード系からリード管系(パイプオルガンのリードストップ)へ変化する。

この上流共鳴管として市販のおもちゃのラッパ37本を流用する案へ発展。

### Phase 7: ベル形状の本質

ユーザーから「ベル上がポイント、筒じゃダメ、円錐漏斗+筒でもダメ」との鋭い指摘。Webster方程式の観点から3形状を比較:

- 筒: 開口端でインピーダンス不連続、反射70-90%、放射効率1.0(基準)
- 円錐+筒: dS/dx不連続、部分反射、放射効率3.5
- ベル(指数ホーン): 連続整合、放射効率8-10

放射効率が8倍違う。これはおもちゃのラッパ選定の絶対基準(ビューグル型=OK、ホイッスル型=NG)に直結。

### Phase 8: 低音域とホーン長の関係

- 閉管/開管規範: L = λ/4 または λ/2、C2で1.3〜2.6m
- 指数ホーン規範: L は m と S(L)/S0 で独立決定、C2でも400mm程度
- F3〜F6の37鍵では38〜486mmで収まり、折り返し不要

### Phase 9: エンクロージャー最終形態

「アップライトピアノ/足踏みオルガン型」の筐体に以下を組み込む:

- 上段: 松天板(二次放射)
- 中段: 37本のラッパ(垂直配置、小→大)
- 下段: ピアニカ本体(リード+空気室)
- 最下段: ブロワ or 足踏みペダル

外形 120×50×100 cm、材料費約7万円。

### Phase 10: 設計仕様書 v1.0 作成

A4 20頁相当のMarkdown設計仕様書を作成。章構成:

1. はじめに(楽器の位置づけ、設計目標)
2. 音源の物理(フリーリード、自励振動、倍音構造、加圧/吸引)
3. 共鳴体の物理(Webster方程式、指数ホーン、カットオフ、ベル形状の必然性)
4. 全体構成(外形、音響経路、制御工学的表現、材質)
5. 音域別寸法表(37鍵の周波数、fc、m、管長)
6. 段階的試作計画(Phase 0〜3)
7. 材料と予算(約7万円)
8. 統合と展望(engineeringknowledgeとの接続、将来展開)
9. 付録(Webster方程式・指数ホーン解・片持ち梁の完全誘導、数値計算)

KaTeX対応HTML版も同時生成。

## 5. 技術的知見

### Webster方程式とベル形状の本質

指数ホーンがなぜ必要か、を物理から決着できた:

- S(x) が連続かつ微分も連続であることが反射を最小化する
- 筒はインピーダンス不連続で大部分のエネルギーが戻る
- 円錐+筒は境界で部分反射、指数カーブの利点を失う
- 指数ホーン(ベル)のみが連続整合で放射効率最大

### フリーリードの駆動方式と音色

加圧型と吸引型は外見が似ていても音色が根本的に違う:

- 加圧型: 気流がリードを叩く → 立ち上がり速、高次倍音豊富、THD 20-35%
- 吸引型: 気流がリードを撫でる → 立ち上がり緩、高次倍音少、THD 8-15%

リコーダー志向は吸引型一択。ピアニカは加圧型なので、Phase 3で改造が必要。

### 柱の役割の同定

外見上「ホーン」に見える柱構造でも、波長と寸法の比較(λ/柱幅=40倍)から音響には寄与しないと判定。柱の実機能は気流制御・乱流抑制・構造支持・異物防止。この見極めが実物のアメリカン・オルガン構造の読解に決定的。

### おもちゃのラッパ流用

自作木製V字ホーンより、既成のおもちゃラッパ(ビューグル型)のほうが物理的に優れる可能性がある。理由は既にベル形状になっているから。自作では指数カーブの精密成形が困難だが、市販品は量産精度で達成されている。

### 低音ホーンの実用限界

指数ホーンはカットオフを広がり率で決めるため閉管・開管規範より短い。ただし開口面積要件(ka>1)から低音ほどベルが大きくなる必要があり、低音限界は実質的に存在する。C2以下はエンクロージャー2-3mの巨大楽器が必要で、F3以上の3オクターブが実用限界。

## 6. 本セッションの設計判断

### 判断1: 楽器名を「ピアニカ・オルガン(mini-pipe-organ)」に

親しみやすさ(ピアニカ)と実態(パイプオルガン類似)を両立。

### 判断2: 音域をF3〜F6に限定

C2までの拡張は楽器が巨大化するため除外。3オクターブで讃美歌、民謡、童謡、バッハのコラールなど大半のレパートリーをカバー可能。

### 判断3: 駆動方式は3段階

Phase 1(呼気)→ Phase 2(加圧ブロワ)→ Phase 3(吸引)。段階を追って改造し、最終的に吸引でアメリカン・オルガンの柔らかい音色を実現。

### 判断4: 設計仕様書は完全版(A4 20頁)

将来の自分、試作協力者、engineeringknowledge参照者のいずれに対しても読める詳細度として完全版を選択。計算根拠を付録に置き、本文の可読性と検証可能性を両立。
