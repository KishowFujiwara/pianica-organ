# Session Journal: 2026-04-30 (Session C)
# Opus 4.6 -- 共鳴箱設計への到達: 管楽器->スピーカー->共鳴箱

## セッション概要

CAHAYA底面パネルの詳細設計から出発し、パネルモード周波数の
ボトルネック(短辺77mm)を発見。張出し設計(三角形共鳴板)、
ヘルムホルツの鼓膜理論、スタインウェイのダイアフラグマティック
響板、トイピアノ型構造を経て、最終的に「桐の三角形共鳴箱」に到達。
v1(ホーン)->v2(ホーン)->v3(エンクロージャ->ピアノ->共鳴箱)の
設計思想の進化が完結。

**思想的転換**: ピアニカは管楽器ではなくスピーカーだった。
そしてスピーカーではなく共鳴箱だった。箱が鳴ることが楽器の本質。

## 作業ログ

### Phase 1: パネルモード周波数のボトルネック

底面カバー400x75mmのパネルでモード計算:
- f_11が全材料で900Hz以上。楽音帯域(175-1047Hz)に入らない。
- 原因: f_11 = ... x ((1/a)^2 + (1/b)^2) の**短辺成分が支配的**
- 長辺をいくら伸ばしても(1/b)^2が残る
- b=75mmでは楽音帯域に届かない

### Phase 2: 張出し設計

底面カバーの枠を超えて板を左右に張り出す方式を考案:

設計目標: f_11 <= F3(175Hz) = 全音域で共振以上(グランドピアノと同じ思想)

| 片側張出し | 板寸法 | 桐3mm f_11 |
|-----------|--------|-----------|
| なし | 400x75 | 1018Hz |
| 50mm | 400x175 | 215Hz |
| 75mm | 400x225 | 144Hz [OK] |

推奨: 桐3mm 400x225mm (片側75mm張出し)
- f_11 = 144Hz < F3 = 175Hz
- Gamma = 1.1 (入力の48%が音)
- 重さ 70g

### Phase 3: 板厚の影響

厚くすると全部悪化:
```
f_11   ∝ h   厚くすると共振が上がる [NG]
Gamma  ∝ h   厚くすると損失が増える [NG]
重さ    ∝ h   厚くすると重い
```

構造強度の下限: 桐2mmでも自重たわみ0.06mm [OK]
結論: できるだけ薄く。桐3mmが実用下限。

### Phase 4: ヘルムホルツとスタインウェイ

松尾楽器の松尾氏の知見: スタインウェイ兄弟がヘルムホルツに
アドバイスされて設計した。

**(a) 鼓膜のように作れ**
- 薄い -> 面密度を下げる -> Gammaの分子が小さく
- 広い -> 放射面積を大きく -> Gammaの分母が大きく
- 鼓膜のZ=691Rayl、空気のZ=420Rayl。比1.6。ほぼ整合。

**(b) 中央厚、周辺薄(ダイアフラグマティック響板)**
- 入力点(駒)は厚くして剛に
- 周辺は薄くして振動しやすく
- 鼓膜のツチ骨付着点と同じ構造
- スタインウェイは指先カンナを発明して実現

**(c) 鋼鉄で松材を支える**
- 側頭骨(硬い) -> 鼓膜(薄い膜) = 耳の構造
- 鉄骨フレーム -> スプルース響板 = ピアノ
- 構造体と振動体の分離

**(d) 因果関係の正しい理解**
- 鉄骨は「響板を薄くするため」にある
- フェルト打弦 -> 鋼鉄弦が必要 -> 張力巨大 -> 木枠では無理
  -> 鉄骨導入 -> 響板が構造から解放 -> 薄くできる -> 大音量

フリーリードは弦張力ゼロ。ピアノが鉄骨を発明してやっと手に入れた
「響板の構造からの解放」を最初から持っている。

### Phase 5: ブリッジ位置の最適化

**黄金分割(0.382)**が最適。12モード中11モードを励振。
- 中央(0.50): 偶数モード全滅(5/12)
- 端(0.10): 基本モードが弱い(8/12)
- 黄金分割: 全ての節線を避ける(11/12)

鼓膜のツチ骨付着点もこの位置。スタインウェイの駒もこの位置。

張出しの非対称化で実現:
```
左86mm + 本体75mm + 右64mm = 225mm -> y/b = 0.382
```

### Phase 6: リードプレートの制約

各リード周波数ごとに最適位置を計算したが、リードプレートは
一枚の平板で全リードが一直線。鍵盤バルブと直結しているため
位置を変えられない。

ブリッジは直線1本、黄金分割位置に固定が最適解。

### Phase 7: スピネットとトイピアノの発想

**(a) スピネット型**
リードプレートを斜めに配置 -> 鍵盤バルブとの対応が崩れる -> NG
ただし空気管で繋ぐ方式なら可能 -> v3本設計の検討課題

**(b) トイピアノ型**
CAHAYA空気室を「部品」としてトイピアノ型木箱に収める
-> 共鳴板の寸法が完全に自由
-> 三角形共鳴板で低音側を広く

**(c) 三角形(台形)共鳴板**
- 低音側: 張出し広い -> 放射面積大 -> f_11低い -> 低音が豊か
- 高音側: 張出し狭い -> 十分鳴る
- グランドピアノと同じ形、しかし理由が違う(弦長ではなく放射面積)

### Phase 8: 音色設計 -- トランペットストップ

目標: パイプオルガンでトランペットレバーを引いた時の音
サン=サーンス交響曲第3番「オルガン付き」に似合う音

**発見: Gammaは周波数に比例する -> 共鳴板が自然に倍音フィルタになる**

```
Gamma(f) = eta * 2*pi*f * m_s / (2*sigma*Z_air)
```

基音のGamma=1なら、第2倍音=2、第3倍音=3...
-> 高い倍音ほど熱で消える -> 自然なロールオフ

音色マップ:
| 音色 | 倍音減衰率 | 条件 |
|------|-----------|------|
| フルート | -12 dB/oct | Gamma@基音 >> 10 |
| ディアパソン | -6 dB/oct | Gamma@基音 ~ 5 |
| トランペット | -3 dB/oct | Gamma@基音 < 2 |

桐2mm: Gamma=1.8, -2.5dB/oct -> トランペット [OK]
桐3mm: Gamma=2.8, -2.7dB/oct -> トランペット [OK]

### Phase 9: 桐箱 -- 共鳴箱への到達

「桐箱で空気室を作ってもいいのか?」という問いから:

フリーリードは荷重12g。構造体と振動体を分ける必要がない。
桐箱が空気室壁と共鳴板を兼ねる = 全面が放射面。

**エンクロージャ vs 共鳴箱**:
```
エンクロージャ(スピーカー):
  箱は鳴ってはいけない。MDF、厚い、高損失。
  振動板(コーン)だけが鳴る。箱の共振=有害。

共鳴箱(楽器):
  箱は鳴らなければいけない。桐、薄い、低損失。
  全面が放射面。箱の共振=音色。
```

ピアニカのABS箱: エンクロージャのつもりが鳴ってしまった「壊れたスピーカー」
桐箱v3: 最初から鳴るために作る「共鳴箱」

### Phase 10: v1-v3の設計思想の進化

```
v1: ホーン(管楽器の思想)
  「空気の柱で音を増幅する」 -> 37本のトランペット管

v2: ホーン(同じ思想の改良)
  「1本の楔形ホーンにまとめる」 -> まだ空気柱が主役

  === 22経路解析: 音の99%は壁面放射 ===

v3初期: エンクロージャ(スピーカーの思想)
  「壁面を制御する」 -> 硬材箱+スプルース共鳴板

v3中期: ピアノ(弦楽器の思想)
  「ブリッジで共鳴板に伝達」 -> Gamma理論、ヘルムホルツ

  === CAHAYA中華製の一体構造からの気づき ===

v3最終: 共鳴箱(楽器の本来の姿)
  「箱全体が鳴ればいい」 -> 桐の三角形共鳴箱
```

転換点が2つ:
1. ピアニカは管楽器ではなくスピーカーだ(v1/v2->v3初期)
2. スピーカーではなく共鳴箱だ(v3中期->v3最終)

中華製CAHAYAの筐体一体構造が、2番目の転換を引き起こした。
底面を木板に変えようとして -> 張出し -> 三角形 -> トイピアノ型
-> 桐箱で全部作れる -> 共鳴箱。

## 本セッションで確立された物理的理解

1. f_11は短辺に支配される。長辺を伸ばしても限界がある
2. 板は薄いほど良い (f_11, Gamma, 重さ 全てhに比例して悪化)
3. ヘルムホルツの鼓膜理論: 薄く、広く、入力点は偏心
4. スタインウェイの鉄骨は「響板を薄くするため」にある
5. フリーリードは弦張力ゼロ -> 最初から「構造からの解放」を持つ
6. 構造体と振動体を分ける必要がない -> 桐箱が全部兼ねる
7. 黄金分割(0.382)がモード励振の最適位置
8. Gammaはfに比例 -> 共鳴板が自然に倍音フィルタ -> 音色を決定
9. Gamma<2でトランペット音色 -> 桐2-3mmで実現
10. エンクロージャ(鳴らさない)ではなく共鳴箱(鳴らす)
