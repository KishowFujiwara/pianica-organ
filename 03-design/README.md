# 03-design: 設計

## 章の目的

本章は、本プロジェクトの**設計仕様書**と**設計セッションの記録(ジャーナル)**を集約する。
v1(2026-04-23) → v2(同日、楔形ホーン代替案) → 撤回(spinet) → v3(2026-04-30) の設計進化を記録する。

## 主要発見の要約 (ジャーナルから)

### 思想的転換 (3段階)

ジャーナル0430cで明文化:

1. **管楽器 → スピーカー** (v1/v2 → v3初期): 22経路解析で音の99%は壁面放射
2. **スピーカー → 共鳴箱** (v3中期 → v3最終): CAHAYA一体構造からの気づき
3. **共鳴箱 → ギター流結合系** (0430h以降): T0/T1パラダイム + フラットトップ3DOF

### エネルギー階層分類 (0430h)

楽器を励振エネルギーで3階層に分類:
- クラスI 微弱励振 (1-50 mW): ギター撥弦、**ピアニカリード**
- クラスII 中規模励振 (10-100 mW): バイオリン、ピアノ
- クラスIII 強力励振 (1W+): スピーカー

ピアニカが学ぶべきはギター製作。バイオリンではない。

### 0423原点との対応 (0430i, 0430j)

設計v3は0423原点 (アメリカン・オルガンの現代的復活) から逸脱せず、以下を継承:
- 教会用途、20-30人収容
- 37鍵想定 → 32鍵に縮小 (CAHAYA/P-32D流用)
- 家具級1m → 卓上型580mmに小型化 (機能維持、構造単純化)
- フットポンプ廃止 → ピアニカ原理駆動
