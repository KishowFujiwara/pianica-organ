- **固体伝導モデル(0427)**: ピアニカの音は経路②(固体伝導 -> ABS壁面放射) が支配的で、経路①(スロット直接放射) の **約286倍**

これらの知見は v1/v2 設計の前提を揺るがす可能性がある。本文書はその整合性を5観点から検証する。

## 2. 観点[1]: 音響パワー収支

### 2.1 v1/v2 の前提

両設計はピアニカのマウスピース取付口 (パイプ口) を密閉してホーン入口に接続する。これにより:

- 経路 A1 (リード -> 箱内空気 -> 鍵盤穴 -> 外部) の上面開口は楽音用に維持
- ピアニカ背面パイプ口は密閉、その下流にホーンを接続

しかし22経路解析が示す通り、ピアニカの音響パワーの大部分は経路②(固体伝導 -> ABS壁面放射) で生じる。ホーン側に流入する空気流は、リードのスロットを通った気流のみ。

### 2.2 各経路のパワー推定

過去セッション(0411, 0427) の固体伝導モデル計算結果:

| 経路 | メカニズム | 音響パワー | 比率 |
|------|-----------|-----------|------|
| ① スロット直接 | リード -> 気流変調 -> スロット放射 | 約 168 μW | 1 |
| ② 固体伝導 | リード -> リードプレート(金属) -> パッキン -> ABS壁 -> 面放射 | 約 48,000 μW | **286** |

経路①は経路②の 1/286 (約 -24.6 dB) しかない。

### 2.3 v1/v2 で達成可能な音圧の理論上限

```
ホーン経由放射 (経路①ベース)
  入力音響パワー   : 168 μW
  ホーン利得       : +20 dB (理想的指数ホーン整合の場合の上限)
  出力音響パワー   : 16,800 μW = 16.8 mW
  音圧 @1m (球面波): 約 70 dB SPL
```

球面波放射での音圧計算:

$$
L_p = 10 \log_{10}\left(\frac{W}{4\pi r^2 I_0}\right)
$$

$W = 16.8\,\mathrm{mW}, r = 1\,\mathrm{m}, I_0 = 10^{-12}\,\mathrm{W/m^2}$ で 約 71 dB SPL。

**目標 (パイプオルガン級 95 dB) との差: -24 dB**

これは出力電力で約250倍の差。**現実的に達成不可能**。

### 2.4 ホーン利得の上限

理想的な指数ホーンの利得は、開口面積比 $S_\text{mouth}/S_\text{throat}$ の対数で決まる:

$$
G = 10 \log_{10}\left(\frac{S_\text{mouth}}{S_\text{throat}}\right)
$$

$S_\text{mouth} = 1000\,\mathrm{cm^2}, S_\text{throat} = 10\,\mathrm{cm^2}$ で +20 dB。これ以上の利得を得るには、ホーンを物理的に巨大化する必要があり、実用的な楽器サイズを超える。

### 2.5 結論[1]

**v1/v2 設計でパイプオルガン級音量 (95dB@1m) は達成不可能。**

主因はピアニカ固有の音響パワー分布で、リードからのスロット気流が音源として 1/286 しか寄与しないこと。これは v1/v2 の根本的な前提誤り。

## 3. 観点[2]: ホーン入力の質

### 3.1 フリーリードのスロット気流の特性

リードがスロットを出入りする半周期型運動による気流変調は、以下の特性を持つ:

```
- 整流作用                : 半周期非対称のパルス流 (DC成分含む)
- 偶数倍音豊富             : 鋸歯状波様のスペクトル (奇数倍音のみのクラリネットと対照)
- 非整数倍音               : リード梁モード比 1 : 6.27 : 17.55 (片持ち梁の固有値)
- 整数倍音                 : 気流整流作用による調和系列 1 : 2 : 3 : ...
- 高調波エネルギー減衰     : 約 1/n (鋸歯波の典型)
```

### 3.2 ホーンの線形音響フィルタ性

指数ホーンは線形音響フィルタ。Webster方程式より:

$$
\frac{\partial^2 p}{\partial t^2} = c^2 \left(\frac{\partial^2 p}{\partial x^2} + \frac{1}{S}\frac{dS}{dx}\frac{\partial p}{\partial x}\right)
$$

これは線形偏微分方程式。**ホーンは入力スペクトルを変えない** (倍音構造を新たに生成も削除もしない)。ホーンの役割は:

- カットオフ周波数 $f_c = mc/(4\pi)$ 以上を放射
- 開口端での放射インピーダンス整合
- 指向性の付与

### 3.3 v1/v2 におけるホーンの理論的役割

過去会話 (5db2c9b3, 2026-04-23) で確立された原則:

> 「フリーリードは自己発振器、ホーンは制御器・放射器に徹する」

この原則は v1/v2 の設計でも守られている。しかし観点[1]で示した通り、**ホーン入口に到達する音響パワーが極めて小さい** ため、ホーンの理論的役割は正しいが入力が枯渇している状態。

### 3.4 結論[2]

ホーンの**役割設定は正しい**が、**入力が不十分**。これは v1/v2 設計の論理整合性の問題ではなく、観点[1]の音響パワー収支問題の派生。

## 4. 観点[3]: 自励振動の安定性

### 4.1 ホーン負荷の周波数特性

リードの下流にホーンを取り付けた場合、リードが感じる音響負荷は:

$$
Z_\text{load}(\omega) = Z_\text{horn}(\omega) + Z_\text{cassotto}(\omega) + Z_\text{box}(\omega)
$$

指数ホーンの入口インピーダンスは $f_c$ を境に大きく変わる:

```
f >> f_c   : 抵抗性負荷  Z = ρc/S (実数)
                         -> リードに同位相の力, 位相遅れなし
                         -> 自励振動安定

f ~ f_c    : 遷移領域    複素インピーダンス
                         -> 共振挙動, 位相シフト発生

f << f_c   : リアクタンス負荷 Z = i ρc/S × something (純虚数)
                         -> リードに位相遅れ
                         -> ピッチプル, 自励振動不安定化
```

### 4.2 v2 設計での具体的問題

v2 設計では $f_\text{min} \approx 264\,\mathrm{Hz}$ (約C4) と想定されている。一方、ピアニカP-32D の音域は **F3 (約175 Hz) ~ F6 (約1397 Hz)**。

```
F3 - B3 (175 - 247 Hz) : ホーンの $f_c$ 以下 -> 不安定領域 (4音, 約11%)
C4 以上 (262 Hz 以上)  : 抵抗性負荷 -> 安定 (33音, 約89%)
```

低音域4音 (F3, F#3, G3, G#3, A3, A#3, B3) で **位相シフトとピッチプル** が発生する可能性が高い。具体的には:

- リードの実効振動周波数が公称値からずれる (ピッチプル)
- 振動振幅の不安定化
- 楽音の調律破綻

### 4.3 v1 設計での対応

v1 (個別37トランペット) では各鍵ごとに独立したホーンを設計する。各ホーンの $f_c$ をその鍵の基音以下に設定すれば原理的に問題は起きないが、低音用ホーン (例: F3 = 175 Hz) のサイズは:

$$
f_c = \frac{mc}{4\pi}, \quad m = \frac{4\pi f_c}{c}
$$

$f_c = 100\,\mathrm{Hz}, c = 343\,\mathrm{m/s}$ で $m \approx 3.66\,\mathrm{m^{-1}}$。$S_\text{mouth}/S_\text{throat} = 100$ を得るには軸長:

$$
L = \frac{\ln(100)}{m} \approx 1.26\,\mathrm{m}
$$

低音域のホーンは 1m 超の軸長が必要となり、おもちゃのトランペット (典型30-40cm) では物理的に不足する。**v1 設計の低音域ホーンも実現困難**。

### 4.4 結論[3]

v2 は約11%の音域で自励振動不安定化リスク。v1 は低音域でホーンサイズ不足。**両設計とも低音域で物理的破綻**。

## 5. 観点[4]: 主要経路 (経路②) の処理

### 5.1 v1/v2 が処理対象としている経路

v1/v2 はピアニカのマウスピース取付口を密閉し、ホーンに接続する。これは22経路のうち以下を変更する:

```
[影響あり]
A1 (リード -> 箱内 -> 穴 -> 外部) : 上面開口は維持, 背面開口は密閉

[影響なし - すべて筐体内で完結]
A2-A5 : 箱内空気経由の伝搬
B1a-B7: 固体伝導の各段階
C1-C5 : 振動の再放射
D1-D3 : フィードバックループ
```

つまり v1/v2 は **22経路中 1経路のみを処理** している。残り21経路は無処理。

### 5.2 経路②の発生源

経路②(固体伝導 -> ABS壁面放射) の主要構成:

```
B1a + B1b -> B2/B3 -> B4 -> C1 -> 外部放射
   |
   +-> B4 -> C2 -> 箱内再放射 (元穴は密閉済なので筐体内に閉じ込められる)
```

これらはすべてピアニカ筐体内部で完結するため、ピアニカを「楽器筐体に組み込む」だけでは止められない。

### 5.3 楽器筐体内での経路②の挙動

v1/v2 では、ピアニカは木製楽器筐体内部に置かれる。すると経路 C1 (ABS底面/側面 -> 外部放射) は:

```
ABS壁の振動 (Q ~ 33, 底面共振 ~ 323 Hz) 
  -> ABS壁から外部空気へ放射 (元の状態)
  -> 楽器筐体内部空気へ放射 (新状況)
  -> 楽器筐体壁を励振
  -> 楽器筐体壁から外部へ放射
```

これは楽器筐体が **ピアニカ内壁の振動を受ける二次キャビティ** として機能することを意味する。経路②のエネルギーは消えず、楽器筐体経由で別の音色に変換されて放射される可能性が高い。

### 5.4 アコーディオンとの対比

22経路解析の表7 が明確に示す通り:

| 制御対象 | プロ用アコーディオン | v1/v2 |
|---------|------------------|-------|
| B1a (リベット結合) | 蜜蝋 (粘弾性) で減衰 | リベット直結, 無制御 |
| B4 (壁体伝導) | 硬材 + ニス塗装 | ABS薄壁, 無制御 |
| C1 (壁面外部放射) | ニス塗装で Q低下 | Q ~ 33 維持 |
| C2 (壁面再放射) | cassotto で音場制御 | 無処理 |

プロ用アコーディオン職人が **核心的に制御する4経路 (B1a, B4, C1, C2)**を、v1/v2 はすべて手付かずのまま継承している。

### 5.5 結論[4]

**v1/v2 はパイプオルガンを目指しながら、ピアニカ特有の最重要音響経路を放置している**。

22経路中、v1/v2 が能動的に処理するのは A1 のみ (1/22 = 4.5%)。他の95.5%の経路は v1/v2 の音響設計に組み込まれていない。

## 6. 観点[5]: VE原則 (V=F/C) との整合性

### 6.1 VE原則の確認

過去のメモリに記録された原則:

> Pursuing pipe organ imitation in a reed organ simply recreates a pipe organ with no added value; the correct goal is to maximize the free reed's inherent character (non-integer beam mode harmonics).

つまり「リードオルガンでパイプオルガンを模倣することは付加価値を生まない。フリーリードの非整数倍音を最大化することが正しい目標」。

### 6.2 v1/v2 で設定されている F (機能)

両設計とも目標として:

```
F1: パイプオルガン的な大音量 (95 dB)
F2: 整数倍音による豊かな音色
F3: 鍵盤奏法可能な動的応答
```

を掲げている。しかしこれは:

- F1 は観点[1]より達成不可能
- F2 はフリーリードの非整数倍音性 (1:6.27:17.55) と対立する
- F3 はリード自体の特性で十分達成済 (ホーンの寄与は小)

つまり **v1/v2 の F は VE原則と矛盾する** 方向に設定されている。

### 6.3 真にVE的な F の候補

フリーリードでしか出せない音色を最大化する F:

```
F (代替案):
  F1': 非整数倍音 (1:6.27:17.55) を最大限に響かせる
  F2': 整数倍音 (鋸歯波系列) と非整数倍音の混合バランス制御
  F3': dipole+monopole 混合音源としての指向性活用
  F4': 自己発振器として、外部音響負荷なしで完結する独立性
  F5': 圧力式と吸引式の音色対比 (アメリカン・オルガン的柔音)
```

### 6.4 結論[5]

v1/v2 の F 設定は VE原則に反している。「パイプオルガンの劣化版」を作る方向に設計が向いている。

## 8. 推奨される方向転換

### 方向 A: 経路②活用型 (cassotto 思想)

ピアニカの音響実態 (経路②支配) を受け入れ、ABS箱を木製cassottoに置換する:

```
新構成:
  リードプレート (ピアニカから取り出し)
    |
    | [蜜蝋でリベット部減衰]
    v
  木製cassottoチャンバー (松または硬材)
    |
    | [ニス塗装でQ制御]
    v
  響板放射 (松, 既存設計と同様)
    + 共鳴管 (オプション)
```

利点: 経路②を主音源として正しく扱える, パイプオルガン的"エンクロージャ放射"と類似
欠点: ピアニカ筐体流用のメリット消失, 工作難度上昇, 材料費増加

### 方向 B: ピアニカ内部処理型 (テープ処置の拡張)

ABS箱を残しつつ、内部音響処理を加える:

```
処置:
  - メンディングテープ全面貼付 (既存知見, Q低下)
  - ABS内壁に粘弾性材 (シリコンゴムシート薄貼り)
  - リードプレート/ABS間パッキンを蜜蝋系に交換
  - ピアニカ全体を木製エンクロージャ内に置き, 二次共鳴調整
```

ホーン (v2) は補助的役割に降格。主音源はピアニカ自体の改造後音響。

利点: 工作変更最小, ピアニカ流用メリット維持, 経路②を制御
欠点: パイプオルガン級音量は依然不可, 「リードオルガン+α」の位置付け

### 方向 C: 完全リード材料化 (リードのみ流用)

ピアニカの ABS箱もリードプレートも捨て、リード単体だけを取り出す:

```
新構成:
  リード (ピアニカから抽出)
    |
    v
  自作リードプレート (真鍮または木製ベース)
    |
    | [蜜蝋クランプ]
    v
  自作木製キャビネット
    |
    v
  音響制御 (cassotto, ホーン, 響板を自由に設計)
```

利点: 設計自由度最大, パイプオルガン的設計思想を貫ける
欠点: ピアニカ流用は名目のみ, 部品費+工作費が大幅増加

### 各方向の VE評価

```
方向 | F (機能)                    | C (コスト)         | V = F/C
-----|----------------------------|------------------|--------
A    | フリーリード+cassotto音色  | 中 (ABS箱置換工作) | 高
B    | リードオルガン強化版        | 低 (テープ等少量)  | 中-高
C    | 設計自由度最大              | 高 (ほぼゼロから) | 中
v1   | パイプオルガン劣化版        | 中 (37ベル+筐体)  | 低
v2   | パイプオルガン劣化版        | 低 (1ホーン)     | 低-中
```

VE観点では **方向 A または B** が最も価値が高い。

## 9. 結論

**v1/v2 設計は、本セッションで明らかになった音響実態と整合しない。** 具体的には:

1. パイプオルガン級音量 (95 dB) は物理的に達成不可能 (理論上限 ~70 dB)
2. 低音域 (F3-B3) で自励振動不安定化リスク
3. 22経路中21経路を無処理 (経路②支配の事実を反映していない)
4. VE原則「フリーリード固有性の最大化」と矛盾する F 設定

これは v1/v2 の作業を否定するものではなく、**設計の探索段階として有用な発見** である。22経路解析 (本セッション 0427) と固体伝導モデル (0427) の知見は、v1/v2 を経たからこそ位置付けが明確になった。

**次の設計反復 (v3) では方向 A または B のいずれかを採用すべき**。両者ともピアニカ流用のメリットを部分的に保ちつつ、フリーリードの音響実態に整合する設計が可能。

## 10. 検証範囲外の論点 (将来の検討課題)

本検証で踏み込まなかった論点:

- **吸引式 (アメリカン・オルガン型) への変換時の経路②挙動**: 圧力式と吸引式で固体伝導経路は同じか?
- **複数鍵同時押しのスケーリング**: 22経路解析は単音前提。和音時の干渉は?
- **温湿度変動の経路別感度**: 各経路の温湿度依存性は均一か?
- **奏者の口腔結合 (経路A4)**: 圧力式特有の奏者音響負荷の定量化

これらは v3 設計を進める際に並行して検討すべき。
