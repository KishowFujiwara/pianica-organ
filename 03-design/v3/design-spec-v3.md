## v3 の位置づけ

v1 (37本ラッパ) と v2 (楔形ホーン1本) はいずれもリード**下流**にホーンを置く構成だった。v3 はフラットトップアコースティックギターの 3DOF 結合系を翻訳し、**ボディ放射 + サウンドホール** によって全音域を共通の共鳴箱で支援する構成へ転換する。

| 項目 | v1 (前期) | v2 (中期) | **v3 (後期、本書)** |
|---|---|---|---|
| 共鳴方式 | 37本ラッパ (各音独立) | 楔形ホーン1本 | **共鳴箱 14.2L + ボディ放射** |

### v1/v2 から v3 への転換根拠

1. **0428h** の 22経路解析: 音の99%は壁面放射 → 管共鳴は支配的でない
2. **0428i** スピネット形状の検討: 共鳴箱の物理が決定的
3. **0430f** 3楽器(ギター・ピアノ・バイオリン)共鳴構造の物理モデル比較: 弦楽器型が最も無駄が少ない
4. **0430h** T0/T1 パラダイム: ヘルムホルツ単独でなく、結合系として設計すべき
5. **0430i** フラットトップギター 3DOF 適用: 桐0.7mmの限界 → 1.5mm + X-brace へ転換
6. 本セッション (0430j 相当): 容積整合性検証 + 14.2L 確定、ピアニカ原理駆動への回帰

### 2.2b 側面材料の選択根拠 (2026-05-01 追加)

側面の物理的役割は「表板の境界条件を定義する」こと。表板端部でのエネルギー反射率 $R$ が高いほど、表板のモードが明確に定義され、S1倍音列が表板だけから放射される。

**反射率の計算** (詳細は 3.6節 数理モデル):

表板 (桐 1.5mm) のインピーダンス $Z_p = \sqrt{\rho_p E_p} \cdot h_p = 1860$ N-s/m-per-m に対し:

| 材料 | 厚 [mm] | $Z_{side}$ [N-s/m-per-m] | 反射率 $R$ | 特徴 |

| 桐 | 4.0 | 4,960 | 0.21 | 表板と近い、モード不明瞭 |
| ラワン | 4.0 | 8,720 | 0.42 | 旧設計、中間的 |
| ウォルナット | 4.0 | 10,680 | 0.50 | 鈴木W-37Cの選択 |
| 楓 (メイプル) | 4.0 | 11,600 | 0.52 | ギター標準 |
| ローズウッド | 4.0 | 14,280 | 0.59 | ギター高級材 |
| **アルミ** | **3.0** | **41,100** | **0.83** | 設置型なら最適 |
| 鉄 | 2.0 | 79,000 | 0.91 | 過剰品質、重い |

**設計判断**: 本機は卓上設置型であり重量制約がない。アルミ L材 (3mm厚、ホームセンター入手可) を第1候補とする。金属側面は表板の境界を完全固定端に近づけ、S1倍音列を桐表板のみで放射する最もクリーンな構成を実現する。

ラワンは反射率 0.42 で桐 (0.21) より良いが、楓/アルミに劣る。旧設計からの変更点。

**制振実験との整合**: CAHAYA制振実験 (2026-05-01) で「ABS壁面を殺すとS1倍音列が露出する」ことを確認。同じ原理で、側面が鳴らない (= 高反射率) ほど表板の音色が明瞭になる。

## 第3章 物理計算結果

### 3.1 ヘルムホルツ A0

```
A0 = (c / 2π) × √(S / (V × L_eff))

V = 14.2 L
ポート: d=38mm 単一
S = π(d/2)² = 11.34 cm²
L_eff = t_板 + 1.7×(d/2) = 36.8 mm

```

### 3.2 トップ板基本モード

```
板単体 (1,1) モード:
  D = E h³ / (12(1-ν²))
  f_p = (π/2) √(D/(ρh)) × (1/a² + 1/b²)
       = 47.2 Hz  (ブレース無し、桐1.5mm、振動域502x302mm)

X-brace + Bass bar + Tone bar (剛性12倍):
  f_p_braced = f_p × √12 = 164 Hz
```

### 3.3 結合系 T0/T1 (Christensen 2DOF)

```
入力: f_p = 164 Hz, f_a (= A0) = 80 Hz, V = 14.2L

板コンプライアンス比: κ² = C_p / C_a = 13.6
  (5以下なら理想的な結合、>20で板が柔らかすぎ)

結合解:
  T0 = 38 Hz   (低音側結合モード、聞こえない)
```

### 3.6 表板振動の数理モデル -- 偏微分方程式の初期値・境界値問題 (2026-05-01 追加)

本節では、共鳴箱トップ板の振動を薄板の Kirchhoff-Love 方程式として定式化し、側面材料が境界条件を通じて振動モードと放射特性に与える影響を数理的に記述する。

#### 3.6.1 支配方程式 (Kirchhoff-Love 薄板方程式)

トップ板の横変位 $w(x, y, t)$ は以下の偏微分方程式に従う:

$$\rho_p h \frac{\partial^2 w}{\partial t^2} + D \nabla^4 w + 2 \gamma \rho_p h \frac{\partial w}{\partial t} = p_{reed}(x, y, t) - p_{cav}(x, y, t)$$

ここで:

- $\rho_p = 280$ kg/m³ (桐の密度)
- $h = 1.5 \times 10^{-3}$ m (板厚)
- $D = \frac{E_p h^3}{12(1 - \nu^2)} = \frac{5.5 \times 10^9 \times (1.5 \times 10^{-3})^3}{12(1 - 0.3^2)} = 1.55$ N-m (曲げ剛性)
- $\gamma$ : 減衰係数 [1/s] (桐の損失係数 $\eta \approx 0.01$ に対し $\gamma = \eta \omega / 2$)
- $\nabla^4 = \frac{\partial^4}{\partial x^4} + 2\frac{\partial^4}{\partial x^2 \partial y^2} + \frac{\partial^4}{\partial y^4}$ (重調和作用素)
- $p_{reed}(x, y, t)$ : リード S1 流量変調による駆動圧 (Bridge patch 領域に局在)
- $p_{cav}(x, y, t)$ : 共鳴箱内部空気からの背圧

#### 3.6.2 初期条件

$$w(x, y, 0) = 0, \quad \frac{\partial w}{\partial t}(x, y, 0) = 0$$

板は静止状態から出発する。$t > 0$ でリード S1 が駆動を開始し、板の応答が時間発展する。

#### 3.6.3 境界条件 -- 側面材料への依存

板端 $\Gamma$ (= 側面との接合線) における境界条件は、側面のインピーダンス $Z_{side}$ に依存する。

**(a) 金属側面 (アルミ, $Z_{side} \gg Z_p$): 固定端 (Clamped)**

$$w\big|_\Gamma = 0, \quad \frac{\partial w}{\partial n}\bigg|_\Gamma = 0$$

表板の端は動かない。モード形状が最も明確に定義される。固有周波数は最も高くなる。

**(b) 硬木側面 (楓/ウォルナット, $Z_{side} > Z_p$): 弾性支持端 (Robin)**

$$D \frac{\partial^2 w}{\partial n^2}\bigg|_\Gamma = -k_{rot} \cdot \frac{\partial w}{\partial n}\bigg|_\Gamma$$

$$D \frac{\partial^3 w}{\partial n^3}\bigg|_\Gamma = k_{trans} \cdot w\big|_\Gamma$$

ここで $k_{rot}$, $k_{trans}$ は側面の回転・並進バネ定数。側面の曲げ剛性 $D_{side}$ から:

$$k_{rot} \approx \frac{4 D_{side}}{h_{side}}, \quad k_{trans} \approx \frac{E_{side} \cdot h_{side}}{L_{side}}$$

固定端と自由端の中間的な挙動。エネルギーの一部が側面に漏洩する。

**(c) 桐側面 ($Z_{side} \approx Z_p$): 単純支持に近い**

$$w\big|_\Gamma \neq 0, \quad D \frac{\partial^2 w}{\partial n^2}\bigg|_\Gamma \approx 0$$

端が自由に回転。モード形状がぼやけ、有効面積が不明確になる。表板から側面へのエネルギー透過が最大。

#### 3.6.4 境界でのエネルギー反射率

表板端部でのエネルギー反射率 $R$ :

$$R = \left( \frac{Z_{side} - Z_p}{Z_{side} + Z_p} \right)^2$$

板インピーダンス:

$$Z = \sqrt{\rho E} \cdot h \quad \text{[N-s/m per unit edge length]}$$

桐表板 (1.5mm): $Z_p = \sqrt{280 \times 5.5 \times 10^9} \times 1.5 \times 10^{-3} = 1860$ N-s/m-per-m

$R \to 1$ (金属) では表板にエネルギーが閉じ込められ、S1 倍音列が表板のみから放射される。$R \to 0$ (同材) ではエネルギーが側面に漏れ、放射源が分散してモード構造が不明瞭になる。

#### 3.6.5 S1 駆動項

リードの流量変調 (S1音源、全パワーの 80-85%) を Bridge patch 領域 $\Omega_B$ に空間局在させる:

$$p_{reed}(x, y, t) = P_0 \sum_{n=1}^{N} a_n \sin(n \omega_0 t) \cdot \chi_{\Omega_B}(x, y)$$

ここで:

- $P_0$ : 励振圧振幅 [Pa] (静圧 200-500 Pa の変動成分)
- $\omega_0 = 2\pi f_0$ : リードの基本角周波数
- $a_n$ : 第 $n$ 次倍音のフーリエ係数 ($a_1 = 1.0, a_2 \approx 0.5, a_3 \approx 0.3, ...$)
- $\chi_{\Omega_B}(x,y)$ : Bridge patch 領域 (200 x 60 mm) の指示関数

#### 3.6.6 共鳴箱内部空気の結合 (Helmholtz 方程式)

箱内部の音圧 $p_{cav}(\mathbf{r}, t)$ :

$$\nabla^2 p_{cav} - \frac{1}{c^2} \frac{\partial^2 p_{cav}}{\partial t^2} = 0 \quad \text{(箱内部)}$$

トップ板面での連続条件:

$$\frac{\partial p_{cav}}{\partial z}\bigg|_{z=0} = -\rho_0 \frac{\partial^2 w}{\partial t^2}$$

側面での境界条件 (側面材料依存):

$$\frac{\partial p_{cav}}{\partial n}\bigg|_{sides} = \begin{cases} 0 & \text{(金属側面: 剛壁)} \\ -\rho_0 \frac{\partial^2 w_s}{\partial t^2} & \text{(木材側面: 側面振動)} \end{cases}$$

サウンドホール開口での放射条件 (ヘルムホルツポート):

$$p_{cav}\big|_{SH} = Z_{rad} \cdot v_{SH}$$

#### 3.6.7 結合系の固有値問題

定常状態 ($w = W(x,y) e^{i\omega t}$) を仮定すると、支配方程式は固有値問題に帰着:

$$D \nabla^4 W - \omega^2 \rho_p h \, W = P_{cav}(x,y) + P_{reed}(x,y)$$

境界条件 (金属側面の場合):

$$W\big|_\Gamma = 0, \quad \frac{\partial W}{\partial n}\bigg|_\Gamma = 0$$

固有モード $W_{mn}(x,y)$ と固有周波数 $f_{mn}$ の概算 (矩形固定端板):

$$f_{mn} = \frac{1}{2\pi} \sqrt{\frac{D}{\rho_p h}} \left[ \left(\frac{m\pi}{a}\right)^2 + \left(\frac{n\pi}{b}\right)^2 \right]$$

ブレースなし桐 1.5mm、振動域 $a = 502$ mm, $b = 302$ mm の場合:

| モード (m,n) | 固定端 $f_{mn}$ [Hz] | 単純支持 $f_{mn}$ [Hz] | 差 |
|---|---|---|---|
| (1,1) | 108 | 47 | x2.3 |
| (2,1) | 207 | 100 | x2.1 |
| (1,2) | 275 | 142 | x1.9 |
| (2,2) | 374 | 195 | x1.9 |
| (3,1) | 368 | 186 | x2.0 |

固定端 (アルミ側面) では全モードが約2倍の周波数にシフトする。ブレース (剛性12倍) を加えると:

$$f_{mn,braced} \approx f_{mn} \times \sqrt{12} \approx f_{mn} \times 3.46$$

(1,1) モードが固定端で $108 \times 3.46 = 374$ Hz (F4-F#4 近傍) となり、T1 = 345 Hz の設計と整合する。

#### 3.6.8 側面材料がモード構造に与える影響のまとめ

$$\text{固定端 (金属)} \xrightarrow{R=0.83} \text{弾性支持 (硬木)} \xrightarrow{R=0.52} \text{単純支持 (桐)} \xrightarrow{R=0.21}$$

| 特性 | 金属側面 ($R \approx 0.83$) | 硬木側面 ($R \approx 0.52$) | 桐側面 ($R \approx 0.21$) |
|---|---|---|---|
| モード周波数 | 最も高い (固定端) | 中間 | 最も低い (単純支持) |
| モード形状 | 明確 (端 = 節) | やや不明確 | ぼやける |
| エネルギー閉じ込め | 最大 (表板集中) | 中間 | 漏洩大 |
| 音色 | クリーン、明瞭 | バランス | 温かいが不明瞭 |
| 3DOF結合への影響 | 板剛性 $k_p$ 最大 -> $\kappa^2$ 低下 -> 結合改善 | 中間 | $k_p$ 低下 -> $\kappa^2$ 増大 |

**設計結論**: 設置型楽器では金属 (アルミ) 側面が最適。表板のモードが明確に定義され、S1 倍音列の放射が表板に集中する。$\kappa^2$ の低下は 3DOF 結合の改善方向であり、T1 設計値の精度も向上する。

| リードプレート | CAHAYA 流用 (P-32D / P-32E 流用も可、いずれも32鍵F3-C6で同一規格) |
| 鍵盤数 | 32 (F3-C6) |
| 鍵ピッチ | 8 mm |
| バルブ機構 | ピアニカ標準 (鍵が上、リード上面側) |
| 内部空気室 | 50-100 mL (脈動平滑) |

ピアニカ部品流用は「リードプレートのみ取り出し、空気室は新規製作」(本セッション確定3点より)。

### 4.2 吹き口

| 項目 | 値 |
|---|---|
| 径 | 12 mm |
| 長さ | 50 mm |
| 材 | ABS 管 |
| 配置 | 鍵盤ユニットの低音側端 |

### 4.3 共鳴箱との接続

鍵盤ユニットのリードプレート出口を、共鳴箱内部 (Bridge patch の位置) に直接マウントする。リードから出る空気振動が共鳴箱の14.2L空間を励振し、トップ板・バック板・空気の3DOF結合振動を起こす。

```
[空気の流れ]

吹き口 -> 鍵盤ユニット内空気室 -> 鍵バルブ -> リードプレート -> リード振動
                                                              |
                                                              v
                                共鳴箱内部空気 (14.2L) -> トップ板駆動
                                                              |
                                                              v
                                              結合振動 (T0/T1)
                                                              |
                                                              v
                                            サウンドホール + ボディ板放射
```

## 第6章 残課題と検証項目

### 6.1 製作前検証

- [ ] CAHAYAリードプレートの実測寸法確認
- [ ] 桐1.5mm板の入手 (柾目品)
- [ ] スプルース ブレース材の入手

### 6.2 製作後検証

- [ ] T0/T1 の実測 (Spectroid)
- [ ] バスレフ動作の確認 (A0 ≈ 80Hz)
- [ ] 内部空気モードの実測 (316Hz 付近のピーク)
- [ ] 教会小礼拝室での実測 SPL (目標 70-80 dB at 1m)

### 6.3 既知のリスク

1. **κ² = 13.6** は理想値 (5以下) より大きい。板を厚くする (1.5→2.0mm) 余地あり。
2. **T1 = 345Hz** が F4 (349Hz) と接近。wolf-note 的不安定の可能性。実測後にブレース調整で対応。
3. **内部空気1次モード 316Hz (D#4近傍)** が音域内。曲面壁・吸音材で対策必須。

## 第7章 0423原点との対応

design-spec-v1 (0423) で宣言された目標に対する達成度:

| 0423目標 | v3達成度 | 備考 |
|---|---|---|
| 音域 F3-F6 (37鍵) | **F3-C6 (32鍵)に縮小** | CAHAYA/P-32D流用のため32鍵に整合 (5鍵分は将来拡張余地) |
| 音色 フルートストップ系 | 検証必要 | 製作後評価 |
| 音量 小ホール30-50人 | 教会20-30人で十分 | 規模合わせ |
| サイズ 120×50×100cm 以内 | **大幅達成** | 58×35×8cm、可搬式に |
| コスト 7万円以下 | **大幅達成** | 約16,000円 |
| 駆動 電動ブロワ→足踏みペダル | 不採用 | ピアニカ原理に回帰 |

「7万円・家具級」という当初の規模感は、ボディ放射への転換とフットポンプ廃止により大幅に小型化された。**機能は維持しつつ、構造は単純化**。
