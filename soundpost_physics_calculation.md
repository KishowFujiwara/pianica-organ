# 魂柱構造の物理計算

## 出発点: 全面接着に隠された問題

前回「ABS壁面に桐を全面接着すれば T=97% でエネルギーが渡る」
と結論した。しかし全面接着には見落としていた副作用がある。

**マスローディング（質量負荷）**

全面接着では桐がABSと一体となって振動する。
桐が空気を押すとき、ABSの質量も一緒に動かさなければならない。
軽い桐が重いABSを背負って振動することになる。

```
面密度:
  m_s_kiri = 280 * 0.002 = 0.56 kg/m^2
  m_s_ABS  = 1050 * 0.003 = 3.15 kg/m^2

全面接着時の複合板:
  m_s_composite = 0.56 + 3.15 = 3.71 kg/m^2  (桐単体の6.6倍)
```

Gamma は面密度に比例するから、複合板のGammaは桐単体の
数十倍に跳ね上がる可能性がある。


### Gamma の再計算 (全面接着)

複合板の損失係数は質量加重平均で近似:

```
eta_composite = (eta_ABS * m_ABS + eta_kiri * m_kiri) / (m_ABS + m_kiri)
              = (0.03 * 3.15 + 0.008 * 0.56) / 3.71
              = (0.0945 + 0.0045) / 3.71
              = 0.0267
```

Gamma は eta * m_s に比例するから:

```
Gamma_composite / Gamma_kiri = (eta_c * m_s_c) / (eta_k * m_s_k)
                              = (0.0267 * 3.71) / (0.008 * 0.56)
                              = 0.0990 / 0.00448
                              = 22.1

Gamma_composite = 1.8 * 22.1 = 40
eta_rad_composite = 1 / (1 + 40) = 2.4%
```

**全面接着での放射効率は36%ではなく2.4%。**
T=97%でエネルギーを渡しても、複合板のGamma=40が
放射を殺す。

```
全面接着の実効効率: T * eta_rad = 0.97 * 0.024 = 2.3%
前版の主張(35%)は 0.97 * 0.36 = 35% だったが、
桐がABSを背負う質量負荷効果を無視していた。
```

ただしこれは最悪ケース(低周波で完全結合)の見積もり。
高周波では部分的に分離する可能性がある。
実際の値は 2.4% ~ 36% の間のどこかにある。


---

## 魂柱が解決すること

魂柱（サウンドポスト）は面接触を点接触に変える。

桐板はフレームのエッジだけで支持され、
ABS壁面には接触しない。
桐板は自分の質量(0.56 kg/m^2)だけで振動する。

```
全面接着: 桐 + ABS が一体振動 -> Gamma = 40
魂柱:     桐のみ自由振動      -> Gamma = 1.8 (保証)
```

代わりに、伝達率 T は面接触の97%から点接触の値に下がる。
その T がいくらになるかを計算する。


---

## 駆動点インピーダンスの計算

無限板の駆動点インピーダンス(周波数に依存しない定数):

```
Z_point = 8 * sqrt(D * rho_s)

D = E * h^3 / (12 * (1 - nu^2))   ... 曲げ剛性
rho_s = rho * h                    ... 面密度
```


### リードプレート (真鍮, 1mm)

プロジェクト既存値(pianica_22paths_system_identification.md):

```
E = 100 GPa
h = 0.001 m
rho = 8400 kg/m^3
nu = 0.34

D_plate = 100e9 * (0.001)^3 / (12 * 0.884)
        = 100e9 * 1e-9 / 10.61
        = 9.43e-3 Nm

rho_s = 8400 * 0.001 = 8.4 kg/m^2

Z_plate = 8 * sqrt(9.43e-3 * 8.4)
        = 8 * sqrt(0.0792)
        = 8 * 0.281
        = 2.25 Ns/m
```


### ABS壁面 (1.5mm)

プロジェクト既存値:

```
E = 2.3 GPa
h = 0.0015 m
rho = 1050 kg/m^3
nu = 0.35

D_ABS = 2.3e9 * (0.0015)^3 / (12 * 0.878)
      = 2.3e9 * 3.375e-9 / 10.53
      = 7.76e-3 / 10.53
      = 7.37e-4 Nm

rho_s = 1050 * 0.0015 = 1.575 kg/m^2

Z_ABS = 8 * sqrt(7.37e-4 * 1.575)
      = 8 * sqrt(1.16e-3)
      = 8 * 1.077
      = 8.62 Ns/m
```


### 桐板 (2mm) --- 新規計算

```
E_kiri_eff = 5 GPa  (L方向7.5, R方向0.5の幾何平均的な板曲げ有効値)
h = 0.002 m
rho = 280 kg/m^3
nu = 0.3

D_kiri = 5e9 * (0.002)^3 / (12 * (1 - 0.09))
       = 5e9 * 8e-9 / 10.92
       = 0.040 / 10.92
       = 3.66e-3 Nm

rho_s = 280 * 0.002 = 0.56 kg/m^2

Z_kiri = 8 * sqrt(3.66e-3 * 0.56)
       = 8 * sqrt(2.05e-3)
       = 8 * 0.0453
       = 0.362 Ns/m
```


### 3材料の駆動点インピーダンス一覧

```
| 材料 | h [mm] | rho_s [kg/m^2] | D [Nm] | Z_point [Ns/m] |
|------|--------|----------------|--------|----------------|
| 真鍮プレート | 1.0 | 8.4 | 9.43e-3 | 2.25 |
| ABS壁面 | 1.5 | 1.575 | 7.37e-4 | 0.69 |
| 桐板 | 2.0 | 0.56 | 3.66e-3 | 0.362 |
```

重要な注: Z_point は「その材料の板を1点で押したとき、
どれだけ押し返すか」を表す。大きいほど硬くて重い。
周波数に依存しない定数である(無限板近似)。

## 魂柱の伝達率計算

魂柱は剛体棒と近似する(柱長 < 波長の条件で成立)。
力がそのまま伝わるので、系は2つのインピーダンスの直列接続。

```
伝達率 T = 4 * Z_source * Z_load / (Z_source + Z_load)^2
```


### Case 1: リードプレート -> 魂柱 -> 桐板 (直結)

```
Z_source = Z_plate = 2.25 Ns/m
Z_load   = Z_kiri  = 0.362 Ns/m

T = 4 * 2.25 * 0.362 / (2.25 + 0.362)^2
  = 3.258 / (2.612)^2
  = 3.258 / 6.823
  = 47.8%
```


### Case 2: ABS壁面 -> 魂柱 -> 桐板

```
Z_source = Z_ABS  = 0.69 Ns/m
Z_load   = Z_kiri = 0.362 Ns/m

T = 4 * 0.69 * 0.362 / (0.69 + 0.362)^2
  = 0.999 / (1.052)^2
  = 0.999 / 1.107
  = 90.3%
```

ABS壁面から桐への魂柱伝達は Z 比が小さいため 90% を超える。


### Case 3: リードプレート -> 魂柱 -> ABS -> 魂柱 -> 桐板

```
T_total = T_plate-ABS * T_ABS-kiri

T_plate-ABS:
  Z_source = 2.25, Z_load = 0.69
  T = 4 * 2.25 * 0.69 / (2.25 + 0.69)^2
    = 6.21 / 8.64
    = 71.9%

... 既存データでは BN2 の実効 T = 53% (多段経路+モード重なり)
... 71.9% は理想値。実効では 53% が妥当。

T_ABS-kiri(魂柱): 90.3%

T_total = 0.53 * 0.903 = 47.9%
```

## 全面接着 vs 魂柱: 正味効率の比較

BN2 以降(リード振動 -> 音)の効率で比較する。


### 全面接着 (現行 v3 設計)

```
BN2(53%) * T_ABS-kiri(97%) * eta_rad(??%) = ?

eta_rad は Gamma に依存し、Gamma はマスローディング状況に依存。

最悪ケース (低周波、完全結合):
  Gamma_composite = 40
  eta_rad = 1/(1+40) = 2.4%
  正味: 0.53 * 0.97 * 0.024 = 1.2%

最良ケース (高周波、完全分離):
  Gamma_kiri = 1.8
  eta_rad = 36%
  正味: 0.53 * 0.97 * 0.36 = 18.5%

実際値: 1.2% ~ 18.5% のどこか
```


### 魂柱 (新設計)

```
BN2_post(53%) * T_ABS-kiri_post(90%) * eta_rad_kiri(36%) = ?

ここで eta_rad = 36% は確定値。
桐板がABSから力学的に分離されているため、
マスローディングは発生しない。Gamma = 1.8 が保証される。

正味: 0.53 * 0.90 * 0.36 = 17.2%
```


### 比較表

```
| 設計 | BN2 | T_interface | Gamma | eta_rad | 正味 |
|------|-----|-------------|-------|---------|------|
| 全面接着(最悪) | 53% | 97% | 40 | 2.4% | 1.2% |
| 全面接着(最良) | 53% | 97% | 1.8 | 36% | 18.5% |
| 魂柱 | 53% | 90% | 1.8 | 36% | 17.2% |
```

**魂柱は全面接着の最良ケースにわずかに劣るが、
最悪ケースの14倍良い。**

そして最良ケース(Gamma=1.8)が実現する条件は
「高周波で桐がABSから完全に分離振動すること」であり、
実際にこれが全帯域で成立する保証はない。

**魂柱は全帯域でGamma=1.8を保証する。**
これが魂柱の本質的な優位性。

## 魂柱直結: リードプレートから桐へ直接

BN2経路を短縮する可能性がある。
リードプレートから直接魂柱を立てて桐板に渡せば、
ABS壁面を経由しない。

```
リードプレート -> 魂柱 -> 桐板 (ABS経由なし)

T = 47.8% (Case 1 の値)

正味: T_post(47.8%) * eta_rad(36%) = 17.2%
```

ABS経由の場合:

```
BN2(53%) * T_post(90%) * eta_rad(36%) = 17.2%
```

偶然にも同じ値になる。

```
直結:  0.478 * 0.36 = 17.2%
ABS経由: 0.53 * 0.90 * 0.36 = 17.2%
```

直結の方がシンプルだが、物理的にリードプレートから
桐板まで魂柱を渡す構造が必要。空気室の気密を維持
しながら柱を通す工夫が要る。

## 魂柱の設計パラメータ

### 材料

魂柱は剛体近似が成立する条件:
柱の長さ l が曲げ波長 lambda_b より十分短いこと。

```
lambda_b = sqrt(2*pi / f) * (D / (rho*h))^(1/4)
```

柱長 l ~ 5-10mm (ABS壁厚 + 隙間) とすると、
500Hz以下で lambda_b >> l が成立し、剛体近似は妥当。

材料候補:
- スプルース (バイオリンの魂柱と同じ): Z = 400*5600 = 2.2 MRayl
- 竹: Z = 700*5000 = 3.5 MRayl
- 桐 (桐板と同素材): Z = 280*3600 = 1.0 MRayl
- ABS (成型品として): Z = 1050*2760 = 2.9 MRayl

魂柱の材料選択は伝達率に影響しない(剛体近似なので)。
ただし魂柱自体の内部損失は小さい方がよいので、
木材(スプルースまたは桐)が適切。


### 直径

バイオリンの魂柱: 直径 6mm。
表板の有効振動面積 ~350 cm^2 に対して、
接触面積 0.28 cm^2。面積比 = 0.08%。

ピアニカ桐板: 有効振動面積 ~900 cm^2 (400x225mm) に対して、
同じ面積比なら接触面積 0.72 cm^2 -> 直径 ~9.6mm。

ただしピアニカは複数の音が同時に鳴るため、
1本の魂柱ではなく複数本が適切。


### 本数と配置

バイオリンは弦4本に対して魂柱1本+駒2脚。
駒の高音側脚の直下に魂柱を配置し、表板と裏板を直結する。

ピアニカは最大37音が同時に鳴り得る。
仕切り壁31枚のうち、主要な振動伝達点は限られる。

提案: 仕切り壁の位置に合わせて 5-8 本の魂柱を配置。
仕切り壁自体を「面接触」から「点接触(柱)」に置き換える
イメージ。

```
現行: [プレート]===仕切り壁31枚(面接触)===[ABS壁]===[桐全面接着]
改良: [プレート]---魂柱5-8本(点接触)---[桐板(自由振動)]
```


### 位置の最適化

バイオリンでは魂柱の位置が音色を決定的に支配する。
数ミリの移動で音が劇的に変わる。

物理的には、魂柱を板のモード節線(振動しない線)に
置いてはならない。節線上では板の速度がゼロなので
エネルギーが渡らない。

桐板 400x225mm の低次モードの節線:

```
(1,1) モード: f_11 = 96 Hz -> 節線なし(全面が同位相)
(2,1) モード: f_21 ~ 190 Hz -> 長手方向中央に節線1本
(1,2) モード: f_12 ~ 280 Hz -> 短辺方向中央に節線1本
(3,1) モード: f_31 ~ 340 Hz -> 長手方向に節線2本
```

魂柱を長手方向中央に置くと、(2,1)モード(190Hz)が
駆動されない。(2,1)の節線を避けて、中央から少しずらす。

バイオリンの魂柱が駒の「少し後ろ」に置かれるのは、
まさにこのモード節線回避のためである。

## 魂柱による表裏2面放射

バイオリンの魂柱が表板と裏板を結合するように、
ピアニカでも上面と下面の2枚の桐板を魂柱で結合し、
放射面積を2倍にできる。

```
      [桐板(上面)] -- 放射面1
          |
        魂柱
          |
  [リードプレート/ABS構造体]
          |
        魂柱
          |
      [桐板(下面)] -- 放射面2
```

放射面積が2倍になると:
- 放射インピーダンス Z_rad = rho_air * c * S が2倍
- Gamma = eta * m_s / (rho_air * c * S/補正) が 1/2 に
- Gamma = 1.8 / 2 = 0.9
- eta_rad = 1/(1+0.9) = 53%

2面放射の正味効率:

```
BN2(53%) * T_post(90%) * eta_rad_2面(53%) = 25.3%
```

1面の 17.2% から 25.3% に改善。

## 結論

### 全面接着の問題

ABS壁面に桐を全面接着する設計は、T=97%という高い
界面伝達率を持つが、マスローディングにより桐板の
実効Gammaが1.8から最大40まで悪化する危険がある。

低周波域(F3=175Hz付近)では完全結合に近く、Gamma悪化が深刻。
高周波域では部分的に分離し、状況は改善される。
全帯域での実効Gammaは不確定。

### 魂柱の優位性

魂柱構造は面接触を点接触に変えることで:

1. 桐板をABSの質量から力学的に解放する
2. Gamma = 1.8 を全帯域で保証する
3. 界面伝達率は97%から90%に下がるが、
   Gammaの確実な低減が補って余りある
4. 2面放射に拡張すれば Gamma = 0.9, eta_rad = 53% に到達

### バイオリンとの構造的同型性

```
バイオリン:
  弦 -> 駒(2脚) -> 表板 + 魂柱 -> 裏板
  2面放射、点入力、板は自由振動

ピアニカ理想形:
  リード -> プレート -> 魂柱(数本) -> 桐板(上) + 魂柱 -> 桐板(下)
  2面放射、点入力、板は自由振動
```

ピアニカの理想的な空気箱は「桐の膜を張ったフレーム構造」であり、
膜はエッジだけで支持され、魂柱だけでエネルギーを受け取る。
ABS空気室は圧力容器としてのみ機能し、音の放射には関与しない。
