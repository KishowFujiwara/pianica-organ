# スピネット形状ピアニカ: 桐箱空気室 + 桐箱共鳴箱 + バスレフ

## 設計思想

圧力と放射を構造的に分離する。

```
内箱(空気室): 密閉・加圧。リードを鳴らすだけが仕事。
外箱(共鳴箱): 大気圧。音を放射するのが仕事。バスレフ穴あり。
```

空気室に穴は開けない。共鳴箱に穴を開ける。
共鳴箱は加圧されていないから、穴を開けても何の問題もない。

これはパイプオルガンの風箱+共鳴管の構造と同じ。
ピアニカの共鳴管を「箱型」に折り畳んだもの。


---

## スピネット形状

スピネット(小型チェンバロ)は台形の筐体を持つ。
低音側が広く、高音側が狭い。弦長の違いに対応する形。

ピアニカでも同じ原理が使える。
低音域は波長が長く、共鳴箱の容積が大きい方が有利。
高音域は波長が短く、小さな容積で十分。

```
平面図:
         250mm
  +------------------+
  |    Bass zone     |  <- 広い(F3-B3)
  |                  |
  |   Mid zone       |
  |                  |
  |  Treble zone     |  <- 狭い(C5-C7)
  +--------+---------+
          120mm

  <--- 400mm (keyboard direction) --->
```


### 寸法

```
| 部位 | 長さ | 幅(低音端) | 幅(高音端) | 高さ | 容積 |
|------|------|-----------|-----------|------|------|
| 空気室 | 400mm | 250mm | 120mm | 25mm | 1.85L |
| 共鳴箱 | 400mm | 250mm | 120mm | 50mm | 3.70L |
| 合計高さ |  |  |  | 75mm |  |
```

空気室は共鳴箱の上に載る。リードのバルブ開口部は
下向きに共鳴箱内部に面する。

## エネルギーフロー

```
息 -> [空気室(密閉)] -> 鍵を押す -> バルブ開く
   -> 空気がリードスロット通過 -> リード自励振動 (BN1: 18%)
   -> 音がバルブ開口部から下向きに放出
   -> [共鳴箱(大気圧)] に音が入る
   -> 3つの放射メカニズム:
      a. 桐壁の板振動 (Gamma=0.66, eta_rad=60%)
      b. バスレフポート (f_H=175Hz)
      c. 内部圧力による壁面モノポール駆動
```

**v3の固体伝導鎖が、空気経由の伝達に置き換わる。**

v3設計: リード振動 -> プレート -> 仕切り壁 -> ABS壁 -> 桐
  (固体-固体のインピーダンス不整合が複数段)

スピネット: リード -> スロット開口 -> 共鳴箱内空気 -> 壁/ポート
  (リードプレートが反射壁となり、音はスロット開口部に集中して放出される)

> **注記 (0430g 検証で更新)**:
> 0430g ジャーナルにて、リード周りの3層(系A/系B/系C)を独立検証した結果、
> 「BN2 53% から 95% への改善」という以前の数値主張は撤回された。
> 比較対象が固体伝導鎖と音響伝達で異なり、同じBN2ラベルで並べられない。
>
> ただし系Bにおいて、リードプレート(黄銅1mm)が空気の83.5%を反射する
> 「ほぼ完全な壁」として機能することが判明し、
> リードはスロット開口部40mm²のみから放射するモノポール音源として
> 確定した。これはスピネット2重箱構造の方向性そのものを支持する。

## ゾーン別共鳴箱設計

共鳴箱をスピネット形状に従い3ゾーンに分割。

```
| ゾーン | 帯域 | 長さ | 平均幅 | 容積 | ヘルムホルツ |
|--------|------|------|--------|------|------------|
| Bass | F3-B3 (175-247Hz) | 133mm | 207mm | 1.37L | 必要 |
| Mid | C4-B4 (262-494Hz) | 133mm | 185mm | 1.23L | 有効 |
| Treble | C5-C7 (523-2093Hz) | 134mm | 163mm | 1.09L | 不要 |
```


### バスレフポート

共鳴箱は大気圧。ポートは自由に配置可能。

Bass ゾーン(V=1.37L)に f_H=175Hz のポート:

```
ポート: 1 x d=20mm (側面に配置)
l_eff = 5mm(壁厚) + 0.85*20mm = 22mm
f_H = 343/(2*pi) * sqrt(3.14cm^2 / (22mm * 1.37L)) = 176 Hz

[PASS] F3(175Hz) にほぼ一致
```

20mm 穴が1つ。極めてシンプル。
設計Cの弁リードは不要。穴を開けるだけ。


### 空気モード

共鳴箱Bass ゾーンの空気モード:

```
f_100 = c / (2 * 133mm) = 1289 Hz (長さ方向)
f_010 = c / (2 * 207mm) =  829 Hz (幅方向)
f_001 = c / (2 * 50mm)  = 3430 Hz (高さ方向)
```

175Hz(F3) は全ての空気モードより遥かに低い。
この帯域ではヘルムホルツモードが支配的であり、
バスレフポートが主たる低音放射体として機能する。

## なぜスピネット形状か

### 理由1: 容積の自然な勾配

低音域は波長が長く、ヘルムホルツ共鳴には大きな容積が必要。
高音域は板モード放射で十分であり、容積は不要。

台形のスピネット形状は、低音側に容積を、高音側にコンパクトさを
自然に割り当てる。

### 理由2: 壁面積の自然な勾配

低音域は板のsigmaが低い(F3で29%)ため、
放射面積が大きいほど有利。低音側の幅250mmは
高音側の120mmの2倍以上の壁面積を提供する。

### 理由3: 楽器としての自然な形

スピネット/クラビコード/小型ピアノは全て
低音側が大きい台形。演奏者にとって自然な形状。
ピアニカの既存の長方形から、台形への変更は
視覚的にも「本格的な楽器」の印象を与える。

## 空気室の材料選択

空気室は音を放射しないため、材料に音響的要件はない。
必要なのは気密性と加圧への耐久性のみ。

選択肢:

```
A. ABS(現行P-32Dの空気室をそのまま流用)
   + 実績あり、気密確実
   + CAHAYAなど既存品を「部品」として利用可能
   - 重い

B. 桐+気密処理(柿渋/漆/ラップ)
   + 軽い
   + 全体が桐で統一された美しさ
   - 気密処理の信頼性が未検証

C. 3Dプリント(PLA/PETG)
   + 自由な形状
   + 気密性良好
   - 音響的には不要な精度
```

推奨: まずABS(既存品流用)で実験し、動作を確認してから
桐製空気室を検討する。

## 空気室から共鳴箱への音の伝達

バルブが開くと、リードのスロット開口部が
共鳴箱の内部に直接面する。

```
断面図:

  [息 ->]  [空気室(加圧)]
           [リードプレート + リード]
           [バルブ(開)] <-- 鍵を押すと開く
  --------|  開口部  |--------  <-- 空気室の底面
           +--------+
  |                            |
  |     共鳴箱(大気圧)          |
  |                            |
  |     桐壁が振動して放射      |
  |                            |
  +---[バスレフポート]----------+
```

リードの音は開口部から共鳴箱に「落ちる」。
共鳴箱がリードの音をキャッチし、
桐壁の振動とバスレフポートで外部に放射する。

鍵を離すとバルブが閉じ、音は止まる。
バルブが閉じた状態では共鳴箱は密閉されず
(バスレフポートが開いている)、残響は短い。

## ピアニカの新しい3要素モデル

```
従来の3要素:
  発生体(リード) -> 伝達体(壁面) -> 放射体(壁面/空気)
  問題: 伝達体と放射体が未分離。ABS壁が両方を兼ねて両方失敗。

スピネットの3要素:
  発生体(リード+空気室) -> 伝達体(音響結合) -> 放射体(桐共鳴箱)
  解決: 伝達は空気(損失ほぼゼロ)、放射は桐(Gamma=0.66)。
```

パイプオルガンもこの3要素:
```
  発生体(リード+風箱) -> 伝達体(管内空気柱) -> 放射体(管開口+管壁)
```

バイオリンも:
```
  発生体(弦+弓) -> 伝達体(駒) -> 放射体(表板+裏板)
```

全ての優れた楽器は、発生・伝達・放射を分離し、
各段階を独立に最適化している。
P-32Dはこの分離に失敗していた。
スピネット設計はこの分離を構造的に実現する。
