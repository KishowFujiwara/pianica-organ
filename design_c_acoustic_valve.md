# 設計C: DC遮断・AC透過の音響弁

## 問題の再認識

空気室にヘルムホルツのポート(穴)を開けたら、
息の圧力(500Pa)がポートから抜け、リードが鳴らなくなる。
気密なしには発音なし。これがフリーリードの絶対条件。

設計C: ポートに「DC圧をブロックしAC音波だけ通す弁」を付ける。


---

## アプローチ1: 毛細管(粘性によるDCブロック)

空気の粘性抵抗は管径の4乗に反比例する(ポアズイユ流)。
極細管ならDC気流を実質ゼロにできる。

しかし計算すると:

```
f_H=175Hz に必要なポート総面積: 23.1 cm^2
DC漏れ < 1mL/s に必要な管径: 2.5 マイクロメートル
必要な管の本数: 1億1800万本
```

2.5マイクロメートルは赤血球(7um)の半分以下。
実質的に多孔質膜。しかしそのような膜はACも大幅に減衰する。

**判定: 不可能。** DC/AC選択性を粘性で達成するには
微視的な孔が必要であり、ACも通さない。


---

## アプローチ2: カンチレバーリード弁

ポートをスロット(細長い穴)にし、カンチレバー(片持ち梁)の
リードで蓋をする。DC圧がリードを閉位置に押し付け、
AC圧(音波)がリードを共振周波数で微振動させる。

これは発音リードと物理的に同じ構造だが、役割が異なる:

```
発音リード(能動): DC気流で自励振動 -> 音を生成
弁リード(受動):   AC音圧で共振振動 -> 音を透過/再放射
```


### 普遍的制約の発見

カンチレバーリードのDC変位(500Pa下での静的たわみ)を
計算すると、驚くべきことにヤング率Eが消える:

```
defl_DC = P / (omega^2 * 0.236 * rho * h)

P:     DC圧力 [Pa]
omega: 角周波数 = 2*pi*f1
rho:   材料密度 [kg/m^3]
h:     板厚 [m]
0.236: カンチレバー1次モードの有効質量係数
```

**DC変位はE(弾性率)に依存しない。**
rho(密度)とh(厚さ)だけで決まる。

これは「硬い材料を使えばDC変位を減らせる」が
幻想であることを意味する。硬い材料は確かに
バネ定数kが高いが、同じf1を得るにはリードが短くなり、
受圧面積が小さくなるものの、kの増加と面積の減少が
ちょうど相殺して、変位は変わらない。

**DC変位を小さくする唯一の方法は rho*h を大きくすること。**
つまりリードを重くすること。


### 材料別のDC変位とGamma

```
| 材料 | rho*h [kg/m^2] | DC変位 [mm] | Gamma | 判定 |
|------|---------------|------------|-------|------|
| 桐 0.5mm | 0.14 | 12.5 | 0.45 | NG (12mm変位) |
| 桐 2.0mm | 0.56 | 3.1 | 1.80 | NG (3mm変位) |
| 竹 1.0mm | 0.70 | 2.5 | 1.41 | NG (2.5mm変位) |
| 真鍮 0.1mm | 0.84 | 2.1 | 0.34 | NG (2.1mm変位) |
| 真鍮 0.15mm | 1.26 | 1.4 | 0.51 | OK |
| 真鍮 0.2mm | 1.68 | 1.0 | 0.68 | OK |
| 鋼 0.15mm | 1.17 | 1.5 | 0.15 | OK (境界) |

判定基準: DC変位 < 1.5mm (スロット深さ2mmに対して余裕0.5mm)
```


### ジレンマ

```
軽い材料: Gamma 小(放射良い) + DC変位 大(シール不可)
重い材料: Gamma 大(放射悪い) + DC変位 小(シール可能)
```

これは弦楽器の「サスティン vs 放射」のジレンマと
構造的に同型の問題。

```
弦楽器:   軽い弦 = 良い放射 + 短いサスティン
弁リード: 軽い弁 = 良い放射 + DC保持不可
```

どちらも「軽さ」と「エネルギー保持」が背反する。


---

## 解決: 弁は放射しなくていい

ここで発想を転換する。

弁リードの仕事は「DCをブロックしてACを通すこと」であり、
「音を放射すること」ではない。

音の放射は桐の壁面が担当する。
f_Hでは箱内部の空気圧が全壁面を同位相で駆動し、
桐壁(Gamma=0.66)がモノポール放射する。
弁リードは密閉条件を作るだけ。

だから弁リードは重くてよい。Gammaが大きくてよい。
放射面積も小さくてよい。ただ「閉じていて、音波で振動する」
だけでよい。

```
弁リードの要件:
  [必須] DC変位 < 1.5mm (密閉維持)
  [必須] f1 ~ 175Hz (低音域の共振)
  [不要] 低Gamma (放射は壁が担当)
  [不要] 大面積 (放射は壁が担当)
```


### 最適材料: 真鍮 0.15mm

```
寸法: 22 x 8 x 0.15 mm (カンチレバー)
rho*h = 1.26 kg/m^2
DC変位 = 1.4mm at 500Pa -> OK
f1 = 175 Hz
```

この小さな真鍮片が実現すること:

```
1. DC圧(500Pa)でスロットに押し付けられ、気密を維持
2. 内部のAC音圧で175Hz付近で微振動
3. 振動により箱の密閉容積が維持されつつAC圧力変動が発生
4. AC圧力変動が全桐壁をモノポール駆動
5. 桐壁(Gamma=0.66)がF3帯域で+4.3dBの放射
6. 弁リード自体の放射は無視できる(重い、小さい)
```

**弁は門番(GATE)であり、放送局(SPEAKER)ではない。**

## シタールの共鳴弦との同型性

この設計は楽器の歴史に先例がある。

シタールには「共鳴弦(sympathetic strings)」がある。
奏者が弾く弦(主弦)の音に共鳴して、弾かれていない弦が
自発的に振動し、音に豊かさと残響を加える。

```
シタール:
  主弦(能動): 指で弾く -> 音を生成
  共鳴弦(受動): 主弦の音に共鳴 -> 音色を豊かにする

ピアニカ設計C:
  発音リード(能動): 息で自励振動 -> 音を生成
  弁リード(受動): 発音リードの音に共鳴 -> 低音を増強
```

弁リードは「受動リード(passive reed)」であり、
シタールの共鳴弦と物理的に同じ役割を果たす。

弁リードを複数の長さで用意すれば、各音程に対応する
共鳴系を作れる:

```
| 音名 | f [Hz] | 弁リード長 [mm] | DC変位 [mm] |
|------|--------|----------------|------------|
| F3 | 175 | 22 | 1.4 |
| A3 | 220 | 19 | 1.1 |
| C4 | 262 | 18 | 0.9 |
| E4 | 330 | 16 | 0.7 |
| A4 | 440 | 14 | 0.5 |
```

5本の弁リードを桐箱のスロットに配置すれば、
F3からA4まで5つの周波数でヘルムホルツ共鳴が使える。


---

## 設計まとめ

```
桐薄板箱: 400 x 225 x 50mm
壁厚: 0.7mm + リブ(60mm間隔)
弁リード: 真鍮0.15mm, 5本(F3/A3/C4/E4/A4にチューニング)
弁スロット: 各弁リード長 x 8mm, 側面に配置

エネルギーフロー:
  息 -> [空気室(密閉)] -> リード自励振動
      -> 音波が箱内部に伝搬
      -> [弁リードが共振] -> 密閉容積のAC圧力変動
      -> 全桐壁がモノポール放射 (Gamma=0.66)
      -> 低音域 +4.3dB

帯域別:
  F3-A3: 弁リード共鳴 + 壁面モノポール駆動 (ヘルムホルツ型)
  B3-C4: 弁リード共鳴(弱) + 板モード放射
  E4以上: 板モード放射のみ (十分)
```


---

## 発見の階層

```
1. 空気室に穴は開けられない (DC圧が逃げる)
2. 穴の代わりにリード弁を付ける (DC遮断+AC透過)
3. DC変位はrho*hだけで決まる (Eは無関係)
4. 軽い弁は変位が大きすぎ、重い弁は放射が悪い (ジレンマ)
5. しかし弁は放射しなくてよい (壁面が放射する)
6. だから重い弁(真鍮)で問題ない
7. これはシタールの共鳴弦と同型の物理構造
```
