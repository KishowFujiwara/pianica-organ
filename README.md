# pianica-organ

ピアニカ・オルガン設計プロジェクト + フリーリード音響学研究

## 概要

ヤマハ ピアニカP-32Dを音源とした、教会小礼拝室用リードオルガンの設計と、その基盤となるフリーリード音響学の研究成果を集約するリポジトリ。

設計v3 (採用版) のサマリー:
- 共鳴箱14.2L (桐1.5mm + X-brace) + ボディ放射 + ピアニカ原理駆動
- 32鍵 F3-C6 (CAHAYA/P-32D流用)
- 卓上型 550×350×80mm、約16,000円

## リポジトリ構成 (3章)

```
01-specification/   楽器の仕様調査 (Yamaha P-32D, CAHAYA, 鈴木 W-37C)
02-physics/         原理・物理的検証 (22経路解析, パワーフロー, 因果機構, 楽器比較)
03-design/          設計 (v1/v2/v3仕様書 + ジャーナル19本)
```

各章のREADMEに詳細:
- `01-specification/README.md` -- 楽器仕様の集約
- `02-physics/README.md` -- 物理検証の集約
- `03-design/README.md` -- 設計進化の軌跡

## 主要な発見

- ピアニカP-32Dは「振動板型エンクロージャ・スピーカー」
- 外部放射の95.2%はABS壁面由来 (C1+C2)、鍵盤穴経由 (A1) はわずか0.8%
- ABS底面(1,1)曲げモード 323Hz (E4近傍) が音色の核
- 22の振動伝達経路を同定・分類・定量分析
- 楽器のエネルギー階層分類: ピアニカはクラスI微弱励振 (ギターと同類)
- 設計v3はフラットトップアコースティックギターの3DOF結合系を翻訳

## 設計進化の軌跡

```
v1 (37本ラッパ)
  -> 反省: 99%は壁面放射 (22経路解析)
v2 (楔形ホーン1本)
  -> 反省: ホーン不要、共鳴箱が本質
spinet (撤回)
  -> 反省: B系3層構造の物理誤り
v3 (共鳴箱14.2L + ボディ放射)  <- 採用
```
