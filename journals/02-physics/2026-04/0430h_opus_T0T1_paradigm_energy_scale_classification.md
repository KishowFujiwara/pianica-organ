# 2026-04-30h: ヘルムホルツポートからT0/T1結合系へのパラダイムシフト —
#               エネルギースケールで楽器を分類する

## セッションの目的

0430g で dfc79ff の数値モデルを撤回した後、次の論点として
「桐箱空気室と桐箱共鳴室の理想的体積」を起点に設計を再構築。
その過程で「ヘルムホルツ共鳴と板振動の選択」という根本的な
論点に行き着き、最終的に**ユーザの核心的洞察**:

  「ギターが参考になるのは、撥弦エネルギーのロスが
   ピアニカのフリーリードのエネルギーロスと近しいから」

によって、**励振エネルギーで楽器を階層化する** という新しい
設計フレームワークに到達した。

## 発見の要旨

1. **桐箱空気室 V_air = 2.0L (推奨レンジ 1.5〜3.5L)** — 物理拘束から導出
   現行スピネット 1.85L は範囲内で適切。

2. **桐0.7mm はヘルムホルツポートとして単独で機能しない**
   板の音響コンプライアンスが空気の27倍。剛体時 f_H=176Hz が
   結合系では f_H_eff ≈ 33Hz まで下がる。
   → 「20mm ポートで F3 整合」という設計言説は物理的に誤り。

3. **正しい物理は T0/T1 結合共鳴系** (ギター・リュート・三味線と同型)
   板柔らかさによる結合で、共振が2峰に分裂する。
   ピアニカでは T0 ~ 80-100 Hz、T1 ~ 200-325 Hz が予想される。

4. **励振エネルギーで楽器は3階層に分かれる** (本セッション最大の発見)
   - クラスI 微弱励振 (1-20 mW): ギター撥弦、ピアニカリード
   - クラスII 中規模励振 (10-100 mW): バイオリン、ピアノ
   - クラスIII 強力励振 (1W+): スピーカー
   ピアニカが学ぶべきはギター製作。バイオリンではない。
   理由は「励振の連続性」ではなく「エネルギースケール」。

5. **設計指針の単純化**
   × 「ピアニカ独自の最適化を一から考える」
   ○ 「ギター製作1000年の蓄積を、励振方式だけ
      フリーリードに置き換えて転用する」

## Phase 1: 体積設計の物理拘束

### 空気室 V_air

物理拘束を4軸で整理:

| 拘束 | 方向 | 値 |
|------|------|----|
| A1 単音25ms過渡 (ΔP≤10%) | 下限 | ≥ 0.5 L |
| A2 和音(10音)過渡 | 下限 | (極限) 79 L だが息直接駆動で無視可 |
| A3 脈動5Hzカットオフ | 下限 | R次第で1〜9 L |
| A4 応答 ≤5ms | 上限 | ≤ 3.5 L |

ピアニカは息直接駆動でブロワ応答時間がほぼゼロのため、
A1・A2 の極限値は実用上不要。

**推奨レンジ 1.5〜3.5 L、中央 2.0 L**。
現行スピネット 1.85L は妥当 [PASS]。

### 共鳴箱 V_res

ヘルムホルツ整合式 V = A/(l_eff·(2πf/c)²) より、
F3=175Hz 整合のためのポート径と V のトレードオフ:

| ポート径 | 必要 V (F3整合) |
|---------|----------------|
| 15 mm   | 0.97 L         |
| 20 mm   | 1.39 L (現行)   |
| 25 mm   | 1.82 L         |
| 30 mm   | 2.26 L         |

**しかしこれは『剛体箱』前提の計算**。本物の桐0.7mm では
この式が成立しないことが Phase 3 で判明する。

## Phase 3: 結合の発見 — 桐0.7mm は『柔らかすぎる』

### 計算 (音響系で統一)

```
m_a (音響質量, ポート空気塊) = ρ·l/A = 84.7 kg/m⁴
C_air (空気のコンプライアンス) = V/ρc² = 9.6 nL/Pa
C_plate (板のコンプライアンス) = η·A_plate/(ω²·ρh) = 259 nL/Pa
                                 (η_mode=0.45, A_plate=0.148 m²)

比 C_plate / C_air = 27 倍

結合系:
  C_total = C_air + C_plate = 268 nL/Pa
  f_H_eff = √(1/(m_a·C_total))/(2π) ≈ 33 Hz
  Q_eff   ≈ 10 × √(C_air/C_total) ≈ 2
```

**剛体時 f_H=176Hz が、桐板の柔らかさで f_H_eff ≈ 33Hz まで下がる**。
Q も 10 → 2 に落ちる。単純なヘルムホルツとして機能しない。

### 含意

dfc79ff の文書中の主張:
> ポート1 x d=20mm (側面に配置)
> f_H = 343/(2*pi) * sqrt(3.14cm^2 / (22mm * 1.37L)) = 176 Hz
> [PASS] F3(175Hz) にほぼ一致

これは**剛体箱という暗黙の前提**で計算されていた。桐0.7mm板を使う
スピネット設計では、この計算は意味を持たない。

## Phase 4: T0/T1 結合モデル — ギター物理に学ぶ

板柔らかさによる結合系では、共鳴が**2つのピークに分裂**する。
これは Christensen-Vistisen モデルとして弦楽器音響学で確立されている:

```
T0 (低音側ピーク) ≈ ヘルムホルツと板の混合
T1 (高音側ピーク) ≈ 板モード単独に近い
```

ギターの典型値: T0 = 100Hz、T1 = 200Hz。
ピアニカでは桐板パラメータから T0 ~ 80-100Hz、T1 ~ 200-325Hz が予想される。

### ギター・リュート・三味線・琵琶 — 千年の経験的最適化

これらの楽器すべてが T0/T1 結合系で動作している。
人類は弦楽器の千年の歴史で、この構造を経験的に発見し最適化してきた。

### 設計案の修正

| 案 | 構成 | 楽器性 | 低音SPL | 用途 |
|----|------|--------|---------|------|
| A 板振動マスター | 桐0.7mm + ポート無 | ◎ | △ | シンプル実験機 |
| B' ヘルムホルツマスター | 桐3mm/MDF + 大ポート | △ | ◎ | 効率特化 |
| **★ C T0/T1 結合系** | **桐0.7mm + 20mmポート** | **◎** | **○** | **楽器の正解** |

dfc79ff の構造選択 (案C) は、**解釈を T0/T1 結合系に置き換えれば正当**。
撤回したのは数値モデルであり、構造提案そのものではない。

## Phase 5: 励振エネルギーで楽器を階層化 (本セッション最大の発見)

### ユーザの核心的洞察

> ギターが参考になるのは、撥弦エネルギーのロスが
> ピアニカのフリーリードのエネルギーロスと近しいから

これは構造的アナロジーよりも深い洞察。**励振エネルギーのスケールで
楽器が分類される**ことを示唆している。

### 各楽器の励振パワー定量化

```
ギター撥弦:
  弦の蓄積エネルギー (1/2)Tx²/L = 1.54 mJ
  減衰時間 τ ≈ 3s
  平均放射相当パワー = 0.51 mW

バイオリン弓:
  弓のメカパワー F·v = 300 mW
  弦への伝達 (stick-slip効率) 5%
  弦の有効励振パワー = 15 mW

ピアニカ・フリーリード:
  息のパワー = 120 mW
  リード周りに入るパワー (BN1=18%) = 21.6 mW
  リード本体維持 = 1.05 mW
  実効音響源パワー = 20.6 mW

ピアノ打鍵:
  打鍵エネルギー (1/2)mv² = 100 mJ
  減衰時間 τ ≈ 15s
  平均パワー = 6.7 mW

スピーカー:
  アンプ駆動 = 5000 mW
```

### 3階層に分かれる

```
クラスI 微弱励振 (1-20 mW)         必要効率: 高 (≥25%)
  ギター撥弦    0.5 mW
  ピアニカ       17 mW   ← 同じ階層
  → 板+空洞の結合系で 30%効率必須

クラスII 中規模励振 (10-100 mW)    必要効率: 中 (5-15%)
  バイオリン弓  15 mW
  ピアノ打鍵     7 mW
  → 共鳴箱は大型化、効率より帯域・音色重視

クラスIII 強力励振 (1W+)            必要効率: 低 (1-5%)
  スピーカー   5000 mW
  → 剛体箱で f_H ピンポイント、効率より制御性
```

### バイオリンを参考にしてはいけない理由

0430f ジャーナルでは「ピアニカv3はバイオリンに最も近い」と結論されていた。
共通項は「励振の連続性」だった。しかし**励振の連続性は枝葉の特徴**であり、
**根本基準は励振エネルギースケール**であることが本セッションで判明。

バイオリンはクラスII (15mW級) のため、F孔・F-hole は
「弓のエネルギーが余っている前提」で設計されている。
これをクラスI (1-20mW) のピアニカに転用すると、共鳴箱効率が
不足する。

### スピーカー型を参考にしてはいけない理由

クラスIII (5W) では効率より制御性。剛体箱+大ポートの構造は
電気増幅の前提。ピアニカに転用するとエネルギーが圧倒的に
不足する。

### ギターを参考にすべき理由

クラスIに属する数少ない楽器。励振エネルギーが希少という根本制約を
ピアニカと共有しており、共鳴箱で 30% 効率を稼ぐ最適解として
T0/T1 結合系を獲得している。1000年の蓄積を転用可能。

3. **bridge的構造** — リードプレート→桐板への結合点を
   ギターの駒に倣って設計
4. **ポート設計** — ローズウィンドウ的装飾も含めてギター実装を流用可能
