## 検算で確定した事実

### Gamma の式の次元誤り (原典0430f 自体に存在)

```
原典の式: Gamma = eta * rho_mat * h / (rho_air * c)
単位: [s] (時間)、無次元ではない

ピアニカv3 のパラメータでの値: 6.8e-5 [s]
原典が記載する Gamma 値: 1.8 (無次元)
差倍率: 約 2.65e+04
```

**結論**: 原典の Gamma 値は eta_rad から `Gamma = 1/eta_rad - 1` で逆算した
無次元数。原典0430f の式自体に誤りがある。

### eta_rad = 1/(1+Gamma) (数学的整合)

ピアニカv3: 1/(1+1.8) = 0.357 ≒ 36% (原典と一致)
ただし Gamma 自体が逆算値なので、これは「逆算値の再変換」。

### f_11 の検算 (妥当範囲)

300x300mm, h=2mm, 桐, 単純支持仮定:
- E=5 GPa: f_11 = 89.3 Hz
- E=6 GPa: f_11 = 97.8 Hz
- E=7 GPa: f_11 = 105.6 Hz

原典の 96 Hz は E≒6 GPa の範囲。妥当な設計値。
ただし境界条件は実際には単純支持ではないため、実機での実測値は前後する。

### T_coupling = 97% (検算不可)

ピアニカv3 については原典に Z_source, Z_board の値の記載がなく、検算不可能。
実測待ち。

### リード->板 1-2mW、足踏み 80mW (出典不明)

原典0430f に記載なし。旧版 speaker_vs_resonance_box.md (Appendix B) で
