## ジャーナル間の相対パス規則

```
源:  journals/{src_chap}/{src_ym}/foo.md
先:  journals/{tgt_chap}/{tgt_ym}/bar.md

同章同月: ./bar.md
同章別月: ../{tgt_ym}/bar.md
別章    : ../../{tgt_chap}/{tgt_ym}/bar.md
```

## ジャーナル → 各章への参照

```
源:  journals/{chap}/{ym}/foo.md
先:  03-design/v3/design-spec-v3.md

相対: ../../../03-design/v3/design-spec-v3.md  (深さ4)
```
