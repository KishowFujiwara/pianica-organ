本ディレクトリは、ピアニカ・オルガン設計プロジェクトにおける**全セッションジャーナル**を集約する。設計成果物 (`03-design/v1, v2, v3, spinet/`) と並列の独立した記録体系。

## 来歴


## ジャーナルの命名規則

```
YYYYMM[suffix]_modelname_short_topic.md

YYYYMM    : 4桁年月 + 2桁日 (例: 0430)
[suffix]  : 同日内通番 (a, b, c, ... 必要時のみ)
modelname : 担当モデル名 (opus, sonnet, haiku)
short_topic : スネークケースの主題
```

例:
- `0423_opus_pianika_organ_design.md` (Claude Opus 4.7、2026-04-23)
- `0430g_opus_B_system_decomposition_dfc79ff_verification.md` (同日7本目)
