## 3つの重大な構造的発見

### 1. B2c: 仕切り壁31枚の面接触 (写真#1, #10)
元モデルで完全に見落とし。ABS仕切り壁頂部がリードプレート裏面に
直接接触(金属-ABS)。接触面積1163mm^2はネジ(106mm^2)の11倍。
BN2の伝達パワーの88%がこの経路を通る。

### 2. B4: 二部品+スポンジ遮断 (写真#1, #2, #3)
ABS箱は一体成型品ではなく、白ABS空気室+黒スポンジ帯+青ABS底面ケース。
元モデルのコーナー伝達(-30dB)は誤りで、実際はスポンジ遮断(-38~-42dB)。

### 3. C1: 二重壁放射 (写真#1, #2)
壁面放射は単壁ではなく、白ABS->スポンジ->青ABSの二重壁を通過。
放射レベルは-15~-20dBから-22~-28dBに悪化。

## BN2の正確な再定義

旧: BN2 = リード -> 壁面 (Z_metal vs Z_ABS = 20:1)
新: BN2 = リード -> [リベット] -> リードプレート -> [3並列接触] -> ABS壁面

3並列接触の寄与:
- B2a ネジ15本 (点接触, T=18%): 寄与 8%
- B2b ゴムガスケット (面接触, 高損失): 寄与 4%
- B2c 仕切り壁31枚 (面接触, T=18%): 寄与 88% [支配的]

## Phase 2: 共鳴箱理想形の検証 (本セッション後半)

### 出発点

エネルギー経路モデルv2.0が明らかにした事実:
「ABSの空気箱の中でフリーリードが発声する」

この一文から共鳴板/共鳴箱の理想形を物理的要件の逆算で導出。

### 4つの物理的要件

1. **気密性**: リードの自励振動には箱内圧力が必要。気密と「鳴る」は矛盾しない(ドラム、バイオリンも気密かつ鳴る)
2. **低Gamma**: eta * m_s < 0.0051 (Gamma<2, 放射効率>33%)。桐2mm(Gamma=1.8)、薄桐1.5mm(Gamma=1.4)が合格
3. **モード帯域**: f_11 < 175Hz(最低音F3)。桐2mm 400x225mmで f_11=96Hz [OK]
4. **上記3要件の同時達成**

### 3つの構造タイプの定量比較

| 構造 | BN2経路 | BN3(Gamma) | 放射面積 | 改善 |
|------|---------|-----------|---------|------|
| A: 分離型 (ABS箱+板) | 金属->ABS->Bridge->板 (3段) | 4.1 | 板のみ | +0.5dB |
| B: 一体型 (桐共鳴箱) | 金属->桐仕切->桐壁 (1段+一体) | 1.8 | 全面 | +9.5dB |
| C: ハイブリッド (ABS+桐) | 金属->ABS->桐壁 (1段+T=97%) | 1.8 | 外面 | +8.5~+13dB |
| P-32D現行 | 金属->ABS->スポンジ->ABS (2段+遮断) | 93 | 底面のみ | 基準 |

### 予想外の発見: ABS-桐のインピーダンス整合

Z_ABS = 1.55 MRayl vs Z_kiri = 1.06 MRayl -> 比 1.5
**伝達率 T = 96.5%**

ABSと桐のインピーダンスがほぼ同じ。ABS壁に桐板を直接接着すれば
振動がほぼ無損失で桐に伝達される。

これにより構造C(ハイブリッド)が構造B(全桐)より有利になる:
- 構造B: 金属->桐 T=12.7% -> +9.5dB
- 構造C: 金属->ABS T=18.1% -> ABS->桐 T=96.5% -> +13dB

理由: ABSのZは金属と桐の「中間」にあり、
意図せずインピーダンス整合層として機能している。

### +15dBに届かない理由

当初推定(BN3だけ33倍改善で+15dB)は楽観的だった。
v2.0モデルで判明したBN2(仕切り壁31枚)の実態により、
BN2がボトルネックとして残ることが明確になった。

### 理想形の再定義

```
理想形 = 構造C: 既存ABS空気室 + 桐外箱(密着接着)

  既存ABS空気室をそのまま使用
    -> 仕切り壁31枚のBN2伝達(T=18%)を維持
    -> 気密はABSが担保
  桐外箱をABS壁面に直接接着(スポンジなし)
    -> T=97%でABS振動をほぼ無損失で受け取る
    -> Gamma=1.8で36%を音に変換
    -> 三角形で低音側を広く

  推定SPL@1m = 85dB (P-32D 72dB比 +13dB)
```

### ABSは敵ではなく「インピーダンス整合層」

P-32Dが「壊れたスピーカー」だった真因は、ABSが悪いのではなく:
1. ABSの先にスポンジ+追加ABS壁があった(二重壁)
2. ABS壁面のGamma=93で99%が熱になっていた

ABSの先に桐を貼れば:
- スポンジを排除(直接接着)
- 桐がGamma=1.8で放射
- ABS->桐 T=97%で界面損失ほぼゼロ

### 実験計画への影響

CAHAYA改造の方針がさらに明確化:
1. まず底面のスポンジ+青ABSを除去
2. ABS空気室底面に桐2mm板を直接接着
3. Spectroidで測定 -> BN3改善を確認
4. 全面桐貼りに拡大
