# 2026-04-30g: dfc79ff 検証セッション — B系3層分解とリードプレート反射の発見

## セッションの目的

直前のセッション(73dbe6dd)で作成した `dfc79ff [design] スピネット形状ピアニカ` の検証。
ユーザからの指摘:「音響結合95%は大きすぎる、計算だと疑わしい」を起点に、
B系(BN系)の力学を**リード本体・プレート反射・空気場挙動**の3層に分解して
それぞれ独立に物理計算を行った。

## 発見の要旨

1. **系B (リードプレート反射) で新発見**:
   黄銅1mmプレートは Z_plate/Z0 = 22.3 で 83.5%反射。
   リードの音はプレート方向には事実上出ず、スロット開口部40mm²
   からのみ放射される。**リードはモノポール音源として確定**。

2. **dfc79ff の数値主張は撤回が必要**:
   「BN2: 53% -> 95%」は比較対象が別物理。
   「全体効率10.3%」「P-32D比98倍」は単一値モデルが破綻。
   ただし「2重箱でBN2バイパス」という構造提案自体は有効。

3. **共振帯域と非共振帯域で物理が違う**:
   F3共振: スロット出力 -> 箱内増幅(Q=10倍) -> ポート集中放射
   非共振: 箱内圧弱、桐壁の板モード放射(Gamma=0.66)が支配
   単一の「BN2 x BN3」連鎖では表現不可能。

---

## Step1: 観測 — 何を疑うべきだったか

dfc79ff コミット時点での主張(spinet_pianica_design.md より):

```
v3:      リード -> 固体伝導(53%) -> 壁面放射(60%) = 合計 7.3%(誤、3.4%)
スピネット: リード -> 音響結合(95%) -> 壁面放射(60%) = 合計 10.3%
```

ユーザの一次指摘:「音響結合95%は大きすぎる」。
これに対し独立検証を行った結果、95%という値そのものよりも
**比較対象がすり替わっていること**が真の問題と判明。

| 何を測っているか | v3 BN2 = 53% | スピネット BN2 = 95% |
|----------------|--------------|---------------------|
| エネルギー形態 | リード機械振動 -> 桐板機械振動 | リード音波 -> 箱内空気 |
| 経路 | 固体5段(プレート->仕切り->ABS->桐) | バルブ開口から空気場へ |
| 物理的支配要因 | 多段インピーダンス不整合 | バルブ周りの漏れのみ |

**異なる物理量を同じ「BN2」ラベルで並べていた**。

---

## Step2: B系の3層分解

ユーザ提案により、B系を以下の3層で独立に検証:

```
[系A] リードが振動すること自体のエネルギー使用
[系B] 振動した空気の金属リードプレート反射
[系C] 振動した空気自体の振る舞い
```

前提値 (F3 = 175Hz):
- リード: 黄銅 20x3x0.5mm, 0.252g, 振幅500um(強奏)
- プレート: 黄銅 1mm, 面密度 8.4 kg/m^2
- スロット: 40 mm^2 (20x2)
- 共鳴箱: 1.37L (Bass zone)
- 息: ΔP=1kPa, Q=100mL/s, 入力100mW

---

## 系A: リード自励振動のエネルギー使用 [PASS]

### 計算

```
最大速度 v_max = ωX = 2π·175·5e-4 = 0.55 m/s
最大運動エネルギー KE = (1/2)m·v² = 0.5·2.52e-4·0.55² = 38.1 μJ
吹奏中Q = 40 (Tarnopolsky系文献から)
1周期あたり減衰損失 = 2π/Q · KE = 5.98 μJ
維持パワー = 5.98 μJ · 175 Hz = 1.05 mW
息に対する割合 = 1.05/100 = 1.0%
```

### 含意

リードは1〜3 mWで維持される共振器。BN1=18%(18mW)の内訳:
- リード本体の振動維持: 1%
- リード周りの空気を直接駆動: 17%

これは Tarnopolsky/Ricot 系文献および 0430d ジャーナルと整合。
**系Aは確定。本検証で動かす必要なし**。

---

## 系B: リードプレートでの空気反射 [INSIGHT]

### 計算

プレート(黄銅1mm)の音響インピーダンス:
```
Z_plate = ω·ρh = 1100 · 8.4 = 9236 Pa·s/m
Z0(空気) = ρc = 1.21 · 343 = 415 Pa·s/m
Z_plate / Z0 = 22.3
```

圧力反射係数とエネルギー反射率:
```
R = (Z_plate - Z0) / (Z_plate + Z0) = 0.914
R^2 = 83.5%  (反射率)
T = 1 - R^2 = 16.5%  (透過率)
```

比較として桐0.7mm板:
```
Z_kiri = 1100 · 0.21 = 231 Pa·s/m
Z_kiri/Z0 = 0.556 → R^2 = 8.1% のみ
```

### 発見

**リードプレートはほぼ「壁」**。
リードが空気を振動させても、プレート方向には83.5%が反射される。
音は事実上、スロット開口部(40mm^2)のみから放射される。

これは以下を意味する:
- リードは「両面放射のダイポール」ではなく「スロット集中のモノポール音源」
- プレート裏側(空気室側)とプレート表側(共鳴箱側)は音響的に別世界
- バルブが開いた瞬間、リードの音は「下向きに集中して」共鳴箱へ降る

桐板の反射率8.1%との比較が示唆的:
**桐板は壁になりきれない。だから振動して放射する。これが楽器の物理**。
逆にリードプレートは壁になりきってしまうから、放射には参加しない。

### 既存知見への接続

0429ジャーナル「魂柱構造の物理計算」でのGamma=0.66議論と整合。
桐は「軽い板」として空気とのインピーダンスが近く、振動伝達できる。
リードプレートは「重い板」として空気を反射する。
**この対比こそが、リード音響系の構造的非対称性の核**。

## 系C: 空気場挙動 [WARN — 帯域依存]

### 計算

リードからの体積流速:
```
リード受圧面積 A_reed = 60 mm^2 (20x3)
実効速度 v_eff = v_max/2 = 0.27 m/s
体積流速 Q = A·v = 16.5 mL/s = 1.65e-5 m^3/s
ka = k·sqrt(A_slot/π) = 0.011 (<<1, 完全無指向)
```

自由空間放射(参考):
```
W_free = ρω²Q² / (4πc) = 0.09 μW
```

共鳴箱内の応答(V=1.37L):
```
非共振時 P = ρc²Q/(ωV) = 1.56 Pa  ← 単純コンプライアンス
共振時(Q_helm=10) P = 15.6 Pa     ← ヘルムホルツ共振による増幅
```

バスレフポート放射(共振時粗推定):
```
A_port = π·(10mm)² = 3.14 cm^2
v_port = P/Z0 = 15.6/415 = 0.038 m/s (整合点)
W_port ≈ 0.5·Z0·v²·A = 0.09 mW
```

### 物理の分岐

**F3共振帯域 (バスレフ稼働、175Hz±10Hz)**:
- スロット出力 -> 箱内ヘルムホルツ共振 (Q=10倍増幅)
- ポート(20mm穴1つ)から集中放射
- 壁面振動は副次的
- 想定パワー ~0.09 mW

**非共振帯域 (中音〜高音)**:
- 箱内圧は弱い (蓄積のみ)
- 桐壁の板モード放射(Gamma=0.66) が支配
- 0429ジャーナルの eta_rad=53〜60% 体系の延長
- 周波数依存の放射効率 σ(f)

### 既存モデルへの含意

**「BN2 x BN3」の単一連鎖では表現不可能**。
- 共振時: 「Q値増幅 + ポート整合放射」の物理
- 非共振時: 「板モード + 放射効率」の物理
- 両者は別の数式系。掛け算で合成できない。

帯域別モデルが必要。

### 残すべきもの

- 2重箱の構造提案 (空気室と共鳴箱の分離)

- CAHAYA改造実験の方針

### 撤回すべきもの

- BN1×BN2×BN3 の単一値連鎖モデル (全表)
- 「98倍」「10.3%」の定量主張
- 「BN2を53%から95%に改善」の主張
- 「+20dB」の SPL 改善見積もり
