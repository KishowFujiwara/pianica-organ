## 1. セッション概要

直前のセッションで v1/v2 設計の検証 (design-verification-v1.md) を実施したが、その途中で「22経路解析の各経路を物理的に定量化する必要がある」との認識に至り、ユーザー指示「22経路の音響的分析を包括的に行う」を受けて本セッションを実施した。

検証文書 (design-verification-v1) は v1/v2 を 22経路解析と照らして評価したが、その時点では各経路の音響パラメータが定量化されていなかった。本セッションで初めて全22経路に物理的数値を割り当て、5軸マトリクスで網羅的に評価した。

正: 経路①パワー × 結合係数(0.05) ≈ 8 μW

## 3. 本セッションの核心的発見

### 3.1 「楽音主経路 A1」の誤誘導の発見

22経路解析の元文書では A1 を「楽音の主経路、ピアニカの音のほぼすべてがこの経路を通る」と記述しているが、定量計算で:

```
外部放射の寄与:
  C1 (ABS壁面外部放射)   : 71.4%  <- 真の主経路
  C2 (壁->穴経由再放射)  : 23.8%  <- 楽音の着色を担う
  C4 (外装放射)         :  3.8%
  A1 (楽音穴経由)        :  0.8%  <- 実は微小
  C5 (パイプ放射)       :  0.1%
  A4 (口腔逆流)         : <0.1%
```

A1は0.8%に過ぎず、**外部放射の95.2%は ABS壁面由来 (C1+C2)**。

これは:
- 過去の固体伝導モデル(0411): 経路②/経路① ~ 286倍
- 22経路解析の表5「不快な共鳴」議論: ABS壁モード323Hz が主役
- アコーディオン職人の経験的知見: B1a, B4, C1, C2 を蜜蝋・ニス・cassottoで制御

これらすべてと整合する。**定量分析が定性的議論を裏付けた**。

### 3.2 ABS壁(1,1)モード323Hz = E4 の意義

ABS底面の共振点323Hzは:

- 楽音帯域 (F3=175Hz ~ F6=1397Hz) のほぼ中心
- E4 (329Hz) と +6 Hz でほぼ完全一致 -> E4-G4 帯域で「鳴り」最大
- Q=33で帯域 ±10Hz 程度 -> 全楽音に着色を加える
- 外部放射の95%がこの周波数を中心に集中

「ピアニカは E4 共振を中心とする箱鳴り楽器」という認識が定量的に確立された。

### 3.3 B1a と B1b の位相干渉

リードプレートでの実効振動は B1a (即時固体伝導) と B1b (90度遅れスロット空気) のベクトル合成:

```
F_plate = √(30000² + 18000²) cos(ωt - 31°) = 35,000 μW @ -31度
```

つまりリードプレートの振動は **リード変位より31度進んだ位相で振動する**。これが整数倍音 (鋸歯波の2倍音) の生成に寄与している。

### 3.4 5軸スコアによる制御優先順位

```
[1位] C1 (ABS壁面外部放射)        : Q低下処置, 壁面ダンピング
[2位] B4 (ABS全体振動)            : 壁体材種の見直し
[3位] C2 (壁->穴経由再放射)       : 内壁吸音
[4位] B1a (リベット結合)          : 蜜蝋系粘弾性ダンパー
[5位] D1/D2 (FBループ)           : 安定性チューニング
```

これは22経路解析の表7で挙げられた「プロ用アコーディオン職人が制御する4経路 (B1a, B4, C1, C2)」と完全一致。**定量分析と定性的職人知見が独立に同じ結論に到達**したことが確認できた。

## 4. メソドロジー的教訓

### 4.1 概念分類だけでは見えない誤誘導

22経路解析 (0427) は経路の概念分類として優れていたが、「A1=楽音主経路」という当初の表現は実は誤りだった。**定量化を経て初めて誤りが顕在化**。

これは PRODECOSの数式全単射検証(0320)で確立した「概要ページ集中・計算法詳細を見落とす」教訓と同型の現象。

### 4.2 物理計算式のサニティチェックの重要性

A2 の初期計算で 1.0×10⁸ μW (= 100 W) という非物理的値が出た。これは「箱内に100Wの音響パワーが流れる」と言っているに等しく、即座に物理的妥当性をチェックすれば気付ける誤り。

教訓:
- 計算結果が桁オーダー逸脱したら **必ず計算式を再検査**
- 全経路の合計が音源(48,168 μW)を超えたら矛盾
- 「全パワー保存則」を恒等的に確認する習慣を

### 4.3 5軸マトリクスの汎用性

パワー/周波数/位相/結合/音色 の5軸は、他の楽器音響分析にも適用できる汎用フレームワーク。例えば:

- アコーディオン: 18経路 × 5軸 = 90データポイント
- パイプオルガン: 各ストップごとに同様
- 弦楽器: 弦/駒/表板/裏板/エンクロージャ系で同様

将来、こうした楽器音響の比較分析を行う際に再利用できる。
