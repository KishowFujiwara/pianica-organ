### Phase 6: フリーリード物理の深掘り

以下のトピックを体系的に議論:

**(a) Q値の物理的意味**
- Q = 2pi * 蓄積エネルギー / 1サイクル損失
- Q高い = 振動がしぶとく続く、発振閾値が低い、音が澄む
- Q低い = すぐ止まる、強い息が必要、ぼやけた音

**(b) 結合振動と固有振動数**
- 2つの振動子を繋ぐとモード分裂が起きる
- f_reed ≒ f_board のとき激しいエネルギー交換 -> ウルフトーン
- 実効結合 ∝ k_c * sqrt(Q_reed * Q_board)

**(c) フリーリードの発振原理**
- [誤] 息が押す -> 戻る -> また押す (パルス駆動)
- [正] 一定気流からリード自身の動きが自動的にエネルギーを抜き取る (自励振動)
- ベルヌーイ引き込み + 正帰還
- DCからACを生み出す非線形過程。周波数応答ではない

**(d) B1aとB1bの位相差31度**
- B1a: 変位に比例する力 (リベット直接伝達)
- B1b: 速度に比例する力 (スロット空気ハンマー)
- sin と cos の性質の違いが31度を生む (距離の遅れではない)
- 位相差が倍音の偶数/奇数バランスを決定

**(e) 倍音生成メカニズム**
- 片持ち梁の構造倍音は非整数 (6.27f, 17.5f ...)
- 楽器中では整数倍音のみ出現
- 原因: スロット開閉の非線形がフーリエ級数を自動生成
- 非線形(1): スロット遮蔽 g(x) = max(0, g0 - |x|)
- 非線形(2): オリフィス流れ Q ∝ g^1.5

**(f) 空気が振動する瞬間**
- 第1段階: スロットで空気が振動 (体積速度音源) -> 寄与0.8%
- 第2段階: 壁面が空気を押す (面音源) -> 寄与99.8%
- ピアニカはスピーカー構造。リード=ボイスコイル、壁=コーン紙

### Phase 8: 真空 -> 静止空気(1atm, 重力)の効果

| 効果 | Q値 | 影響度 |
|------|------|--------|
| 材料内部損失 | 500 | 支配的 (80%) |
| Stokes粘性抗力(自由面) | ~2000 | 中程度 (20%) |
| Couette剪断(スロット側面) | ~15000 | 小さい (3%) |
| 放射減衰 | >100,000 | 無視可能 |
| 音響放射 | >50,000 | 無視可能 |

結論: 静止空気ではQ=500 -> Q=400 (20%低下のみ)。材料Qが依然支配的。

**Tarnopolsky Q=16-55との乖離は「吹奏中の空気流力学的効果」によるもの。**
これはナビエ-ストークス方程式の領域であり解析的には計算不能。

修正された材料設計の結論:
- 定常吹奏中: 材料Qの差は空気流減衰に埋もれる (Blue Steel Q≒39 vs C5210 Q≒37)
- リリース(息を止めた瞬間): 材料Qが効く (Blue Steel 480 vs C5210 400)
- 材料選定の真の判断基準: 疲労寿命、加工精度、耐食性、コスト

### Phase 9: 重力の発見

G1低音リード: 重力たわみ = 88um = スロット隙間150umの59%!

水平設置(ピアニカ): 上側238um / 下側62um -> 非対称 -> 音色変化
垂直設置(アコーディオン/v3): 幅方向たわみゼロ -> 問題なし

v3でアップライト筐体を選択したのは(意図せず)正しい。

### Phase 10: 気室設計

**(a) ピアニカの問題**
- パイプからのジェットが位置依存の不均一を作る
- 箱内の圧力脈動: ~100Pa (静圧の10%)
- 和音時にリード間干渉

**(b) 2層傾斜気室設計**

```
層1: 共通プレナム (大体積、ブロワ直結、脈動吸収)
層2: 個別チャンバー (仕切り板で37室分割、傾斜底板)
     接続: 絞り穴 (f_c ≒ 10Hz のRCローパスフィルタ)
```

傾斜の意味: ヘルムホルツ共鳴チューニングではなく、
低音リードの大きな空気消費に対するバッファリング。

パイプオルガンのウインドチェスト(風箱)と同じ設計思想。

**(c) 和音時の挙動**
- 各チャンバーが独立 -> リード間干渉なし
- プレナム圧低下 -> ブロワ容量で対応
- 絞り穴がRCフィルタ: DC供給は通す、脈動(49Hz以上)は遮断
- 最大同時10音: 必要風量 159 L/min, 必要圧力 1200 Pa

## 本セッションで確立された物理的理解

1. フリーリードは自励振動。DC気流からACを生成する非線形過程
2. 周波数はリードのk/mで決まる。息は関係ない
3. 倍音はスロット開閉の非線形が自動生成。梁の構造倍音ではない
4. B1a(変位比例) + B1b(速度比例) の位相差が音色を決定
5. 材料Qは静止空気で支配的だが、吹奏中は空気流減衰に埋もれる
6. 吹奏中のQ=30-60はナビエ-ストークス領域。計算不能。実測で決める
7. 重力がG1低音リードの隙間を59%変える。垂直設置が必須
8. 気室設計はRCローパスフィルタ。ヘルムホルツ共鳴チューニングではない
