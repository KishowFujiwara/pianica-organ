## セッション概要

L5d共鳴板設計を「空気箱の中でフリーリードが発声する」条件で実施。
Gammaパラメータの導出、3つの構造案の物理的評価、推奨構成の決定。

**最大の発見**: Gamma = eta * omega * m_s / (2 * sigma * Z_air)
この無次元数一つで「壁面到達パワーのうち何%が音になるか」が決まる。
ABS: Gamma=106(0.9%が音)、Spruce: Gamma=2.4(30%が音)。

## 作業ログ

### Phase 1: ABS壁面の問題の定量化

パワーフローv2.0から出発:
- 壁面到達パワー P_wall_in = 25,500 uW
- ABS損失係数 eta = 0.03, Q = 33
- 放射効率 sigma = 0.001
- 外部放射 P_rad = 224 uW (0.9%)
- 内部損失(熱) = 25,053 uW (98.2%)

**問題: 到達パワーの98%が熱で消える。**

### Phase 2: 共鳴板材料の音響定数比較

8材料を比較。主要指標:
- c_L = sqrt(E/rho): 縦波速度
- R = c_L/rho: 放射比
- Q = 1/eta: 品質係数

| 材料 | rho | eta | Q | R (放射比) |
|------|-----|------|-----|-----------|
| ABS | 1050 | 0.030 | 33 | 1.41 |
| Sitka Spruce | 400 | 0.008 | 125 | 13.46 |
| European Spruce | 440 | 0.006 | 167 | 11.87 |
| W. Red Cedar | 340 | 0.007 | 143 | 14.00 |
| Paulownia (桐) | 260 | 0.009 | 111 | 15.64 |

### Phase 3: Gammaパラメータの導出 [本セッション最大の発見]

壁面(共鳴板)のエネルギーバランスから導出:

```
定常状態: P_in = P_rad + P_diss
P_rad = sigma * rho_air * c * S * <v^2>
P_diss = eta * omega * m_s * S * <v^2> / 2

解くと:
P_rad / P_in = 1 / (1 + Gamma)

Gamma = eta * omega * m_s / (2 * sigma * Z_air)
Z_air = rho_air * c = 420 Rayl
```

Gammaの物理的意味:
- Gamma >> 1: ほぼ全部が熱 (ABS: Gamma=106)
- Gamma = 1: 半分が音、半分が熱
- Gamma << 1: ほぼ全部が音 (理想)

**重要発見**: sigma(放射効率)が同じでも、etaが低いだけで
Gammaが下がり、放射効率が改善される。
直感: 低損失材 = 振動が長持ち = 長く放射 = 累積放射量が増える。

### Phase 4: 共鳴板構造の3案評価

条件: フリーリードは気室(2層構造)の中で発声する

| 案 | 構造 | 判定 | 理由 |
|----|------|------|------|
| 案1 | 共鳴板=気室の蓋 | NG | 固体伝導が切れる(空気越し=1/286) |
| **案2** | **ブリッジ接続** | **採用** | **固体伝導が活きる。T=98.7%** |
| 案3 | プレート直接マウント | NG | プレート質量(189g)が板(141g)を抑圧 |

ブリッジのインピーダンス整合:
```
Z_plate (真鍮1mm) = 67.3 N*s/m
Z_board (Spruce 4mm) = 84.9 N*s/m
T = 4*Z1*Z2 / (Z1+Z2)^2 = 98.7%
```

偶然にも非常に良い整合。

### Phase 5: 推奨構成の決定

### Phase 6: CAHAYA CY0341-1 改造実験計画

**重要な構造の違い**:
```
日本式(ヤマハ/鈴木): カバー + 空気室 = 別部品
中華式(CAHAYA):      筐体 + 空気室 = 一体成形
```

一体型は底面を切り取れば空気室壁そのものを木板に置換できる。
日本式ではカバーを外しても中にABS空気室がもう一つ残る。

実験計画:
```
Step A: 改造前ベースライン (Spectroid FFT)
Step B: 底面ABS切除 (SPL低下を確認)
Step C: 木板底面装着 (SPL回復+音色変化を確認)
```

底面材料の予測:
| 材料 | Gamma | vs ABS | SPL差 |
|------|-------|--------|-------|
| ABS 2mm (現状) | 141 | 1.0x | 0dB |
| 桐板 5mm | 5.2 | 14.0x | +11dB |
| ヒノキ板 4mm | 5.7 | 11.9x | +11dB |
| シナベニヤ 4mm | 17.9 | 4.1x | +6dB |
| スプルース 4mm | 4.7 | 12.3x | +11dB |

必要経費: ~3,500円(鍵盤ハーモニカ+木板+接着剤)

### Phase 7: 弦楽器の駒との比較 [物理的洞察]

弦楽器の駒と同じインピーダンスミスマッチの物理:

| 楽器 | 駒の伝達率 | 反射率 |
|------|-----------|-------|
| ピアノ | 2.4% | 97.6% |
| バイオリン | 3.2% | 96.8% |
| 改造ケンハモ(桐) | 61.7% | 38.3% |

弦楽器は伝達率2-6%でも十分な音が出る。
理由: 弦のQ=3000で何千回も振動し、毎回少しずつ渡す。
定常状態では弦エネルギーの98.4%が最終的に響板に移る。

フリーリードはさらに有利:
- 自励振動 -> 反射エネルギーは「損失」ではなく「振動維持」
- 息のDCパワーから常に補充 -> 弦は一発、リードは持続的

結論: 接合部の反射50%は問題にならない。

### Phase 8: 鈴木W-37Cとの比較分析

鈴木W-37C:
- カバー(上面)をウォルナットに置換
- 空気室はABSのまま (Melodica World: "internal parts are plastic")
- 設計目的: 柔らかい音色(アンサンブル調和)
- ウォルナット選択理由: 強度+耐湿+加工精度+音色バランス

構造の違い:
```
鈴木W-37C (弦楽器/木管楽器型):
  リード -> プレート -> ABS空気室壁 -> [駒1] -> 木カバー -> 空気
  駒が2段。非効率だが製品信頼性が高い。

CAHAYA改造 (グランドピアノ型):
  リード -> プレート -> ABS側壁 -> [駒] -> 木底面(=空気室壁) -> 空気
  駒が1段。最短経路。さらにA2経路(音圧駆動)も直接効く。
```

鈴木の設計思想への評価:
- 「カバーが音づくりを左右する大きな要素」 = 壁面放射99%を一文で表現
- 22経路解析で我々が導いた結論を、楽器職人は経験から知っていた
- 製品制約の中での最適解。我々は制約なしの物理最適を攻める。

## 本セッションで確立された物理的理解

1. Gamma = eta*omega*m_s / (2*sigma*Z_air) で放射効率が決まる
2. 低損失材は sigma が同じでも放射効率を劇的に改善する
3. 共鳴板はブリッジ接続が最適(案1:空気越しNG、案3:質量過大NG)
4. 接合部の反射50%は問題にならない(弦楽器は2%で成立)
5. フリーリードは自励振動なので反射=損失ではなく振動維持
6. 中華一体型は底面置換実験に最適(日本式は空気室が別部品で残る)
7. 鈴木W-37Cは「弦楽器型2段駒」、CAHAYA改造は「グランドピアノ型1段駒」
8. 鈴木は壁面放射の物理を経験的に知っていた
