# 2026-04-30q: ガスケット問題 - 観測ミスの訂正と CAHAYAガスケットレス実態の発見

## セッションの目的

ユーザの問い「ガスケットって空気箱なのよね? リードの音には影響ないの?」に答えるため、CAHAYA画像を観察してガスケット位置を確認したところ、**Claudeの観測ミスが連続発覚** した。本セッションは認知バイアスの実例として記録する。

## 認知バイアスの実例 (3段階の訂正)

### 第1ミス: 透明ビニール膜の解釈

```
[Claude初見] 上ケース内の透明ビニール膜 = CAHAYA出荷時のガスケット?
[実態]      ユーザが後付けで貼った制振材 (DIY改造)
```

ユーザが後付けで貼った制振材を、CAHAYAの工場仕様と誤認した。

### 第2ミス: 白色シーリング材の解釈

```
[Claude初見] 上ケース内中央の白い帯 = ガスケット
[実態]      これも制振材の一部、または別目的の素材
```

仮説に合わせて画像を解釈し続けた典型例。

### 第3ミス: そもそものガスケット存在の前提

```
[Claude初見] 革ガスケット 0.5mm + ネジ4本 (リードオルガン文献由来)
[実態]      CAHAYA は ガスケットレス設計
            プラ枠 vs プラ枠 を直接ネジで締結のみ
```

L6-1 ドキュメント全体が「ガスケットあり前提」で書かれていたが、CAHAYAは出荷時ガスケットなし。

## ユーザの一言で訂正

```

  「シーリングテープ 実はユーザが制振のために貼った」
  「ガスケットらしきものは見当たらない (プラ同士を直接ネジで締めている)」

```

これで Claudeの3段階の誤認がすべて訂正された。再度 0430k末尾の教訓:

> 物理オブジェクトの観測では:
> 1. 数えるべきものは数える
> 2. 構造を仮説で埋めない
> 3. ユーザの一言を最高優先で受け取る

を再確認。

## 物理分析: ガスケットレスがなぜ成立するか

CAHAYAの設計が物理的に成立する条件:

```
A. 接触面の平面度が極めて高い (射出成形品の精度: 表面粗さ Ra 1.6μm)
B. 接触面積が大きい (面圧が分散)
C. ネジ圧着力が十分 (M3-M4 で 0.3-0.5 Nm)
D. 微小漏れは許容 (リード励振に支障ない範囲)
```

### 漏れ量の試算

圧力差 ΔP = 400 Pa、リード流量 Q_reed = 0.5 L/s として:

| 状態 | 隙間 [m²] | 漏れ流量 [L/s] | リード流量比 | 評価 |
|---|---|---|---|---|
| 理想 (面接触)         | 0     | 0      | 0%      | 完全密閉 |
| 実用 (微小隙間)       | 1e-6  | 0.026  | 5.2%    | CAHAYA出荷時 |
| 劣化 (5年後)          | 1e-5  | 0.258  | 51.6%   | 音圧 -0.4dB |
| 漏れあり (破損)       | 1e-4  | 2.582  | 516%    | 修理必要 |

**結論**: ガスケットレス設計は中華量産品の合理的最適化。革ガスケットは19世紀リードオルガンの伝統だが、現代では過剰品質。

## 本機 (v3) への翻訳

CAHAYAと本機の相違点:

```
1. 共鳴箱 = 桐板 (柔らかい木材)
   CAHAYA = ABS (硬い射出成形品)
   → 桐板の表面精度はCAHAYAより劣る (鉋仕上げで±0.1mm)

2. 接合先がCAHAYA鍵盤ユニットのプラ枠
   → 「桐板 vs ABSプラ」異種材料接合

3. 接合面積が小さい
   CAHAYA = 筐体全体周縁 (大面積)
   本機  = Bridge patch周辺 (200x60mm のみ)
```

→ 本機ではガスケットレスは**リスク高**。何らかのシール材または平滑化処理が必要。

## シール材の選択肢 (本機 v3)

| 案 | 方式 | 利点 | 欠点 | リスク | 用途 |
|---|---|---|---|---|---|
| A | ガスケットレス (CAHAYA踏襲) | シンプル | 桐板平滑化が必要 | 高 | 桐板を完全平滑化できる場合のみ |
| B | 革ガスケット 0.5mm | 長寿命50年級 | プラ枠との相性未検証 | 低-中 | 本機長期使用 |
| C | シリコンOリング | 圧縮設計可、再現性高 | 矩形には加工要 | 中 | 比較対象 |
| **D** | **シリコンシート 0.5-1mm** | **安価、加工容易** | **5-10年で劣化** | **低** | **プロト推奨** |
| E | フェルト | 伝統的 | 多孔質で漏れる | 高 | 不採用 |

### ユーザ判断 (2026-04-30q)

> 「まだ決めない (プロトで実験)」

プロトで段階的に試す方針:

```
[プロト 段階1] ガスケットレス (案A)
  桐板表面を鉋 + サンドペーパー800番で平滑化
  → 漏れ計測 (Spectroid + 圧力計、または音量低下の聴感)
  → OK なら採用、NG なら次へ

[プロト 段階2] シリコンシート 0.5mm (案D)
  ホームセンターで入手可
  → 漏れ計測
  → ほぼ確実にOK

[プロト 段階3] 革ガスケット 0.5mm (案B、本機長期用)
  StewMac等で入手、本格組立時
  → 50年級の長寿命を狙う
```
