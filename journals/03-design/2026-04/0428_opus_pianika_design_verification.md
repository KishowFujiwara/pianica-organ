## 3. 検証の核心的発見

### 3.1 v1/v2 の前提誤り

両設計は「ピアニカからの空気流をホーンが処理して大音量化」という前提に立つが、これは:

- 経路①(スロット直接放射) = 168 μW
- 経路②(固体伝導 -> ABS壁面放射) = 48,000 μW (約286倍)

の事実と矛盾する。**v1/v2 は全音響パワーの 0.35% しか処理対象にしていない**。

### 3.2 物理限界

ホーンの理想的利得 +20dB を考慮しても:

```
ホーン経由出力 = 168 μW × 100 = 16,800 μW = 16.8 mW
音圧 @1m       = 約 70 dB SPL
パイプオルガン目標 = 95 dB SPL
差              = -25 dB (達成不可能)
```

### 3.3 21経路の無処理

v1/v2 が能動的に処理するのは A1 (1経路) のみ。残り21経路は:

- B1a, B1b (リードからリードプレートへの2系統駆動) -> 無制御
- B2-B7 (固体伝導の各段階) -> 無処理
- C1-C5 (再放射) -> 無処理 (アコーディオンではcassotto+ニスで制御する核心経路)
- D1-D3 (フィードバック) -> 無処理

### 3.4 VE原則違反

memoryに記録された原則「フリーリードの非整数倍音(1:6.27:17.55) を最大化すべき、パイプオルガン模倣は付加価値を生まない」と、v1/v2 の「パイプオルガン化」目標が真っ向から対立している。

## 4. 推奨される方向転換

3つの代替方向を提示:

| 方向 | 概要 | 利点 | 欠点 |
|------|------|------|------|
| A | 経路②活用型 (cassotto化) | 物理的整合, パイプオルガン的設計 | ABS箱置換の工作 |
| B | ピアニカ内部処理拡張 | 工作変更最小, 流用メリット維持 | 95dB達成は不可 |
| C | リードのみ流用 | 設計自由度最大 | コスト大幅増加 |

VE評価では **方向 A または B が高評価**。

## 7. 技術的知見

### 検証メソドロジーの教訓

設計文書と物理解析文書が同じディレクトリに併存するとき、**両者の整合性は明示的にチェックしないと矛盾が放置される**。本セッションでは:

- design-spec-v1.md (2026-04-23 作成)
- design-spec-v2.md (2026-04-23 作成)
- pianica_22paths_analysis.md (2026-04-27 作成、別文脈で発生)

の3者が同じ pianika-organ プロジェクトに属していたが、それぞれ独立に書かれていた。検証作業によって初めて、**v1/v2 と 22経路解析が音響モデル前提について不整合** であることが顕在化した。

### Bijection的思考の応用

PRODECOSの数式全単射検証(0320)で確立した「悉皆検証」の発想を、設計文書の検証にも適用した:

- 22経路すべてに対して v1/v2 の処理状況をチェック
- 各経路の「処理 / 未処理」を表形式で全件確認
- 結果として「v1/v2 が処理するのは 1/22 のみ」という定量的指摘が可能になった
