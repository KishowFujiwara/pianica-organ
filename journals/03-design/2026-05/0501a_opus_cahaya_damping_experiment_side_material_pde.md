# 2026-05-01a: CAHAYA制振実験の洞察 + 側面材料選択 + PDE数理モデル

## セッションの目的

CAHAYAの観測から始まり、ユーザーの制振実験の報告を経て、側面材料の設計判断と表板振動の偏微分方程式モデルの策定に至った。

## 主要な発見と洞察

### 1. CAHAYA制振実験 (ユーザー実施)

ユーザーがCAHAYA上ケース内壁にアルミテープを貼り、制振した状態で演奏。

```
観測結果: 「明らかに倍音列のデシベルが大きくなる」
         (スペクトラム未計測、聴感による判断)
```

### 2. 「エンクロージャ化」の誤った結論を回避

制振実験の結果から2つの解釈が可能だった:

```
[誤った結論] 制振が良い -> エンクロージャ方向
  音量は下がる、倍音は「見えるだけ」

[正しい結論] ABSが邪魔だった -> 取り除く -> 代わりに桐板
  倍音は「見える」のではなく「増幅される」
```

ユーザーの一言で確定:
> 「ABSから解放して、薄い素材で共鳴しまくる材料で、共鳴させて、また共鳴させる」

### 3. 楽器カテゴリの転換

```
CAHAYA現状 = ハーモニカ + 偶然鳴るプラ箱 (安物スピーカ)
v3設計     = リードオルガン (設計された共鳴箱)
```

制振実験は「改造」ではない。CAHAYAをハーモニカに戻し、そこから共鳴箱付きリードオルガンへ転換する。

### 4. 02-physicsの自分たちの分析を引かなかった反省

Claudeがフリーリードの倍音生成メカニズムについて「推測」と述べた項目の多くが、
02-physics/22-paths/ に既に解明済みだった:

```
S1 (流量変調): 80-85% -> 整数倍音 f, 2f, 3f... の起源
S2 (ダイポール): 10-15%
S3 (梁高次モード): 3-5% -> 非整数倍音 6.27f, 17.55f... の起源
```

リポジトリの既存分析を参照せず推測で答えたのは手順の不備。

### 5. CAHAYA風箱構造の確認

画像観察で3点を確認:
1. 単一風箱 (仕切りなし)
2. ABS射出成形品が風箱の壁材
3. ABS風箱がそのまま外壁 (= C1+C2の95.2%の実物)

### 6. CAHAYA風箱の側面流用は不可

ユーザー提案「ABS風箱の側面を再利用して表板を桐にする」を検討:

```
風箱内部寸法: 約 380 x 50 x 20 mm
桐板 380 x 50 x 1.5mm の f(1,1) ≈ 1200+ Hz
-> 幅50mmでは低音域モードが存在しない
-> 材料は正しいが形状が間違っている
-> v3の 550 x 350mm が必要な理由はここにある
```

### 7. 側面材料の選択

ユーザーとの議論:

```
[問] ABS vs 金属 vs 桐、側面はどう違う?
[答] 反射率 R = ((Z_side - Z_plate)/(Z_side + Z_plate))² で決まる

  桐 4mm:        R = 0.21 (漏洩大、モード不明瞭)
  ラワン 4mm:    R = 0.42 (旧設計)
  楓 4mm:        R = 0.52 (ギター標準)
  アルミ 3mm:    R = 0.83 (ほぼ固定端)

[問] じゃあ金属が良いのね
[答] 設置型だから重量問題なし -> 金属が最適

[設計決定] アルミ L材 3mm (第1候補) / 楓 4mm (第2候補)
```

### 8. PDE数理モデルの追加

design-spec-v3.md 第3章 3.6節に以下を新設:

```
3.6.1 支配方程式 (Kirchhoff-Love薄板方程式)
3.6.2 初期条件 (静止状態からの出発)
3.6.3 境界条件 (金属/硬木/桐の3ケース)
3.6.4 境界でのエネルギー反射率
3.6.5 S1駆動項 (リード流量変調のフーリエ展開)
3.6.6 共鳴箱内部空気の結合 (Helmholtz方程式)
3.6.7 結合系の固有値問題
3.6.8 側面材料がモード構造に与える影響のまとめ
```

固定端 (アルミ) の場合、ブレース付き(1,1)モードが 374 Hz となり、T1 = 345 Hz と整合することを確認。

## メタ考察

本セッションで最も重要だったのはユーザーの以下の発言:

> 「迷いがなくなった。ABSから解放して、薄い素材で共鳴しまくる材料で、共鳴させて、また共鳴させる」

ユーザーの手が楽器に触れ、耳で倍音を聴き、「ABSが邪魔」と判断したプロセスこそが、このプロジェクトの核心にある。
