# 2026-05-01b: 桐空気箱 + 共鳴箱 2段構造の発見 -- 設計パラダイムの根本転換

## セッションの目的

CAHAYA制振実験の洞察から出発し、側面材料選択とPDE数理モデルを経て、最終的に**設計の根本的な誤りを発見・修正**した。本セッションはプロジェクト史上最大のパラダイム転換を含む。

## 最大の発見: 2段構造

```
[桐空気箱]         スパン60mm, Gamma=0.66, 効率41%
  全壁面がS1倍音列を全帯域で効率的に放射
  フリーリードは連続励振だから放射し放題 (T=97%)
    |
[リードプレート]   CAHAYA流用 (溝構造含む)
    |
    v
[共鳴箱]           ほしい倍音列を選んで共鳴増幅
  教会で使える音量と音色を作る
```

空気箱 = 「S1倍音列を効率よく出す」。共鳴箱 = 「出た音を増幅する」。役割が違う。

## 発見に至る道筋

### 1. CAHAYA制振実験 (ユーザー実施)

アルミテープで上ケース内壁を制振 -> 「倍音列のデシベルが明らかに大きくなる」

解釈: ABS 323Hz共振がフィルタとして倍音列をマスクしていた。制振でフィルタが消え、S1倍音列が露出した。

ユーザーの確信:
> 「ABSから解放して、薄い素材で共鳴しまくる材料で、共鳴させて、また共鳴させる」

### 2. 側面材料の議論

表板の境界条件 = 側面材料のインピーダンスで決まる。

| 材料 | 反射率 R |
|---|---|
| 桐 4mm | 0.21 |
| 楓 4mm | 0.52 |
| アルミ 3mm | 0.83 |

設置型楽器だから重量制約なし -> アルミが最適 -> PDE数理モデル (Kirchhoff-Love) をdesign-spec-v3に追加。

### 3. L字構想と挫折

CAHAYA溝構造の流用 + 桐板プレナム + アルミ側面のL字形空気箱を検討。

```
+====表板 (桐)==============+
|                            |
|        プレナム            |
|                            |
+桐板+                       |
     |アルミ                 |
     |    溝 32ch            |
     +=======RP==============+
```

計算結果:
- 表板 内圧撓み (X-braceあり): 0.5mm -> OK
- 水平棚 内圧撓み: 0.02mm -> OK
- 水平棚 最低モード: 457Hz -> 副放射面として機能?

しかしユーザーの指摘: **水平棚も表板もプレナムの密閉壁。振動したら空気が漏れる。共鳴板にはなれない。**

-> 「桐箱空気箱は断念」

### 4. 誤りの遡行

「桐箱空気箱は断念」から遡って誤りの連鎖を追跡:

```
[02-physics/materials] Gamma計算: 桐 スパン60mm -> 効率41%
    |
    v
[0430h] 「ギターが参考になる」 -> T0/T1結合、3DOFモデル
    |
    v
[0430i] 550 x 350 桐板 + X-brace + 14.2L共鳴箱
    |
    v
[0430j] 「リードプレートのみ流用、空気室は新規製作」
    |
    v
[0430k] 「上ケース = 廃棄」
```

### 5. 核心的な誤り: 550 x 350 の桐板で共振させようとしたこと

Gamma計算が示していた答え:

```
| スパン | Gamma | 効率 |
| 350mm  |  >8   | <1%  |  <- 550 x 350 桐板
|  60mm  | 0.66  | 41%  |  <- CAHAYAの空気箱サイズ
```

550mmの板はスパンが大きすぎてGammaが悪い。だからX-braceで分割して剛性を上げて共振モードを制御して... という回り道をしていた。

共振は周波数を選ぶ。S1倍音列は f, 2f, 3f, 4f... と全帯域にある。特定周波数だけ増幅しても他は落ちる。

**求めていたのは共振ではなく全帯域の効率的放射。Gamma計算が最初から答えを出していた。**

### 6. フリーリードの持続性 (02-physics 因果機構分析)

02-physics/causal-mechanism/ に既に分析済み:

```
弦楽器: 1回の励振 -> エネルギーが減衰 -> T を上げすぎると弦が死ぬ
  -> 放射と持続のトレードオフ -> 共鳴箱で控えめに増幅

フリーリード: 連続励振 -> エネルギーは息が供給し続ける -> T に上限なし
  -> T = 97% で壁から全力放射していい -> トレードオフが存在しない
```

ギター型設計 (共鳴箱で控えめに増幅) は弦楽器の「放射しすぎると死ぬ」問題を解決する設計。フリーリードにはその問題がない。

### 7. しかし共鳴箱は要る

桐空気箱だけでは鍵盤ハーモニカのサイズで放射面積が小さく音量が足りない。共鳴箱で「ほしい倍音列を選んで増幅」する必要がある。

```
桐空気箱だけ: S1を効率よく出す (Gamma最適) が音量不足
共鳴箱だけ:   増幅はできるが入力がABSフィルタで汚れている
両方:         桐空気箱でクリーンなS1を出し、共鳴箱で増幅 -> 正解
```

## 設計の修正方針

### 旧設計 (0430j-0501a前半)

```
CAHAYA上ケース = 廃棄
空気室 = 新規 (共鳴箱と一体化を想定)
共鳴箱 = 14.2L 桐板 550x350 + X-brace + アルミ側面
```

### 新設計 (2段構造)

```
桐空気箱 = 新規製作 (CAHAYAの寸法に合わせた長方形)
  スパン ~60mm, 桐 0.7mm, Gamma=0.66
  CAHAYAの溝構造 + 鍵盤ユニット + リードプレートを内蔵

共鳴箱 = 新規製作
  桐空気箱から出た音を受けて、ほしい倍音列を共鳴増幅
  寸法・構造は再設計が必要 (ギター理論は使える)

2段構造の接続方法は次セッションで検討
```

### 回り道で得た知識の活用

```
[使える]
  Gamma計算           -> 桐空気箱の壁厚・スパン設計
  T0/T1結合           -> 共鳴箱の設計
  3DOFモデル           -> 共鳴箱の設計
  PDE数理モデル        -> 共鳴箱の表板振動
  側面アルミ (R=0.83)  -> 共鳴箱の側面
  X-brace             -> 共鳴箱のブレース

[修正が必要]
  550 x 350 桐表板     -> 共鳴箱として再設計 (寸法未定)
  design-spec-v3       -> 2段構造に全面改訂
  空気室は新規製作     -> 桐空気箱として新規製作 (意味が変わった)
  上ケース廃棄         -> 溝構造+鍵盤ユニットは流用、ABSケースのみ廃棄
```

## 知見の再整理

### プロジェクトの本質

> このプロジェクトは19世紀の物理学最高峰の一つの成果の実装をAIが出来るかという実験である。

Helmholtz (共鳴、倍音列)、Kirchhoff (板振動PDE)、Rayleigh (音の理論) が確立した数理音響学を、フリーリード楽器として実装する。

### AIの役割と限界

```
AIが出来ること:
  Gamma計算、PDE立式、固有モード計算、設計文書化

AIが出来なかったこと:
  制振実験で「ABSが邪魔」と気づく
  溝を見て「自作できない」と判断する
  「密閉と振動は両立する (CAHAYAが証明)」に気づく
  Gamma計算の答えを見ながらギターのロマンに走る
  自分のリポジトリに答えがあるのに引けない

ユーザーが担ったこと:
  実験と直観
  誤りの修正 (「ちがうちがう」「だめじゃんwwww」)
  物理的本質への回帰 (「欲張りすぎたのでしょwww」)
```

19世紀の物理学者は計算と実験と直観を一人で持っていた。本プロジェクトではそれがユーザーとClaude (Opus) に分かれている。そしてClaudeだから出来る: リポジトリの読み書き、セッションジャーナルの蓄積、訂正を受け入れて前に進む、そしてwwwwと笑われながらも次の計算を正確に出す。
